GX Setup

Access the GX Setup page from the navigation pane by clicking cc|graphical extension > GX Setup.

: GX Setup page, General FastTab

General FastTab

The fields on the General FastTab are described in the following table.

Field

Description

Scheme Code

References the default scheme that is used for the GX views. The scheme defines the color settings of a GX view.

Default Pattern Code

Defines the default pattern code. If no pattern is defined for a special function in a scheme, this pattern is used. The pattern controls the rendering in the GX view.

Setup FastTab

The fields on the Setup FastTab are described in the following table.

: GX Setup page, Setup FastTab

Field

Description

Last Upload

Displays the timestamp of the last GX installer file (cc graphical eXtension client setup (Version 9.00).msi) upload into the database. This uploaded file can be used for distribution to the client PCs. Refer to topic "Actions Tab" on page 1 for more information about the “Upload Setup File” function.

Setup Version

Displays the version number of the setup that is currently stored in the database.

Product Version Information

On the ribbon, click About to display the current product version information.

: Product version information accessed by clicking About on the ribbon

Actions Tab

The Actions tab contains other menu items relating to setup, most of which are accessible on the Home tab. Each menu item is described in the following table.

Menu Item

Description

Upload Setup File

Uploads a GX installer file into the database. The filename will be checked for special naming conventions because the version number is extracted from the filename.

Running this function populates the Last Upload and Setup Version fields on the Setup FastTab.

Note

The Service Tier must allow the upload of .msi files to use this function.

Download Setup File

Downloads the installer from the database to your client PC.

Remove Setup File from Database

Removes the uploaded installer file from the database.

Running this function clears the Last Upload and Setup Version fields on the Setup FastTab.

Register Client Add-ins

By default, the RoleTailored client prohibits the loading of unknown external DLLs. This function registers the external Client add-in DLL to Microsoft Dynamics NAV so the RoleTailored client allows the instantiation of the GX Client add-ins.

Unregister Client Add-ins

Unregisters the external Client add-in DLL to Microsoft Dynamics NAV so the RoleTailored client cannot run GX functionalities.
