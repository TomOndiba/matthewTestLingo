License Setup

The Client add-in and the external DLLs are protected by a separate license. The license can be either a demonstration license with an expiration date or an unlimited license that references the Microsoft Dynamics NAV Serial Number.

You can store multiple licenses in a single database. When launching GX functions, the license is automatically picked. A special routine looks for a license that references the current enabled Microsoft Dynamics NAV license’s serial number. If no matching license is found, the function looks for a Demo license and picks the license with the latest Expire Date.

A demo license is included in the standard settings file. Therefore, it is unnecessary to import a demo license if you import the standard settings file (refer to topic "GX Setup Import/Export" on page 1).

To import, export or check a license, open the GX License Information page, accessed from the cc|graphical extension menu. This page displays all imported licenses.

: GX License Information with an expired demo license and a valid demo license

On the ribbon, access the import/export functions as described in the following table.

Function

Description

Import

Opens an Import window where you can select a license file to be imported into the database.

Export

Exports the current selected license into a text file. This file can be imported by the “Import” function.
