Introduction

cc|graphical extension is a graphical interface that features a connection of graphics and data in Microsoft Dynamics NAV. It allows building views that show data and records of Microsoft Dynamics NAV in a visual way. The standard Microsoft Dynamics NAV functions – related to the displayed dataset – can be accessed for an easy-to-use graphical interface.

Key Features

  • User-optimized operator interface
  • Uses Microsoft Dynamics NAV’s integrated business logic
  • Faster and easier processes
  • Easy customization
  • Quick setup with minimal steps

Access the cc|graphical extension module in the navigation pane from the Departments menu.
