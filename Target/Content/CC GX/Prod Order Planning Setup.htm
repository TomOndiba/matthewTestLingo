Prod. Order Planning Setup

The settings in the Prod. Order Planning Setup page control the features of the GX Prod. Order Planning View page, which shows the operations of production orders in combination with their work centers (or machine centers).

In the navigation pane, click cc|graphical extension > Prod. Order Planning Setup.

: Prod. Order Planning Setup page, General FastTab

General FastTab

The fields on the General FastTab are the same fields as in the base scheme setup (refer to topic "General FastTab" on page 1). However, any values changed on the Prod. Order Planning Setup page override the settings in the scheme.

Routing Lines FastTab

The fields on the Routing Lines FastTab are described in the following table.

: Prod. Order Planning Setup page, Routing Lines FastTab

Field

Description

Show Routing Lines

Enables or disables the drawing of the routing lines. Each production order can contain production order lines. And each production order line can contain multiple production order routing lines (= operations).

Planned Rtng Lines Pattern

Each production order routing line has a routing status. You can link each possible routing status to a pattern code, so you can see the status at first sight.

This field contains the pattern for routing lines with status = Planned. If this field is empty, the pattern code of a routing line is taken from the status of the production order.

In Progress Rtng Lines Pattern

Contains the pattern for routing lines with status = In progress. If this field is empty, the pattern code of a routing line is taken from the status of the production order.

Finished Rtng Lines Pattern

Contains the pattern for routing lines with status = Finished. If this field is empty, the pattern code of a routing line is taken from the status of the production order.

Hard Linking Dependencies

Routing lines are linked by their “Next Operation” and “Previous Operation” properties. This relation is also displayed in the GX view.

If selected, the relation is updated (like a preview) while dragging a job in the GX view. Otherwise, the view is updated after you finished dragging a job.

Check Item Availability

If selected, an item availability check is performed for each job. If some components are missing at the starting date, a red exclamation mark is drawn within the job to warn the user.

Show Progress

Determines whether to show a progress in the routing lines. The progress is rendered in the bottom part of the job if the routing status = In progress. The options are:

  • Disabled – draws no progress bar

  • Duration % – posted duration divided by planned duration

Pattern for with to render the progress bar. You can set the height of the progress bar in the Job Height field of the pattern.

Capacity FastTab

The fields on the Capacity FastTab are described in the following table.

: Prod. Order Planning Setup page, Capacity FastTab

Field

Description

Show Capacities

Disables or enables the capacity rendering. If selected, each row has a special rendering area where a capacity profile is drawn.

Capacity Render Mode

Indicates how a capacity profile is drawn. The options are:

  • Normal

  • Days – Shows the capacity and the capacity load per day with a vertical progress bar.


  • Weeks – Shows the capacity and the capacity load per week with a vertical progress bar.


  • Months – Shows the capacity and the capacity load per month with a vertical progress bar.
  • Days Bars, Weeks Bars, Months Bars – Shows the capacity load (left) and capacity (right) per period. The caption of both bars is always in minutes.

Capacity Period Offset

If the value in the Capacity Render Mode field is:

  • Other than Normal, this value defines an offset that can be used to modify the starting day the chosen period.
  • Weeks – the period starts on a Sunday.

If you enter a value of one in this field, the week period starts on a Monday.

Capacity Caption Mask

If the Capacity Render Mode field = Days, Weeks, or Months, this value is a macro for the caption within the capacity. The following placeholders can be used within the caption:

  • %RATIO – the ratio: capacity load divided by capacity
  • %L – capacity load
  • %LM – capacity load in minutes without decimal places
  • %LH – capacity load in hours with two decimal places
  • %LD – capacity load in days with two decimal places
  • %P – capacity
  • %PM – capacity in minutes without decimal places
  • %PH – capacity in hours with two decimal places
  • %PD – capacity in days with two decimal places
  • %WEEKDAY – name of the day
  • %MONTH – name of the month
  • %DAY, %D – number of the day within the month
  • %DD – number of the day with two places
  • %YEAR, %YYYY – year with four places
  • %YY – year with only two places
  • %CW, %WEEK – calendar week
