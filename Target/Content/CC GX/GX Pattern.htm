GX Pattern

The GX Pattern List page displays all patterns. A pattern contains rendering information, such as background color, border color, hatch style, font size, etc.

: Filtered list of GX Patterns

GX Pattern Card

The GX Pattern properties are defined on the GX Pattern Card page. Open this page from the GX Pattern List page by clicking New to create a new GX Pattern or double-click a pattern to view or edit an existing GX Pattern.

: GX Pattern Card page, General FastTab

General FastTab

The fields on the General FastTab are described in the following table.

Field

Description

Code

Unique identifier of the GX Pattern.

Description

Short description of the pattern, which is only displayed in the overview information.

 

Background FastTab

The fields on the Background FastTab determine the background settings that are used to render the inner part of displayed regions and the hatch style brush over the background.

: GX Pattern Card page, Background FastTab

The fields on this FastTab are described in the following table.

Field

Description

Define Background Settings

Enables or disables the Background settings.

Background Type

Defines the brush style for the filled region. The region can be painted in a single color (Solid) or by using two colors for a gradient style.

From Color

First color used to render this pattern.

From Alpha

Alpha channel of the first color. This value must be in a range from 0 (complete transparency) to 255 (completely opaque).

To Color

Second color used to render this pattern.

To Alpha

Alpha value of the second color.

Gradient Angle

Angle that is used for the gradient direction in degrees:

0 = from left to right

90 = from top to bottom

…

Offset X Pixels / Offset Y Pixels

An additional offset of the gradient filled region; an offset can have a negative value. These fields control a horizontal (X) pattern offset and a vertical (Y) pattern offset.

Offset X Relative / Offset Y Relative

Relative offsets use the region size to calculate a pixel offset. The values should be in a range from -1.0 to +1.0. These fields control a horizontal (X) pattern offset and a vertical (Y) pattern offset

Hatch Enabled

Enables or disables the rendering of a hatch style brush over the background.

Hatch Style

Style to render.

Hatch Style Description

Description of the selected style.

Hatched Color

Color of the style.

Hatched Alpha

Alpha value of the style.

Inverse Hatch

If selected, reverses the rendering of a hatch, since the style brush consists of two different states (color and no color).

 

Border FastTab

The fields on the Border FastTab are described in the following table.

: GX Pattern Card page, Border FastTab

Field

Description

Render Border

Enables or disables a special border rendering pass.

Border Color

Color of the border.

Border Color Alpha

Alpha value of the border color.

Border Width

Width of the border in pixels. This value can be fractional.

 

Jobs FastTab

The fields on the Jobs FastTab are described in the following table.

: GX Pattern Card page, Jobs FastTab

Field

Description

Round Jobs

Enables or disables a rounded shape when rendering jobs with this pattern.

Edge Radius

Edge radius for rounded jobs.

Job Height

Specific height (in pixels) for jobs that are rendered with this pattern. A value of zero disables this function and renders jobs in the default height.

Disable Job Shadows

If selected, renders no shadows for jobs with this pattern.

 

Text FastTab

The fields on the Text FastTab are described in the following table.

: GX Pattern Card page, Text FastTab

Field

Description

Define Text Settings

Enables or disables the following additional text settings. If not selected, text will still be rendered (in a default way).

Text Color

Color of the text.

Text Color Alpha

Alpha value of the text color.

Font Size

Size of the text in pixels.

Bold

Bold property of the text

Italic

Italic property of the text

Underline

Underline property of the text

Text Horizontal Alignment

Horizontal alignment of the text (Left, Center, or Right)

Text Vertical Alignment

Vertical alignment of the text (Top, Center, or Bottom)

Render Text Border

Enables or disables rendering of a border that surrounds the text (like a glow).

Text Border Color

Color of the text border.

Text Border Alpha

Alpha value of the text border color.

Text Border Width

Width of the text border in pixels.
