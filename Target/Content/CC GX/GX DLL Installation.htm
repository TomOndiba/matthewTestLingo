GX DLL Installation

The client add-in must be installed on each client machine that will be used to run GX. This can be done by running the client installation on individual client machines, or done universally from the server installation, which will distribute the client add-in to all client machines. The installation processes are essentially the same except for where the files are installed.

Run GX DLL Client Installation Program

Follow these steps to run the GX client installation file:

  1. Locate and double-click the “cc graphical eXtension client setup (Version 9.00).msi” file. The Setup Wizard window opens.

: Setup Wizard, Welcome window

  1. Click Next. The Destination Folder window appears.

: Setup Wizard, installation destination folder selection

  1. Accept the default target folder or specify a new folder for the GX installation and then click Next. The Ready to install cc|graphical eXtension window appears.

: Setup Wizard, ready to start the installation

  1. Click Install. The Installing cc|graphical eXtension window appears with a progress bar that indicates the status of the installation process.

: Setup Wizard, installation progress

When the installation is finished, the installation completed window appears.

: Setup Wizard, installation completed

  1. Click Finish. The Setup Wizard closes.

After the installation, multiple files are located in the installation folder specified in the setup wizard.

: Files in the GX installation folder (client)

Uninstall GX

The GX DLLs are uninstalled with the standard uninstall program function from the Control Panel.
