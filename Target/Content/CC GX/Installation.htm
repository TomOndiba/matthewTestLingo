Installation

For a complete cc|graphical extension (GX) installation, the following steps must be performed:

  1. Install the external components on the server or your client PC (refer to topic "GX DLL Installation" on page 1).
  2. Register the Client add-in in your Microsoft Dynamics NAV database (refer to topic "Register Client Add-Ins" on page 1).
  3. Create or import settings (refer to topic "GX Setup Import/Export" on page 1) if the demo database is not used.
  4. Import a GX license file (refer to topic "License Setup" on page 1) if the demo database is not used.

Information about removing the GX installation is also included in this section.
