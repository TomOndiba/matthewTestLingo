Register Client Add-Ins

The Client add-ins must be registered to Microsoft Dynamics NAV otherwise it will deny the instantiation of the Client add-ins at runtime. This registration is needed only once per database. In the navigation pane, click Departments > cc|graphical extension > GX Setup > Actions > Register Client Add-ins.

: Accessing the Register Client Add-ins function
