GX Prod. Order Planning View

To open the GX view for Manufacturing, click cc|graphical extension > Prod. Order Planning.

: GX Prod. Order Planning View page

The GX Prod. Order Planning View page shows production orders and their operations. The operations are shown in the same row as their corresponding work center or machine center. The operations can be altered using the drag and drop method and their work center (or machine center) can be changed by dragging them vertically.  The duration of an operation and the starting or ending date and time can also be modified using the mouse.

This view also displays the current capacity load state of the work centers and machine centers.

Calendar Control

The Calendar control shows the same date range that is visible in the Timeline. The displayed date area can be changed by clicking and dragging the mouse on the Calendar control.

: GX Prod. Order Planning View page, Calendar Control

Timeline Control

The Timeline controls the date area for the rendered entries. Scroll horizontally by clicking and holding the mouse on the Timeline, and the dragging it to the proper date range. The Previous and Next buttons can also be used to scroll to the next or the previous period.

To scale the Timeline, hold down the right mouse button and drag the mouse cursor horizontally or click the Day, Week or Month buttons.

: GX Prod. Order Planning View page, Timeline Controls

The Timeline can also be adjusted by clicking the following menu items on the ribbon:

  • Center Work Date – Moves the work date to the center of the Timeline
  • Current Week – Adjusts the Timeline to show the current week

Detail View Control

The Detail View control contains rectangles that are referred to as jobs in our context. Jobs can be picked and dragged by holding down the left mouse button.

: GX Prod. Order Planning View page, Detail View Control

While dragging, press the CTRL or SHIFT key to lock a horizontal or vertical movement; press the ESC key to cancel the dragging process.

It is also possible to alter the duration of jobs by dragging the left or right edge of a job. To allow this alteration capability, ensure that the Runtime Editable check box is selected on the Routing Lines FastTab in the Prod. Order Planning Setup page.

Note

Jobs in the Operations sections (routings) can be altered; the jobs in the Production Orders/Prod. Order Lines sections cannot be altered.

Explorer Control

The Explorer control contains various types of data:

  • The top section lists the work center groups, work centers, and machine centers; or production orders
  • The bottom section switches the Detail View between Work Center Groups or Production Orders

: GX Prod. Order Planning View page, Explorer Control

If an entry in the top section is selected, the Detail View scrolls to the selected entry.

Search Control

In the Search field, enter a search description (using the standard Microsoft Dynamics NAV filtering syntax for the search text) and then click the Find icon. When the matching criterion is met, the view automatically scrolls to the record. Click the Find icon again to move to the next record that meets the search criteria.

: GX Prod. Order Planning View page, Search Control

Context Menus

Normally the displayed entries are linked to a Microsoft Dynamics NAV record, so you can open a context menu by right-clicking the entries.

: GX Prod. Order Planning View page, Context Menus

The context menu show functions related to the Microsoft Dynamics NAV record that is linked to the selected entry. These functions are executed by standard Microsoft Dynamics NAV functions.

Tooltips

When a displayed item is linked to a Microsoft Dynamics NAV record, there might also be a tooltip available. To activate a tooltip, set the mouse cursor over an entry and wait a short while. The tooltip shows information related to the entry under the mouse cursor.

: GX Prod. Order Planning View page, Tooltip in Detail View control

The tooltip also works in the Explorer control.

: GX Prod. Order Planning View page, Tooltip in Explorer control

Refresh Function

On the ribbon, click Refresh View to update the GX view with new or modified data (from the GX view or the Microsoft Dynamics NAV application).
