By-Product Sequence Planning

If an item’s routing contains a work center that is a sequence work center and a production order is created for this item, the production order line is shown in the Sequence Planning page for that work center.

If the item has by-products, the production order contains an additional production order line for each by-product. These lines are attached to the production order line for the main item (the line shown in sequence planning).

The additional lines are not shown in the Sequence Planning page for the work center. But if a user changes the starting or ending times of the production order line of the main item manually – by re-scheduling or by moving the line to a different sequence position – the starting and ending times of the additional production order lines are changed as well.

Refer to topic “Sequence Planning” on page 1 for more information.
