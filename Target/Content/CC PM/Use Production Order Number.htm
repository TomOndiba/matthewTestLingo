Use Production Order Number as Lot Number

In cc|process manufacturing, the production order number can be set up to automatically be used as the lot number. For this to occur, the Use Prod. Order No. as Lot No. check box must be selected on the Process Manufacturing Setup page.

: Setup required to use the production order number as the lot number

If this check box is selected, production order numbers are used as lot numbers when production orders are posted.

: Production order number used as the lot number

Under the following conditions the production order number is not used as the lot number, even if the “Use Prod. Order No. as Lot No.” function is activated:

  • If a lot number is assigned manually, this number has priority and is used as lot number.
  • If a lot number series is entered on the item card, the production order number is not automatically used as lot number.
  • If a production component that is set up as lot determining is used, the lot determining lot number takes priority over the production order number as lot number.
