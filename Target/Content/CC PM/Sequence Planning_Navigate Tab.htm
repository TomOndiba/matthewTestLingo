Navigate Tab

The actions on the Navigate tab relate to the work center selected in the General section of the Sequence Planning page.

In the following table, each cc|process manufacturing page is described in detail while standard Microsoft Dynamics NAV pages are not; refer to online help for additional information about standard pages.

Action

Description

List

Displays a list of all sequence work centers.

Card

Displays the card of the selected work center. Refer to topic “Capacities” on page 1 for more information.

Note

A card will only open if there is a work center sequence on the line.

Capacity Ledger Entries

Displays the capacity ledger entries for the selected work center.

Capacity ledger entries are created if capacity consumption is entered for a production order. They are created in time units (run time, stop time, setup time) or in quantities (scrap quantity, output quantity).

Dimensions

Displays the dimensions assigned to the selected work center.

Manufacturing Dimensions

Displays the default manufacturing dimensions for the selected work center (refer to topic “Manufacturing Dimensions” on page 1).

Planning Dimensions

Planning dimensions are used to better handle sequence planning. For example, based on planning dimension priorities of color, machines will be sequences so lighter colors are processed first, followed by darker colors (refer to topic “Manufacturing Dimensions with Sequence Work Centers” on page 1).

As illustrated in the above screen shot, when the “Priority Planning” function is run, the Sequence Planning lines will be re-sorted by due date, then color and so on (refer to topic “Optimize Planning Functions in Sequence Planning” on page 1).

Comments

Displays comments for the selected work center.

Load

Contains a drop-down list with the following menu items:

  • List – Opens the Work Center Load page, which provides an overview of availability at the selected work center.
  • Load/Setup - Chart – Opens the Work Center Load/Setup - Chart page, which provides a chart-style overview of availability (Load) or Setup-/Run Times (Setup) at the selected work center.

  • The items in the menu groups on the Home and Actions tab determine the displayed information:
    • Chart – switch between Load and Setup views
    • Period Length – time periods to be displayed
    • Options – values to calculate (capacity, load or both) and type of chart (column, pie, line and so on)

Statistics

Displays the statistics for the selected work center.

Text Module

Opens the Text Module List page where special text can be added for the Sequence Work Center or existing text can be viewed or edited (refer to the current “cc|text module” documentation).
