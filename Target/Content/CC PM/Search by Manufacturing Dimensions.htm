Search by Manufacturing Dimensions

The Search by Mfg. Dimensions page can be used to search items, item variants, work centers, tools and inserts by their specified manufacturing dimensions. It is accessed from the Product Design or the Manufacturing, Tasks menu.

: Searching by manufacturing dimensions

In the Find What field, enter what the function needs to search for (items, work centers, tools or inserts).

In the Mfg. Dimension Code field, enter the manufacturing dimension to be searched for and then set at least one value filter in the Mfg. Dimension Value Filter field.

To search for manufacturing dimensions specified in a manufacturing dimension template, click Get Mfg. Dimension Template on the ribbon.

Once all searching criteria has been set, click Find on the ribbon to run the search function.

: Search results of the items and item variants that contain the RED manufacturing dimensions in the SHEET manufacturing dimension template

The Item Card for selected items/item variants can be displayed by clicking Show on the ribbon.

The search for work centers, tools and inserts is performed as instructed for the items.
