Change Routing Lines with Production Rates

The Work Center, Tool and Insert combination with setup and run times can be changed on a routing line by selecting a Work Center Production Rate.

The “Get Work Center by Production Rate” function is accessed from the Prod. Order Routing page.

: Accessing the Get Work Center by Production Rate function

This function opens the Work Center Production Rate list page, which displays records with a work center number. Depending on the Work Center / Tool / Insert combination in the Production Order Routing, certain Work Center Production Rate records are shown:

  • Only a Work Center in the Production Routing Line:
    • Work Center Production Rates with Work Center Numbers and without Tool or Insert Numbers.
    • No Item filter is used.
  • Work Center with attached Tool Number in the Production Routing Line:
    • Work Center Production Rates with Work Center and Tool Numbers but without Insert Numbers.
    • Only records with the Item Number of the Production Order Line are shown.
  • Work Center with attached Tool and Insert Number in the Production Routing Line:
    • Work Center Production Rates with Work Center, Tool and Insert Numbers.
    • Only records with the Item Number of the Production Order Line are shown.

: Work Center Production Rates available for item P-10401

The Work Center Production Rate Card can be opened from the Work Center Production Rates page to view existing records or new records can be added by clicking New on the ribbon.

On selection of a Work Center Production Rate with a Work Center / Tool / Insert combination different than the current Production Order Routing, the Work Center / Tool / Insert Numbers and the Setup Time and Run Time are changed.
