Send Note on Production Order Delay

If the ending date of a production order line linked to a sales order line via reservation is changed to a date later than the production order line due date, a note will be created and attached to the sales order. The notification function can be enabled or disabled using the Send Note on Prod. Order Delay field in Manufacturing Setup.

: Send Note on Prod. Order Delay setting on Manufacturing Setup

If the salesperson entered in the sales order is linked to a user via User Setup, the note will be shown in the My Notifications part of the salesperson’s role center.

: User Setup with link to salesperson

The sales order can be opened directly from this notification.

: Delay notice in the My Notifications part of salesperson’s role center
