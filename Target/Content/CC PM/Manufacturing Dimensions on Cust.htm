Manufacturing Dimensions on Customers

Manufacturing dimension values can be assigned to customers. A manufacturing dimension must be defined as the customer manufacturing dimension in the Customer Mfg. Dimension Code field in the Planning Extension Setup page (refer to topic “General FastTab” on page 1). The manufacturing dimension values of this manufacturing dimension can be assigned to customers.

: Dimension value of the Customer Priority customer dimension assigned to customer 10000

The customer manufacturing dimension values can be used to define customer priorities which can be used for priority planning (refer to topic “Priority Planning” on page 1).   

When a sales order is created for a customer, the value in the Customer Mfg. Dim. Value Code field is transferred to the Customer Mfg. Dim. Value Code field on the Shipping FastTab of the Sales Order.
