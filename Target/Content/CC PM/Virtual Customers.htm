Virtual Customers

Since Microsoft Dynamics NAV is only able to create sales documents (Sample Order = Sales Order) for customers and not for contacts that are not yet linked to a customer, the system uses a Virtual Customer in the sales header to be able to create orders for contacts.

A virtual customer must be created for every existing combination of country codes and currency codes. To do so, select the Virtual Customer check box on the Contingents FastTab of the Customer Card.

: Virtual Customer information on the Customer Card

Note

A default virtual customer can be assigned in the Virtual Customer field on the General FastTab in the Marketing Setup page.

If the system should find nonconformity, the user will have to select an appropriate record from a list of virtual customers. Refer to topic “Contingent Management for a Contact” on page 1 for more information.
