Calculation Unit of Measure

Since the concept of unit of measures (weights, package and so on) provides an automatic conversion, it is difficult to use, for example, a recipe based on KG to enter a packaging unit in the item list.

The following fields on the Production Structure lines provide the automated calculation of the optimal quantity per unit of measure:

Required Quantity

Allocation Quantity

Unit of Measure Code

Allocation Unit of Measure Code

 

: Production Structure lines with calculation units of measure

As Fig. 1 illustrates, for item P-10600, Adhesion AT5 (100-500), one can of red paint is required but the production structure only refers to kilograms. With the conversion of cans to kilograms, the needed quantity (Quantity per) of the component will be calculated.

Note

This function is only available with items, not with parts of a recipe since a different calculation is needed.
