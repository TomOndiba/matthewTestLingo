Incorporation Usage

Incorporation usage shows all production orders into which a user can incorporate an item.

The assignment of incorporation to a production order can be made directly in the Incorporation Usage page, opened from the Item Card by clicking Navigate > Production > Incorporation Usage. This page displays the maximum incorporation percentage and quantity for each production order.

: Incorporation Usage listing valid production orders

In the Incorporation Usage page, enter a percentage up to the maximum value in the Incorporation % field or the maximum quantity in the Incorporation Quantity field.

Alternatively, click Automatic Allocation on the ribbon to automatically distribute the available inventory amongst the available production orders. After using this function, manual changes can be done on the lines as needed.

: Incorporation percentages automatically allocated to production orders by demand date

Click OK to transfer the usage to the production order(s). The Qty. on Component Lines field on the Item Card is updated based on the Incorporation Quantity.

: Incorporation Usage updated on the Item Card

Production Order Status Filter

Use the production order Status filter to restrict the production order lines for the incorporation usage allocation.

: Incorporation Usage without status filtering

After a status filter is set, the incorporation usage lines are updated accordingly.
