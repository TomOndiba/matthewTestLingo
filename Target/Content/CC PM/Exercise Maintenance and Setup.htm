Exercise – Maintenance and Setup Order

Several items are processed on machine M001. When priority planning is run, there is always a change from P-10001 to P-10005 and P-10005 to P-10006.

The changing from P-10001 to P-10005 has been defined in the Setup Matrix. You need to define the change of six hours from P-10005 to P-10006 with a new Setup Order named Setup Order Material Change 3.

Set up the Maintenance-/Setup Job Type and define the change in the Setup Matrix. In Sequence Planning, run priority planning and then test the results.

Create the Maintenance-/Setup Job Type

Follow these steps to create the Maintenance-/Setup Job Type:

  1. In the navigation pane, under Manufacturing, click Maintenance-/Setup Job > Maintenance-/Setup Job Types.
  2. On the ribbon, click New.
  3. In the No. field, type WA009.
  4. In the Description field, type Setup Order Material Change 3.
  5. In the Order Type Code field, enter SETUP.
  6. In the Standstill Code field, enter SETUP.
  7. In the Setup Time field, type 6.
  8. In the Setup Time Unit of Meas. Code field, enter HOURS.
  9. Click OK to close the Maintenance-/Setup Job Type page and then close the Maint.-/Setup Job Type List page.

Define the Machine Change in the Setup Matrix

Follow these steps to define the machine change in the Setup Matrix:

  1. On the Maintenance-/Setup Job menu, click Maintenance-/Setup Matrix.
  2. Right-click the first line from item P-10005 to item P-953-KP001 and then click New Line.
  3. In the Work Center No. field, enter M001.
  4. In the From Item No. field, enter P-10005.
  5. In the To Item No. field, enter P-10006.
  6. In the Maint.-Setup Job Type No. field, enter WA009.
  7. Click OK to close the Maintenance-/Setup Matrix page.

Review the Maintenance-/Setup Orders in Sequence Planning

Follow these steps to review the maintenance-/setup orders in the Sequence Planning page:

  1. In the navigation pane, under Manufacturing, click Planning > Sequence Planning.
  2. Click Priority Planning on the ribbon, select the Schedule Deferred Prod. Order Lines check box and then click OK.
  3. On the ribbon, click Actions > Maintenance-/Setup Matrix > Planning Maintenance-/Setup Matrix.
  4. Click Yes in the message to run the function.
  5. Confirm that a production order for “Setup Order Material Change 3” was inserted between all changes from item P-10005 to P-10006.
  6. Select one of the new production orders for “Setup Order Material Change 3” and then click Line > Routing on the Lines FastTab.
  7. Confirm that the Setup Time is set to 6 hours.
