Returnable Containers in Production

Returnable containers can be added to BOMs.

: Production Structure with returnable container

When a production order is created, the container is transferred as a component and the Returnable Container check box is selected on the line.

: Reusable container listed as a production order component

The container is grouped separately from the other lines in the Production Journal and is identified as a Returnable Container.

: Production Journal with reusable container

When the production order is posted, the container(s) is transferred to the finished location (where the output is posted), not posted from inventory.
