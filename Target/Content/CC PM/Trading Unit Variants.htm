Trading Unit – Variants

The concept of the Variants Trading Unit is based on the standard item unit of measure. When a new item unit of measure is created, a new variant is created in the Item Variants page.

: Item Card with unit of measures and variants

The Item Availability by Variant page has been enhanced to accommodate this functionality. In the standard page, all values are displayed in the base unit of measure.

: Item Availability by Variant with base unit of measure quantity displayed

On the Options FastTab, the Quantity field has been added to view the trading unit quantity per unit of measure. The Quantity per Unit of Measure option displays the number of trading units of measure for each unit of measure/variant.

: Item Availability by Variant with quantity per unit of measure displayed

Since a variant is created for each item unit of measure, the item variant represents exactly one item unit of measure. When posting to inventory, these values must be equal or the following error message is displayed.

: Posting error for different trading units
