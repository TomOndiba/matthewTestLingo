Select Lot for Production Components by QC Test Values

The item lots that should be used for a production order component can be chosen by viewing their QC test values. On the Prod. Order Components page, select the component and then click Actions > Select Lot for Prod. Component on the ribbon.

: Select Lot for Prod. Component page

The Select Lot for Prod. Component page contains four areas:

  • Header for filtering
  • Component lot lines
  • QC Test values (Lines FastTab)
  • Search options and component quantity information

In the Select Lot for Prod. Component page, select a component lot line and then review the QC test values.

Click the Selected check box on the component lot line that you want to use; the Qty. to Handle field is automatically updated to match the Remaining Quantity. Update the Qty. to Handle field to the needed component quantity.

: Lot No. CH-3046 selected for a quantity of 475 for component P-30000

After the lot is selected, click Assign on the ribbon to create a reservation between the production order component, inventory and item tracking entry. Click OK to close the Select Lot for Prod. Component page.
