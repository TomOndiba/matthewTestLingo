Work Center Production Rates

The setup times and production rate values of the machines – or different combinations with one machine – are maintained in the Work Center Prod. Rates Card page. These values are only used for calculating the production orders if the routing has no data in this line. Otherwise, times have the highest priority in the routing (refer to topic “Work Time Priorities” on page 1).

The page is opened from the Work Center Card or Sequence Work Center Card by clicking Navigate > Work Center Production Rates or from the Item Card by clicking Navigate > Production > Work Center Production Rates.

: Work Center Prod. Rate Card for Item P501, Work Center M015

On the right side of the General FastTab, setup times and setup time units of measure as well as production rate values are maintained.

In Fig. 1, work center M015 has a setup time of 45 minutes and a production rate of 1,500 kilograms per eight hours.

Note

The unit of measure of the production rate must be the same unit of measure as defined in the Item or the Work Center Prod. Rate Card. If using an item, you can only select the item’s UOM. If using a work center, you can only use correct capacity UOMs. If using an item category, it is possible to select a unit of measure that is not available in all items of this category. However, this will cause an error on production order calculation.

The use of the Standard Task Code field on the Standard Task FastTab is described in topic “Comments, Tools, Personnel and Quality Measures” on page 1.
