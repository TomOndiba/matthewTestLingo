Item Ingredients

Ingredients can be assigned to any type of item: manufactured, component, finished goods, and so on. While this section is primarily focused on manufactured and component items, the process for assigning ingredients to the different items is the same.

Ingredients can be allocated to the manufactured items. The entered ingredients and ingredient quantities represent the nominal condition of the product, which means that the specified ingredient quantities should be achieved when producing the according intermediate or final product. The defined ingredients (and their respective quantities) are transferred to the header of the production structure.

To ensure that the ingredient calculation is correct for output, expected ingredients must be assigned to any component item with monitored ingredients. The ingredients and ingredient quantities that are allocated to the component should mirror the fair quality of the purchased or produced component items. The defined ingredients (and their respective quantities) are transferred to the lines of the production structure.

Additionally, ingredients can be assigned to finished good items for creating declaration documents.

The allocation for items are entered on the Item Card by clicking Actions> Ingredients.

: Item Ingredients assigned to a manufactured item that has base unit of measure = KG

The pre-defined ingredients are selected in the Ingredient No. field. The Description and Unit of Measure Code (equals the Base Unit of Measure Code of the ingredient) fields are inherited from the master data of the ingredient. 

The Quantity field must be filled with the quantity of the ingredient which the item should contain. This quantity is used as the standard quantity in the Production Structure header when creating a Production Structure.

: Item Ingredients assigned to a component items that have base unit of measure = KG

Allocation Unit of Measure

The Alloc. Unit of Measure Code field is inherited from the Base Unit of Measure Code from the item. However, this value or the Unit of Measure Code can be changed here or later in the production process to handle quantity conversions. For example, if an item has a base unit of measure of KG, but the ingredient rate included in the raw material is very low, it would be useful to change the allocation unit of measure to T (metric ton) so the quantity will not be small, like 0.1. Conversely, if the ingredient rate included in the raw material is very high, the unit of measure or allocation unit of measure can be changed to GR.

Exercise – Create Manufactured and Component Items with Ingredients

Create three lot tracked component items with base unit of measure of KG and a unit of measure conversion for GR. Ingredient 1 will be assigned to Item A for a quantity of 10, Ingredient 2 will be assigned to Item B for a quantity of 30, and no ingredients will be assigned to the Filler item.

Then create a lot tracked manufactured item with the same unit of measure setup. Ingredient 1 will be assigned with a quantity of 1 and Ingredient 2 will be assigned with a quantity of 5.

Create Component Items with Ingredients

Follow these steps to create the component items with ingredients:

  1. In the navigation pane, click Manufacturing > Product Design > Items.
  2. On the ribbon, click New.
  3. Click Cancel to close the template selection page.
  4. In the General FastTab, click Show more fields if the No. field is not displayed.
  5. In the No. field, type P-I01.
  6. In the Description field, type Item A.
  7. In the Base Unit of Measure Code field, enter KG.
  8. Expand the Price & Posting FastTab.
  9. In the Gen. Prod. Posting Group field, enter RMS-PMW.
  10. In the Inventory Posting Group field, enter RAW MAT.
  11. Expand the Item Tracking FastTab.
  12. In the Item Tracking Code field, enter LOT_QUAR .
  13. In the Lot Nos. field, enter LOT.
  14. On the ribbon, click Navigate > Units of Measure.
  15. On the ribbon, click New.
  16. In the Code field, enter GR and then enter .001 in the Qty. per Unit of Measure field.
  17. Click OK to close the Item Units of Measure page.
  18. On the Actions tab, click Ingredients.
  19. In the Ingredient No. field, enter INH-0001 (Ingredient 1).
  20. In the Quantity field, type 10.

: Item Ingredient assigned to component Item A

  1. Click OK to close the Item Ingredients page.
  2. On the Actions tab, click Copy Item (in the Functions menu group; use the Customize Ribbon function to add the Copy Item function if not accessible by default).
  3. In the Target Item No. field, enter P-I02.
  4. Select the Set All check box.
  5. Click OK and then click Yes to the message to edit the item.
    Notice the new item card for item P-I02 is active.
  6. In the Description field, type Item B.
  7. On the Actions tab, click Ingredients.
  8. In the Ingredient No. field, enter INH-0002 (Ingredient 2).
  9. In the Quantity field, type 30.

: Item Ingredient assigned to component Item B

  1. Click OK to close the Item Ingredients page.
  2. On the Actions tab, click Copy Item.
  3. In the Target Item No. field, enter P-I07.
  4. Ensure that Set All check box is selected.
  5. Click OK and then click Yes to the message to edit the item.
    Notice the new item card for item P-I07 is active.
  6. In the Description field, type Filler.
  7. Do not close the Item Card.

Create Manufacturing Item with Ingredients

Follow these steps to create the manufacturing item with ingredients:

  1. On the Item Card, click Home > New.
  2. In the template selection page, select the ITEM01, Item FINISHED template and then click OK.
  3. In the No. field, type P-I10 and then click Yes to rename the record.
  4. In the Description field, type Product.
  5. In the Base Unit of Measure Code field, enter KG.
  6. Expand the Invoicing FastTab.
  7. In the Standard Cost field, type 2 and click Yes to change standard cost.
  8. Expand the Replenishment FastTab.
  9. In the Replenishment System field, enter Prod. Order.
  10. Expand the Planning FastTab.
  11. In the Reordering Policy field, enter Order.
  12. On the Item Tracking FastTab and enter LOT_QUAR in the Item Tracking Code field.
  13. In the Lot Nos. field, enter LOT.
  14. On the Navigate tab, click Units of Measure.
  15. On the ribbon, click New.
  16. In the Code field, enter GR and then enter .001 in the Qty. per Unit of Measure field.
  17. Click OK to close the Item Units of Measure page.
  18. On the Actions tab, click Ingredients.
  19. In the Ingredient No. field, enter INH-0001 (Ingredient 1).
  20. In the Quantity field, type 1.
  21. Go to the next line.
  22. In the Ingredient No. field, enter INH-0002 (Ingredient 2).
  23. In the Quantity field, type 5.

  1. Click OK to close the Item Ingredients page.
  2. Click OK to close the Item Card.

Ingredients with Lot Tracking

Ideally, ingredients should be assigned to lot tracked items before initial inventory is posted so that the default ingredient quantities are automatically created in the Lot No. Ingredients page (opened from the Lot No. Information Card page).

If ingredients are added to lot tracked items after inventory has been posted, the ingredient quantities must be manually assigned to the posted Item Tracking Entries (Item Card > Navigate > Entries > Item Tracking Entries > Navigate > Lot No. Information Card > Edit > Navigate > Ingredients).

Refer to topics “Receipt Testing” on page 1 and “Item Delivery” on page 1 for more information about ingredients and item tracking.

Item Variants

Ingredients can be assigned to item variants in the same way ingredients are assigned to the master items. On the Item Variants page, click Navigate > Ingredients.
