Setup

To open the Maintenance-/Setup Matrix page, on the Manufacturing menu, click Maintenance-/Setup Job > Maintenance-/Setup Matrix.

: Maintenance-/Setup Matrix page filtered on work center M002

The fields in the General and Options sections only serve for filtering. Enter the values here, for example a certain tool, to automatically insert this data on the lines during data entry.

In the lines, users can select a work center and a change; for example, item P-953-KP001 to P-10001. In the same line, users can enter the time and unit of measure that is needed for the change. Users can also configure:

Work Center Number

Tool Number

Item Category

Manufacturing Dimension Code and Value

Item Number

Starting Date

Item Variant Code

Ending Date

Note

The values in the From Mfg. Dimension Code and To Mfg. Dimension Code fields must be equal if Manufacturing Dimensions are used.

Maintenance-/Setup Job Field

The Maint.-/Setup Job Type No. field on the Maintenance-/Setup Matrix page allows users to select an existing maintenance-/setup job. If the field remains empty, it has a different effect on the calculation:

  • If the field is empty, the predefined value is transferred to the related routing during the calculation of setup times.
  • If the field contains a number, a separate maintenance-/setup job is created in the sequence planning in the calculation.
