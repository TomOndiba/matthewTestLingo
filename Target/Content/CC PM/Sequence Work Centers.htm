Sequence Work Centers

The sequence of the work centers is stored in the Sequence Work Center Card, which has generally the same functions as the Work Center Card.

: Sequence Work Center Card for work center M001

If a Sequence Work Center is created manually, a Work Center Card is automatically created. A Sequence Work Center is only created from a Work Center if the Work Center Type field is set to Sequence Planning.

Data entered in the sequence work center card is automatically identified for Sequence Planning.

Work Center Production Rates can be viewed from this page (refer to topic “Work Center Production Rates” on page 1).

For more information about the Rescheduling and Scheduling Required fields on the Scheduling FastTab, refer to topic “Manual Rescheduling of Sequence Planning” on page 1.

Additionally, a graphical view of the Sequence Work Center load and setup information is displayed in the Work Center Chart FactBox. The menus above the chart determine the displayed information:

  • Chart – switch between Load and Setup views
  • Period Length – time periods to be displayed
  • Options – values to calculate (capacity, load or both) and type of chart (column, pie, line and so on)
