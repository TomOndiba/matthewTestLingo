Fixed in Sequence Planning

With this function, users can fix production orders in the sequence planning so that the current calculated start and end dates are not changed during the sequence planning process.

: Sequence Position 3 set as fixed

The scheduling process checks if the production order can be planned before the fixed production order. If there is not enough time, the production order is planned after the fixed production order.

If the free time before the fixed production order is greater than or equal to the value in the Min. Duration Fixed Maint.-Job field in the Planning Extension Setup page, a maintenance order is inserted to fill the free time.

: Maintenance order position added to fixed planning lines

As Fig. 2 illustrates, after scheduling was run, the original calculated time remained on the fixed production order and a maintenance order was inserted.

Whether the maintenance order appears and under which conditions it can be viewed is determined in the Planning Extension Setup page (refer to topic “Planning Extension Setup, Maintenance FastTab” on page 1).
