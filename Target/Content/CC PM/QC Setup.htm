QC Setup

In the QC Setup page, define how the system will administrate the Quality Control functionality.

: QC Setup page

In addition to the number series that must be set up to use the quality control functionality, there are four fields used to validate the quality check status. Each is described in the following table.

Field

Description

Check QC Lot Test

If selected, the QC Checks are verified based on the question status.

Question Status All Right

Identifies the display color in the Question Status field in the lines of the QC Test, Lot No. Information Card, or Select Lot for Prod. Component pages if the answer to a question allows the release of the QC Check.

Note

The color is selected from the Data Color Control List page (refer to topic “Data Control Colors” on page 1).

Question Status Restriction

Identifies the display color in the Question Status field in the lines of the QC Test, Lot No. Information Card, or Select Lot for Prod. Component pages if the answer to a question restricts the release of a QC Check.

Note

The color is selected from the Data Color Control List page (refer to topic “Data Control Colors” on page 1).

Question Status Blocked

Identifies the display color in the Question Status field in the lines of the QC Test, Lot No. Information Card, or Select Lot for Prod. Component pages if the answer to a question blocks the release of a QC Check.

Note

The color is selected from the Data Color Control List page (refer to topic “Data Control Colors” on page 1).
