Status of Lot Numbers

In cc|process manufacturing, lot numbers can be assigned a status that controls whether inventory can be sold, transferred or consumed. This can be done using the QC Status with locations (refer to topic “QC Status Validation” on page 1).

: QC Status setup for lot tracking

Each QC Status/location combination can be set up to allow transfers, sales and consumption. The relevant check box is selected on the QC Status. For locations, the following check boxes are selected:

  • Use As Transfer Location – only inventory with a QC Status that has the Transfer check box selected can be transferred to this location.
  • Use As Sales Location – only inventory with a QC Status that has the Sales check box selected can be transferred to this location.
  • Use As Consumption Location – only inventory with a QC Status that has the Consumption check box selected can be transferred to this location.
