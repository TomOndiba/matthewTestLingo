Planning Coordination

The Planning Coordination Overview page collects all system data grouped by orders and provides an overview of all relevant information. This page is opened from the Manufacturing menu by clicking Planning > Planning Coordination.

The core information in the order structure is the warning. It should also serve as a differentiation for the order structure for all users, such as sales staff, order processors, purchasing agents and production staff. All warnings have urgent need for action in respect to other orders without warnings.

For example, the following information is shown for purchase and sales order lines:

  • For sales order lines:
    • Display of warehouse shipments that should be delivered, but are not delivered yet.
    • Display of sales shipment lines, which are not invoiced yet.
  • For purchase order lines – display of warehouse receipts that should be delivered, but are not delivered yet.

: Planning Coordination Overview page

The Planning Coordination Overview page contains no data when it is opened. On the Options section, define filters to reduce the time necessary to collect data. After setting filters, click Find on the ribbon to populate the page. If filters are changed, the “Find” function must be run again.

Note

If the Find As You Type check box is selected in the Search Options section, the search is started directly after entering a filter in the Options section. Therefore, it is unnecessary to use the “Find” function.

: Planning Coordination Overview filtering on shipment dates

After the search is complete, the number of orders and warnings is calculated on the Options section. The Warning % is based on the ratio of orders to warnings and provides an overview of the current situation of the company. The difference should not be greater than 90%. This percentage will vary slightly and should be included as a goal factor for the group leader.

The lines display a list of all orders. The important columns are:

  • Type of information (order, production order and so on)
  • Warning (warning received, warning)
  • Source Information (customers name, vendor name and so on)
  • Information to content (item, resources, quantities and so on)
  • Demand Date and Planned Date
  • Person responsible (salesperson, buying agent)
