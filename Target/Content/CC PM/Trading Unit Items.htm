Trading Unit – Items

Item Trading Units require more settings than variants. In the Units of Measure page, one item extension must be entered for each unit of measure.

: Item extension for units of measure

When a new item unit of measure is added to an item that has the Item trading unit selected, a new item is created automatically and is linked to the base item. The entry made in the Item Extension field of the corresponding unit of measure is added to the end of the new item number.

: New unit of measure 4 L added to item P-10502

As illustrated in Fig. 2, the 4 L unit of measure was added to item P-10502. The 4 L unit of measure is set up with an item extension of -4L.

: Item P-10502-4L created from item P-10502

As Fig. 3 illustrates, item P-10502-4L was created from item P-10502. The same information from P-10502 was transferred to item P-10502-4L with some exceptions, including but not limited to:

  • Item No. contains the item extension
  • Base Unit of Measure from the unit of measure added to item P-10502 and the base unit of measure of the base item has been converted to 4 L
  • Trading Unit per Item UOM is blank
  • Trading Unit Item check box is selected
  • Base Trading Unit Item links to item P-10502

Base Trading Unit Item Sorting

In the Item List page (or Items list page), the Base Trading Unit Item field has been added to provide an overview of the item trading unit structure.

Note

This field must be added manually using the “Choose Columns” function.

: Base Trading Unit Item field added to Item List page, sorted by item number

Click the Base Trading Unit Item field name to re-sort the Item List page and indent the sub-items below the base item.

: Sub-items indented below the base item P-10502

Trading Unit – Item Adjustments

To simplify the maintenance of the data that is copied to the new sub-item that is created for each item unit of measure of a base item, use the Trading Unit Item Adjustment page to define the fields that should be copied for all sub-items during changes in the base item.

The Trading Unit Item Adjustment page is accessed from the Manufacturing, Administration menu, under Setup.

: Trading unit item adjustment setup

When creating a sub-item, the quantity per unit of measure related to the base item can be taken into consideration by selecting the Consider Quantity check box. For example, a sub-item is created for a pallet (1 pallet = 10 pieces). If the unit price is 7 EUR for one piece, then the unit price of the sub-item for pallets is 70 because the unit price from the base unit was multiplied by 10.

Select the Skip check box on any fields that should not be copied to the sub-item. As illustrated in Fig. 6, the value in the Shelf No. field will not be transferred from base items when sub-items are created.
