Creating a Production Order

When creating a production order, the by-products are calculated in relation to the main product.

: Firm Planned Production Order for item P-953-KP001

In Fig. 1, item P-953-KP001 has a production quantity of 650 units and the by-products P-954-KP-10002 and P-955-KP003 have quantities of 1950 units and 1300 units, respectively.

Manual By-Production on Production Orders

By-production lines can be added to production orders manually using the “Add By-Product” function, accessed from the Lines FastTab by clicking Line > Add By-Product.

: Selection of by-product to be manually added to production order line

In the Item List page, select the by-product to add to the production order and then click OK.

The quantity is not automatically calculated on the by-product line, so the user must manually enter the necessary quantity in the Quantity field.

Note

Manually added by-production lines are deleted when production orders are recalculated.
