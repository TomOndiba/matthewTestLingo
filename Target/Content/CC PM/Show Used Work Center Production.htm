Show Used Work Center Production Rates

Work center production rates used to calculate the setup and run times of a production order routing line can be shown from the Prod. Order Routing page by clicking Navigate > Show used Work Center Production Rate.

: Accessing the Show Used Work Center Production Rate function

The Work Center Prod. Rate Card page opens for the selected line.

: Work Center Prod. Rate Card page for selected routing line

If this function is run on a line without a Work Center Production Rate, the following message appears.

: Message received for routing line without a Work Center Production Rate

Note

The used Work Center Production Rates can also be shown from the Production Order line.

For sequence planned production order lines, the used Work Center Production Rate can also be shown from the Sequence Planning page as well as from the Firm Planned Production Order and Released Production Order pages (Lines FastTab > Line > Show used Work Center Production Rates).
