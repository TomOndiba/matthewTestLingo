Tools in the Work Center Prod. Rate Card

The values of a tool are copied to the Work Center Prod. Rate Card after their creation and can be maintained separately for every machine (work center).

: Tool/Insert FastTab on the Work Center Prod. Rate Card for Item P-10400, Work Center M003

For information about the fields on the Tool/Insert FastTab, refer to topic “Tool Management” on page 1.

Tool Time Calculations

All times from the tool are calculated to the setup times and production rate values in the Work Center Prod. Rate Card using the following formulas:

  • Setup time = Ramp-up Time + Tool Setup Time
  • Production rate time = (Cavity * (Cycle Quantity / Cycle Time))

Different time units will be converted.
