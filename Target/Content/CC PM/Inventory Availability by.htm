Inventory Availability by QC Status

Inventory availability can also be controlled using the QC Status functionality.

: Available inventory control on QC Status

Only inventory with a QC Status that has the Inventory Available check box selected will be considered available, and all others will appear in the Inventory blocked by QC field on the Item Card.

 

: Total inventory for item P-10802 is 600, and 100 units are currently blocked

The value in the Inventory blocked by QC field is considered in the calculation of the planning worksheet. With all further functions that use item inventory, the value of this field is not considered. This function does not affect the Inventory field on the Item Card or the inventory valuation.
