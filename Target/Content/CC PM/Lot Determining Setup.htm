Lot Determining Setup

Lot Determining is specified by selecting the Lot Determining check box on the item, production structure lines and the production order components. In each instance, the check box can be cleared as needed.

On the Item Card, the Lot Determining check box is located on the Item Tracking FastTab.

: Lot Determining specified on an Item Card

If this check box is selected on the Item Card, it is automatically transferred to the Production Structure.

: Lot Determining transferred from an item to the production structure

In a recipe or production structure, like in the components, only one raw material in a BOM can be set up as Lot Determining.

: Lot Determining transferred from the production structure to the production order components

After consumption of the lot determining component and the completion of the production order, the Item Tracing page displays the results, confirming that the lot number has been carried from purchasing to the finished product.

: Lot number carried from purchase to output
