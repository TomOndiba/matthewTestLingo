Navigate Tab

Click the Navigate tab to access actions relating to the entire production structure.

Comments

This action displays comments for the selected production structure. Comments are for internal use only; they are not used on any documents.

Work Center Production Rates

This action displays the work center production rates for the work centers on the production structure. Changes made in this page are transferred back to the work center production rates on the item.

Incorporation Items

This action allows users to determine the incorporation items for this production structure (refer to topic “Incorporation” on page 1).

By-Production

This action opens the By-Production List page, which is used to create and maintain general data for a by-product (refer to topic “By-Production” on page 1).

Detail Calc. Standard Cost

This action opens the Detail Calculation page, which displays an overview of the standard cost calculation for the production structure (refer to topic “Detail Calculations” on page 1).

Calculation

This action opens the Calculation List page, displaying all calculations for the production structure. From this page, an existing Calculation Card can be opened or a new calculation can be created (refer to topic “cc|calculation” on page 1).

Sample Shipment

This action opens the Sample Shipment page, if the Sample Shipment check box is selected on the General FastTab (refer to topic “Sample Shipment” on page 1).

Note

Sample shipments are not shipments for which an inventory change is posted. They are neither opened from nor connected to standard Microsoft Dynamics NAV shipment functions.

Ingredients

This action opens the Prod. Structure Ingredients page, which displays the sum of ingredients that:

  • Should be included in the components
  • Are included in the entered components

Refer to topic “Production Structures ” on page 1 for more information.

Verification Entries

This action opens the Verification Entries page, which displays verified production structure status changes (refer to topic “cc|regulatory affairs” on page 1).
