Create Production Order from Maintenance Order

Production Orders can be created directly from the maintenance order by clicking Create Prod. Order on the ribbon. Before running this function, the Status on the maintenance order must be set to Released. The following dialog appears.

: Create production order dialog

Set any other filter as needed and then click OK to create the production order for the maintenance order. After the production order is created, a message appears that states how many production orders were created and production order information is transferred to the Manufacturing FastTab on the maintenance order.

: Production order information on the Manufacturing FastTab

The maintenance order and production order are now linked and can be compared with each other.
