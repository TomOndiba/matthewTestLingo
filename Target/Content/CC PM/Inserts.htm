Inserts

Inserts are also Work Centers and are used in additional to the tools, meaning that one or more inserts that are running with a tool on a machine can be defined. Inserts are also accessed from the Capacities menu.

: Insert Work Center Card for insert E005

Because an insert is a work center, the standard capacities can be used, which provides insight into bottlenecks within the tools.

In the production structures, the inserts are entered below the tools and appear indented.

: Insert E006 indented below the W004 tool

When a new insert is created, the system verifies whether a tool already exists. If more than one insert is created, only one is designated as the main insert (same function as in the tool management). The main insert has more features in the sequence planning.

When a production order is created, the inserts are scheduled at the same time as the tool and the machine.

: Machine, tool and insert lines planned simultaneously

During the machine and tool runs, the capacities of the inserts are blocked as well.
