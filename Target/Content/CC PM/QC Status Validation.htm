QC Status Validation

A QC Status for inventory with lot number information and QC Test can only be set if the QC Test is validated.

: Validation control on QC Status

For each QC status, a QC test validation of All Right, Restriction, or Blocked can be specified. These statuses are based on the answers defined in the QC Answer Card page.

: QC Answer Card for a positive answer

A positive answer can be defined for each answer of a QC Test. The display colors for positive or negative answers are managed in the QC Setup page (refer to topic “QC Setup” on page 1).

: QC Test highlighting QC validation fields

Several QC Status can be locked when some of the answers of the attached QC Test are not positive or without answer. If a QC Test is not validated or unclear, a comment can be entered in the QC Status Comment field on the QC Information FastTab on the QC Test page.

Note

The Question Status and Positive Answer fields are also used in the Lot No. Information Card Adv. page.
