Utilization Work Center Group

If Work Center Groups have been defined, the Utilization Work Center Group page displays a global matrix view of the Work Center capacity. Utilization Work Center Groups summarize Work Centers with a Work Center Group based on the following options (as defined in the View field on the Options FastTab):

Planned

Available %

Planned %

Capacity

Available

 

: Utilization Work Center Group showing planned percentages by month

Also on the Options FastTab, define the following for the displayed values:

  • Rounding factors in the Rounding Factor field.
  • Time interval in the View by field.
  • Type of amounts (Net Change or Balance at Date) in the View as field.

The column names can also be shown by clicking the Show Column Name check box.

On the Utilization Work Center Group Matrix FastTab, the Work Center Groups are listed on the left side of the page. Work Center Groups are set up in the Work Center Groups page and are assigned to work centers on the Work Center Card page.

: Work Center Groups and Work Center Card highlighting the Work Center Group Code field

Utilization Work Center Group Matrix FastTab, Work Center Group Menu

On the Utilization Work Center Group Matrix FastTab, click Work Center Group to access menu items relating to the selected Work Center Group. Each menu item is described in the following table.

Menu Item

Description

Calendar

Opens the Work Ctr. Group Calendar page where matrix filters are set. Click Show Matrix to open the Work Ctr. Group Calendar Matrix page, which provides a list of all work center groups and the capacity (calendar entries) of each work center group in a time period.

Load

Opens the Work Center Group Load page, which provides an overview of availability at the selected work center group. Refer to online help for more information about this page.

Utilization Work Center

Opens the Utilization Work Center page, filtered by the selected work center group (refer to topic “Utilization Work Center” on page 1 for more information).
