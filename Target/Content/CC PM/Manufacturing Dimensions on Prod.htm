Manufacturing Dimensions on Production Order Lines

When production order lines are created, the default manufacturing dimensions from the item are transferred to the production order line. If the production order line contains an item variant code and the manufacturing dimension setup for the item variant exists, the default manufacturing dimensions of the item variant are copied to the production order line instead of the default manufacturing dimension of the item.

If the production order lines are sequence planned, the default manufacturing dimensions from any existing sequence work center, tool, or insert are also transferred to the production order line.

If the production order line is related to a sales order line, the customer manufacturing dimension value of the sales order is also transferred to the production order line. If multiple production order lines are combined that are connected to different sales orders, and customer manufacturing dimensions exist with different priorities, then only the dimension with the highest priority is transferred to the production order line.

The Priority field in the Manufacturing Dimensions Setup page defines the order of the different default manufacturing dimensions to transfer.

For example, assume that the item and the work center in a production order line have the same default manufacturing dimension. Only the default manufacturing dimension value of the type with the highest priority in the manufacturing dimension setup is transferred to the production order line.
