Trading Units and Trading Unit Numbers – Warehouse Entry

When adding an item using the trading unit number administration, users can assign different trading unit numbers if he or she has selected the item tracking for that item.

The Item Journal is used to explain this process.

: Trading units in an item journal

In Fig. 1, the first line was entered with five trading units containing 10 liters each. Open the Item Tracking Lines page to enter single trading unit numbers either manually or automatically.

: Item tracking lines for first line in Item Journal

To automatically assign trading unit numbers, click Actions > Assign Lot/Trading Unit No.

: Batch job set up to automatically assign number of trading units and starting number

The number of lots and the lot number can be assigned manually or automatically depending on the number of lots that need to be created.

For manual assignment:

  • If only one lot needs to be created, specify a value in the Lot No.  field for the lot number. The Lots field is set to 1.
  • If more than one lot needs to be created, then enter the number in the Lots field and leave the Lot No. field empty (the number series will be used).  

For automatic assignment, the Use Lot No. Series check box can be used with the Lots field to auto-populate the Lot No. field with a value from the Lot No. series assigned to the item.

  • If selected, the Lots field is automatically set to 1 and the Lot No. field displays the next value from the number series.
  • If selected and a value greater than 1 is entered in the Lots field, a message is displayed stating that the lot number will be deleted.

Accepting the message clears the Lot No. field and the lot numbers from the number series (e.g. Lot1, Lot2, etc.) are automatically assigned upon closing the Assign Lot/Trading Unit page.

The number of trading units per lot and the starting number may be assigned manually. Alternatively, a default Start Trading Unit No. can be defined in Inventory Setup (refer to topic “Trading Unit – Item Setup” on page 1).
