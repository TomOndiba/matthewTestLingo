Data Control Colors

To color the position lines for the evaluation, users must first setup the colors. Several colors can be defined for the analysis point and symbols can also be used.

Note

Symbols should not exceed 15 x 15 pixels.

The data control colors appear in the DC field to indicate the status of the production order, based on the selected condition.

: Data Control Colors page

Follow these steps to set up data control colors:

  1. On the Manufacturing menu, click Administration > Data Control Colors.
  2. On the ribbon, click New.
  3. In the Code field, type the color.
  4. In the Description field, type a description for this color.
  5. On the ribbon, click Picture.
  6. Right-click the Picture box and then click Select Picture.
  7. Locate and select the picture file to import and then click Open.
  8. Click OK to close the Data Control Color Picture page.
  9. Repeat steps 2–8 to add more colors and pictures.
