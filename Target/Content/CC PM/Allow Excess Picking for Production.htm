Allow Excess Picking for Production Order Consumption

Pick quantities can exceed the needed quantities for production order consumption in order to put bulk items in manufacturing locations. This excess quantity is only used for the warehouse movement, not for consumption (can pick more than needed but cannot use more than needed).

To use this functionality, the Excess Pick for Consumption check box must be selected on the General FastTab in the Warehouse Setup page.

: Warehouse Setup page

Once this check box is selected, a higher quantity value can be entered in the Qty. to Handle field in a pick worksheet than specified by the quantity of the respective Sales or Production Order. Thus, it is possible to take a higher quantity from the inventory than necessary for the order.

However, the withdrawn excess quantity is not allocated to the specific order, but remains in the specified picking location as available inventory. Therefore, the original order is not changed.

Example

Component item 30300 has a quantity of 344 on a production order, which is transferred to the Warehouse Pick document line.

: Warehouse Pick line with actual component quantities

In this example, 350 KG need to be picked and moved to the manufacturing location.

: Warehouse Pick line quantity increased

The component quantity remains at 344 before and after the Warehouse Pick is registered.

: Registered Pick and related Prod. Order Component line

In the Production Journal, the user is prevented from increasing the Consumption Quantity to 350, even though the Pick quantity was increased to that amount.
