Identify Operation for Sequence Planning

In Production Structures, more than one sequence work center can be entered in the lines. It is also possible to select which sequence work center in the Production Structure lines should be used for sequence planning if more than one sequence work center is added.

: Sequence Planning check box indicating that Work Center M021 is the sequence work center for item P-10500

Only one sequence work center can be selected in the Production Structure. If the Sequence Planning check box is selected for another sequence work center, the new work center will be selected and the Sequence Planning check box will be cleared on the previous work center.

: Sequence Planning check box changed to make Work Center M016 the sequence work center for item P-10500

Additionally, the Sequence Planning check box can only be selected for work centers that are set up for sequence planning.

: Error generated by selecting a non-sequence planning work center as the sequence work center for item P-10500

When a production order is created for this item, the sequence planning is done only on the sequence work center selected in the Production Structure, in this case M021.

: Sequence Planning for new production order on Work Center M021 for item P-10500

The Sequence Planning indicator is transferred to the Prod. Order Routing. It is not possible to change the Sequence Planning check box in the Prod. Order Routing.

: Sequence Planning set on Work Center M021 for item P-10500
