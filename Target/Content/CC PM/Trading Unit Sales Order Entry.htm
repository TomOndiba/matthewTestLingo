Trading Unit – Sales Order Entry

During sales order entry, the configuration of the different trading units is possible using the units of measure:

  • Variants – the variant code on the line is updated automatically to match the selection in the Unit of Measure Code field. For example, if a user enters item P-10503 on the sales line and changes the Unit of Measure Code from the base of KG to 10 L, when he or she leaves the field, the variant code is updated to 10 L.
  • Items – the base item can be entered on the line and then adjusted automatically by changing the unit of measure. For example, if a user enters P-10502 on the sales line and changes the Unit of Measure Code from the base of KG to 4 L, when he or she leaves the field, the item will change to P-10502-4L.
