Exercise – Create and Verify a Production Structure for a New Item

Set up a new item with the following information:

  • Number = P-10002, Name = Panel
  • Base Unit of Measure = PCS

Then create a new Production Structure for this item using the following line information:

  • Item = P-30008, Quantity per = 10 kg
  • Item = P-30009, Quantity per = 15 kg
  • Work Center = M002, Setup Time = 120 minutes, Run Time = 50 minutes

Certify the Production Structure and then manually create a firm planned production order for item P-10002. Confirm that the components and routings are correct.

Create a New Item

Follow these steps to create a new item:

  1. In the navigation pane, click Departments > Manufacturing > Product Design.
  2. On the Product Design menu, click Items.
  3. On the ribbon, click New.
  4. Click Cancel to close the template selection page.
  5. In the No. field, type P-10002.
  6. In the Description field, type Panel.
  7. In the Base Unit of Measure field, enter PCS.
  8. Expand the Price & Posting FastTab.
  9. In the Gen. Prod. Posting Group field, enter RMS-PMW.
  10. In the Inventory Posting Group field, enter RAW MAT.
  11. Do not close the Item Card.

Create the Production Structure

Follow these steps to create the Production Structure:

  1. On the ribbon of the Item Card, click Production Structures.
  2. On the ribbon, click New.
  3. Press ENTER to insert a new production structure number.
  4. On the Lines FastTab, in the Type field, enter Item.
  5. In the No. field, enter P-30008.
  6. In the Quantity per field, type 10.
  7. On the next line, in the Type field, enter Item.
  8. In the No. field, enter P-30009.
  9. In the Quantity per field, type 15.
  10. On the next line, in the Type field, enter Work Center.
  11. In the No. field, enter M002.
  12. In the Setup Time field, type 120.
  13. In the Run Time field, type 50.
  14. On the General FastTab, in the Status field, enter Certified.
  15. Click OK to close the Production Structure page and then close the Production Structure List page.
  16. Expand the Replenishment FastTab.

: Item Card, Replenishment FastTab updated from Production Structure certification

When the Production Structure is certified, the Replenishment System field is set to Prod. Order and the corresponding Production Structure No., Routing No., Production BOM Type and Production BOM No. fields are updated on the Item Card.

  1. Click OK to close the Item Card.

Create a Production Order and Verify Setup

Follow these steps to verify the Production Structure setup:

  1. In the navigation pane, under Manufacturing, click Execution > Firm Planned Prod. Orders.
  2. On the ribbon, click New.
  3. Press ENTER to insert a new production order number.
  4. In the Source No. field, enter P-10002.
  5. In the Quantity field, type 1.
  6. Expand the Posting FastTab and enter MAINWH in the Location Code field.
  7. On the ribbon, click Refresh Production Order.
  8. In the Refresh Production Order batch job, click OK.
  9. Ensure that the first line is selected on the Lines FastTab, and then click Line > Routing.

: Production Order routing information based on production structure lines

  1. Verify that the work center information entered on the Production Structure appears and then close the Prod. Order Routing page.
  2. On the Lines FastTab, click Line > Components.

: Production Order component information based on production structure lines

  1. Verify that the item information entered on the Production Structure appears and then close the Prod. Order Components page.
  2. Close the Firm Planned Prod. Order page.
