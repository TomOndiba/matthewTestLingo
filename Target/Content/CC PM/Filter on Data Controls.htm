Filter on Data Controls

After a data control has been defined, it can be used to filter the lines in the Planning Pool or Sequence Planning pages.

Follow these steps to set a data control filter in Sequence Planning:

  1. Open the Sequence Planning page.
  2. In the General section, select a work center.
  3. In the Data Control section, enter the condition to filter the lines in the Condition field.
  4. Press ENTER or TAB.

The DC field now contains the color picture assigned in the Data Control Color Picture page.
