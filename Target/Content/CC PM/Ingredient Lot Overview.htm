Ingredient Lot Overview

The Ingredient Lot Overview page provides two functions:

  • Displays the current ingredient structure of a production order in a single view
  • Assignment of lots for a production order component based on the Lot No. Ingredients

This page can be opened from the production order header or lines (Ingredient Lot Assignment) or directly from the Ingredients Management menu (Ingredient Lot Overview).

A range of filter criteria is available for restricting the display to certain production orders. Once filter criteria is entered or changed, click Get Records on the ribbon to populate the lines. Expand and collapse all lines to increase or reduce detail; individual lines can be collapsed or expanded by clicking the arrows in the Entry Type column.

: Ingredient Lot Overview accessed from production order line

The benefit of accessing the page from the production order header or line is that all necessary filters are preset and line detail information is populated. The difference between opening the page from header and lines is all production order lines are included in the view or only the selected line is included.

When accessing the page from the Ingredients Management menu, populate the Prod. Order Status Filter, Prod. Order Filter and, if necessary, the Prod. Order Line Filter fields and then click Get Records on the ribbon.

: Ingredient Lot Overview accessed from Ingredients Management menu with filters manually set

The fields in the General section are described in the following table.

Field

Description

Prod. Order Status Filter

Indicates the status of the production order to be included in the view.

Prod. Order Filter

Indicates the production order number(s) to be included in the view.

Standard Microsoft Dynamics NAV filters can be used, such as a filter string like PO1..PO2 where PO1 is the number of the first production order and PO2 is the number of the second production order created from Sales Order lines.

Prod. Order Line Filter

Indicates the production order line number(s) to be included in the view.

Standard Microsoft Dynamics NAV filters can be used.

Due Date Filter / Item Filter / Variant Filter / Location Filter

Additional filters to be included in the overview.

Show Component Ingredients

If selected, ingredients assigned to the components are displayed in the lines.

Note

The “Get Records” function must be run to update the lines.

Include Item Ledger Entries

If selected, posted item ledger entry information is displayed in the lines.

Note

The “Get Records” function must be run to update the lines.

The Ingredient Lot Overview page shows several elements of a Production Order qualified by their Entry Type. The valid Entry Types are described in the following table.

Entry Type

Source Information

Production Order

Output

  • Prod. Order Line Ingredient (related to Prod. Order Line)
  • Prod. Order Component Ingredient (related to Prod. Order Component)
