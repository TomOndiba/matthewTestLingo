Allow Picking for Prod. Order Lines Reversed in Sequence Planning

The No Pick for Reversed Prod. Order Lines field in Planning Extension Setup determines if a warehouse pick is created for production order lines reversed in Sequence Planning (sequence position 999,999,999,999).

: No Pick for Reversed Prod. Order Lines field disabled in Planning Extension Setup

Example – Picking for Production Order Lines Reversed in Sequence Planning

In this example, a sales order has been created for 10 units of item P-10001 to the YELLOW location that requires picking. A production order was created but not scheduled for planning.

: Unscheduled production order ready for picking

The No Pick for Reversed Prod. Order Lines field is disabled in Planning Extension Setup to allow picking for the unscheduled released production order.

: Pick successfully created for the unscheduled production order

The created pick is deleted and then the No Pick for Reversed Prod. Order Lines field is enabled in Planning Extension Setup to prevent picking for the unscheduled released production order.

: No Pick for Reversed Prod. Order Lines field enabled in Planning Extension Setup

When the “Create Whse. Pick” function is run for the production order, the pick will not be created.

: Pick not created for unplanned production order

Now, the production order is planned in Sequence Planning.

: Scheduled production order ready for picking

With the No Pick for Reversed Prod. Order Lines field still enabled, the “Create Whse. Pick” function is run again for the production order and the pick is created.

: Pick successfully created for the scheduled production order
