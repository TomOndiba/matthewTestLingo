Actions Tab

The actions in the Functions and Report groups on the Actions tab relate to functionality that is relevant for the selected sequence work center. Each action is explained in the following table.

Action

Description

Schedule

Schedules the work centers as of a certain point of time.

The fields are described as follows:

  • Start Date / Start Time – automatically filled with the current time and date. If the production is planned to a certain date or, for example, to the beginning of a shift, these fields must be modified.
  • To Due Date – Schedules the sequence production order lines to a specified due date.
  • Schedule Deferred Prod. Order Lines – Select to indicate that the priority planning should schedule deferred production orders in the sequence planning of the work center.

Maintenance-/Setup Matrix

For the automatic calculation of setup times, the setup matrix can be used. The following functions are available in sequence planning:

  • Maintenance-/Setup Matrix (creation of a maintenance-/setup matrix)
  • Planning Maintenance-/Setup Matrix
  • Delete Maintenance-/Setup Times

For more information, refer to topic “Maintenance Order & Setup Order” on page 1.

Optimize Planning

To optimize the Sequence Planning, users can set up Planning Dimensions. Refer to topic “Optimize Planning Functions in Sequence Planning” on page 1 for more information.

Planning Pool

Opens the Planning Pool page, which is a list of all production orders, sorted by assignment to the sequence planning work center.

Show Capacity Warning

Opens the Capacity Conflict Log page, which shows all capacity warnings for all production orders attached to the current work center (refer to topic “Visualize Capacity Currently Utilized on Work Centers/Tools” on page 1).

Print

On the ribbon, click Print to generate the Sequence Production Plan report.
