Order Network

The Order Network provides an overview of the current process. The page can be opened from the Customer Card by clicking Order Network on the ribbon.

: Opening the Order Network for customer 10000

The Order Network shows information trails, for example, from a sales order to a production order to consumption/output posting to shipment and invoicing. Expand and collapse lines to increase or reduce detail. Click Show on the ribbon to open the page for the selected line.
