Automatic Lot Proposal at Warehouse Picking

When a Warehouse Pick document is created, it is possible to have the system propose a lot-, trading unit- or serial number automatically. This is only necessary if no item tracking has been assigned to the picked item in the original document (sales- or production order); if item tracking was already assigned, then it will be transferred to the picking document.

Note

This function does not split lots; therefore, the Lot No. field will not be populated on the pick line if a single lot does not have enough quantity on hand to meet the demand.

For the automatic assignment, the Suggest Lot No./Serial No. check box must be selected in the Whse. Shipment - Create Pick request page on the Warehouse Shipment or the Whse. Source - Create Pick request page from the Production Order during the pick creation process.
