Lines FastTab

The Lines FastTab contains two menus: Functions and Line. The menu items for each are explained in the following sections.

Functions Menu

The menu items on the Functions menu relate to the selected line. Each function is described in the following sections.

Move

This function moves the selected production orders before or behind the current position. After the move, all production orders are rescheduled that are currently in planning in the work center.

To mark production orders to move, select the orders by clicking the line while pressing the CTRL key. The last line selected is where the first marked orders are moved.

Reverse

This function reverses planned production orders. The production order is moved to the end of the sequence in the work center and then all other planned production orders are rescheduled.

: Reversed production order lines

Note

During the “Reverse” process, planned maintenance-/setup jobs are deleted.

Change Work Center

This function changes the work center on a sequence order. Initially, a list of all sequence work centers is displayed in the Work Center List page.

: Work Center List

In this page, select the work center to where the current production order in the sequence planning is moved. After clicking OK, the following message appears.

: Message to change the Work Center on the production order line

After accepting the message, the production order line is moved to the new work center. The production orders of the current work center will be re-scheduled.

: Production order line moved from work center M002 to M001

During the work center change, all current production rates are considered. If the new item/work center combination has different production rates, they are used and stored in the routing sheet.

The work center is also changed in the routing sheet.

: Production order routing with updated work center information

In addition to the production rates, the work center costs are updated.

Change Tool

Generally, the change of tools is like the change of a work center. A list of available tools that can be changed is displayed.

During the change process, the tool is changed in the routing sheet and new costs and production rates are recalculated.

Change Insert

Generally, the change of inserts is like the change of a work center. A list of available inserts that can be changed is displayed.

During the change process, the insert is changed in the routing sheet and new costs and production rates are recalculated.

Split Prod. Order Line

This function splits production order lines into multiple production orders and approaches. For more information, refer to topic “Split Production Orders” on page 1.

Combine Prod. Order Line

This function combines several production orders into one production order. For more information, refer to topic “Combine Production Orders” on page 1.

Change Status

This standard manufacturing function in Microsoft Dynamics NAV is used in the Sequence Planning to change the status of one or more production orders. To change several production orders, each must have the same status. Multiple orders can be selected by clicking the line while pressing the CTRL key.

: Change status dialog

Select the relevant status (to which this production order is to be assigned) and the appropriate Posting Date.

To update the unit cost of the item that is produced, select the Update Unit Cost check box. The unit cost of the item will also be updated in any production orders or sales orders for which this item has been reserved.

Planning Maintenance-/Setup Job

During the planning of a maintenance and setup job, a list of all maintenance and setup job types is displayed. From this list, predefined maintenance and setup jobs may be selected.

: List of Maintenance / Setup Job Types

This function moves the new maintenance-/setup job before or after the current production order. After the move, a new maintenance-/setup job is created for the current work center. For more information, refer to topic “Maintenance-/Setup Job Types” on page 1.

Set Start Order

If no production order has been planned for a work center in the sequence planning (all sequence positions are 999,999,999,999), one order must be set as the start order.

When the “Set Start Order” function is run, the Sequence Pl. Start dialog box appears. In the Start Date and Start Time fields, enter the point of time on which the current production order should be scheduled to.

These fields are automatically filled in with the current time and date. If the production is planned to a certain date or, for example to the beginning of a shift, these fields must be modified. By clicking OK, the selected production order only is rescheduled.

Line Menu

The menu items on the Line menu relate to the selected line. Each function is described in the following table.

Menu Item

Description

Prod. Order

Opens the Production Order of the selected production order line.

Item Availability by

Contains a drop-down list with the following menu items:

Event

Location

Period

Lot

Variant

BOM Level

These options check the availability of the item that is currently produced.

Reservation Entries

Opens the Reservation Entries page, which contains one entry for each manually or automatically created reservation of the current production order line.

Dimensions

When a production order, a production order line, or a production order component is created, document dimension values can be added that can be used later in the analysis. For two shortcut dimensions, the dimension values may be entered directly in a production order, a production order line, or a production order component.

Values for dimensions that are not available in a production order, a production order line, or a production order component, can be entered in this page.

Manufacturing Dimensions

On creation of production order lines, the default manufacturing dimensions from the item are transferred to the production order line. If the production order lines are sequence planned, the default manufacturing dimensions from the possibly existing sequence work center, tool or insert are transferred to the production order line as well.

The Priority field in the Manufacturing Dimension Setup page defines the order of the different default manufacturing dimensions to transfer (refer to topic “Manufacturing Dimension Priority” on page 1).

Example

The item and the work center in a production order line have the same default manufacturing dimension. Only the default manufacturing dimension value of the type with the highest priority in the manufacturing dimension setup is transferred to the production order line.

Routing

Opens the Prod. Order Routing page for the selected production order line, where users can enter routing sheets before or after a resource sequence.

Components

Opens the Prod. Order Components page, where substitution items can be checked and replaced, and new components for the production order can be entered manually.

Components Planning Pool

Opens the Components Planning Pool page, where the availability of components can also be checked in the planning pool (refer to topic “Components Planning Pool” on page 1).

Item Tracking Lines

Opens the Item Tracking Lines page, used to track planned items that are Lot No. / Serial No. tracked.

Show used Work Center Production Rate

Opens the Work Center Prod. Rate Card page for sequence planned production order lines to show the Work Center Production Rates used to calculate the setup and run times of a production order routing line.

If this function is run on a sequence planning line without a Work Center Production Rate, a message will appear.

Show Capacity Warning

Opens the Capacity Conflict Log page, which shows all capacity warnings for the current production order (refer to topic “Visualize Capacity Currently Utilized on Work Centers/Tools” on page 1).

Calculation

Contains the Calculation (Expected) and Calculation (Actual) options. Refer to topic “cc|calculation” on page 1 for more information.

Production Journal

The production journal is standard manufacturing functionality in Microsoft Dynamics NAV but has been enhanced with some cc|process manufacturing features. For more information about the cc|process manufacturing-related functionality, refer to topic “Production Journal” on page 1.

For more information about the standard functionality, refer to the Production Order Processing content in the current Microsoft Dynamics NAV Manufacturing training material.
