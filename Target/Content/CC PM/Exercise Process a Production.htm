Exercise – Process a Production Order

Create a Production Order from the Production Structure for item I10, Product. Confirm that the ingredients from the recipe were transferred correctly. Add lot numbers and confirm captured ingredient quantities. Post consumption and output and confirm posted ingredient quantities.

Create the Production Order

Follow these steps to create the production order from the production structure:

  1. On the Manufacturing menu, click Product Design > Production Structures.
  2. Open the Production Structure created in the “Create and Certify Production Structure” exercise for item P-I10.
  3. On the ribbon, click Navigate > Ingredients.
  4. Note the calculated quantities of 1 and 6, respectively.

: Prod. Structure Ingredients

  1. Click OK to close the Prod. Structure Ingredients page.
  2. On the ribbon, click Actions > Create Prod. Order.
  3. Select the Released option and then click OK.
  4. In the Released Production Order page, click Line > Ingredients on the Lines FastTab.

: Prod. Order Line Ingredients

  1. Note that the Expected Quantity (Ratio) and Expected Comp. Qty. (Ratio) fields match the production structure ingredients, and the Expected Quantity (Total) and Expected Comp. Qty. (Total) fields are the ratio multiplied by the produced quantity (100):

1 MG per KG (Ratio) or 100 MG of Production Order line (Total 100 KG is ordered)

  1. Click OK to close the Prod. Order Line Ingredients page.

Add Lot Tracking to Production Order Line

Follow these steps to add lot tracking to the production order line:

  1. On the Lines FastTab, click Line > Item Tracking Lines.
  2. In the Lot No. field, type LOT200.
  3. In the Quantity (Base) field, type 100.
  4. Click OK to close the Item Tracking Lines page.

Review Components and Add Lot Tracking

Follow these steps to review the components and add lot tracking:

  1. On the Lines FastTab, click Line > Components.
  2. On the first line, for Item A, click Navigate > Ingredients.

: Prod. Order Component Ingredients with no captured values

  1. Note that the “Captured…” field values are set to 0 because no lots have been assigned.
  2. Click OK to close the Prod. Order Component Ingredients page.
  3. On the Navigate tab, click Item Tracking Lines.
  4. On the ribbon, click Select Entries.
  5. With the line for lot LOT001 selected, click OK.
  6. Click OK to close the Item Tracking Lines page.
  7. On the Navigate tab, click Ingredients.

: Prod. Order Component Ingredients with captured values

  1. Note that the Contains Expected Values check box is selected because no measurements exist for the assigned lots and the Captured Quantity (Ratio) and Captured Quantity (Total) fields are populated with the expected quantity values.
  2. Click OK to close the Prod. Order Component Ingredients page and then close the Prod. Order Components page.
  3. On the Lines FastTab, click Line > Ingredients.

: Prod. Order Line Ingredients with captured values

  1. On the first line, note that the Contains Expected Values check box is selected and the Captured Quantity (Ratio) and Captured Quantity (Total) fields are populated with the expected quantity values.
  2. Click OK to close the Prod. Order Line Ingredients page.
  3. On the Lines FastTab, click Line > Components.
  4. Select the second line for Item B and then click Navigate > Item Tracking Lines.
  5. On the ribbon, click Select Entries.
  6. With the line for lot LOT002 selected, click OK.
  7. Click OK to close the Item Tracking Lines page.
  8. On the Navigate tab, click Ingredients.

: Prod. Order Component Ingredients with captured values

  1. Click OK to close the Prod. Order Component Ingredients page and then close the Prod. Order Components page.
  2. On the Lines FastTab, click Line > Ingredients.

: Prod. Order Line Ingredients with captured values

  1. Note that both lines have captured quantities now.
  2. Click OK to close the Prod. Order Line Ingredients page.
  3. On the Lines FastTab, click Line > Components.
  4. Select the third line for the Filler item and then click Navigate > Item Tracking Lines.
  5. On the ribbon, click Select Entries.
  6. With the line for lot LOT003 selected, click OK.
  7. Click OK to close the Item Tracking Lines page.
  8. Click OK to close the Prod. Order Components page.

Note

Item Tracking for components with ingredients can also be assigned using the “Ingredient Lot Selection” function (refer to topic “Ingredient Lot Selection” on page 1).

Post Consumption and Output

Follow these steps to post consumption and output through the Production Journal:

  1. On the Lines FastTab, click Line > Production Journal.
  2. On the output line, select the Finished check box.
  3. On the ribbon, click Post.
  4. Click Yes to post the journal lines.

Review Posted Ingredient Quantities

Follow these steps to review the posted ingredient quantities:

  1. On the Lines FastTab, click Line > Ingredients.

: Prod. Order Line Ingredients with posted values

  1. Note that the Posted Quantity (Ratio) and Posted Quantity (Total) fields are populated with the expected quantity values and the “Captured…” fields are reset to 0.
  2. Click OK to close the Prod. Order Line Ingredients page.
  3. On the Lines FastTab, click Line > Components.
  4. On the first line, for Item A, click Navigate > Ingredients.

: Prod. Order Component Ingredients with posted values, no captured values

  1. Note that the “Captured…” field values are set to 0 because the consumption/output was posted, as indicated in the “Posted…” fields.
  2. Click OK to close the Prod. Order Component Ingredients page.
  3. On the second line, for Item B, click Navigate > Ingredients.

: Prod. Order Component Ingredients with posted values, no captured values

  1. Click OK to close the Prod. Order Component Ingredients page and then close the Prod. Order Components page.

Review Lot Ingredients

Follow these steps to review the Lot No. Ingredients:

  1. On the Lines FastTab, click Line > Item Tracking Lines.
  2. On the ribbon, click Navigate > Lot No. Information Card and then click Edit on the ribbon.
  3. On the ribbon, click Navigate > Ingredients.

: Lot No. Ingredients created upon posting the Production Journal

  1. Note that the Quantity (Expected), Unit of Measure Code and Alloc. Unit of Measure Code fields match the values from the Prod. Order Line Ingredients.

The Quantity (Actual) and Actual Value fields are used with cc|quality management (refer to topic “Receipt Testing” on page 1) and direct inventory posting through the Item Journal (refer to topic “Item Delivery” on page 1).
