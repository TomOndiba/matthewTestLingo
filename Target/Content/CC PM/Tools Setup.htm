Tools Setup

The Tool Work Center Card is accessed from the Capacities menu by clicking Tools.

In addition to the standard fields on the Work Center Card, the Tool and Amortization FastTabs have been added to create the Tool Work Center Card. The fields on the Tool FastTab determine the usage times of a tool.

: Tool FastTab on the Tool Work Center Card for the W001 tool

The fields on this FastTab are described in the following table.

Field

Description

Cavity

Factor for the producible units.

Cycle Time

The time, depending on cycle unit of measure, that is needed to produce an item.

Cycle Time Unit of Measure

Cycle unit of measure that was defined in the Capacity Units of Measure page.

Cycle Quantity

Quantity, depending of cycle quantity unit of measure.

Cycle Quantity Unit of Measure

Cycle quantity unit of measure that was defined in the Units of Measure page.

Ramp-up Time

Needed ramp-up time.

Ramp-up Time Unit of Measure

Ramp-up time unit of measure that was defined in the Capacity Units of Measure page.

Setup Time

Needed setup time.

Setup Time Unit of Meas.

Setup time unit of measure that was defined in the Capacity Units of Measure page.

Sprue Bush Weight

Refer to topic “Sprue Bush” on page 1.
