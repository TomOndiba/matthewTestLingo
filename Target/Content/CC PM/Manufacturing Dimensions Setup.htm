Manufacturing Dimensions Setup

Manufacturing dimensions can be added to items, work centers, tools, inserts, production order lines and item variants. The global and shortcut dimensions for each type can be specified in the Manufacturing Dimensions Setup page.

: Manufacturing Dimensions Setup

The global manufacturing dimensions 1 and 2 can be set by using the “Change Global Mfg. Dimensions” function accessed by clicking Actions > Change Global Mfg. Dimensions on the ribbon.

: Changing global manufacturing dimensions for the Item table

This report can be used to change the global manufacturing dimensions 1 and 2 for items, work centers, tools, inserts, production order lines and item variants.
