Utilization Work Center

The Utilization Work Center page displays a list of work centers with free capacity based on the following options (as defined in the View field on the Options FastTab):

Planned

Available %

Planned %

Capacity

Available

 

For the calculation of the values, only needed capacities of un-booked production orders are used.

: Utilization Work Center showing planned percentages by month for the PROD work center group

On the Options FastTab, set filters to restrict the calculation of the capacities on different work center groups or shop calendars, set rounding factors, and define the time interval and type of amount in the View by and View as fields, respectively.

Utilization Work Center Matrix FastTab, Work Ctr. Menu

On the Utilization Work Center Matrix FastTab, click Work Ctr. to access menu items relating to the work center selected in the lines. Each menu item is described in the following table.

Menu Item

Description

Capacity Ledger Entries

Displays all capacity ledger entries of the current work center. Refer to online help for more information about the Capacity Ledger Entries page.

Comment

Allows users to view, add, or modify comments for the selected work center.

Load

Opens the Work Center Load page, which provides an overview of availability at the selected work center. Refer to online help for more information about this page.

Statistics

Displays the statistics for the selected work center. Refer to online help for more information about the Work Center Statistics page.
