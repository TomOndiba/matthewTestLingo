Plan Table Work Center

The Plan Table Work Center page displays an up-to-date list of the queued orders for processing.

: Plan Table Work Center showing Firm Planned production orders by month

On the Options FastTab, set filters to view production orders by status, define the time interval in the View by field and how to display the column names.

Plan Table Work Center Matrix FastTab, Work Ctr. Menu

On the Plan Table Work Center Matrix FastTab, click Work Ctr. to access menu items relating to the work center selected in the matrix lines. Each menu item is described in the following table.

Menu Item

Description

Card

Displays the work center card for the selected work center.

Capacity Ledger Entries

Displays all capacity ledger entries of the current work center. Refer to online help for more information about the Capacity Ledger Entries page.

Comment

Allows users to view, add or modify comments for the selected work center.

Load

Opens the Work Center Load page, which provides an overview of availability at the selected work center. Refer to online help for more information about this page.

Statistics

Displays the statistics for the selected work center. Refer to online help for more information about the Work Center Statistics page.

 

Print Plan Table Work Center

On the ribbon, click Print to generate the Production Plan report, which contains the needed capacities of the work centers as of the Starting Date that was entered in the request page.
