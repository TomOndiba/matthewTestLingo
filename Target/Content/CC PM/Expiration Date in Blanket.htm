Expiration Date in Blanket Sales Order

The Blanket Order Expiration Date field was added to the Invoicing FastTab on the Blanket Sales Order. When the “Make Order” function is run, a warning will appear if the date of the Blanket Sales Order exceeds the work date.

: Expiration Date warning generated by running the Make Order function when Blanket Order Expiration Date of January 15, 2019 is prior to the work date of January 24, 2019 (as noted in the promoted Document Date on the General FastTab)

A warning will also appear when an expired Blanket Sales Order line is manually added to a Sales Order line.

: Expiration Date warning generated by attempting to assign expired Blanket Sales Order 1001 to a Sales Order
