Item Bottling from Sales Order Line

The Item Bottling page can be opened directly from a sales order line.

: Accessing the Item Bottling page from the Sales Order line

The filters in the item bottling are populated with the values from the sales order line. The Requested Quantity FastTab displays the requested, selected and undefined quantities.

: Item Bottling page accessed from the Sales Order line, showing the Requested Quantity FastTab

The item bottling also creates reservations to the sales order line if a production order is created.
