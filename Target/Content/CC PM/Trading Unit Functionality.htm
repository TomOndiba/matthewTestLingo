Trading Unit Functionality

As described in topic “Trading Unit Numbers” on page 1, the standard lot number in cc|process manufacturing is split into Lot No. Trading Unit and Trading Unit No. fields. If a user enters values in both fields, the values are concatenated and copied into the Lot No. field. These two fields are included in the following processes in cc|quality management:

Incoming Goods Inspection

Lot-/Serial No. Administration

Manufacturing Inspection

Inventory Overview

Shipment Inspection

Item Tracing

Stability Tests

8D Report

Test Incidents

Item Tracking

Retained Samples

 

The usage and tracking of these fields is generally the same as the Lot No. field described in the cc|quality management documentation. Refer to this documentation for further information.
