Delay List

The Delay List page displays all delayed production orders. This page is accessed from the navigation pane by clicking Execution > Prod. Order Delay List.

: Delay List page

If the Maint-/Setup Job in Delay List check box is selected in the Planning Extension Setup page, then the maintenance and setup jobs are listed in the delay list, in addition to the production orders.

In the Delay List page, the Delay field is calculated as follows: Delay = Due Date – Ending Date.

Home Tab

The Home tab contains the Prod. Order group. Each action is described in the following table.

Menu Item

Description

Card

Displays the production order card for the selected production order.

Comments

Allows users to view, add, or modify comments for the selected production order.

Statistics

Displays the statistics for the selected production order. Refer to online help for more information about the Production Order Statistics page.

 

Print Delay List

On the ribbon, click Print to generate a production order delay list.
