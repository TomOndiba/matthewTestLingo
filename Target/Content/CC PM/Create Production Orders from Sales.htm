Create Production Orders from Sales Orders

As seen in previous exercises, production orders can be created for selected sales lines. The existing Planning functionality has not changed. After selecting individual sales lines, production orders can be created from the Lines FastTab by clicking Order > Create Prod. Order.

: Preparing to create production orders for selected sales order lines

The Create Order From Sales Line dialog appears.

: Create Order From Sales Line dialog

After confirmation, the production orders for the selected sales lines are created.

: Created order confirmation message

The created production orders are shown if the Show Production Orders check box was selected on the Create Order From Sales Line dialog.

: Production orders created from the sales lines
