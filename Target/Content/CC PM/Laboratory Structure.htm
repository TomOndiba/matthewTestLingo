Laboratory Structure

Similar to the development structure, the laboratory structure does not require an item, just the raw material items. The laboratory structure makes it possible to run a first test production. 

Laboratory Structures created from the development structure may contain lines without item numbers. Laboratory structures cannot be certified without the No. field being populated on the lines.

When filling in the No. field on a line where it had been blank, the following message appears.

: Message generated when entering item numbers on text lines

By accepting this message, the text in the Description field and the recipe shares are maintained. Otherwise, the Description is overwritten with the standard item description and the share information is deleted.

: Message accepted on the third and fourth lines to maintain values and then added work center

If not created from a development structure, a laboratory structure can be created from the Research & Development menu by clicking Laboratory Structure and then clicking New on the ribbon.

Existing production structures can be copied into the Laboratory Structure by clicking Actions > Copy > Production Structure on the ribbon. Copied lines can be modified as needed.

When certifying the laboratory structure, the production BOM/recipe and routing are created – if the New Production BOM/New Routing check boxes were selected on the General FastTab.

R&D Item Number Setup

Prior to creating a production order from a Laboratory Structure, a dummy item must be set up and then assigned in the R&D Item No. field on the Process Manufacturing Setup page.

: Assigned R&D item number

Item numbers from this number series will be used on any production orders created from the laboratory structures.

Production Orders with Laboratory Structures

In the Laboratory Structure, users can create a production order that is considered in the planning and for the raw material planning.

During the creation of a production order, the value entered in the Lot Size field on the production structure header is copied as the manufacturing quantity.

: Lot size on the production structure header

To create the production order from the laboratory structure, click Actions > Create Prod. Order on the ribbon. A dialog appears, asking for the status of the production order.

: Production order status selection

The production order is created for the quantity entered in the Lot Size field on the laboratory structure.

: Production order created from laboratory structure

The production BOM/recipe and routing created when certifying the laboratory structure are transferred to the production order and calculated during the refresh process.

: Production Order Routing and Components from Laboratory Structure

If the sequence resources are used with the routing sheet, the laboratory production is available in the current planning for scheduling. In addition, the required components appear as demand in the requisition worksheet.

Change to Manufacturing Structure

Once the laboratory structure has been created and is ready for the next step in the R&D process, users can create a Manufacturing Structure from the ribbon of the Laboratory Structure page by clicking Create Manufacturing Structure.

The new Manufacturing Structure page appears and is set with a Status of Under Development.

: Manufacturing Structure created from the Laboratory Structure

The R&D FastTab displays the number of the linked Laboratory Structure.
