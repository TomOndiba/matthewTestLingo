Visualize Capacity Currently Utilized on Work Centers/Tools

The same production resource (work center, machine center, tool or insert) can be planned to be used in different production orders at the same time. The results are overlapping time frames in different prod. order routing lines utilizing the same production resource.

Therefore, overload of work center or machine center/overlapping of tools or inserts can now be viewed in Prod. Order Routing lines along with the ability to lookup to work center or machine center load/ tool or insert load. In Sequence Planning, an overload of the sequence work center or machine center/tool or insert in Prod. Order Routing lines can be viewed as well.

Sequence Planning

For sequence planning, the Capacity Warning field shows sequence planned production order lines.

: Sequence Planning with multiple capacity warnings

If the Capacity Warning field contains Yes, one of the work or machine center/tool or insert within the prod. order routing lines of the prod. order line is overloaded. By clicking the Capacity Warning field, the Prod. Order Routing page is shown to check which operations are causing the overload.

Prod. Order Routing

The Capacity Warning field is also on the Prod. Order Routing page.

: Prod. Order Routing opened from the Sequence Planning line showing the operations causing the overload

The options for this field are described in the following table.

Option

Description

Empty

No capacity warning.

Sequence Work Center Overload

The work center is the sequence work center of the routing and on one day between starting and ending date of the operation the load is greater than 100%.

Prod. Resource Overload

The work or machine center is not the sequence work center of the routing and on one day between starting and ending date of the operation the load is greater than 100%.

Tool Overlap

The work center is a tool and the operation is overlapping with an operation of another production order using the same tool.

Insert Overlap

The work center is an insert and the operation is overlapping with an operation of another production order using the same insert.

 

Capacity Conflict Log

The Capacity Conflict Log page shows detailed information about the source and conflicting production orders. This page can be opened from the ribbon of the Sequence Planning page and from the Sequence Planning Lines page.

: Accessing the “Show Capacity Warning” function on the Sequence Planning page

The information in the Capacity Conflict Log page differs depending on from where it is opened in the Sequence Planning page:

  • If opened from the ribbon, all warnings for all production orders attached to the current work center are shown.
  • If opened from the Lines FastTab, all warnings for the current production order are shown.

: Capacity Conflict Log opened from the Lines FastTab, showing all conflicts for production order 101047

In the Prod. Order Routing page, the Capacity Conflict Log can be opened for the selected line by clicking Show Capacity Warning on the ribbon or by clicking the Capacity Warning field.

: Capacity Conflict Log opened from the Prod. Order Routing, showing the Tool Overlap conflict

From the ribbon of the Capacity Conflict Log page, use the actions to open the current production order/production order routing, and the production order/production order routing causing the conflict. Additionally, the Work Center Load page can be opened to view load information for the current work center.
