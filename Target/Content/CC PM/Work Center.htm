Work Center

A work center comprises a number of machine centers.

To activate a work center for use within the Sequence Planning, the Work Center Type field must be set to Sequence Planning on the General FastTab on the Work Center Card.

: Work Center Card for sequence work center M001

After selecting this option, the system automatically creates a corresponding Sequence Work Center Card. Only data records with this setup are considered within the Sequence Planning (refer to topic “Sequence Planning” on page 1).

Work Center Production Rates can be viewed from this page (refer to topic “Work Center Production Rates” on page 1).

Additional data is entered for work centers that are used for accounting and calculations. Work centers can also represent production capacities. Refer to online help for more information about creating work centers.
