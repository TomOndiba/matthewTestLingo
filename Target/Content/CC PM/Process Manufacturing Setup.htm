Process Manufacturing Setup

The Process Manufacturing Setup page is opened from the navigation pane by clicking Departments > Manufacturing > Administration > Process Manufacturing Setup. This page provides the basic settings for the cc|process manufacturing module.

The following sections explain the fields and the About action on this page.

General FastTab

The fields on the General FastTab are described in the following table.

: Process Manufacturing Setup page, General FastTab

Field

Description

Use Prod. Order No. as Lot No.

If selected, production order numbers are used as lot numbers on posting of production orders (refer to topic “Use Production Order Number as Lot Number” on page 1).

Edit QC Test in Lot No. Info.

If selected, QC tests can be edited in the Lot No. Information Card Adv. page. If not selected, the user must open the QC Test page to edit the lines.

Create Item Tracking at Reservation

If selected, an item tracking line in the source document is created if a reservation is made for a lot, serial number, or trading unit.

If the reservation is canceled, the item tracking line will be deleted. However, if a reservation and an item tracking line are available and the item tracking line is deleted, the reservation is NOT deleted.

Allow Product Design Change in Prod. Order Line

If selected, the Production Structure No., Production BOM No., Production BOM Version No., Routing No. and Routing Version No. can be deleted from a production order line.

Work Center for Tool Planning

This field must be filled in if tools are inserted in routings by a production structure.

On creation of routings, tools are inserted to another work center for parallel planning. Because a parallel planning always must begin and end with a non-parallel planned work center, this work center is inserted automatically in the routing if a work center before and after the parallel planned range is missing.

Work Center for Parallel Routing

This field must be filled in if parallel operations are used in production structure lines (refer to topic “Parallel Operations” on page 1).

Routing Link Code

This code links the item and the routing line in the production order. If using a production structure, this field must be populated.

Note

The value of the field is handled like a number in a number series; therefore, the value must end with a number e.g. RLC00001 because it will be incremented by the application.

Reclaimed Base Unit of Meas.

Base unit of measure code for the reclaimed item. This code is used for the calculation of the scrap and is compared with the unit of measure of the manufactured item. A calculation is only possible if both unit of measure codes are equal.

R&D Item No.

Dummy item number series that is used in the development orders.

Weight Unit of Measure

Unit of measure used for the weight calculation (refer to topic “Unit of Weight and Unit of Volume” on page 1).

Volume Unit of Measure

Unit of measure used for the volume calculation (refer to topic “Unit of Weight and Unit of Volume” on page 1).

Template Toolkits

Default quality management template that serves as the template for the toolkits of the tools. This template is used if no template is entered in the Toolkits field on the Tool Work Center Card.

Method of Production Code

This code is transferred to the production order lines that are created by item bottling.

Use Location and Bin Defaults

If selected, default Location and Bin Codes from items and production structures are defaulted to production order components (refer to topic “Production Inbound/Outbound Location Code and Bin Code” on page 1 and topic “Default Location/Bin Codes” on page 1).

By-Product Cap. Cost Alloc.

Determines how by-product capacity cost is allocated to output posting. The options are:

  • Quantity Proportional – All posted capacity costs of a production order are totaled and then allocated, quantity proportional, on all output postings.
  • None – Capacity costs are only allocated to the output posting that creates the capacity cost.

Refer to topic “Capacity Cost Allocation” on page 1 for more information.
