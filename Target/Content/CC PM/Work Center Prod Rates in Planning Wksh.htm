Work Center Production Rates in Planning Worksheet

Work center production rates can be used in Planning Worksheets and the attached Planning Routing lines. Tools and inserts are also considered.

Example

In production structure PST-004, no setup or run times are defined for work center M003.

: Empty setup and run times for work center M003

When a sales order is generated for item P-10400, the demand can be processed in the Planning Worksheet.

In the Planning Routing, the setup and runs times from the Work Center Production Rates are used. These are the same rates that would be used if a production order was created directly from the sales line.

: Setup and Run Times from Work Center Production Rates

These times can be checked with the “Show used Work Center Production Rate” function on the Navigate tab of the Planning Routing page.

: Work Center Prod. Rate Card used to calculate setup and run times

The setup and run times in the Planning Routing are transferred to the production order, even if the times are changed in the Planning Routing.

: Setup and Run Times in production order routing created from Planning Worksheet
