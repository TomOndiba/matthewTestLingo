Manufacturing Dimensions with Sequence Work Centers

Sequence Work Centers can be set up with Planning Dimensions or Manufacturing Dimensions.

: Manufacturing dimensions set up on sequence work center M001

If Planning Dimensions are not set up, the Manufacturing Dimensions are used. However, Planning Dimensions offer priorities that are important in sequence planning (refer to topic “Optimize Planning Functions in Sequence Planning” on page 1).

 

: Planning dimensions set up on sequence work center M001

In the Work Center Plan. Dimensions page, priorities as well as default values can be specified.

Additionally, the “Get Planning Dimensions” function (Navigate > Get Plan. Dimensions) can be used to copy all Manufacturing Dimensions with Default Priority <> 0 into the page and thereby to the Work Center Plan. Dimensions of the current sequence work center.

Caution

Running the “Get Planning Dimensions” function will delete existing work center planning dimensions.
