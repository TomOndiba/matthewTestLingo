Amortization of Tools

The fields on the Amortization FastTab on the Tool Work Center Card contain information for the acquisition of a tool, the usage and physical life, customer and item, inventory and Quality Control.

: Amortization FastTab on the Tool Work Center Card for the W001 tool

The fields on this FastTab are described in the following table.

Field

Description

Acquisition Cost

Acquisition cost of the tool.

Acquisition Date

Acquisition date of the tool.

Physical Life

Operating life expectancy.

Projected Disposal Date

Automatically populated with the projected disposal date after the Acquisition Date and Physical Life fields are entered.

Guaranteed Output Quantity

Guaranteed output quantity of the tool.

Guaranteed Output Quantity Unit

Unit of measure for the guaranteed output quantity.

Output Quantity

Automatically populated with the already produced quantity.

Customer No.

Number of the customer for which the tool is exclusively produced.

Item No.

Number of the item that is exclusive produced with this tool.

Tool Location Code / Tool Bin Code

Location code / Bin code where the tool is stored.

Toolkits

A QC template with relevant question and answer data for the tool.
