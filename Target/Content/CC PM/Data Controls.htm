Data Controls

Data Controls provide visual evaluation of the existing data and answers the user’s daily questions about their value. The existing data is marked with colors with the help of an analysis point. In cc|process manufacturing, Data Control is integrated in the Planning Pool and in the Sequence Planning.

Note

This functionality can be used with all other tables in Microsoft Dynamics NAV.

Two Data Control setup pages are accessed from the Manufacturing, Administration menu:

  • Data Control Colors
  • Data Control Conditions
