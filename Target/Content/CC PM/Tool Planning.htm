Tool Planning

A tool is planned parallel to the machine in the sequence planning. During this, a dummy routing line is created before and after the routing lines of the machine and tool (if no other routing lines are available). The same setup and run times are calculated in the production order for the machine and the corresponding tool.

: Dummy routing lines added before and after machine and tool

When a new tool is created, the system verifies whether a tool already exists. If more than one tool is created, only one is designated as the main tool. Only the main tools are displayed in the Sequence Planning page.

When a production order is created, the tools are scheduled at the same time as the machine.

Tools can be changed on the Sequence Planning lines by selecting the line to change on the Lines FastTab and then clicking Functions > Change Tool. Select the new tool from the list that appears, click OK and then click Yes to assign the new tool. The sequence planning and the production order routing lines are updated with the new tool. Refer to topic “Change Tool” on page 1 for more information.
