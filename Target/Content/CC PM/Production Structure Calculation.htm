Production Structure Calculation

On the Calculation Card or Calculation List pages, click Prod. Structure Calculation on the ribbon to compare different production structures.

: Prod. Structure Calculation action on Calculation Card

In the Prod. Structure Calc. Lines page, enter the various production structures to compare in the Production Structure No. field.

: Three production structures added for comparison

To run the price comparison, click Show on the ribbon and the click Yes in the message to copy the information from the calculation. The Prod. Structure Calc. Matrix page opens, displaying the calculation lines based on the different production structures.

 

: Sales calculation lines for the three production structures

On the General FastTab, select how to display the column names. The options are:

  • Prod. Structure No. – Production Structure number specified in the Prod. Structure Calc. Lines page
  • Description – Description from the Prod. Structure Calc. Lines page
  • Calculation No. – Calculation No. from Production Structure or automatically assigned to each lot size

Close the Prod. Structure Calc. Matrix page and note that the following values were transferred to the Prod. Structure Calc. Lines page:

  • net cost calculation line for each production structure transfers to the Standard Cost field
  • net price calculation line for each production structure transfers to the Unit Price field

: Calculated cost and price values from the matrix

The production structure calculation can be transferred to the item by selecting the line and then clicking Transfer Value on the ribbon. The Transfer Calculation Value page appears, displaying the value to transfer; click OK to complete the transfer process.

Calculated lines can be deleted as needed; the related calculation is also deleted.
