Sprue Bush

The following points are relevant when explaining sprue bush weight:

  • Raw material consumption is dependent on the tool
  • Raw materials requirements must be considered
  • Raw material costs must be considered
  • If changing a tool, a recalculation of the sprue bush is needed
  • Possible dependence between machine and tool

The Sprue Bush Weight for each tool can be specified on the Tool FastTab of the Tool Work Center Card.

: Sprue Bush Weight specified on the W004 tool

As with the other fields on this FastTab, the sprue bush weight is transferred to the Work Center Prod. Rate Card page where it can be customized for each work center.

: Sprue Bush Weight transferred from Tool W004 to Work Center Prod. Rate Card for work center M003

The value entered in the Weight Unit of Measure field on the Process Manufacturing Setup page ensures that the sprue bush weight and the item are calculated using the same unit of measure.

In addition to ensuring the correct calculation, the Weight Unit of Measure field must be filled in because the sprue bush weight from the tool must be converted into a percentage for the raw materials in the component lines of the production order. This is due to the increasing amount of raw materials because of the scrap %.
