Actions Tab

On the Actions tab, the Functions and Report groups contain actions relating to the selected Planning Pool line. Each action is explained in the following sections.

Reverse

This function reverses planned production orders. The production order is moved to the end of the sequence in the work center and then all other planned production orders are rescheduled.

: Reversed production order line

Note

During the Reverse process, planned maintenance-/setup jobs are deleted.

Change Work Center

This function changes the work center on a sequence order. Initially, a list of all sequence work centers is displayed in the Work Center List page.

: Work Center List page

In this page, select the work center to where the current production order in the sequence planning is moved. After clicking OK, the following message appears.

: Message to change the Work Center on the production order line

After accepting the message, the production order line is moved to the new work center. The production orders of the current work center will be re-scheduled.

: Production order line moved from work center M002 to M001

During the work center change, all current production rates are considered. If the new item/work center combination has different production rates, they are used and stored in the routing sheet.

The work center is also changed in the routing sheet.

: Production order routing with updated work center information

In addition to the production rates, the work center costs are updated.

Change Tool

Generally, the change of tools is like the change of a work center. A list of available tools that can be changed is displayed. During the change process, the tool is changed in the routing sheet and new costs and production rates are recalculated.

Change Insert

Generally, the change of inserts is like the change of a work center or tool. A list of available inserts that can be changed is displayed. During the change process, the insert is changed in the routing sheet and new costs and production rates are recalculated.

Split Prod. Order Line

This function splits production order lines into multiple production orders and approaches (refer to topic “Split Production Orders” on page 1).

Combine Prod. Order Line

This function combines several production orders into one production order (refer to topic “Combine Production Orders” on page 1).

Change Status

This standard manufacturing function in Microsoft Dynamics NAV is used in the Planning Pool to change the status of one or more production orders. To change several production orders, each must have the same status. Various orders can be selected by clicking the line while pressing the CTRL key.

: Change status dialog box

Select the relevant status (to which the production order(s) is to be assigned) and the appropriate Posting Date.

To update the unit cost of the item that is produced, select the Update Unit Cost check box. The unit cost of the item will also be updated in any production orders or sales orders for which this item has been reserved.

Print Planning Pool

Click Print to generate a production overview from the planning pool.
