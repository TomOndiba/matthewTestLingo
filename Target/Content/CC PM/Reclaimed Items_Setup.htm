Setup

This section covers the setup needed to use the reclaimed item functionality.

Process Manufacturing Setup

In the Process Manufacturing Setup page, a unit of measure must be entered in the Reclaimed Base Unit of Meas. field. The unit specified here must be entered for both items (production item and reclaimed item) as the item unit of measure for the calculation.

: Process Manufacturing Setup, Reclaimed Base Unit of Measure field

With a fixed conversion unit, such as KG, a unique conversion is possible if this unit is used as a base unit of measure in both items. If the reclaimed item was entered with the unit PCS, then the reference is no longer unique.

For example, a production quantity of 10 pieces (at 5 kg each) with 10% scrap may result in a scrap of pieces (at 1 kg each).

List Reclaimed Item

In the List Reclaimed Item page, the reclaimed item is assigned to the main item. Open this page for editing from the Product Design menu by clicking List Reclaimed Item > Edit List. In the Edit - List Reclaimed Item page, fill in the Base Unit of Measure, Scrap % and Reclaimed Item No. fields on the relevant item lines.

: List Reclaimed Item page in edit mode

As Fig. 2 illustrates, when pieces of item P-950-RG001 are produced, 10 percent is scrap and item P-951-RG002 is produced from that scrap.

Note

Only one reclaimed item is possible for each item.
