Ingredient Card

On the Ingredient Card – accessed in the navigation pane by clicking Manufacturing > Ingredients Management > Ingredients – all necessary ingredients are defined and maintained.

: Ingredient Card

The fields on the General FastTab are described in the following table.

Field

Description

No.

Unique identifier for the ingredient.

Description

Description of the ingredient.

Description 2

An additional description of the ingredient.

Base Unit of Measure Code

Base unit of measure code of the ingredient. Click the drop-down arrow to access the Ingredient Unit of Measure list where one of the predefined units of measure can be selected and conversion factors can be defined between the several units.

Test Property Code

A link to the test property in cc|quality management (refer to topic “cc|quality management” on page 1). Click the drop-down arrow to enter / allocate a test property that will be applied when the determined actual value is to be allocated in a receipt test. A test property code is only to be allocated exactly to one ingredient.

Note

The Test Property must be set up with a specific Alloc. Unit of Measure for ingredients processing for proper value transfer to the lot ingredients (refer to topic “Receipt Testing” on page 1).

On the ribbon, click Navigate > Translations to open the Ingredient Translations page, where translations for the Description and Description 2 fields can be entered. Therefore, one of the predefined Language Codes must be selected and the according translated descriptions must be entered. The translation can e.g. be printed on several reports.

Exercise – Create Ingredients

Create two ingredients, each with the following ingredient unit of measure records:

  • MG = 1 (base)
  • GR = 1000

Assign separate Test Property codes to each ingredient.

Create the Ingredient Cards

Follow these steps to set up the ingredient cards:

  1. In the navigation pane, click Manufacturing > Ingredients Management > Ingredients.
  2. On the ribbon, click New and then press ENTER to automatically populate the No. field.
  3. In the Description field, type Ingredient 1.
  4. In the Base Unit of Measure Code field, click the drop-down arrow and then click New.
  5. In the Code field, enter MG.
  6. Go to the next line.
  7. In the Code field, enter GR.
  8. In the Qty. per Unit of Measure field, type 1000.
  9. Select the first line for MG to ensure that it is selected as the base unit of measure.

: Ingredient Unit of Measure

  1. Click OK to close the Ingredient Unit of Measure page.
    Note that the Base Unit of Measure field is populated with MG.
  2. In the Test Property Code field, enter TP0000127.

: Ingredient Card

  1. Click the drop-down arrow on the OK button and select OK & New.
  2. Press ENTER to automatically populate the No. field.
  3. In the Description field, type Ingredient 2.
  4. In the Base Unit of Measure Code field, click the drop-down arrow and then click New.
  5. In the Code field, enter MG.
  6. Go to the next line.
  7. In the Code field, enter GR.
  8. In the Qty. per Unit of Measure field, type 1000.
  9. Select the first line for MG and then click OK.
    Note that the Base Unit of Measure field is populated with MG.
  10. In the Test Property Code field, enter TP0000128.
  11. Click OK to close the Ingredient Card.
