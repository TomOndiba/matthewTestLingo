Permanently Deferred in Sequence Planning

With this function, users can defer production order lines in sequence planning permanently so that the re-scheduling process does not consider the production order line.

To defer a production order in the Sequence Planning page, select the Permanently Deferred check box on the relevant line.

: Permanently deferred production order line in sequence planning

The marked production order line is not re-scheduled even if the Schedule Deferred Prod. Order Lines check box is selected in the Priority Sequence Planning batch job (refer to topic “Priority Planning” on page 1) or the Sequence Pl. Start Advanced batch job (refer to topic “Actions Tab” on page 1).
