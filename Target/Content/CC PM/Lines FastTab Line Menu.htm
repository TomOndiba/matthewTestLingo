Lines FastTab, Line Menu

On the Lines FastTab, click the Line menu to access menu items relating to the matrix lines.

Substitutions

This menu item displays item substitutions for the selected production structure line. Item substitutions are only available when the line Type is set to Item or Share of Recipe.

: Item substitutions for item P-30009

The Item Substitution Entry page can also be opened from the item card. The corresponding item substitutions that are entered here are not only valid for the production process line but also for the entire item.

Use the “Choose Columns” function on the production structure lines to display the Substitution Available field for better visibility. If necessary, items can also be exchanged in the component lines in a production order.

Item Routing

This menu item displays the routing for the item on the selected production structure line, if one exists. Use the “Choose Columns” function on the production structure lines to display the Routing Exists field for better visibility.

Item Prod. BOM

This menu item displays the production BOM for the item on the selected production structure line, if one exists. Use the “Choose Columns” function on the production structure lines to display the Production BOM Exists field for better visibility.

Search by Mfg. Dimensions

This menu item opens the Search by Mfg. Dimensions page, where users can search for Items, Share of Recipe, Work Center, Tools and Inserts that can then be inserted into the Production Structure lines.

To insert a selection from the Search by Mfg. Dimensions page, select an empty line in the Production Structure and enter the relevant value in the Type field (e.g., Item or Share of Recipe). Next, open the Search by Mfg. Dimensions page, enter the search criteria in the header and then click Find.

: Searching for items with the Manufacturing Dimension of COLOR = BLUE

Select the item, work center, tool, or insert from the Mfg. Dimension Search Results FastTab and then click OK. The selected result is added to the Production Structure line.

Note

An existing production structure line will be overwritten with the selected result if a new line is not inserted first.

Refer to topic “Search by Manufacturing Dimensions” on page 1 for more information.

Ingredients

This menu item opens the Prod. Structure Line Ingredients page, which displays the calculated quantities for ingredients on a production structure line (refer to topic “Production Structure Component Ingredients” on page 1).

Line Texts

This menu item opens the Edit Text page where special text can be added for the selected production structure line or existing text can be viewed or edited (refer to the current “cc|text module” documentation).

Create Item

This function allows users to create a new item with the information from the production structure line, including ingredients and line texts. This function can be used only with production structure lines that have no item number.
