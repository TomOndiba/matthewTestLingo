Development Structure

The development structure is a composition of substances and materials that does not require the use of existing items.

In the Production Structure, when the Production Structure Type field is set to Development Structure, an item is not entered, only the raw materials. Other fields, however, such as unit of measure and lot size, can be entered. In the Content Specification on field, users can choose to calculate the share of recipe by mass percent or volume share percent. Refer to topic “Content Specification” on page 1 for more information about using these calculation percentages.

: Development Structure

In the lines, enter existing items and text to create the recipe.

Note

To enter a Share % without an existing item number, the line Type of Share of Recipe must be used.

Use text to specify instructions, such as a guide for the mixing of a recipe. When the Status is changed to Certified, a calculation is done on lines with Type = Share of Recipe. Text type lines are not verified.

In the development structure stage, users can enter work centers and machine centers, but they do not affect the capacity planning.

On the ribbon, click Print to generate the result or mixing guide with the laboratory structure.

Change to Laboratory Structure

Once the Development Structure has been created and is ready for the next step in the R&D process, users can create a Laboratory Structure from the ribbon of the Development Structure page by clicking Create Laboratory Structure.

The new Laboratory Structure page appears and is set with a Status of Under Development.

: Laboratory Structure created from the Development Structure

The R&D FastTab displays the number of the linked Development Structure.
