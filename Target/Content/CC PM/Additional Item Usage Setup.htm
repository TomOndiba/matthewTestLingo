Additional Item Usage Setup with Tools and Inserts

This section includes important setup information about working with items in tools and inserts.

Tool with Main Item

When working with tools, ensure the following:

  • Set up the tool with the main item; the cavity is just for the main item.
  • Set up the item so that it will produce with the same tool.
  • Set up the By-Production for all other cavities.
  • Set up the cost accordingly (standard for the main item but the raw material cost can be split).

Working with Inserts

When working with inserts, ensure the following:

  • Set up the insert with the main item; the cavity is just for the main item.
  • Set up item and inserts and compare to the By-Production; it will not be set up automatically at this time.
