Returnable Containers on Sales Orders

When a sales order is created with a line that has returnable containers assigned, the returnable containers can be selected directly from the order.

: Preparing to insert returnable containers on the sales order to customer 10000

If only one container has been assigned to the item, it is added to the sales line with the calculated quantity.

Note

The number of returnable containers is rounded up.

If multiple containers have been assigned to the item, the Returnable Container Selection page appears. 

Depending on the possibilities, the number of needed returnable containers are calculated and displayed. Enter the relevant quantity in the Selected Quantity (Container) field.

: Selection of three 500 kg containers for the sales order line

The fields at the bottom of the page are updated. Each field is described in the following table.

Field

Description

Selected Container

Total number of currently selected containers.

Max. Quantity

Quantity in the order line for which containers should be selected.

Allocated Quantity

Total number of the containers that have been selected.

Undefined Quantity

Difference between the Max. Quantity and the Allocated Quantity.

Unit of Measure Code

Unit of measure in which the container must be chosen.

Once the container(s) is selected, click OK. The container(s) is added to the sales line with the Returnable Container check box selected.

: Addition of three 500 kg containers to the sales order line

During posting, the container is transferred to the Customer Location but remains in inventory.

The Returnable Cont. by Location page shows all returnable containers that were transferred to customer locations. Open this page from the Warehouse menu by clicking Planning & Execution > Tasks > Returnable Cont. by Location. On the Options FastTab, the column name can be shown.
