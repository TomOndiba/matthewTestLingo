Verification Setup, Manufacturing FastTab

The Manufacturing FastTab includes verification required to complete the cc|process manufacturing processes described in the following table.

: Verification Setup

Field

Description

Verify Certify Prod. Structure

Verification is required to certify a Production Structure.

Verify Reset Prod. Structure Status

Verification is required to reset a Production Structure from the Certified status.

Note

Only Password Check and Password Check+Documentation verifications are available for this process.

Verify Certify By-Production

Verification is required to certify a By-Production.

Verify Reset By-Production

Verification is required to reset a By-Production from the Certified status.

Note

Only Password Check and Password Check+Documentation verifications are available for this process.
