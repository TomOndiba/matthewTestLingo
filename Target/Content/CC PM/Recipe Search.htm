Recipe Search

This function helps users locate recipes. Enter different materials and a range (minimum share % and maximum share %) to search.

When opened from the Production Structure, all necessary data is transferred to the Recipe Search page so less data entry is needed.

: Recipe Search from production structure PST-V-002

Use the filters in the Options section to refine the search:

  • Select the Exact Match check box to locate only recipes that have all items within the minimum and maximum share ranges.
  • Select the Only certified Production BOM check box to locate production BOMs whose status is certified.
  • Select the Only active Versions check box to locate active versions of production BOMs.

Without using filters, the system finds all recipe combinations.

In the lines, enter the different values and a minimum and maximum share % and then click Find on the ribbon to receive the results in the Recipe Search Results page.

: Recipe Search Results

From the Recipe Search Result page, open the recipe cards by clicking Navigate > Card on the ribbon.
