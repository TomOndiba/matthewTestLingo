Manufacturing Dimension Templates

Manufacturing dimension templates can be used to standardized manufacturing dimensions for items, work centers, tools and inserts. These templates consist of a template header with a unique code and a description. Manufacturing dimensions can be allocated to the templates.

: Manufacturing dimension template for plastic sheets

If a manufacturing dimension template is added to an item, work center, tool or insert, the manufacturing dimensions entered in the template are transferred to the default manufacturing dimensions of the current record.

: Default manufacturing dimensions assigned for sheet item

The value in the Mfg. Dimension Template Code field cannot be changed if default manufacturing dimensions already exist.

Item Categories

Manufacturing dimension templates can be added to item categories.

: Def. Mfg. Dim. Template Code field on item categories

If the item category is used for an item, the Def. Mfg. Dim. Template Code specified on the item category is transferred to the Mfg. Dimension Template Code field on the item.
