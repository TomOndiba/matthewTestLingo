Navigate Tab

On the Navigate tab, there are two groups: Work Ctr. and Line. In the following sections, each cc|process manufacturing page is described in detail while standard Microsoft Dynamics NAV pages are not; refer to online help for additional information about standard pages.

Work Center Group

The actions in the Work Ctr. group relate to the work center selected on the General section of the Planning Pool page.

Action

Description

List

Displays a list of all work centers.

Card

Displays the card of the selected work center. Refer to topic “Capacities” on page 1 for more information.

Note

A card will only open if there is a work center sequence on the line.

Capacity Ledger Entries

Displays the capacity ledger entries for the selected work center.

Capacity ledger entries are created if capacity consumption is entered for a production order. They are created in time units (run time, stop time, setup time) or in quantities (scrap quantity, output quantity).

Dimensions

Displays the dimensions assigned to the selected work center.

Manufacturing Dimensions

Displays the default manufacturing dimensions for the selected work center (refer to topic “Manufacturing Dimensions” on page 1).

Manufacturing dimensions are used as global and shortcut dimensions just like the standard dimensions. Manufacturing dimensions can be added to items, work centers, tools, inserts and production order lines. The global and shortcut dimensions for each type can be specified in the manufacturing dimension setup.

Planning Dimensions

Planning dimensions are used to better handle sequence planning. For example, based on planning dimension priorities of color, machines will be sequences so lighter colors are processed first, followed by darker colors (refer to topic “Manufacturing Dimensions with Sequence Work Centers” on page 1).

As illustrated in the above screen shot, when the “Priority Planning” function is run, the Sequence Planning lines will be re-sorted by due date, then color and so on (refer to topic “Optimize Planning Functions in Sequence Planning” on page 1).

Comments

Displays comments for the selected work center. Comments are for internal use only; they are not used on any documents.

Load

Contains a drop-down list with the following menu items:

  • List – Opens the Work Center Load page, which provides an overview of availability at the selected work center.
  • Load/Setup - Chart – Opens the Work Center Load/Setup - Chart page, which provides a chart-style overview of availability (Load) or Setup-/Run Times (Setup) at the selected work center.

  • The items in the menu groups on the Home and Actions tab determine the displayed information:
    •  Chart – switch between Load and Setup views
    •  Period Length – time periods to be displayed
    •  Options – values to calculate (capacity, load or both) and type of chart (column, pie, line and so on)

Note

The chart will only open if there is a sequence work center on the line.

Statistics

Displays the statistics for the selected work center.

Text Module

Opens the Text Module List page where special text can be added for the Sequence Work Center of the Planning Pool line (if a Sequence Work Center exists) or existing text can be viewed or edited (refer to the current “cc|text module” documentation).

 

Line Group

The actions in the Line group relate to the selected line and are described in the following table.

Description

Description

Prod. Order

Opens the Production Order of the selected production order line.

Sequence Planning

Opens the Sequence Planning page. If the selected line contains a sequence work center, this work center will be the selected line in the Sequence Planning page.

Item Availability by

Contains a drop-down list with the following menu items:

Event

Location

Period

Lot

Variant

BOM Level

These options check the availability of the item that is currently produced.

Reservation Entries

Opens the Reservation Entries page, which contains one entry for each manually or automatically created reservation of the current production order line.

Dimensions

When a production order, a production order line, or a production order component is created, document dimension values can be added that can be used later in the analysis. For two shortcut dimensions, the dimension values may be entered directly in a production order, a production order line, or a production order component.

Values for dimensions that are not available in a production order, a production order line, or a production order component, can be entered in this page.

Manufacturing Dimensions

On creation of production order lines, the default manufacturing dimensions from the item are transferred to the production order line. If the production order lines are sequence planned, the default manufacturing dimensions from the possibly existing sequence work center, tool or insert are transferred to the production order line as well.

The Priority field in the Manufacturing Dimension Setup page defines the order of the different default manufacturing dimensions to transfer (refer to topic “Manufacturing Dimension Priority” on page 1).

Example

The item and the work center in a production order line have the same default manufacturing dimension. Only the default manufacturing dimension value of the type with the highest priority in the manufacturing dimension setup is transferred to the production order line.

Routings

Opens the Prod. Order Routing page for the selected production order line, where users can enter routing sheets before or after a resource sequence.

Components

Opens the Prod. Order Components page, where substitution items can be checked and replaced, and new components for the production order can be entered manually.

Components Planning Pool

Opens the Components Planning Pool page, where the availability of components can also be checked in the planning pool (refer to topic “Components Planning Pool” on page 1).

Item Tracking Lines

Opens the Item Tracking Lines page, used to track planned items that are Lot No. / Serial No. tracked.

Show used Work Center Production Rate

Opens the Work Center Prod. Rate Card page for planned production order lines to show the Work Center Production Rates used to calculate the setup and run times of a production order routing line.

If this function is run on a planning line without a Work Center Production Rate, a message will appear.

Calculation

Contains the Calculation (Expected) and Calculation (Actual) options. Refer to topic “cc|calculation” on page 1 for more information.

Production Journal

The production journal is standard manufacturing functionality in Microsoft Dynamics NAV but has been enhanced with some cc|process manufacturing features. For more information about the cc|process manufacturing-related functionality, refer to topic “Production Journal” on page 1.

For more information about the standard functionality, refer to the Production Order Processing content in the current Microsoft Dynamics NAV Manufacturing training material.
