Work Center Efficiency

The Work Center Efficiency report displays the productive and nonproductive time per work center with listing per Order Type Code and Standstill Reason Code. Access this report from the Manufacturing menu by clicking Archive > Work Center Efficiency.

: Work Center Efficiency Report showing productive and nonproductive time for work center M007

The report can also be set up to print more detailed information by selecting the Print Single-Position check box on the Options FastTab on the report request page.

: Work Center Efficiency Report showing detailed nonproductive time for work center M007
