Item Card, Production Actions

On the Navigate tab, click the Production icon () in the Assembly/Production group to access actions relating to process manufacturing. Each is described in the following table.

Function

Description

Production Structures

Opens the Production Structure List page, which is used to create or open existing production structures (refer to topic “Production Structure Setup” on page 1).

This function is also available on the Home tab.

Production BOM

Opens the standard Microsoft Dynamics NAV Production BOM page.

Where-Used

Opens the standard Microsoft Dynamics NAV Prod. BOM Where-Used page, which shows items that contain the selected item. If recipes are used, the corresponding values are calculated and listed based on the net weight.

Calc. Standard Cost

Calculates the unit of measure cost of the item using the production structure (BOM, recipes and routings).

Detail Calc. Standard Cost

Opens the Detail Calculation page, which shows a detail calculation considering the unit cost for the selected item (refer to topic “Detail Calculations” on page 1).

Calculation

Opens the Calculation List, where calculation cards can be created and/or opened (refer to topic “cc|calculation” on page 1).

This function is also available on the Home tab.

Manufacturing Dimensions

Opens the Default Mfg. Dimensions page, which displays the default manufacturing dimensions for the selected item (refer to topic “Manufacturing Dimensions” on page 1).

Work Center Production Rates

Opens the Work Center Production Rates page where the setup and the run time of items or item categories per work center can be viewed or edited (refer to topic “Work Center Production Rates” on page 1).

Production Simulation

Calculates the required routing operations and the necessary material for a specific quantity of an item. The calculation will include all production stages (refer to topic “Production Structure Simulations” on page 1).

Item Bottling

Opens the Item Bottling page, which is filtered by the item and will display the current stock of inventory (refer to topic “Bottling Orders” on page 1).

Incorporation Usage

Opens the Incorporation Usage page, which allows users to check into which production orders an item could be incorporated (refer to topic “Incorporation Usage” on page 1).
