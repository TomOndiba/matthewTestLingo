Bottling Production Structure

The Bottling Structure type on the Production Structure provides the ability to bottle items in different units of measure. Thus, a rapid handling of the filling or decanting is possible.

In this type of production structure, only the packaging materials or filling periods that are necessary for bottling are defined. The Item Bottling page handles the transformation from one unit of measure to another, such as 10 liter units of paint to 4 liter units of paint; but it does not handle the 4 liter package and the processing time. To handle this, a Bottling Structure is created with the 4 liter package and the time for packaging.

When certifying the Bottling Structure, the Production BOM and Routing are not copied to the item.

: Bottling Structure selected on a production structure

In Fig. 1, a Bottling Production Structure has been set up for the OSNALUX gloss paint item. This production structure is also a production variant. To bottle 10 liters of this paint, materials (10 liter pail) and processing (bottling plant) are required.

Only certified Bottling Structures can be used in the Item Bottling page.

: Item Bottling for item P-10503

When a value is entered in the New Unit of Measure Code field, the system looks for certified production structures that match the unit of measure.

: Item Bottling for item P-10503 set to 10 L unit of measure

The Avail. Production Structures field displays all the production structures that match the unit of measure. If only one exists, the Production Structure and related Production BOM and Routing are assigned in the Production Structure FastTab.

: Production Structure information for 10 L of item P-10503

If multiple production structures match the unit of measure, one must be selected manually in this FastTab. The Production BOM and Routing can also be selected manually. When the production order is created, the information in this FastTab is used.

If there is a lot of movement, such as to KG, there is no initial difference in the outcome. For example, 300 KG from lot OS-KG-FX300 must be filled into 10 L buckets, so 300 is entered into the Qty. to Handle field on the relevant KG line.

: 300 kg of lot OS-KG-FX300 selected and prepared for posting

When this item bottling selection is posted, 30 10 L buckets are produced from the same lot and the Remaining Quantity on the originating line is reduced by the 300 units.

: Thirty 10 L buckets of item P-10503 posted

To review the postings for the 30 10 L buckets, select the line and then click Navigate on the ribbon.

: Posted entries for the thirty 10 L bucket of item P503

Select the Item Ledger Entry table and then click Show Related Entries on the ribbon.

: Item Ledger Entries for the Item Bottling

300 units of item P-10503 (OSNALUX gloss paint) have been produced and 30 units of item P-30260 (Pail, 10 L), has been consumed.

Close the Item Ledger Entries page and then open the Capacity Ledger Entry from the Navigate page.

: Capacity Ledger Entries for the Item Bottling

The capacity of the work center in the Bottling Structure has been posted. Using the Bottling Structure does not change the speed of production but provides a substantial advantage by using a correct unit of measure calculation.
