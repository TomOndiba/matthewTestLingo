Component Availability

In the lines of the Planning Pool, Sequence Planning and Production Order pages, the Components Available field assists with component planning.

  • “Yes” indicates that the availability is assured or there is no component requirement on the line. If the production orders of a work center use the same raw materials, users can see at what point of time they run out of raw materials.
  • “No” indicates that availability is not assured. Missing raw materials can be located using the Components Planning Pool.
