Exercise – Data Control Conditions

Set up a new condition called EXERCISE for the Production Order Line table, 5406 with the following controls:

  • If the Status is Released, the Planning Pool line needs to display a green picture.
  • If the Status is Firm Planned, the Planning Pool line needs to display a yellow picture.
  • In all other cases, the Planning Pool line needs to display a red picture.

Create a New Data Control Condition

Follow these steps to create a new condition:

  1. On the Manufacturing, Administration menu, click Data Control Conditions.
  2. On the ribbon, click New.
  3. In the No. field, type EXERCISE.
  4. In the Table No. field, enter 5406.
  5. In the Description field, type Production Order Status.
  6. In the Def. Color field, enter RED.
  7. On the Lines FastTab, in the Field No. field, enter 1.
  8. In the Operator field, enter =.
  9. In the Type field, enter Value.
  10. In the Value field, enter 3.
  11. In the Color field, enter GREEN.
  12. On the next line, in the Field No. field, enter 1.
  13. In the Operator field, enter =.
  14. In the Type field, enter Value.
  15. In the Value field, enter 2.
  16. In the Color field, enter YELLOW.
  17. Click OK to close the Data Control Conditions page and then Data Control Condition List page.

Set the Filter in the Planning Pool

Follow these steps to set the filter in the Planning Pool page:

  1. In the navigation pane, under Manufacturing, click Planning > Planning Pool.
  2. In the Data Control section, enter EXERCISE in the Condition field.
  3. Press ENTER or TAB.
  4. Verify that the correct colors are used in the DC field by setting a filter in the Prod. Order Status Filter field on the General FastTab.
