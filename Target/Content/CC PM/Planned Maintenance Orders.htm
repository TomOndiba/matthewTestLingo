Planned Maintenance Orders / Maintenance Order Templates

A maintenance order template can be created manually. The Status is set to Planning when created manually.

: Maintenance Order Template for machine number W-M-002

If the maintenance is performed by an external service provider, the vendor is entered in the Vendor No. field on the General FastTab and only G/L accounts can be entered in the lines.

To insert comments into lines, leave the Type field blank and enter the comment in the Description field.

The validity dates of the template can be specified in the Starting Date and Ending Date fields on the General FastTab. The Starting Date determines the Next Maintenance Date, which is when the next service should be performed. This date can be manually changed.

After service has been performed, the Next Maintenance Date field is recalculated (based on the Maintenance Interval), and the Last Maintenance Date field contains the date of the most recent service. This date cannot be edited.

Create Maintenance Orders

Maintenance Orders can be created using either of the following methods:

  • Single order directly from the maintenance template
  • Multiple orders using the “Create Maintenance Orders” periodic activity
Create Single Maintenance Order

After the template is set up, a maintenance order can be created by clicking Create Maintenance Order on the ribbon. Click Yes to create the maintenance order on the date specified in the Next Maintenance Date field. This date appears as the Starting Date on the new maintenance order.

If created from a template, the number appears in the Created-by Template No. field on the maintenance order.

Create Multiple Maintenance Orders

The Create Maintenance Orders batch job is accessed from the Plant Maintenance menu.

: Create Maintenance Orders batch job request page

Limit the maintenance orders created by entering start and end dates on the Options FastTab. Select the Create all orders in period check box to include all the templates where the Next Maintenance Date falls within the specified period.

On the Maintenance Order FastTab, filter on the templates, machine numbers and so on to include in the creation process. Click OK to begin the creation process.
