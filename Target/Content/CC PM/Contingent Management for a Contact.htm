Contingent Management for a Contact

For a contact that has not yet been linked to a customer, users can create a sales order from the Contact Card by clicking Actions > Create Sales Document. Here, the system checks whether a Virtual Customer with corresponding country code or currency code exists.

Single Matching Record

If one matching record exists, the user can specify the type of the document to be created in the Create Sales Document from Contact page.

: Document type selection request page for matching customer

If a customer relation with a contact already exists, the system will provide the according customer number as a default value. This number cannot be changed by the user.

Multiple Matching Records

If there is more than one valid record, the system requests the user to select an appropriate one in the Create Sales Document from Contact page.

: Multiple matching customer message and request page

No Matching Customers

If the system is unable to find a suitable virtual customer, a warning message will be displayed.

: No matching customers warning message
