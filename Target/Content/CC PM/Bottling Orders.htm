Bottling Orders

To use item bottling orders in cc|process manufacturing, the items must be configured as an item or variant storage unit.

Depending on from where the Item Bottling page was opened, the current warehouse stock is displayed automatically or must be defined:

  • If opened from the Item Card, by clicking Navigate > Production > Item Bottling, nothing must be done.
  • If opened from the Execution menu, enter the item number and then click Find on the ribbon.

: Item Bottling page opened from the item card, filtered on 25 KG and 10 L units of measure

In addition to the item number, other filters on the Options FastTab can be set to narrow the warehouse stock displayed.

To change the unit of measure (the trading unit) for specific inventories, follow these steps:

  1. On the Options FastTab, enter the new unit of measure in the New Unit of Measure Code field.
  2. In the Qty. to Handle field, enter a subset of the line for the bottling; the Selected check box is automatically selected on the line(s).

: New item unit of measure and line set for handling

After entering the changes, there are two options of proceeding: 

  • Post the bottling directly from this page by clicking Post on the ribbon.
  • Create a production order for the bottling, which can be further processed, by clicking Carry Out Item Bottling on the ribbon.

NOTE

Only the production structure shown in the Production Structure FastTab is used with this function.

When using the Post option, a production order is created and posted automatically. All analysis options remain available.

If the Carry Out Item Bottling option is selected, the following dialog appears.

Note

When the Carry Out Item Bottling dialog page is opened, the Type field defaults to Post and the Production Order field is inactive until the Type field is set to Create Prod. Order.

: Carry Out Item Bottling with the Create Prod. Order option selected

In the Type field, select the option to Create Prod. Order and then select the type of production order to create. Select the Show Document check box to show the production order after clicking OK.

After creating a production order, the reserved quantities are displayed in the Item Bottling page.

: Reserved quantity relating to the firm planned production order

Production orders generated from the Item Bottling page can also be viewed by selecting the relevant line, and then clicking Actions > Show Prod. Order.
