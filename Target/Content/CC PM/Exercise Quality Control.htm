Exercise – Quality Control

In this exercise, set up a QC Template for a new tool check using the following information:

  • QC Type = TOOL
  • Headerline with the description of New Tool Check
  • Create a question with two answers without using existing questions/answers
  • Add two more question lines by selecting these existing questions (and the related answers):
    • QC-F-005
    • QC-F-006

Allocate the QC Template to the work centers and then run the QC test for tool W002.

Create a New Tool Check QC Template

Follow these steps to create the New Tool Check QC Template:

  1. In the navigation pane, under Manufacturing, click Quality Control > QC Template.
  2. On the ribbon, click New.
  3. Press ENTER or TAB to insert a new number.
  4. In the Description field, type New Tool Check.
  5. In the QC Type Code field, enter TOOL
  6. On the first line, in the Type field, enter Headerline.
  7. In the Description field, type New Tool Check.
  8. On the second line, enter the question.
  9. On the third line, enter the first answer.
  10. On the fourth line, enter the second answer.
  11. On the fifth line, in the No. field, enter QC-F-005.
  12. Click Yes to transfer the assigned answers.
  13. On the seventh line, in the No. field, enter QC-F-006.
  14. Click Yes to transfer the assigned answers.
  15. In the Status field, enter Certified.

Allocate the QC Template to Work Center

Follow these steps to allocate the QC Template to the Work Center table:

  1. On the ribbon, click Navigate > QC Template Allocations.
  2. In the Source Type field, enter 99000754.
  3. Click OK to close the QC Template Allocations page, click OK to close the QC Template page and then close the QC Template List page.

Run the QC Test for Work Center W001

Follow these steps to run the New Tool Check test for work center W001:

  1. In the navigation pane, under Manufacturing, click Capacities > Tools.
  2. Locate tool W001 and open the Tool Work Center Card.
  3. On the ribbon, click QC Test.
  4. Confirm that the questions and answers defined in the new QC Template appear.

Note

If you previously allocated the existing QC Template to the Work Center, then open the W002 work center and note that the QC Test List page opens with the existing and new templates when you click QC Test on the ribbon.
