Tools and Inserts

Just as with work centers and tools, inserts can be assigned to specific tools.

: Inserts assigned to the W004 tool

With this setup, the assigned inserts can be used with this tool, but other inserts can be assigned to this tool in the sequence planning. To ensure that only these inserts are used with this machine in sequence planning, the Check Insert Usage check box must be selected on the Scheduling FastTab on the Tool Work Center Card.

: Check Insert Usage selected for tool W004

With this setup, only the assigned inserts can be selected with the “Change Insert” function in Sequence Planning (refer to topic “Change Insert” on page 1).
