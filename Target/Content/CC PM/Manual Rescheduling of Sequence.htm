Manual Rescheduling of Sequence Planning

On each action in sequence planning (move of production orders, change of routings and insert of maintenance orders) the data is updated. That means that the sequence work center and all production orders attached to the sequence work center are re-scheduled.

This is not problematic – and can be necessary – for larger amounts of data because otherwise there is incorrect, outdated data in the system that can cause false results if further processes (such as, creating a requisition worksheet for the purchase) use this data.

However, if there are several changes to make for a machine (Work Center) or for its production orders, the processing time can be reduced if, after all changes, the new scheduling is done once.

The Rescheduling field is located on the Scheduling FastTab of the work center and contains two options for how existing production orders for a sequence work center are handled when changes are made to a production order that is attached to the sequence work center:

  • Automatic – All production orders for that work center are immediately rescheduled.
  • Manual – None of the production orders are rescheduled and the Scheduling Required check box is selected for the work center in Sequence Planning.

When the “Scheduling” function is run, the Scheduling Required check box will be cleared on the work center line.
