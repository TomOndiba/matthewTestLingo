Reclaimed Items

If scrap is a material that can be re-used in production or can be sold separately, cc|process manufacturing supports this with the Reclaimed Item functionality. With this functionality, scrap is maintained in inventory.

Note

This functionality is referred to as Regranulation in the plastics industries.

 

: Reclaimed item process flow

When an item is produced, the scrap is registered based on the setup in the Scrap % field on the Item Card.

As Fig. 1 illustrates, 10% scrap has been set up on Item A. When it is produced, the work hours and material planning receive an additional 10% so that the time and material usage are correctly planned and calculated as demand.

The 10% scrap of item A is then posted on inventory using item B. The arrow from item B to the Production Order illustrates that the scrap can be re-used on production orders.
