Sales Orders with Production Variants

If a sales order is placed for an item with different production variants containing the aforementioned criteria, a production order is created under consideration of the different production structures.

For example, item P-10600 has different production structures.

: Production Structures for item P-10600

If item P-10600 with variant LOS-A and location MAINWH is entered on a sales order and a production order is created for this line, the production structure PST-V-006 – with its Production BOM and Routing No. – is used to calculate the production order routing and components.

: Items with different criteria on a sales order

Production order lines are calculated using the different production structure variants and their specific BOM and routing information.

: Production Order for sales order with different item criteria
