Lot Splitting in Lot-/Serial No. Administration

If – for partial quantities of lots or trading units – data changes are done (such as changes of status, expiration date, entry date and so on) in the Lot-/Serial No. Administration page, the system will automatically assign a new number for the partial quantity.

The number assignment for data records containing lot-/trading unit numbers happens according to the logic in the following table.
