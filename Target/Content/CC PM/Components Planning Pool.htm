Components Planning Pool

The Components Planning Pool functionality is comparable to the Planning Pool. In contrast to the Planning Pool – which displays all Production Orders based on work centers and machine capacity – the Components Planning Pool is based on materials.

The Components Planning Pool page displays the availability of the components used in the BOM or Recipes of the corresponding production order. In the General section, various fields (Date Filter, Item Filter or Production Order Filters) are available to filter the component lines. In the Data Control section, set conditions for the production order components (refer to topic “Data Controls” on page 1).

: Components Planning Pool page with no filtering

This page can be opened in many ways:

  • From the Manufacturing menu by clicking Planning > Components Planning Pool.
  • From the action on the Home or Navigate tab on the Planning Pool page.
  • From the Lines FastTab (Line > Components Planning Pool) on the Sequence Planning page.
  • From the Lines FastTab (Line > Components Planning Pool) on all Production Order (Simulated, Planned, Firm Planned and Released) pages.
  • From the Components Available field on the lines of the Planning Pool, Sequence Planning and all Production Order (Simulated, Planned, Firm Planned and Released) pages (refer to topic “Component Availability” on page 1).
