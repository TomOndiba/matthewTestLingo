Configuration

In the By-Production page – accessed from the Product Design menu – the main item is entered on the General FastTab. The by-products with units of measure and quantities are added in the lines. The quantity and unit of the by-products always relate to the base unit of measure of the main item.

: By-production main item with two by-products

The fields on the General FastTab are described in the following table.

Field

Description

No.

Number of the by-production; can be entered manually or by using a number series. This number cannot be changed once it is entered.

Item No.

Item for which the items entered in the by-production lines are by-products.

Variant Code

Variant of this item; allows users to create one by-production header for each item-variant combination.

Description / Description 2

Descriptions of the by-production.

Unit of Measure Code

Unit of measure in which the main item is produced.

Status

The options are:

  • New – Choose this option for a new By-Production. With this status, the fields can be edited.
  • Certified – Choose this option for a certified By-Production; the fields cannot be edited in this status.
  • Under Development – Choose this option for a By-Production that is in development. With this status, the fields can be edited.
  • Closed – Choose this option for a By-Production that is no longer used.

Search Name

Description used to search for a by-production by name instead of the number.

Last Date Modified

Last date on which the by-production card was modified.

The Lines FastTab contains the co-products with the relevant unit of measure and quantity. The quantity and the unit of co-products are related to the unit of measure on the General FastTab.

In Fig. 1, the production process produces one base unit of measure of the main item P-953-KP001 and two by-products: 3 units of the item P-954-KP-10002 and 2 units of P-955-KP003.

The quantities in the lines can be entered using allocation quantity and allocation unit of measure. The relevant fields are described in the following table.

Field

Description

Quantity

Quantity of the by-production that occurs when the finished item (in the General FastTab) is manufactured.

Unit of Measure Code

Unit of measure for the by-production.

Output Quantity

Quantity of the by-production output based on the unit of measure.

Allocation Quantity

Quantity of the finished product based on the output quantity.

Alloc. Unit of Measure Code

Unit of measure of the finished product to which the reference quantity applies.
