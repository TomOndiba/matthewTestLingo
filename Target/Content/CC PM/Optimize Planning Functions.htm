Optimize Planning Functions in Sequence Planning

Two functions are available in the Sequence Planning page to assist with planning optimization:

  • Priority Planning
  • Setup Optimization

Each function is accessed from the Home or Actions tab on the ribbon.

: Optimize planning functions on the Sequence Planning page

Priority Planning

The “Priority Planning” function opens the Priority Sequence Planning page.

: Priority Sequence Planning page

This page displays all the default and setup values from the Planning Dimensions. In the header, enter the start date/time and due date for the sequence planning.

Select the Schedule Deferred Prod. Order Lines check box to calculate all the production order lines.

After clicking OK (and thus triggering the sequence planning process), the Sequence Planning lines are sorted based on the planning dimension priority and scheduled.

Setup Optimization

The “Setup Optimization” function deletes the setup time from the routing if a production order exists in the sequence with the same:

  • Machine
  • Item
  • Tool
  • Insert

For example, if there are three production order lines on the same sequence work center with the same item, tool and insert, only the first production order keeps the setup times. The setup times in the second and third lines are deleted.
