QC Answers

In the QC Answer Card page, standardized answers can be created that can later be used in the QC templates. Refer to online help for a description of the fields on this page.

: QC Answer Card that provides a check box indicating a positive answer

When answers are transferred to QC Template lines, the values in the Curvature Precision, Curvature Method, Lower Limit, Upper Limit and Positive Answer fields are also transferred.
