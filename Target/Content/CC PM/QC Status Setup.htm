QC Status Setup

On the Quality Control menu, click QC Status.

: QC Status setup page

The following fields are used in item tracking:

  • Standard Location Code – refer to topic “QC Status on Locations” on page 1.
  • Transfer
  • Sales
  • Consumption
  • Inventory Available – refer to topic “Inventory Availability by QC Status” on page 1.

The Transfer, Sales and Consumption fields do not prevent the actual posting; they are always used in conjunction with the Use As Transfer Location, Use As Sales Location and Use As Consumption Location fields in the Location table. Refer to topic “Status of Lot Numbers” on page 1 for more information.

In terms of consumption, an Item Ledger Entry with a QC Status where the Consumption field is not activated cannot be transferred to a Location where Use As Consumption Location is activated. The same applies accordingly to transfer and sales.

The QC Status provides organizational support. In order to prevent that a warning is shown that inventory should not have been used after the physical consumption has already been executed, inventory is transferred to a consumption location before it is consumed.

The remaining fields are used in validation (refer to topic “QC Status Validation” on page 1).
