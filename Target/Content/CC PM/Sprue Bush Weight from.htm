Sprue Bush Weight from Work Center Production Rate

Sprue Bush increases the expected quantity of Production Order components like scrap. It represents additional material needed for production. It is entered on Work Center Production Rates as weight related to the Prod. Rate Unit of Measure field and will be converted into a percentage in the Production Order components on calculation.

Example

In production structure PST-014, a Sprue Bush Weight has been defined in the Work Center Production Rate for work center M007.

: Empty setup and run times for work center M003

Note

Sprue Bush Weight is converted to Sprue Bush % in Planning Worksheet and Production Order calculation. The Prod. Rate Unit of Measure is KG; therefore, the Sprue Bush Weight of 0.1 will be converted to 10%.

When a sales order is generated for item P-10103, the demand can be processed in the Planning Worksheet.

In the Planning Components, the Expected Quantity is calculated to include the Sprue Bush Weight from the Work Center Production Rate. This is the same calculation that would be used if a production order was created directly from the sales line.

: Planning Components calculated with Sprue Bush Weight from Work Center Production Rate

The Expected Quantity is calculated as:

(Quantity from Planning Worksheet Line * Quantity per from Planning Component) * Sprue Bush %

Note

Modifications to the Planning Components do not include Sprue Bush calculations.

The expected quantities in the Planning Components are transferred to the production order.
