Sequence Planning with cc|graphical extension

cc|graphical extension – designed specifically for manufacturers – provides users with a graphical interface that is easy to use and tailored precisely to the standard production functionality. It contains all the functions of the production module and provides data in a visual presentation.

cc|process manufacturing includes the functionality to perform sequence planning with the cc|graphical extension. The GX Sequence Planning page is accessed from the navigation pane by clicking Manufacturing > Planning > GX Sequence Planning.

: GX Sequence Planning

The way in which sequence planning information is displayed in the GX Sequence Planning view is based on a defined scheme that is assigned in the GX Seq. Planning Scheme page, which is accessed on the cc|graphical extension menu.

For more information about setting up schemes and all other cc|graphical extension functionality, refer to the current “cc|graphical extension” documentation.
