Warning Management

For every line there are two fields for the warning management. The Include Warning check box indicates if there is a warning, but the line itself is not the cause of the warning. The Warning check box is selected on the line that caused the warning.

There are two methods of showing the warnings:

  • On the ribbon, click Included Warnings. Each warning in the structure is displayed.

: Planning Coordination Warning line for Order 1027

  • Expand the lines within the line that has the Included Warning check box selected.

: Expanded Planning Coordination lines for Order 1027

The Demand Date and Planned Date fields are transferred from the corresponding documents. If there is a deviation between those dates, (Planning Date > Demand Date) the Warning check box is selected. The Warning check box is also selected if there is an issue with item availability.
