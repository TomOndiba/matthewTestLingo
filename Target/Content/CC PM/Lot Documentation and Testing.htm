Lot Documentation and Testing

In the standard Microsoft Dynamics NAV manufacturing solution, lot numbers are displayed in all records and documents. In cc|process manufacturing, additional lot documentation has been created and a lot test system has been developed.

On the Item Tracking FastTab of the Item Card, the Lot QC Template No. field specifies the documentation or test.

: Lot QC Template on the Item Card

The Lot QC Template can also be specified on the Misc. FastTab of the Item Tracking Code Card.

: Lot QC Template on the Item Tracking Code Card

From either card, a QC Standard Status can be set up as the default.

When a production order is processed (consumption and output posted), the lot number appears in the Item Ledger Entries. Click the Lot No. field to open the Lot No. Information Card Adv. page. Depending on the QC template selected on the item or item tracking code, the relevant template opens. In Fig. 1, a lot test was assigned to item P-10500.

: Lot No. Information Card Adv. opened from the item ledger entries; QC Status set to BLOCKED, which was the default setting on the item and item tracking code

The Lines FastTab contains the lot test. Information can be entered here or a new test can be opened by clicking QC Test on the ribbon.

: New QC Test opened from the Lot No. Information Card Adv. page

In this page, the information can be printed by clicking Print on the ribbon. In addition, comments can be added to the individual lines by clicking Line > Comments on the Lines FastTab.

All records related to the QC Test can also be displayed from the QC Test page by clicking Navigate > QC Test Allocations.

: QC tests allocated to test QC-P-083

Lot Information Card List

A listing of all the Lot Information Cards can be accessed directly from the Quality Control menu by clicking Lot No. Information Card Adv. This list is useful when searching for lot- or trading unit numbers.

: Lot No. Information List Adv. page in view mode

Test Certification for Multiple Lots

The QC Test assigned to one lot number information record can be allocated to multiple lot number information records using one of the following functions available on the ribbon on the Lot No. Information Card Adv. page:

  • Transfer QC Test to Lot – allocates the actual lot number of the existing QC test to other lot number information records.
  • Get QC Test from Lot – uses the QC test of another lot number information record for the actual lot number information record.

 

: Lot No. Information Card Adv. page highlighting the functions

When using the functions, the Lot No. Information List Adv. page opens for record selection.

: Lot No. Information List Adv. page opened from the Transfer QC Test to Lot function

The selection process is different based on the function:

  • If the “Transfer QC Test to Lot” function is used, multiple records can be selected. 
  • If the “Get QC Test from Lot” function is used, one record can be selected.

Note

If a QC Test exists for a lot number information, one of the following messages will appear.

: Allocation cancellation messages for the Transfer QC Test to Lot and Get QC Test from Lot functions

The QC Test FastTab of the Lot No. Information Card Adv. page displays the allocated QC test number and the number of Lot No. Information records to which this QC Test has been allocated (refer to Fig. 7).

Change QC Status from Test Certification

The “QC Status Change” function allows users to change the QC Status field in the item ledger entries.

This function is available from the ribbon of the Lot No. Information Card. Adv. page and opens the QC Status Change page, which is filtered by the Lot No. Information record.

: QC Status Change page

Set additional filters and then use the “Find” function to narrow the search. To change the status, enter the new status in the New QC Status field and then click Change on the ribbon. Accept the message to change the status. The QC Status is changed on the Lot No. Information record and the item ledger entries.

: QC Status changed to CHECK on the lot number information and the item ledger entries

The “Batch QC Status Change” function can also be run from the QC Status Change page, which allows users to change the status for multiple items based on certain criteria (refer to topic “QC Status Maintenance” on page 1).
