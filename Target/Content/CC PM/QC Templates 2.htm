QC Templates

QC templates can be relevant at various points in cc|process manufacturing. For example, they might be used to gather the following data:

  • Vendor evaluation using the vendor card
  • Receipt control using the order lines
  • Shipment control using the order lines
  • Manufacturing process control

In any instance, a QC Template is defined and then assigned to a table, from where it can be accessed. For example, in cc|process manufacturing, this functionality is available on the Tools Work Center Card. Based on setup in the Process Manufacturing Setup page, a specific template is opened when the QC Test is run for a tool.

Since QC Tests differ by company, this functionality can be easily added to another table. Refer to your COSMO CONSULT consultant for more information about adding this functionality to other tables.

Setup and Usage

In the Template Toolkits field on the Process Manufacturing Setup page, the default QC template that should be used with toolkits is defined.

: Template Toolkits field highlighted on the Process Manufacturing Setup page

In the QC template, a link to a table can be created, which will open when a QC Test is run from that page.

: Opening the QC Template Allocations page

In the QC Template Allocations page, specify one or more tables where this template is to be used.

: Work Center set up to use the QC-V-001 template

With the setup in Fig. 3, when the “QC Test” function is initiated from the Tool Work Center Card, the QC-V-001 template is opened.

: QC-V-001 template opened from the QC Test action on the Tool Work Center Card

To print the QC Test, click Print on the ribbon of the QC Test page.

To add comments to a QC Test line, select the line and then click Line > Comments on the Lines FastTab. Enter comments in the Comment Sheet page and then click OK. The QC Comment field for the line will update to Yes (refer to the first line in Fig. 4).

For an overview of the QC test, click Navigate > QC Test Allocations.

: Overview of the QC test from the QC Test Allocations page

The QC Test Allocations page can also be opened from the Quality Control menu to display all QC Tests.
