Planning Extension Setup

The Planning Extension Setup page is opened from the Manufacturing menu, under Administration, by clicking Planning Extension Setup. This page provides the basic planning and maintenance settings for the process manufacturing module. The following sections explain the fields and the About action on this page.

General FastTab

The fields on the General FastTab are described in the following table.

: Planning Extension Setup page, General FastTab

Field

Description

Start Period Utiliz. Display

Default period of time that is shown when the Work Center Load page is opened.

Start Period Plan Table

Default period of time that is shown when the Plan Table Work Center page is opened.

Delimiter Plan Table

Character that is used as a delimiter for the plan table. Any character can be used.

No Plan. Flex. on Seq. Plan.

If selected, the Planning Flexibility field is set to None on the lines of the Sequence Planning page during scheduling (refer to topic “Planning Flexibility in Sequence Planning” on page 1).

Upd. Prod. Order Line Due Date

By default, the due dates of components are changed when a production order is moved in Sequence Planning.

If this check box is selected, and the components are supplied by another production order line, then the due dates of the supplying production order line are also changed.

Not selecting this check box could result in production order recalculation in MRP because the due dates are not changed.

Update Multilevel Prod. Orders

If a production order is created over several dependent production stages, the scheduling calculates each production stage independently.

If this check box is selected, dependent production order lines of a production order are updated according to schedule. For example, the ending date/time of lower production stages will be transferred to the Starting Date and Starting Time fields of higher production stages.

Omit Reversed Orders in Schedule

If selected, schedules reversed production order lines in Sequence Planning only if their Start Date/Time is before the End Date/Time of the last planned production order line.

No Pick for Reversed Prod. Order Lines

If selected, a warehouse pick is created for production order lines reversed in Sequence Planning (sequence position 999,999,999,999).

Customer Mfg. Dimension Code

Manufacturing Dimension of which its dimension values can be assigned to customers. These values can be used for priority planning.

Setup Hours per Day

Time per day, which is used as the base calculation for the setup time. The number of hours affects the necessary setup personnel.
