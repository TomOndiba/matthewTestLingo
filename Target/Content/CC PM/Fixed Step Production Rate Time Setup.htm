Fixed Step Production Rate Time Setup

The Fixed-Step Prod. Rate Time check box locks in the production rate time.

For example, 1500 pieces of item P-01007 are produced for a period of 60 minutes, namely:

  • 0 – 1500 PCS = 60 minutes
  • 1501 – 3000 PCS = 120 minutes
  • 3001 – 4500 PCS = 180 minutes and so on

: 1500 PCS Fixed Step Production Rate Time set up for item P_1007 and work center M015

When a production order is calculated, the setup time and run time are calculated for each quantity step defined in the Production Rate field, regardless of the exact production order quantity.

: Production order quantity of 2000; setup time is 30 minutes and run time is set to 60 minutes

In the routing, users can see if an operation has a Fixed-Step Prod. Rate Time and the assigned Fixed-Step Production Rate.
