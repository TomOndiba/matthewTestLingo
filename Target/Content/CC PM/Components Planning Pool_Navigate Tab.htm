Navigate Tab

The actions on the Navigate tab provide detailed information for the item in the component lines, such as dimensions, bin contents, where-used, item substitutions, and the availability of items by date, location or variant. These standard functionalities are also available in the Prod. Order Components page; refer to online help for more information.

Select Item Substitution

This function uses the standard Microsoft Dynamics NAV item substitution functionality to substitute a component with a valid substitution set up on the item or production structure line.
