Progress Status on Production Orders

The Last Processed Operation No. and Last Routing Status fields in the Production Order lines show the current operation status, i.e., the last operation that received a booking. This will provide information that an order has started without having to drill into Capacity Ledger Entries.

These fields can be shown in the lines of Released Production Orders and Sequence Planning pages.

: Progress status fields in Sequence Planning line

If the routing status of a routing line is changed manually, by output posting or by work/tool/insert center change in Sequence Planning, the routing operation number and the current routing status is transferred to the new production order line fields.
