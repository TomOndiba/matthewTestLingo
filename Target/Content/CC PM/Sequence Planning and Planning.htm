Sequence Planning and Planning Pool

Default manufacturing dimensions are shown in the Sequence Planning and Planning Pool pages.

: Manufacturing dimensions shown on Sequence Planning and Planning Pool pages

From either page, the default manufacturing dimensions of the selected work center can be shown by clicking Navigate > Manufacturing Dimensions (in the Work Ctr. group).

On the Planning Pool page, the default manufacturing dimensions of the selected production order line can be shown by clicking Navigate > Manufacturing Dimensions (in the Line group).

On the Sequence Planning page, the default manufacturing dimensions of the selected production order line can be shown from the Lines FastTab by clicking Line > Manufacturing Dimensions. All production order pages access the line manufacturing dimensions using this path.

Note

Any changes made to the production order lines are specific to the order; they do not transfer back to the item.
