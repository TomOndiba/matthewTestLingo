Production Orders

When a production order is created for an item that has ingredients, the ingredients of the item’s production BOM:

  • Header are copied to the production order line
  • Lines are copied to the production order components

: Production BOM header ingredients added to production order line

The quantity fields on the Prod. Order Line Ingredients page are described in the following table.

Field

Description

Expected Quantity (Ratio) / Unit of Measure Code (Ratio)

Ingredient values from the production BOM/recipe; can be changed for differing production amounts.

Alloc. Unit of Measure Code

Measurement in which the ingredient is to be produced; can be changed for differing production amounts.

Expected Quantity (Total) / Unit of Measure Code (Total)

Expected ingredients calculated by the production order line quantities in the indicated unit of measure.

Note

The Unit of Measure Code (Total) field can be changed for differing production amounts.

Expected Comp. Quantity (Ratio)

Current ingredient values of the components prior to any lot assignment or production posting.

Expected Comp. Quantity (Total)

Current ingredient values of the components calculated by the production order line quantities.

Captured Quantity (Ratio)

Ingredient values from the components after lot assignment.

Note

This field will not contain a value if a lot has not been assigned to the production order line. Additionally, this field is cleared after consumption/output posting.

Captured Quantity (Total)

Ingredient values of the components calculated by the production order line quantities after lot assignment.

Note

This field is cleared after consumption/output posting.

Posted Quantity (Ratio)

Ingredient values from the components after consumption/output posting.

Posted Quantity (Total)

Ingredient values of the components calculated by the production order line quantities after consumption/output posting.

Contains Expected Values

Selected if lots assigned to components contain the expected values – not measured values from a processed test order (refer to topic “Receipt Testing” on page 1) or direct inventory posting (refer to topic “Item Delivery” on page 1).

This setting is only related to captured or posted quantities.

To access the ingredients for a component, open the Production Order Components page and then click Navigate > Ingredients on the ribbon.

: Production BOM line ingredients added to production order component item P-60018

The quantity fields on the Prod. Order Line Ingredients page are described in the following table.

Field

Description

Expected Quantity (Ratio) / Unit of Measure Code (Ratio) / Alloc. Unit of Measure Code / Expected Quantity (Total) / Unit of Measure Code (Total)

Expected ingredients based on component quantities.

Captured Quantity (Ratio) / Captured Quantity (Total)

Captured ingredients based on Item Tracking Lines.

Posted Quantity (Ratio) / Posted Quantity (Total)

Posted ingredients based on Item Ledger Entries.

Contains Expected Values

Selected if lots assigned to components contain the expected values – not measured values from a processed test order (refer to topic “Receipt Testing” on page 1) or direct inventory posting (refer to topic “Item Delivery” on page 1).

If lots without measured values are assigned to the production order components, the Contains Expected Values check box is selected and the sum of the actual lot ingredient quantities is copied to the Captured Quantity (Ratio) and Captured Quantity (Total) fields on the Prod. Order Component Ingredients page.

: Prod. Order Component Ingredients updated by entering a lot for component item P-60018

On the Prod. Order Line Ingredients page, the Contains Expected Values check box and the Captured Quantity (Total) field is updated with the value from the components.

Note

The Captured Quantity (Ratio) field is updated when the lot is assigned to the production order line.
