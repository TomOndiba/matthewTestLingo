Exercise – Sequence Planning

Create and release a sales order for customer 50000 with the following three lines of item P-10002:

  • Planned Shipment Date February 5, 2019, 25 PCS
  • Planned Shipment Date February 10, 2019, 30 PCS
  • Planned Shipment Date February 28, 2019, 45 PCS

Release the sales order and then create the production orders directly from each sales line. Afterwards, test the Sequence Planning with different functions.

Create the Sales Order

Follow these steps to create the sales order:

  1. In the navigation pane, under Manufacturing, click Planning > Sales Orders.
  2. On the ribbon, click New.
  3. Press ENTER to insert the next number in the series.
  4. In the Customer field, enter 50000.
  5. On the Lines FastTab, in the Type field, enter Item.
  6. In the No. field, enter P-10002.
  7. In the Location Code field, enter MAINWH.
  8. In the Quantity field, type 25.
  9. In the Planned Shipment Date field, enter 02/05/19.
  10. On the next line, in the No. field, enter P-10002.
  11. In the Location Code field, enter MAINWH.
  12. In the Quantity field, type 30.
  13. In the Planned Shipment Date field, enter 02/10/19.
  14. On the next line, in the No. field, enter P-10002.
  15. In the Location Code field, enter MAINWH.
  16. In the Quantity field, type 45.
  17. In the Planned Shipment Date field, enter 02/28/19.
  18. On the ribbon, click Release.

Create the Production Orders from the Sales Order

Follow these steps to create the production order for each order:

  1. Select the first and third sales lines.
  2. On the Lines FastTab, click Order > Create Prod. Order.
  3. In the Create Order From Sales Line dialog box, click Yes to accept the default settings.
  4. Click OK to close the confirmation message.
  5. Select the second sales line.
  6. On the Lines FastTab, click Order > Create Prod. Order.
  7. In the Create Order From Sales Line dialog box, select the status of Released and then click Yes.
  8. Click OK to close the confirmation message.
  9. Click OK to close the Sales Order.

Review the Production Orders in Sequence Planning

Follow these steps to review the production orders in Sequence Planning:

  1. In the navigation pane, under Manufacturing, click Planning > Sequence Planning.
  2. In the General section, select the line for work center M002.
  3. Scroll to the bottom of the Lines FastTab and review the last three lines, which were created from the sales order.
  4. Test various functions.
