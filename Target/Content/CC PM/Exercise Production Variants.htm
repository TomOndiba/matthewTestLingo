Exercise – Production Variants

Create a new Production Variant for item P-10002 by creating a new Production Structure and copying the original one and updating the Production Variant FastTab with the following information:

  • Quantity lesser than 50 PCS should be produced on machine M005.
  • Setup time for machine M005 is 30 minutes but to produce a panel on it uses twice the amount of time (10 minutes) as machine M002.

Next, create a sales order to customer 10000 with the following lines:

  • 15 panels, planned delivery date February 2, 2019
  • 135 panels, planned delivery date March 2, 2019

The final step is to create a Production Order for the sales header and verify that the correct Production Structure is used on the lines.

Create the Production Variant

Follow these steps to set up the production variant for item P-10002:

  1. On the Product Design menu, click Production Structures.
  2. On the ribbon, click New.
  3. Press ENTER or TAB to insert a new number.
  4. In the Item No. field, enter P-10002.
  5. On the ribbon, click Actions > Copy > Production Structure.
  6. Select the production structure previously created for item P-10002 and then click OK.
  7. Expand the Production Variant FastTab.
  8. Select the Production Variant check box.
  9. In the Maximum Lot Size field, type 49.
  10. On the Lines FastTab, change the Work Center to M005.
  11. In the Setup Time field, type 30.
  12. In the Run Time field, type 10.
  13. Change the Status to Certified.
  14. Click OK to close the Production Structure.

Create the Sales Order

Follow these steps to create the sales order:

  1. In the navigation pane, under Manufacturing, click Planning > Sales Orders.
  2. On the ribbon, click New.
  3. Press ENTER or TAB to insert a new number.
  4. In the Sell-to Customer No. field, enter 10000.
  5. On the Lines FastTab, in the Type field, enter Item.
  6. In the No. field, enter P-10002.
  7. Clear the Location Code field.
  8. In the Quantity field, type 15.
  9. In the Planned Delivery Date field, enter 02/02/19.
  10. On the next line, in the No. field, enter P-10002.
  11. Clear the Location Code field.
  12. In the Quantity field, type 135.
  13. In the Planned Delivery Date field, enter 03/02/19.
  14. Release the order.

Create the Production Order and Verify Results

Follow these steps to set up the Production Order and verify the results:

  1. On the ribbon, click Actions > Planning.
  2. In the Sales Order Planning page, click Create Prod. Order.
  3. In the Create Order From Sales page, select Project Order as the Order Type and then click Yes.
  4. Click OK to close the confirmation message.
  5. In the Sales Order Planning page, click Order Tracking.
  6. On the ribbon, click Show.
  7. With the first line selected, click Line > Routing on the Lines FastTab. Confirm that machine M005 is used since less than 50 pieces are being produced.
  8. Escape back to the production order.
  9. Repeat step 7 for the second line, confirming that machine M002 was used since more than 50 pieces are being produced.
  10. Escape back to the production order.
  11. To further confirm this, add the Production Structure No. field to the Lines FastTab using the “Choose Columns” function and verify that the different production structures are used.
