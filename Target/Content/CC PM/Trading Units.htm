Trading Units

Different storage units can be tracked in cc|process manufacturing using a trading unit. On the Item Tracking FastTab of the Item Card, trading units are defined as either Item or Variant.

: Trading units on the Item Card

Select the relevant option in the Trading Unit per Item UOM field as follows:

  • <blank> – for standard items
  • Item – to use the item concept where a new item is created from the base item
  • Variant – to use the variant concept where a new variant is created for the base item

Both concepts automatically create items or variants from the item; thus eliminating the need to manually create new records.
