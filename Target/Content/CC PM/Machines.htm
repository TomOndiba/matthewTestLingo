Machines

All machines that need to be maintained are set up on Machine Cards.

General FastTab

This FastTab contains the fields that define the general features of the machine.

: Machine Card, General FastTab for Machine 001

The fields on the General FastTab are described in the following table.

Field

Description

No.

Unique identifier for the machine.

Fixed Asset No.

Machines that have been set up as Fixed Assets can be linked to the Machine Card by entering the number in this field. The Description, Serial No. and Search Description from the Fixed Asset Card default to the Machine Card.

Resource No.

Machines that have been set up as Resources can be linked to the Machine Card by entering the number in this field. The Description and Search Description from the Resource Card default to the Machine Card.

Description / Description 2

Descriptions of the machine.

Serial No.

Serial number of the machine.

Search Description

Machine description that can be used in search functions.

Blocked

If selected, prevents machines from appearing in maintenance orders.

Machine/Component

Defines the type of machine. The options are:

  • Main Machine
  • Sub Machine
  • Component

Main Machine No.

If the machine is a sub machine or component, this field links it to a main machine.

 

Technical Notes FastTab

The fields on this FastTab allow users to enter specific comments about the machine.

: Machine Card, Technical Notes FastTab for Machine 001

Manufacturing FastTab

The fields on this FastTab link the machine to a work center.

: Machine Card, Manufacturing FastTab for Machine 001

This is necessary if the maintenance contracts need to be included in sequence planning. Therefore, it is also necessary that the machine be linked to a maintenance job type. The setup and run times/units of measure are populated from the combination of work center/maintenance job type but can be changed. This information is transferred to the Maintenance Template.
