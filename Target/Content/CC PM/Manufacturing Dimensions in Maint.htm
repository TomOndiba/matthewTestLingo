Manufacturing Dimensions in Maintenance-/Setup Matrix

Maintenance-/Setup Orders can be inserted between production order lines if there is a change in the manufacturing dimensions between production order lines. Therefore, it is possible to enter a Maintenance-/Setup Matrix line with values in following fields:

  • From Mfg. Dimension Code
  • From Mfg. Dimension Value Code
  • To Mfg. Dimension Code
  • To Mfg. Dimension Value Code

Note

The values in the From Mfg. Dimension Code and To Mfg. Dimension Code fields must match.

A Maintenance-/Setup Matrix line is used only if both production order lines contain a value for the entered manufacturing dimension.
