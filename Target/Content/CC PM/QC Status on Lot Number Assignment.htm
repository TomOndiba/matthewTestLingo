QC Status on Lot Number Assignment

The QC Status assigned to a lot number is shown in the Lot No. List page, which is opened from the Item Tracking Lines page by clicking the AssistEdit button in the Lot No. field.

: QC Status on the Lot No. List page

The QC Status field is populated with the number sign (##########) if there are multiple item ledger entries for a lot number with different QC Statuses.
