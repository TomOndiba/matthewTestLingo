Production Structure Change in Production Order

Users can change the production structure manually on existing production orders.

Note

The Allow Product Design Change in Prod. Order Line check box must be selected in the Process Manufacturing Setup page to change the Production Structure No. on the production order line.

: Selecting a different Production Structure on production order line

After changing the production structure, the “Refresh Production Order” function must be run to recalculate the production order without deleting the production lines.

: Refresh Production Order function with Lines check box cleared
