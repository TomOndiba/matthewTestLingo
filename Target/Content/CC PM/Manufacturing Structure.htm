Manufacturing Structure

The manufacturing structure is the final stage of the R&D process for an item. In this stage, an item must be assigned and the raw materials must be determined. Typically, the number created for the production order from the laboratory structure is used. The manufacturing structure cannot be certified without a value in the Item No. field on the header.

If not created from a laboratory structure, a manufacturing structure can be created from the Research & Development menu by clicking Manufacturing Structure and then clicking New on the ribbon.

Existing production structures can be copied into the Manufacturing Structure by clicking Actions > Copy > Production Structure on the ribbon. Copied lines can be modified as needed.

When certifying the manufacturing structure, the production BOM/recipe and routing are created – if those check boxes were selected on the General FastTab.

Note

If there is no Production BOM No./Routing No. in the production structure, a new production BOM/routing is created regardless of whether the New Production BOM/New Routing check boxes are selected. If there is a Production BOM No./Routing No. in the production structure, a new Production BOM Version/Routing Version is created if the New Production BOM/New Routing check boxes are selected. If the New Production BOM/New Routing check boxes are cleared, the existing BOM is overwritten.

Create Item from Manufacturing Structure

If an item has not been created, it can be created from the Manufacturing Structure by clicking Actions > Create Item on the ribbon. The following message appears.

: Create item message

Click Yes and the Item Card for the new item appears.

: Item created from the manufacturing structure

After updating the item as needed, click OK to assign it to the manufacturing structure.
