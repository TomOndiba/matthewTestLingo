Capacity Calculation Groups

The capacity calculation groups relate to capacity in work centers.

: Capacity Calculation Groups

This information – as well as item calculation groups – is used to make a grouping of capacities within a routing and to represent this value in separate calculation lines.

In addition to the Machine Center Card and Work Center Card, capacity calculation groups can be entered in the Calculation Group Code field of the following cc|process manufacturing pages:

  • Sequence Work Center Card, Posting FastTab
  • Tool Work Center Card, Posting FastTab
  • Insert Work Center Card, Posting FastTab
