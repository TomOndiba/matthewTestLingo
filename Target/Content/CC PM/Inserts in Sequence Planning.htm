Inserts in Sequence Planning

The Sequence Planning offers the same functions for the inserts as offered for tools. Only the main inserts are displayed in the Sequence Planning page (refer to topic “Sequence Planning” on page 1).

Inserts can be changed on the Sequence Planning lines by selecting the line to change on the Lines FastTab and then clicking Functions > Change Insert. Select the new insert from the list that appears, click OK and then click Yes to assign the new insert.

The Sequence Planning and the production order routing lines are updated with the new insert.
