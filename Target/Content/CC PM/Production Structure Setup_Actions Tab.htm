Actions Tab

Click the Actions tab to access actions relating to the Production Structure matrix lines.

Copy

This function contains two options for copying existing data to a production structure:

  • Prod. BOM / Routing – copies lines from existing production BOM, recipes and routings into the current production structure.
  • Production Structure – copies existing production structure to the current production structure.

Create Item

This function allows users to create a new item with the information from the production structure. This function can be used only with the Production Structure Type of Manufacturing Structure that has no item number.

Create Prod. Order

This function allows users to create a production order from a production structure. A dialog appears, allowing the user to select the production order status.

: Create Prod. Order dialog

To create a production order, the production structure must be certified and the Lot Size field must contain a value. If the Production Structure Type field is set to Laboratory Structure, a production order can be created without a production order item (refer to topic “Research and Development” on page 1).

Show Prod. Order

This function displays production orders that were created from the production structure:

  • If only one production order was created, the corresponding production order page opens.
  • If multiple production orders were created, a list page appears from which a production order can be opened and viewed.
  • If no production orders were created, the following message appears.

: No production orders to show for the selected production structure

Check Recipe

This function performs two tasks prior to certifying the Production Structure:

  • Calculates the Share % field if values have been entered in the Share field.
  • Checks if the Share % total equals 100%.

Refer to topic “Check Recipe” on page 1 for more information.

Recipe Comparison

This function compares two different recipes and recognizes the variations (refer to topic “Recipe Comparison” on page 1).

Recipe Search

This function helps users locate recipes. Enter different materials and a range (minimum share % and maximum share %) to search (refer to topic “Recipe Search” on page 1).

Production Simulation

This function allows users to simulate the production structure in multiple steps, starting from the item, from which this function is opened. The quantity of the simulation is determined by the batch size (refer to topic “Production Structure Simulations” on page 1).

Production Structure Report

Click Print to generate the Production Structure report for the selected Production Structure.
