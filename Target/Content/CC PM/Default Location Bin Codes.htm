Default Location/Bin Codes

On the Production Structure lines, a default location- and bin code can be assigned in the Comp. Location Code and Comp. Bin Code fields. When the production structure is certified – and a recipe, recipe version, production BOM, or production BOM version is created – the default location- and bin code are transferred to the Comp. Location Code and Comp. Bin Code fields on the recipe/production BOM lines. This default location and bin is then transferred to the component lines when a production order is created.

Note

The Use Location and Bin Defaults check box must be selected in the Process Manufacturing Setup page to use this functionality.
