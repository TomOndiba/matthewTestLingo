Manufacturing Dimension Priority

All manufacturing dimensions can be assigned a Priority. Priorities are first set on the Manufacturing Dimension and then assigned to the Manufacturing Dimension Values.

Priorities are necessary for the planning. With priorities, planning is better optimized because it calculates the best sequence for the machine.

  • If Numeric Values are used, select Value Ascending or Value Descending in the Value Prioritization field (refer to topic “Manufacturing Dimensions with Numeric Values” on page 1).
  • If Manufacturing Dimension Values are used, the Priority can be specified in the Manufacturing Dimension Values page.
