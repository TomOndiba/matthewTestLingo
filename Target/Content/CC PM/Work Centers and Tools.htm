Work Centers and Tools

Tools can be assigned to specific work centers in the Sequence Work Center Card page.

: Tools assigned to the M003 sequence work center

With this setup, the assigned tools can be used with this machine, but other tools can be assigned to this machine in the sequence planning. To ensure that only these tools are used with this machine in sequence planning, the Check Tool Usage check box must be selected on the Scheduling FastTab on the Sequence Work Center Card.

: Check Tool Usage selected for sequence work center M003

With this setup, only the assigned tools can be selected with the “Change Tool” function in Sequence Planning (refer to topic “Change Tool” on page 1).

: Only assigned tools available for selection using the Change Tool function
