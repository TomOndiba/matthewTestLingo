EFB Calculation

EFB is a German calculation document that sometimes must be delivered with the sales quote document. It includes information about the calculation and the surcharges.

Surcharges to be included in the EFB Calculation are first defined in the EFB Surcharge page (refer to section “EFB Calculation” on page 1 for more information).

: EFB Surcharge page

Note

EFB supports only the five cost types shown in Fig. 54. Since there is no consolidation of "Other" and "Other 2", the "Other 2" cost type is not considered for EFB.

EFB Calculation Page

In the Job Card, click Navigate > EFB Calculation to start the calculation process.

 

: EFB Calculation opened from Job Card

In the EFB Calculation page, the Values for Unit Prices Wage FastTab displays that calculations of wages and surcharges and provides the ability to update values. The fields in this FastTab are described in the following table.

Field

Description

BoQ No.

Indicates the job number for which the EFB Calculation is to be performed.

Average Wage

Displays the average Unit Cost which is calculated based on Charging Wage minus Wage Surcharge minus Additional Wage Costs.

Additional Wage Costs

Specifies additional costs for wages.

Additional Wage Costs %

Displays the Costs % which is calculated based on Additional Wage Costs and Average Wage.

Non Wage Labour Costs

Specifies non wage labour costs which is used to reduce the Average Wage.

Calculation Wage

Displays the Calculation Wage which is calculated based on Charging Wage minus Wage Surcharge.

Calculation Wage Surcharge

Displays the Surcharge Amount which is calculated based on Charging Wage and Calculation Wage Surcharge %.

Calculation Wage Surcharge %

Displays the Wage Surcharge % based on EFB Surcharges for Cost Type wage.

Charging Wage

Specifies the wage sales price per hour.

The Surcharges for Single Costs of Part Efforts FastTab includes the surcharges set up in the EFB Surcharge page. New surcharges can be added and default values can be changed as needed for the specific BoQ Job.

The Quote Total Calculation FastTab provides statistical information about the Sales Quote calculation.

Add Values to EFB Calculation

In the Values for Unit Prices Wage FastTab, enter 40 in the Charging Wage field to specify the Sales Price per hour for the EFB Report.

Values in this FastTab and the Quote Total Calculation FastTab are calculated and updated as well.

: Updated values from entering Charging Wage

In the Non Wage Labour Costs field, enter 3.00 to specify these costs for the EFB Report.

The Average Wage is reduced by the amount entered in the Non Wage Labour Cost field.

: Non Wage Labour Cost added, reducing the Average Wage

In the Additional Wage Costs % field, enter 20.00 to specify these costs for the EFB Report.

The Average Wage and Additional Wage Costs % fields are recalculated.

: Additional Wage Cost % added, adjusting the Average Wage

Once all values are entered, click Update EFB Calculation to make sure all values are up to date. This function must be run if the EFB surcharges are changed in order to update all values.

To generate the report to send with the Sales Quote, click Report > EFB-Price 221 (1a - 311.a). In the EFB Price 221 dialog, click Print to print the report or Preview to view the report.
