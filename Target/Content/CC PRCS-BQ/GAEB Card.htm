GAEB Card

The GAEB Card contains the complete information from the imported GAEB file, as well as any basic Job-related information if imported into a Job. The fields and actions in the GAEB Card are described in this section.

General FastTab

The fields on the General FastTab are described in the following table.

Field

Description

No.

Unique identifier of the GAEB Manager.

Description

Short description of the GAEB Manager. If imported, the description is based on the GAEB file. If exported, the description is based on the job description.

Exchange Format

Indicates the GAEB format of the GAEB import or export file. The options are:

  • GAEB 90
  • GAEB 2000
  • GAEB XML

File Info Version

Indicates the version of the GAEB import or export file. The options are:

  • 90 – used for GAEB 90
  • 1.0, 1.1 and 1.2 – used for GAEB 2000
  • 3.0, 3.1 and 3.2 – used for GAEB XML

Data Exchange Phase

Indicates the Data Exchange Phase of the GAEB import or export file. The options are:

  • 83 Quote Calling
  • 84 Quote Bidding
  • 85 Alternative Bids
  • 86 Order Award
  • 81 Specifications
  • 82 Cost Estimate

Data Exchange Type

Indicates whether the exchange is customer-based or vendor-based.

BoQ Type

Indicates the type of the BoQ. The option are:

  • Short Text
  • Long Text

Bidder Comment Permitted

If selected, the bidder is permitted to comment.

Additional Offer Permitted

If selected, an alternative bid is permitted.

Contract Rules

Indicates if contract rules exist.

No. of BoQ Positions

Indicates the number of BoQ positions.

Project No.

Indicates the assigned Project No.

Job No.

Indicates the assigned Job No.

Purchase No.

Indicates the number of a Purchase Document that is assigned to the GAEB Manager.

In Use

Indicates whether this GAEB Manager is in use in a Job.

No. of Differences

Indicates the number of differences between the GAEB Manager and the BoQ Budget Calculation on the BoQ Position level.

 

Price Splitting FastTab

A maximum of four cost splits can be defined in the Price Splitting FastTab.

: GAEB Card, Price Splitting FastTab

The fields on this FastTab are described in the following table.

Field

Description

Split Costs

If selected, costs can be split for this file.

Selecting this check box prompts a message asking to refresh Line Amounts with Cost Type Values.

No. of Cost-Splits

Indicates how many cost splits are needed for this file. Entering 1–4 provides access to the four cost splits.

Cost Split 1 Description .. Cost Split 6 Description

Indicates the name of the Unit Price Part in the GAEB file.

Cost Split 1 Type .. Cost Split 6 Type

Indicates the allocation of the Cost Split Description from the GAEB file to the internal Cost Types. The options are:

  • Wages
  • Material
  • External Service
  • Equipment
  • Other
  • Other 2

 

BoQ Positions FastTab

The BoQ Positions FastTab contains all the positions (lines) in the GAEB file.

: GAEB Card, BoQ Positions

The fields on this FastTab are described in the following table.

Field

Description

Difference

If selected, the 84/86 comparison found a difference between the GAEB line and the Job line.

Action Message

Indicates the 84/86 comparison result. The options are:

  • Transfer Changes
  • Delete Line
  • Create Line

Accept Action Message

Used to accept the Action Message to transfer changes from GAEB line to Job line by running the “Carry Out Action Message” function.

Position No.

Indicates the ordinal number of the BoQ Position in a BoQ.

Position Type

Indicates the type of the BoQ Position. The options are:

  • Normal Position
  • BoQ-Normal Group
  • Notice
  • Req. Pos w/o Totals
  • Req. Pos with Totals
  • Base Position
  • Alternative Position
  • Reference Position
  • Repeated Position
  • Guiding Description
  • Labour Hour Work
  • Surcharge Pos.
  • Combination Pos.
  • Lump Sum Pos.
  • Executive Description
  • Block of exec. Description
  • Sub Description
  • Lot
  • BoQ-Basegroup
  • BoQ Alternate Group

Description

Indicates the 50 first characters of the Short Text of the BoQ Position.

Short Text

Indicates whether there is a Short Text or not.

Long Text

Indicates whether there is a Long Text or not.

Text Supplement Offerer

Indicates whether the Offerer has included supplemental text.

Text Supplement Bidder

Indicates whether the Bidder has included supplement text.

Quantity

Indicates the Quantity of the BoQ Position.

Unit of Measure

Indicates the Unit of Measure of the BoQ Position. This is case-sensitive.

Unit Price

Indicates the Unit Price of the BoQ Position.

Line Amount

Indicates the Total Price of the BoQ Position.

Sales Line Discount %

Indicates the Discount percentage of the BoQ Position.

Sales Line Discount Amount

Indicates the Discount Amount of the BoQ Position.

Job No.

Indicates to which Job the BoQ Position is allocated.

Unique No.

Indicates to which Job Line Unique No. the BoQ Position is allocated.

UP Part 1 Amount .. UP Part6 Amount

Indicates the Unit Price Part 1 to Unit Price Part 6.

GAEB Line No.

Indicates the automatically created Line No.

GAEB No.

Indicates the GAEB Manager No.

If the bid is accepted, the details are defined in the Placing FastTab.

: GAEB Card, Placing FastTab

Field

Description

Placing Type

Indicates the type of allocation for the BoQ. The options are:

  • Open Procedure
  • Restricted Procedure
  • Negotiation Procedure
  • Open Tendering
  • Restricted Tendering
  • Direct Award
  • Int. NATO-Tendering

Specific Placing Type

Indicates the special type of allocation for the BoQ. The options are:

  • Maintenance
  • Blanket Order
  • B.O. for Maintenance

Placing Currency

Indicates the allocation currency for the BoQ.

Placing Currency Description

Indicates the allocation currency description for the BoQ.

Quote Date

Indicates the date of the submitted quote.

Submission Date

Indicates the date of submission.

Submission Time

Indicates the time of submission.

Acceptance Deadline

Indicates the date until quotes are accepted.

Delivery Location

Indicates the location of the placing.

Estimated Start Execution

Indicates the estimated start date for execution.

Estimated End Execution

Indicates the estimated end date for execution.

Order No.

Indicates the order number of the customer.

Order Date

Indicates the date of the order.

Approval Type

Indicates the type of approval.

Approval Date

Indicates the date of approval.

Period of Warranty

Indicates the period of warranty, which includes only decimal numbers.

Unit of Measure Warranty

Indicates the unit of measure of the warranty period.

End Warranty

Administration FastTab

The Administration FastTab displays date, user and file import and export information.

: GAEB Card, Administration FastTab

Navigate Tab

The actions on the Navigate tab are described in the following table.

Action

Description

GAEB Unit of Measure Assignment

Opens the GAEB Unit of Measure Assignment page, which displays all assigned and missing units of measure between the GAEB file and Microsoft Dynamics NAV.

Job Description

Opens the GAEB Texts - Job Description page, which shows the job description of the BoQ.

Contract Rules Top

Opens the GAEB Texts - Contract Rules Top page, which shows the header contract rules.

Contract Rules Bottom

Opens the GAEB Texts - Contract Rules Bottom page, which shows the footer contract rules.

 

Actions Tab

The actions on the Actions tab are described in the following table.

Action

Description

Import GAEB

Imports the GAEB File into the GAEB List (refer to section “Import GAEB File” on page 1 for more information).

Export GAEB

Exports the current iteration of the GAEB Card into a GAEB file (refer to section “Export GAEB File” on page 1 for more information).

Copy GAEB

Creates a new GAEB record from a copy of the current GAEB Card.

Calculate 84/86 Comparison

Compares the position number, position type, quantities and prices from the customer’s d/p/x86 file (Order Award) or bidder's d/p/x84 file (Quote Bid) to the calculated BoQ Budget Calculation

Carry Out Action Message

Processes any action messages to be carried out in the BoQ Positions FastTab.
