Submission Ranking

The results of all vendors who submitted a bid on a project can be entered in the Submission Ranking page for statistical purposes. If the project was a public BoQ, the submission ranking is posted by a specific date so all vendors know the status of their bids.

In the Job Card, click Navigate > Submission Ranking.

: Submission Ranking page

The fields in this page are described in the following table.

Field

Description

Contact No.

Specifies the bidder for the project. User-defined text can be entered or existing contacts can be selected using the lookup

Gross Quotation Value

Specifies the gross value of the submitted quote.

Discount %

Specifies the discount percentage offered for the submitted quote.

Discount Amount

Specifies a discount amount offered for the submitted quote.

Net Quotation Value

Displays the calculated net value of the submitted quote.

Ranking

Specifies the ranking of the submitted quote, with 1 being the winner.

As illustrated in Fig. 2, five vendors bid on the BoQ Job. One existing contact was selected from the lookup and the others were manually entered in the Contact No. field. Although not the lowest price, CRONUS won the bid due to a variety of reasons, such as quality and reputation.

: Submission Ranking with submitted bids for the BoQ Job, highlighting the winning bid
