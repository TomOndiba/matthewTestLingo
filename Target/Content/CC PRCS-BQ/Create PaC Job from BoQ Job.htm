Create PaC Job from BoQ Job

Creating PaC Jobs from the BoQ Job transfers the BoQ positions to the PaC Job. This is beneficial if actual items and/or services have been included in the BoQ Job so manual entry is reduced.

On the Job Card, click Actions > Create PaC Job. A message appears asking to create calculation group totals. Since calculations have been done in the BoQ Job, click Yes.

: Creating PaC Job message to create calculation group totals

The PaC Job is created with a new job number and Structure Type = Process and Controlling.

: PaC Job

Refer to the section “Job Structure Type = Process and Controlling (PaC)” on page 1 for more information about the PaC-specific fields and functions on the Job Card.
