Detailed Calculation

Detailed calculations are handled through the BoQ Budget Calculation page, which is essentially the same as the Job Budget Calculation page but with additional functions for Construction Works Processing. This page is opened from the Job Card by clicking Budget Calculation or from the BoQ Position List by clicking BoQ Budget Calculation.

: BoQ Budget Calculation

BoQ Budget Calculation Page

Many of the fields on the BoQ Budget Calculation page are the same as fields in the BoQ Position List. Therefore, only the fields that are specific to the BoQ Calculation are described in the following table.

Field

Description

Allocated Quantity to PaC

Indicates the quantity assigned to a PaC Job.

Allocated Sales Line Amount to PaC

Indicates the Sales Line Amount assigned to a PaC Job.

Allocated Total Cost to PaC

Indicates the Total Cost assigned to a PaC Job.

Allocation Status

Indicates the status of allocation. The options are:

  • None – used if the allocation is completely removed.
  • Partially – used if the complete quantity is not assigned.
  • Complete – used if the complete quantity is assigned.

Amendment Type

Indicates the type of amendment. The options are:

  • Amendment
  • Decrease
  • Increase

 

Assigned PaC Job No.

Indicates the Pac Job which is assigned to the Job Line.

Exp. Alloy Price (p. B. U.) FF

Indicates the expected Alloy Price per Base Unit of Measure, which is calculated based on the selected item and the alloy quotation.

Add Items and Services in BoQ Budget Calculation

As previously mentioned, any data entered in the Fast Calculation and/or updated in the BoQ Position List is transferred to the BoQ Budget Calculation.

: BoQ Budget Calculation with data from Fast Calculation

Fig. 2 shows that the entries made for wages and materials in the Fast Calculation for positions 3. 1. 10. and 3. 1. 20. have been created as separate general component lines in the BoQ Budget Calculation.

Actual services and items that are known during the quoting process can be entered in the BoQ Budget Calculation. These positions can then be copied to other positions in the calculation with the “Copy Calculation” function. Any services/items entered in the BoQ Budget Calculation will transfer to the PaC Job, which reduces data entry during the processing phase.

New lines are created in the BoQ Budget Calculation just like in the Job Budget Calculation: select the position to add a line and then click New Level. In the Type field, enter Item or Service, in the No. field, enter the relevant Item or Resource, update quantities as needed and then enter any costs.

: Item added to BoQ Budget Calculation

Add another line by clicking New Position and then filling in the Type, No. , quantities and cost fields.

: Service added to the BoQ Budget Calculation

Scroll to the Calculation Status field and note that the status on the position is set to Progress.

: Calculation Status on position with new lines

With the position selected, click Change Calculation Status to set this position as Calculated.

: Changing calculation status to Calculated on position with new lines

Copy Calculations

Calculations that have been added to positions can be copied to other positions in the BoQ Budget Calculation. This is useful when the same items and/or services are used in multiple positions.

With the position line selected, click Actions > Copy Calculation. The BoQ Position List opens for selection of position(s) to copy the calculations.

: BoQ Position List opened from Copy Calculation function

In the BoQ Position List, select the position(s) in which to copy the calculations from the position selected in the BoQ Budget Calculation, and then click OK.

: Selected positions to copy calculations from budget lines

In the BoQ Budget Calculation, the positions selected in the BoQ Position List are updated with the calculations copied from the originating budget position.

: BoQ Budget Calculation positions with copied calculations

Copy Positions

Similar to the “Copy Calculation” function, one position can be copied to another position using the “Copy to” and “Paste from” functions.

In the BoQ Budget Calculation, select the position to be copied and then click Actions > Copy to.

: Position to be copied to another position

Select the position where to copy the position and then on the Actions tab, click Create Structure > New Level.

: New level for pasting position

Select the new level position and then on the Actions tab, click Paste from. The copied position is pasted below the selected position.

: Copied position pasted to new level

In the empty line above, a line for, e.g., an item can be added.

: Item added to the position

Change Wage Group

Wage Groups can be changed on positions using the Wage Group Change function, which swaps service components and work types or changes the average calculation wage for a title, subtitle, position or the complete BoQ.

A new service calculation is added to a position in the BoQ Budget Calculation.

: Service added to position

All new entries that have not been set with a calculation status of Calculated are selected and the “Change Calculation Status” function is run. Once finished, click Actions > Wage Group Change to open the Wage Group Change page.

: Wage Group Change page

The fields in the Wage Group Change page are described in the following table.

Field

Description

BoQ Pos. No.

Specifies the BoQ Position No. to be included in the Wage Group change.

Complete BoQ

If selected, all positions in the BoQ are to be included in the Wage Group change.

Calculate Fixed BoQ Positions

If selected, BoQ Positions that are set as Fixed will be included in the Wage Group change.

Extra Charge %

Specifies how to update the Extra Charge % field in the BoQ Budget Calculation. The options are:

  • Keep (Calculate Unit Price)
  • Update (Keep Unit Price)
  • Find (Calculate Unit Price)

Wage Hours

Displays the current number of wage hours for the selected position(s).

Wage Cost

Displays the current wage cost for the selected position(s).

Avg. Wage (h)

Displays the average wage (in hours) for the selected position(s).

Avg. Wage (h) (New)

Specifies the new average wage (in hours) to be used for the wage change.

Calc. Res. No.

Specifies the Calculation Resource to be changed.

Calc. Res. No. (New)

Specifies the Calculation Resource to replace the current Calculation Resource during the Wage Group change.

Work Type (New)

Specifies the new Work Type to be added to the Calculation Resource during the Wage Group change, if necessary.

In the BoQ Position No. field, the main position for the new calculations is selected from the BoQ Position List; the Complete BoQ check box is automatically cleared.

: BoQ Position specified in the Wage Group Change page

In the Extra Charge % field, the Find (Calculate Unit Price) option is selected to update the Extra Charge % field from the Job Calculation Group page, which recalculates the Unit Price based on Unit Cost + the new Extra Charge %.

In the Calc. Res. No. field, the service to change is selected from the Resource List, which is filtered by only the resources included within the BoQ Position.

: Selecting the Calculation Resource to change

In the Calc. Res. No. (New) field, the new resource is selected from the Resource List, which now displays all service resources.

: Selecting the new Calculation Resource to replace existing resource

For this change, no new work type is selected.

: Wage Group Change ready to process

In the Wage Group Change page, click OK. The resource for position 3. 2. 70. is changed from S005 to S007.

: Wage Group changed for position 3. 2. 70. from S005 to S007

Create Sales Quote

Now that calculations have been added to some positions in the BoQ Budget Calculation, a Sales Quote can be created.

On the Home tab, click Create Sales Quote.

: Creating a sales quote from the BoQ Budget Calculation

In the dialog, select All positions without components to omit internal calculation components in the Sales Quote, and then click OK.

A message appears asking to budget all sales positions; click No to have the ability to use the "Wage Group Change" or "Vernier Adjustment" functions later in the calculation process.

: Message to budget sales positions

The Sales Quote opens.

: Sales Quote created from the BoQ Budget Calculation

Close the Sales Quote to return to the BoQ Budget Calculation.

View Calculation Statistics and Edit Vernier Adjustments

In the BoQ Budget Calculation page, select a position and then click Navigate > Statistic. The Calculation Statistics page opens, displaying total values as well as values for the selected position.

: Calculation Statistics for complete job and a sub-position

Fields in the General FastTab provide detailed information about total values and position values.

: Flow- and DropDown Fields in General FastTab

Each is described in the following table.

Field

Description

BoQ Positions (Open/In Progress)

Displays the number of BoQ positions for the total job budget and/or for the selected position that have a Calculation Status = Empty or Progress.

BoQ Positions (Calculated/Fixed)

Displays the number of BoQ positions for the total job budget and/or for the selected position that have a Calculation Status = Calculated or Fixed.

BoQ Positions (Fixed)

Displays the number of BoQ positions for the total job budget and/or for the selected position that have a Calculation Status = Fixed.

Wage Hours Calculated

Displays the number of wage hours for the total job budget and/or for the selected position.

Wage Days Calculated

Displays the number of wage days for the total job budget and/or for the selected position.

Average Wage

Displays the average wage for the total job budget and/or for the selected position.

Calculation Group Filter Total / Calculation Group Filter Position

Filters the values for the total job budget and/or selected position by a Calculation Group.

 

: Calculation Group Wages opened for Total Wage Hours Calculated

On the ribbon of the Calculation Statistics page, click Init Preview. This transfers preview values to the Total (Preview) and Position (Preview) FastTabs, which had no values until the function was run.

: Populated Total (Preview) and Position (Preview) FastTabs

Additionally, the “Edit Vernier Adjustment Total”, “Edit Vernier Adjustment Position” and “Transfer Preview” functions become available on the ribbon after the “Init Preview” function is run.

Click Edit Vernier Adjustment Total to make temporary changes in the Calculation Statistics, not directly in the BoQ Budget Calculation. This page is used for trying different calculations without affecting the actual budget calculation. New calculations that are beneficial to the bid can be transferred to the BoQ Budget Calculation.

: Vernier Adjustment Edit Values

The Vernier Adjustment Edit Values page contains basically the same information in all FastTabs, with the Total (External) FastTab containing a few extra fields. This page provides the ability to change average Line Discount %, Line Discount Amount and Target Price for the entire job, or average Extra Charge % and Line Discount % on individual components (Wages, Materials, External Service, Equipment, Others) for the entire job.

For example, to change the average Extra Charge % for Wages from 20.00 to 23.00, enter 23 in the Extra Charge % (Average) field in the Wages FastTab. Leaving the field prompts a message regarding transferring values to positions.

: Changing average Extra Charge % for Wages

Click Yes to transfer the changes or No to stop the process. If Yes is selected, values are updated in the Vernier Adjustment Edit Values page.

: Updated values after Extra Charge % change for wages

Close the Vernier Adjustment Edit Values page to return to the Calculation Statistics page, which has been updated with changes.

Note

Vernier Adjustments can be made for specific positions using the same process with the “Edit Vernier Adjustment Position” function.

In the Calculation Statistics page, click Transfer Values to transfer all changes to the BoQ Budget Calculation.

: Values updated in Calculation Statistics

A message appears asking to transfer the preview values. Click Yes to transfer the values to the BoQ Budget Calculation, click No to cancel the transfer.

The Total (Preview) and Position (Preview) FastTabs are cleared.

: Cleared preview FastTabs in Calculation Statistics

Values in the Total and Position FastTabs are updated with information from the BoQ Budget Calculation.

: Updated values in Total and Position FastTabs

Note

To restart the Vernier Adjustment process, click Refresh Statistics.

Escape back to the BoQ Budget Calculation page.

: BoQ Budget Calculation with Vernier adjustments
