Invoicing Template

There must be a possibility to reduce the invoices with share in costs like construction site power, construction site water, construction site insurances and different kinds of deductions, e.g. security deposits. For this case, an Invoice Schedule is provided.

: CS Invoicing Template for a basic invoice

Refer to section “CS Invoicing Templates” on page 1 for more information about the fields on this page.

A template can be assigned to Job so that a standard calculation can be used on invoicing processes. For example, a customer agrees to a specific percentage of insurance to be applied to every invoice. Once this template has been set up on the Job, the “Get CS Invoicing Calc. from Job” function can be run from the Sales Order/Sales Invoice to open this template with the specific charges.

On the BoQ Job Card, click Navigate > Get Template. The CS Invoicing Templates page opens.

: Getting CS Invoicing Template

Select the correct template and then click OK. The Invoicing page opens.

: Invoicing page opened from CS Invoicing Template

The fields in the Invoicing page are mostly the same as the ones in the CS Invoicing Template page, except the Invoicing page also has references to the originating document in the header and values in the lines.

In the Totaling fields, enter values for the Charge percentages, which can be used to calculate the charge amount for decrease of the invoice amount.

: Values added for Charge percentages

When all values are entered, change the Status to Released to make the template available for invoicing.

: Invoicing released
