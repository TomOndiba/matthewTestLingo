Base Calculation

Before creating the Sales Order, the last step must sometimes be to create/update the base calculation records. These will be given to a solicitor for administration. A report is currently not provided through cc|project.

The base calculation will be opened for clarification only if any price conflict occurs between the customer and the contractor during the building phase.

Note

The EFB Calculation must be created before creating the Base Calculation.

In the Jobs list, select the relevant job and then click Actions > Update Base Calculation (or in the Job Card, click Actions > Calculate Rate Unit Cost).

: Updating Base Calculation for BoQ Job

When the update is complete, open the BoQ Budget Calculation page. Select a row to view the base calculation and then click Navigate > Rate Unit Cost.

: Accessing the Base Calculation

The Base Calculation Card opens for the selected position, displaying quoted sales price values, surcharges and unit prices for components (Wage, Material, Ext. Service, Equipment and Other).
