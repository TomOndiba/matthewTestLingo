Processing

This topic describes how alloy surcharges are processed in sales and purchase documents.

Alloy Management for Sales

The calculation of the expected alloy amount is performed in the Sales Order. Once an item with alloy specifications is entered in the sales order line, click Lines > Alloy Surcharge Lines in the Lines FastTab to open the Alloy Surcharges Lines page.

: Alloy Surcharges Lines on sales order line

The fields in the Alloy Surcharges Lines page are described in the following table.

Field

Description

Alloy Type Code

Alloy Type assigned to the item.

Alloy Weight (per Base Unit)

Defaults from the Alloy Condition assigned to this Alloy Type for this customer.

Total Alloy Weight

Calculated by the Quantity in the sales line + Alloy Weight (per Base Unit).

Exp. Alloy Amount

Automatically calculated using the following formula:

(Alloy Quotation – Alloy Base) / 100 * Total Alloy Weight in KG

If the above fields are not pre-populated with data, this field is not automatically calculated.

Alloy Base

Defaults from the Alloy Condition assigned to this Alloy Type for this customer.

Alloy Quotation

Defaults from the Alloy Quotation of the Alloy Type, if the sales Order Date matches a date in the Alloy Quotation. Otherwise, this field is blank.

Example

The CU alloy type has quotation values for three days in July 2019, one of those days is July 27, which is the order date on the Sales Order in Fig. 1.

The Average Quotation value is used in Fig. 1 because the Alloy Condition MID DAY, which uses an average quotation base, was assigned to the CU alloy type for this customer.

Fix Values

If selected, the calculation of the Exp. Alloy Price and Exp. Alloy Amount is different:

(Exp. Alloy Price (per B. Unit) * Sales Line.Quantity (Base)) / Total Alloy Weight

 

Sales Invoicing

When sales invoices are processed using the “Get Shipment Lines” function, the alloy surcharge is added as an item charge and assigned to the item.

: Alloy surcharge included as Item Charge in Sales Invoice line using Get Shipment Lines function

Alloy Management for Purchases

The process of purchasing items with alloy surcharges is generally the same as for selling items with alloy surcharges, except the Expected Alloy Surcharge amount is visible from the Purchase Order line.

The calculation of the expected alloy amount is performed in the Purchase Order. Once an item with alloy specifications is entered in the purchase order line, click Lines > Alloy Surcharge Lines in the Lines FastTab to open the Alloy Surcharges Lines page.

: Alloy Surcharges Lines on purchase order line

The fields in the Alloy Surcharges Lines page are described in the following table.

Field

Description

Alloy Type Code

Alloy Type assigned to the item.

Alloy Weight (per Base Unit)

Defaults from the Alloy Condition assigned to this Alloy Type for this vendor.

Total Alloy Weight

Calculated by the Quantity in the purchase line + Alloy Weight (per Base Unit).

Exp. Alloy Amount

Automatically calculated using the following formula:

(Alloy Quotation – Alloy Base) / 100 * Total Alloy Weight in KG

If the above fields are not pre-populated with data, this field is not automatically calculated.

Alloy Base

Defaults from the Alloy Condition assigned to this Alloy Type for this vendor.

Alloy Quotation

Defaults from the Alloy Quotation of the Alloy Type, if the purchase Order Date matches a date in the Alloy Quotation. Otherwise, this field is blank.

Example

The CU alloy type has quotation values for three days in July 2019, one of those days is July 27, which is the order date on the Purchase Order in Fig. 3.

The Average Quotation value is used in Fig. 3 because the Alloy Condition MID DAY, which uses an average quotation base, was assigned to the CU alloy type for this vendor.

Fix Values

If selected, the calculation of the Exp. Alloy Price and Exp. Alloy Amount is different:

(Exp. Alloy Price (per B. Unit) * Purchase Line.Quantity (Base)) / Total Alloy Weight

 

Purchase Invoicing

When purchase invoices are processed using the “Get Receipt Lines” function, the alloy surcharge is added as an item charge and assigned to the item.
