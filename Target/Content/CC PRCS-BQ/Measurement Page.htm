Measurement Page

Measurements are recorded in the Measurement page, accessed from the cc|project menu by clicking Construction > Processing > Measurement Recording. Click New to create a new card or Edit to review an existing measurement recording.

: New Measurement page

Note

Measurement Documents can also be created directly from a Sales Order.

The fields in the General FastTab are described in the following table.

Field

Description

No.

Unique identifier for the Measurement Recording.

Status

Indicates the current status of the Measurement Recording. The options are:

  • Recorded – The measurement recording is in progress and can be edited.
  • Registered – The measurement recording has been registered and cannot be edited.

Document Date

Specifies the date the Measurement Recording was created.

Job No.

Specifies the job to which the Measurement Recording applies.

Description

Displays the description of the selected Job No.

Person Responsible

Specifies the user who is responsible for the Measurement Recording; defaults from the Job if one exists.

Measurement Date

Specifies the date of the Measurement Recording.

Measurement Page No.

Specifies the page number of the Measurement Recording.

Measurement Level

Specifies the level of the Measurement Recording.

Measurement Location

Specifies the location of the Measurement Recording.

User ID

Indicates the user who created the Measurement Recording.

Amount (LCY)

Displays the total amount (in local currency) of the Measurement Lines.

The fields in the Lines FastTab are described in the following table.

Field

Description

Measurement Date

Specifies the date of the Measurement Recording line; can be defaulted from the General FastTab.

Measurement Page No.

Specifies the page number of the Measurement Recording line; can be defaulted from the General FastTab.

Job No.

Specifies the job to which the Measurement Recording line applies. This field is defaulted from the General FastTab.

BoQ Position No.

Specifies the BoQ Position to which the measurement is recorded.

BoQ Quantity

Indicates the quantity of the BoQ Position.

BoQ Description

Indicates the description of the BoQ Position.

Quantity

Specifies the quantity to be measured.

Cum. Quantity

Specifies the cumulative quantity to be measured.

Cum. Registered Quantity

Displays the registered measurement quantity.

Unit Price

Displays the Unit Price of the BoQ Position based on Sales Order or BoQ Position, depending on the setting in the Jobs Setup page.

Line Amount

Displays the calculated Line Amount of the measurement line.

Line Discount %

Displays the Line Discount % of the BoQ Position based on Sales Order or BoQ Position, depending on the setting in the Jobs Setup page.

Line Discount Amount

Displays the calculated Line Discount Amount of the measurement line.

Measurement Level

Specifies the level of the measurement recording line; can be defaulted from the General FastTab.

Measurement Location

Specifies the location of the measurement recording line; can be defaulted from the General FastTab.

Unique No.

Indicates the unique identifier of the BoQ Position.

The actions on the Navigate tab are described in the following table.

Action

Description

Measurement List

Opens the Measurement List page, which displays the BoQ that contains recorded and registered measurements.

Position Card

Opens the Position Card page for the selected measurement recording line.

This action is also available from the Lines FastTab (Line > Position Card).

The actions on the Actions tab are described in the following table.

Action

Description

Register

Changes the measurement status from Recorded to Registered.

Assume BoQ Quantity

Transfers the BoQ Quantity to the Quantity field in the measurement lines.

Reset Quantity

Clears the Quantity field in the measurement lines.

Get BoQ Positions

Creates measurement lines based on the complete BoQ.

Delete Zero Quantity Lines

Deletes all measurement lines that have an empty Quantity field.

Print

Opens the Measurement Report that is defined in the Jobs Setup page.
