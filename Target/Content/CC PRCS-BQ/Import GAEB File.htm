Import GAEB File

The process for importing a GAEB File into the GAEB List or a Job is the same except for where to begin the process:

  • GAEB List – on the navigation pane, click Departments >cc|project> Construction > GAEB > GAEB List > click Import GAEB on the ribbon
  • Job – on the navigation pane, click Departments >cc|project> Construction > Processing > Job List > select a job with Structure Type = Bill of Quantities > click Import GAEB on the ribbon

Note

The “Import GAEB” function is only available for jobs with Structure Type = Bill of Quantities.

During the import process, a GAEB record is created in the GAEB List and if importing into the job, created in the BoQ Budget Calculation.

When the import completes, a message appears stating that the import was successful and references the GAEB Manager No.

: GAEB File import process complete

Click OK to continue the process.

When importing into a Job (not into the GAEB List), if units of measure in the GAEB file do not exist in the GAEB Unit of Measure Assign page, a message appears asking to complete the GAEB UoM Allocation.

: : Message to complete missing GAEB UoM Allocation

Clicking Yes opens the GAEB Unit of Measure Assign page, which is the same page located in the Construction, Administration menu, except it lists only the units of measure from the GAEB file with the Unit of Measure Code field blank for the missing units.

: GAEB Unit of Measure Assignment page opened during GAEB file import for missing UOM Allocation; PCS entered in the Unit of Measure Code field for the missing UOM “St”

In the Unit of Measure Code field, enter the unit that corresponds to the unit in the GAEB file.

Note

If a unit of measure is not entered during the import process, the Unit of Measure Code field will be empty in the related line(s) in the BoQ Budget Calculation.

After clicking OK in either the successful import message or the GAEB Unit of Measure Assign page, the process continues and then another message appears asking to open the GAEB Card for the imported record.

: Open GAEB Card message

Clicking Yes opens the related GAEB Card; clicking No completes the import process.

: GAEB Card opened after file import

The BoQ Budget Calculation page is described in section “Detailed Calculation” on page 1.
