Order Award and Comparison

When the customer awards the order to the vendor, they send a GAEB file for phase 86, Order Award, which is imported into the BoQ Job. However, it is common in the Construction Industry that most of the returned 86 files do not include the same positions and quantities as the sent 84 files. Therefore, Construction Works Processing provides a tool to compare the positions, quantities and prices in the 86 file to the calculated 84 file.

In the BoQ Job, import the 86 file received from the customer.

: Importing Order Award (Phase 86) GAEB File into BoQ Job

A message appears stating that the import was successful.

: Import confirmation

Click OK. Another message appears stating that there are existing entries in the Job Line Budget table.

: Message regarding existing Job Line Budget entries

Click OK. Another message appears asking to open the imported GAEB file.

: Message to open imported GAEB file

Click Yes to open the GAEB Card.

: Imported 86 GAEB file

Click Calculate 84/86 Comparison on the ribbon.

: GAEB Card in Comparison view

The information in the GAEB Card now changes, with all positions with differences noted by the selected Difference check box, the Action Message field indicates that the difference can be transferred, and now the information in the Job Line Budget Details FactBox shows the values from the Job Line Budget, which was not available before starting the comparison tool.

For example, in the first row where values were entered in the Fast Calculation, the 86 file shows the value as 172, but the Job Line Budget Details FactBox shows 166: the value entered in the 84 file.

: Reviewing differences between 84/86 files

It is likely that further negotiations will occur between vendor and customer, which means GAEB files could be sent/received multiple times. Ultimately, the Unit Prices from the 86 file should be transferred to the Unit Price field of the BoQ Budget Calculation, as well as the Quantities and BoQ Position Types.

Once negotiations have completed, the differences can be transferred from the 86 file to the BoQ Budget Calculation using the “Carry Out Action Message” function. Select the Accept Action Message check box on the position and then click Carry Out Action Message on the ribbon.

: Carry Out Action Message function on ribbon

In cases where unit price splitting is used, a message appears stating that existing component lines will be replaced with new lines.

: Component replacement message while accepting action messages

Clicking Yes deletes the calculated components, which is usually not the desired result, so the message is not confirmed by clicking No.

When this message is declined, an error message occurs, similar to the following:

: Error message from declined component line replacement

Since the action message was not accepted for this position, the differences between the 86 file and BoQ Budget Calculation must be updated manually.

When an action message has been accepted, the Difference check box and Action Message field are cleared, and the change is visible in the Job Line Budget FactBox.

: Accepted action message on position, updated pricing in Job Line Budget FactBox

When the comparison is complete, click OK to close the GAEB file.

In the BoQ Budget Calculation, transferred pricing from the accepted actions are visible.

: Transferred pricing from GAEB 84/86 comparison tool to BoQ Budget Calculation

In the Job Card, click Navigate > GAEB List to view the related files for this job. Notice that there are now three files: one for phase 83 (Quote Calling), 84 (Quote Bidding) and 86 (Order Award).

: GAEB List with related files for the job
