Setup

This section covers the general setup done in the cc|alloy management module as well as the setup required on the master data (items, customers and vendors).

General Setup

All cc|alloy management setup pages are accessed from the cc|project menu by clicking Construction > Alloy Surcharges.

Alloy Surcharges Setup

The general setup for cc|alloy management is done in the Alloy Surcharges Setup page.

: Alloy Surcharges Setup page

The fields in the Alloy Surcharges Setup page are described in the following table.

Field

Description

Calendar for Alloy Quotation

Indicates the base calendar that is used for calculating the base date for the alloy quotation.

Default Alloy Condition Code

Indicates the defaults alloy condition code that will be used if no vendor or customer-specific code is found; refer to section “Alloy Conditions” on page 1.

Quotations Web Url

Indicates the default website that will open when the “Quotations (WWW)” function is used in the Alloy Quotations page (refer to section “Alloy Quotations” on page 1).

 

Alloy Types

In the Alloy Types page, the different types of alloy that are used in quotations are defined.

: Alloy Types page

The fields in the Alloy Types page are described in the following table.

Field

Description

Code

Unique identifier of the alloy; typically the same as used in the standard Periodic Table.

Description

Short description of the alloy; typically the same as used in the standard Periodic Table.

Item Charge No.

Indicates the item charge that will be used when the alloy is added to the sales or purchase line.

Unit of Measure Code

Indicates the unit of measure that is used when selling/purchasing the alloy.

 

Alloy Conditions

Alloy Conditions are used to define rules for establishing the alloy quotation.

: Alloy Conditions page

The fields in the Alloy Conditions page are described in the following table.

Field

Description

Code

Unique identifier for the alloy condition.

Description

Short description of the alloy condition.

Quotation Base

Determines which base is defined to be used. The options are:

  • Lower
  • Average
  • Upper

Period

Used with the Calculation Formula field to determine period ranges.

Calculation Formula

Indicates the formula for calculating the period range; used with the value in the Period field. The options are:

  • Day
  • Month
  • Quarter
  • Half-Year
  • Year
  • Special Quotation

Reference Date

Determines which document date is the base for establishing the alloy quotation. The options are:

  • Receipt of Order Date
  • Delivery Date

Staggered

If selected, the alloy quotation is established considering one month earlier than calculated. This option is only used with the Calculation Formula of Month, Quarter, Half-Year and Year.

 

Alloy Quotations

In the Alloy Quotations page, the quotation values per period, per 100kg, for each alloy type is specified. On the General FastTab, enter the Alloy Type code and then enter the quotation information in the lines.

: Alloy Quotations with data for Copper

The fields in the lines of the Alloy Quotations page are described in the following table.

Field

Description

Period Date

Indicates the date on which the quotation is valid.

Lower Quotation

Indicates the lower value for the quotation on this date.

Upper Quotation

Indicates the upper value for the quotation on this date.

Average Quotation

Automatically calculated based on the values in the Lower Quotation and Upper Quotation fields.

The “Quotations (WWW)” function on the ribbon opens the website indicated in the Quotations Web Url field in the Alloy Management Setup page, and can be used to gather the lower and upper quotation values based on the period.

Master Data Setup

Once the setup is complete in cc|alloy management, Alloy Weight needs to be assigned to items and Alloy Conditions need to be set up on customers and vendors. This setup is described in this section.

Items

On the Actions tab of the Item Card, click Alloy Weights to open the Item Alloy Weights page, where weight quantity and unit of measure is defined for an item.

: Item Alloy Weights for copper set up on Item Card

The fields in the Item Alloy Weights page are described in the following table.

Field

Description

Alloy Type Code

Indicates the alloy type for which weights are defined for this item.

Alloy Weight

Indicates the weight contained in the item per base unit of measure relating to the Alloy Weight Quantity field.

Alloy Weight Quantity

Indicates the quantity per base unit of measure relating to the Alloy Weight field.

Alloy Weight (per Base Unit)

Automatically calculated based on the values of the Alloy Weight and Alloy Weight Quantity fields.

Alloy Base

Indicates the base price that will be used in the quotations.

 

Customers

On the Navigate tab of the Customer Card, click Alloy Conditions to open the Cust./Vend. - Alloy Conditions page, where the billing condition is defined for a customer.

: Alloy Conditions for copper on the Customer Card

The fields in the Cust./Vend. - Alloy Conditions page are described in the following table.

Field

Description

Alloy Type Code

Indicates the alloy type for which the conditions are defined for this customer.

Alloy Condition Code

Indicates the condition for the alloy type for this customer.

Alloy Condition Description

Automatically populated from the Alloy Condition Code value.

Alloy Base

Indicates the base price that will be used in the quotations for this customer.

Amount Scales Exist

Indicates whether amount scales exist for this alloy condition or not.

Click the value in the field to open the Alloy Surcharges Amount Scales page where a minimum amount and surcharge percentage can be defined for the condition.

Specific Quotation

Indicates a special quotation price that has been agreed to with this customer.

 

Vendors

Alloy Conditions for Vendors are set up the same as on Customers. On the Navigate tab of the Vendor Card, click Alloy Conditions to open the Cust./Vend. - Alloy Conditions page, where the billing condition is defined for a vendor.

: Alloy Conditions for copper on the Vendor Card

Refer to section “Customers” on page 1 for details about the fields on the Cust.-/Vend. - Alloy Conditions page.
