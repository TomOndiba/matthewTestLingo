Structural Plan

The Structural Plan defines the processing structure for the job. The PaC Structural Plan page is accessed from the PaC Job Card by clicking Navigate > Structural Plan.

: PaC Structural Plan page

PaC Structural Plan Page

The PaC Structural Plan page is similar to the BoQ Budget Calculation in that the new positions and levels are used to create the hierarchical structure. Several of the same fields are used in both pages as well, many of which are FlowFields from the BoQ Budget Calculation to the PaC Structural Plan.

The fields in the PaC Structural Plan page that are specific to this page are described in the following table.

Field

Description
Type of Structure (PaC)

Indicates the line type of the automatically created positions. The options are:

  • Cost Type – one position per Cost Type is created as a headline
  • Calculation Group – specifies which calculation group summary is used
  • Material Demand – specifies the positions that are relevant for the material demands based on the BoQ Job
  • Component Group – one headline is created per each Component Group. At least one headline "Without Component Groups" is created if Component Groups are not used in items

Quantity

Specifies the new planning quantity. This amount can be the same as the estimate but is typically changed when the Project Manager or Engineer re-plans the job with better estimates.

Allocated Quantity from BoQ

Displays the value from the Allocated Quantity to PaC field in the BoQ Budget Calculation, which is based on calculated estimates.

Allocated Total Cost from BoQ

Displays the value from the Allocated Total Cost to PaC field in the BoQ Budget Calculation.

Allocated Sales Line Amount from BoQ

Displays the value from the Allocated Sales Line Amount to PaC field in the BoQ Budget Calculation.

The actions on the PaC Structural Plan page that are specific to this page are described in the following table.

Action

Description

Get Calculation Groups

Inserts calculation groups from BoQ positions if the create Calculation Groups option was declined when the PaC Job was created.

Get Material Demands

Calculates items with Calculation Groups that are set up with Structure Type = Material Demand.

Insert Execution Unit

Creates the PaC Structural Plan based on the BoQ Budget Calculation structure defined in the Execution Unit type = Placing Unit.

Update Allocation

Updates the Allocation View with new allocations.

Allocation View

Opens the Job Line Allocations page, which displays all BoQ Positions that have been allocated to the Structural Plan.

 

Create Structure

Depending on how the PaC Job was created, information may have been transferred from the BoQ Job, or information may still need to be transferred. The “Get Calculation Groups” and “Get Material Demands” functions assist in transferring the information.

After the base information is transferred from the BoQ Job, specific services and materials are added to the PaC Structural Plan using the “New Position” and “New Level” functions.

Get Calculation Groups

Since the Calculation Groups were created when the PaC Job was created, the structure is already started in Fig. 9 for Wages, as that cost type had calculated entries.

: PaC Structural Plan with calculation groups created during PaC Job creation

If the calculation groups had not been created during PaC Job creation, first the “New Position” function would be run and then the “Get Calculation Groups” function.

Get Material Demands

To update the costs for material demands, select the row for Cost Type = Material and then click Get Material Demands. A message appears asking to insert the material demands.

: Running the Get Material Demands function in PaC Structural Plan

Click Yes to insert the material demands below the selected Material line.

: Material demand added to the PaC Structural Plan

The calculated material demand from the BoQ Budget Calculation is inserted below the selected position.

Update Quantities

Once the calculation groups and material demand has been added, quantities are updated to include actual values, as opposed to the estimated values for the quote.

: Updated wage and material quantities

View and Update Allocations

The Job Line Allocations page shows the allocations that have been made in the PaC Structural Plan from running the Get Calculation Groups and Get Material Demands functions.

: Job Line Allocations

If something is changed in the BoQ Job after the PaC Structural Plan has been created, e.g. because of an increase of BoQ quantities, use the “Update Allocation” function to update the information.
