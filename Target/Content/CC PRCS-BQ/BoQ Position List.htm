BoQ Position List

Once GAEB files are ready for calculation processing, open the BoQ Position List page from the Job Card ribbon.

: Job Card accessing BoQ Position List action and the BoQ Position List page

The BoQ Position List page shows the complete list of positions in the GAEB file, with the allocated units of measure performed during the import (compare the “St” value in the Unit of Measure field in Fig. 2 with the “PCS” value in the Unit of Measure Code field in Fig. 1).

The BoQ Position List page will display calculations that are entered in the BoQ Budget Calculation or Fast Calculation pages; no calculations are done directly in the BoQ Position List page.

The fields in the BoQ Position List page are described in the following table.

Field

Description

Job No.

Identifies the job related to the BoQ position.

BoQ Pos. No.

Identifies the position number from the GAEB file.

BoQ Position Type

Identifies the position type from the GAEB file.

Description

Displays the position description from the GAEB file.

Long Text

Indicates whether long text exists for the position or not.

Quantity

Displays the position quantity from the GAEB file.

Unit of Measure Code

Displays the allocated unit of measure for the position from the GAEB file.

Responsible

Displays the person responsible for the position from the GAEB file.

Responsible Calculation

Displays the person responsible for calculating the position from the GAEB file.

Calculation Status

Indicates the status of the calculation for the position. The options are:

  • Blank – no calculations have been done for the position.
  • In progress – calculations have been started for the position.
  • Calculated – calculations are done for the position and the “Change Calculation Status” function has been run.
  • Fixed – calculations are done for the position and the prices have been fixed. Refer to section “View Calculation Statistics and Edit Vernier Adjustments” on page 1 for more information.

Assigned PaC Job No.

Identifies the assigned PaC Job to which the position is transferred.

Unit Price (LCY)

Indicates the calculated unit price based on the pricing in the BoQ Budget Calculation.

Total Price Budget (LCY)

Indicates the calculated total budget price based on the quantity and pricing in the BoQ Budget Calculation.

Sales Code

Displays the sales code assigned to the allocated unit of measure for the position.

Sales Line Amount Budget (LCY)

Displays the budgeted amount including discounts assigned to the sales line from the BoQ Budget Calculation.

Sales Line Discount %

Displays discount percentage assigned to the sales line from the BoQ Budget Calculation.

The actions on the Home tab are described in the following table.

Action

Description

Position Card

Opens the Job Position Card, where complete information about the selected position is stored.

Change Calculation Status

Opens the Calculation Status dialog, which allows for the change of Calculation Status for the selected position(s).

Giving Away Units

Opens the Giving Away Unit Allocations page, where Giving Away Units can be assigned; currently has no functionality.

BoQ Budget Calculation

Opens the BoQ Budget Calculation page, where detailed calculations can be entered/viewed for the complete position list.

Fast Calculation

Opens the Fast Calculation page, where calculations can be quickly entered for the complete position list.

Long Text

Opens the LV Job Texts - Long Text page, where long text for the selected positions can be viewed.

Short Text

Opens the LV Job Texts - Short Text page, where short text for the selected positions can be viewed.
