BoQ Manager

The BoQ Manager can be used to import and export GAEB files. GAEB is a German organization that defines standards in electronic data exchange for construction works industries. Refer to http://www.gaeb.de/en/ for more information.

There are three different versions of the GAEB standard:

  • GAEB 90 – file *.d*
  • GAEB 2000 – file *.p*
  • GAEB XML – file *.x*

The GAEB data exchange has different exchange phases, each with their own file definitions. Currently cc|project supports the following exchange phases:

  • 81 Specification
  • 82 Cost Estimate
  • 83 Quote Calling
  • 84 Quote Bidding
  • 85 Alternative Bids
  • 86 Order Award

Example files:

  • *.d81,*.d82,*.d83, *.d84, *.d85, *.d86
  • *.p81,*.p82,*.p83, *.p84, *.p85, *.p86
  • *.x81,*.x82,*.x83, *.x84, *.x85, *.x86

As mentioned previously, the BoQ manager uses a third party component of the GAEB Konverter software to convert GAEB files to a specific TML file format. These TML files are imported and exported to/from Microsoft Dynamics NAV; the third party component works in the background and is invisible for the users.

This topic describes the different usages of the BoQ Manager in cc|project, and provides an overview of the main processes of working with GAEB files. More detailed information about calculation and invoicing are described in later sections.
