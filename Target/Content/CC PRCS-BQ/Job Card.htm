Job Card

A significant number of fields and actions have been added to the Job Card for Construction Works Processing. Additionally, the information dynamically changes depending on the Structure Type of the Job (Bill of Quantities, Process and Controlling, or BoQ and PaC).

Job Structure Type = Bill of Quantities (BoQ)

The fields and actions on the Job Card that are specific to jobs with Structure Type = Bill of Quantities are described in this section.

: Job Card for Structure Type = Bill of Quantities

General FastTab

The fields on the General FastTab that are specific to Construction Works Processing are described in the following table.

Field

Description

Structure Type

Indicates if the job is for calculating (Bill of Quantities) or processing (Processing and Controlling), or both (BoQ and PaC).

Subsection

Indicates if this job is a category of another job structure; for informational purposes only.

Person Responsible 3 / Person Responsible 4

Two additional people can be assigned in the Construction Works Processing job. These individuals are automatically assigned to the Job Team.

Project No.

Indicates the number of the related Project.

BoQ Blocked by

Indicates a user ID who blocked the BoQ by using the calculation statistics vernier adjustment.

 

Posting FastTab

The field on the Posting FastTab that is specific to Construction Works Processing is described in the following table.

: Job Card, Posting FastTab

Field

Description

Invoice Model

Indicates how invoicing will be handled for this job. The options are:

  • Performance per Period – used to invoice the service that was delivered since the last invoice was created.
  • Cumulative Invoicing – used to invoice all delivered services including services from previous invoices. For example:
    • 1st invoice 10 pcs at 100 = 1000, amount to pay 1000
    • 2nd invoice 30 pcs at 100 = 3000, amount to pay 2000
    • 3rd invoice 100 pcs at 100 = 10000, amount to pay 7000

Calculation FastTab

The Calculation FastTab is used specifically for Construction Works Processing and contains one field that relates to the integration with Alloy Management: the Alloy Surcharge Date field, which defines the date for calculating alloy surcharges.

: Job Card, Calculation FastTab

Date FastTab

The Date FastTab is used specifically for Construction Works Processing.

: Job Card, Date FastTab

The fields on this FastTab are described in the following table.

Field

Description

Submission Date

Indicates the date of submission.

Submission Time

Indicates the time of submission.

Quote Valid Until

Indicates the date until the quote is valid.

Quote delivered Date

Indicates the date until the quote must be delivered.

Indicates the date of the submitted quote.

Indicates the date of negotiation.

Indicates the date of the order receipt.

Indicates whether the prices are fixed until the end of the construction period.

Indicates the date of the submitted final invoice.

Actions Tab

The actions on the Actions tab that are specific to Construction Works Processing are described in the following table.

Action

Description
Create PaC Job

Creates a PaC Job for the Project/BoQ Job.
Calculate Rate Unit Cost

Calculates the rate unit cost based on the Budget Calculation and the EFB Surcharges.

Import GAEB

Opens the GAEB File Import page, from where a GAEB file can be selected for import into the job structure (refer to section “Import GAEB File” on page 1 for more information).

Export GAEB

Opens the Data Exchange Phase dialog where the correct phase is selected and then creates a GAEB Export file for the selected phase.

Get GAEB Import

Opens the GAEB List, from which an existing GAEB record can be imported into the job structure. Used when a GAEB file was imported first into the GAEB list instead of a job; prevents the unnecessary creation of another GAEB record.

Create GAEB Export

Opens the Data Exchange Phase dialog where the correct phase is selected and then creates a GAEB Export Manager for the selected phase.

Create Measurement Document

Creates a new measurement document and provides the ability to copy the complete BoQ into the document.

 

Report Tab

The Report tab contains the EFB-Price 221 (1a - 311.a) report, which is used with the EFB Calculation (refer to section “EFB Calculation” on page 1).
