Project Card

The information entered in the Project Card forms the basis of the construction works job and is transferred to the sub-jobs created from the Project Card.

: Project Card

The fields in the Project Card are the same as the fields in the Job Card, but are sometimes in different FastTabs since information is consolidated in fewer FastTabs on the Project Card. Otherwise, the main differences are the actions on the ribbon, and the addition of the Project Information FactBox which also appears on the Job Card when Construction Works Processing is activated in cc|project.

This section describes information that is specific to the Project Card. Refer to topic “Job Card” on page 1 for information about other fields in the Job Card.

Actions Tab

The functions on the Actions Tab are described in the following table.

Action

Description

Create Job

Similar to the “Create Sub-Job” function in the Job Card, the “Create Job” function creates the sub-job for the Project; however, a Job Type must be specified during creation.

Copy Project

Opens the Copy Job page, which copies the current project to a new project.

 

Navigate Tab

The functions on the Navigate Tab that are specific to the Project Card are described in the following table.

Action

Description

Contact Certificates

Opens the Contact Certificates page, where certificates (e.g. tax exemption) can be stored for contacts.

Jobs

Opens the Job List, filtered by related jobs with Structure Type = Empty.

Bill of Quantities

Opens the Job List, filtered by related jobs with Structure Type = Bill of Quantities.

Process and Controlling

Opens the Job List, filtered by related jobs with Structure Type = Process and Controlling.

 

Project Information FactBox

The Project Information FactBox provides an overview of the number of Jobs related to the Project, as well as related purchases, sales and measurement documents.
