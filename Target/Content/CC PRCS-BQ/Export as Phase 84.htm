Export as Phase 84

For the purpose of this overview, only some values were entered to demonstrate how to process entries in the BoQ Job Calculation. Typically, values are entered for all positions before the GAEB file is exported as Phase 84, Quote Bidding, and sent to the customer. The customer will then review the file, and hopefully award the order.

In the Job Card, click Export GAEB on the ribbon. The Data Exchange Phase dialog appears with phase 84 pre-selected since the originating GAEB file was phase 83. It is unusual to change this to a different phase at this time, but it can be done.

: Exporting GAEB file for Quote Bidding (Phase 84)

Click OK to create the export as Phase 84. The Exchange Format dialog opens with the import format selected as default but can be changed as needed.

: Exchange Format dialog

Click OK. The export file is created and the GAEB File Export page opens to the same location where the import file was stored.

: GAEB File Export page for export storage

Notice that the file name has incremented one number since the import GAEB file was created. Click Save to complete the export process. When finished, a confirmation message appears.

: Export confirmation

On the ribbon, click Navigate > GAEB List to view the related files for this job. Notice that there are two files: one for phase 83 (Quote Calling) and 84 (Quote Bidding).

: GAEB List with related files for the job

Select the line for phase 84, Quote Bidding and then click Edit on the ribbon to review the GAEB Card.

: GAEB Card with values from BoQ Position calculations

Escape back to the Job Card.
