BoQ Job Calculations

When a GAEB File is received from a customer, a project is created along with a Bill of Quantities (BoQ) Job.

: Project and BoQ Job for calculation processing

The GAEB file must be imported into the BoQ Job so calculations can be performed for the quote. As previously mentioned, the file can be imported directly into a job using the “Import GAEB” function or a previously imported file can be added to a job using the “Get GAEB File” function.

: Importing GAEB file into BoQ Job

After import, a GAEB record is created in the GAEB List and in the BoQ Budget Calculation of the job.

: GAEB Card and BoQ Budget Calculation created from GAEB file import into a BoQ Job

This section explains the various types of calculations that are performed in the BoQ Jobs after a GAEB file is imported. It includes the basic steps of reviewing the BoQ Position List, adding generic wage and material information for the quote using the Fast Calculation, adding more detailed quote information using the BoQ Budget Calculation, exporting the quote as Phase 84, running the EFB calculation, reviewing the submission ranking, and using the compare tool when the project is awarded.
