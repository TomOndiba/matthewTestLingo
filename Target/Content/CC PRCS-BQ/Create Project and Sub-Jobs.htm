Create Project and Sub-Jobs

This section provides an overview of how to create the Project and BoQ/PaC Job structure used in the previous sections.

Create Project

Follow these steps to create a Project:

  1. On the navigation pane, click Departments > cc|project> Construction > Processing > Projects.
  2. On the ribbon, click New and then press Enter or Tab to insert a new number.
  3. In the Description and Description 2 fields, enter relevant descriptions of the project.
  4. In the Sell-to Customer No. field, enter the number of the customer who sent the quote request.
  5. In the Job Type field, enter STRUCTURE.
  6. In the Person Responsible field, enter the resource responsible for the project.

: Project Card

Create BoQ Job

Follow these steps to create the BoQ Job:

  1. On the ribbon, click Create Job.
  2. In the Copy Job dialog page, select the Job Type associated with Structure Type = Bill of Quantities (refer to section“Job Types Setup” on page 1 for more information).

: Copy Job dialog page with BoQ-related Job Type selected

  1. Click OK.
    The Bill of Quantities job is created with links to the project.

: BoQ Job created from Project

  1. Click OK to close the BoQ Job Card.
  2. Click OK to close the Project Card.

Create PaC Job

Follow these steps to create the PaC Job from the BoQ Job:

  1. On the navigation pane, under cc|project, click Construction > Processing > Job List.
  2. Select the sub-job of the project, with Structure Type = Bill of Quantities.

: Selecting BoQ Job from Job List

  1. On the ribbon, click Actions > Create PaC Job.

: Create PaC Job function on BoQ Job

  1. A message appears asking to create calculation group totals, which would be beneficial if the BoQ is calculated before the PaC Job is created because it creates a new job line structure based on the calculation.

: Message to create calculation group totals in the PaC Job

  1. Since calculations have not been done yet in the BoQ Job, click No in the message.
    The Process and Controlling job is created with links to the project.

: PaC Job created from BoQ Job

  1. Click OK to close the BoQ Job Card.

In the Job List, review the Project and Sub-Job structure.

: Project and Sub-Job (BoQ/PaC) Structure in Job List
