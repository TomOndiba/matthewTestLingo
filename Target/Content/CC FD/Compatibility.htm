Compatibility

cc|formatted documents – as part of the cc|document management pack – is already fully integrated and compatible with all COSMO CONSULT products.

For example, cc|formatted documents is available in all functional cases by using cc|text module. cc|text module allows an extensive text controlling, that exceeds by far the capabilities of standard Microsoft Dynamics NAV. If cc|document configurator is used, it is not necessary to place cc|formatted documents in a report layout via native Report Designer or Visual Studio.
