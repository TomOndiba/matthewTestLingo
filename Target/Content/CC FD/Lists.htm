Lists

Ordered and unordered lists can be added.

  • Numbered lists are important for displaying sequentially executed actions or rankings clearly. The entries in a numbered list are automatically numbered.
  • Bullets can be used to represent, for example, product features clearly. In a bulleted list, all list entries are provided with a bullet.

To insert a list, follow these steps:

  1. Mark the text to be changed.
  2. Click the action or in the START tab in the Paragraph group.

: Lists

To change the indent of a list entry, follow these steps:

  1. Mark all the text to be changed.
  2. Click the action to increase the indent or to decrease the indent. Both actions can be found in the START tab in the group Paragraph group.
