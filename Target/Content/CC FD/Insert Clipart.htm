Insert Clipart

To insert a clipart, follow these steps:

  1. Click where the clipart should be inserted.
  2. Click the dropdown list of the clipart in the INSERT tab in the Graphics group.
  3. Find and select the clipart to insert.

Extend the Clipart Gallery

Additional clipart can be download and stored in the installation directory of cc|formatted documents. cc|formatted documents automatically recognizes the available clipart of file type PNG in the “Cliparts” subfolder.

This clipart can be downloaded from the OpenClipart.org website: https://openclipart.org.
