Layout of the Report (in Report Designer)

Any number of subreports can be placed anywhere in the main report, except in the header. The name of the report can be user-defined; however, the chosen name must match the prefix of the column names.

The RectangleForNo parameter is mandatory and must be added to a report.

Note

If the subreport is placed in a table or matrix, it will be rendered for each line. In this case the value of the parameter “RectangleForNo” for each line is relevant.

Expression for value:
=Fields![<SubreportIdentifier>_]FD_RectangleForNo.Value

If the subreport is placed outside of a table or matrix, the First function must be used. The First function returns the first value in a set of records, after sorting and filtering of a specified area is applied.

Expression for value:
=First(Fields![<SubreportIdentifier>_]FD_RectangleForNo.Value)
