Client Installation

The Installation Wizard for the “cc|formatted documents – Client” must only be run for all Microsoft Dynamics NAV Clients that have a version lower than Microsoft Dynamics NAV 2015 and that should be used for cc|formatted documents. Therefore, if cc|formatted documents is used only in the sales or order department, it is not necessary to install the Microsoft Dynamics NAV Client Layers in the inventory department.

An automated Software Deployment of the Client Software is basically possible.

: Three Tier Architecture

Requirements

Requirements for the correct operation of “cc|formatted documents – Client”:

  • .NET Framework 3.5 or higher
  • Microsoft Dynamics NAV Client

Note

The “cc|formatted documents – Server” must be installed prior to installing the “cc|formatted documents – Clients” because it provides a (required) service to the clients. Without the server, the clients cannot operate (especially the report output).

Installing the server includes the following processes:

  1. Run the file “cc formatted documents <NAV Version ID> - Client Setup.msi”.
  2. Follow the installation wizard (see following pages).
  3. Follow the configuration wizard (see following pages).

Each process is described in the following sections.

Client Installation

Follow these steps to run the cc|formatted documents client installation:

  1. Locate and run the “cc formatted documents <NAV Version ID> - Client Setup.msi” file to start the installation.

: cc|formatted documents Client Setup Wizard, Welcome window

  1. Click Next. The information window opens.

: cc|formatted documents Client Setup Wizard, Information window

  1. Click Next. The License Agreement window opens.

: cc|formatted documents Client Setup Wizard, License Agreement window

  1. Accept the License Agreement and then click Next.

: cc|formatted documents Client Setup Wizard, installation folder selection

  1. Confirm or select the installation folder and then click Next.

: cc|formatted documents Client Setup Wizard, settings and features selection

  1. Confirm the default setting or (de-)select additional features:
    • The “Report Generator” is a required component of cc|formatted documents. Without the Report Generator an output of formatted content is not possible.
    • The “HTML Editor” is a component that is required for text processing.

Note

If you have purchased the HTML Editor only as a flowtext editor of your Microsoft Dynamics NAV Installation, you can deactivate the Report Generator of cc|formatted documents at this point.

  1. Click Install. Required files/components are copied and registered.
    When the installation is finished, the Installation Complete window opens.
  2. When the installation process is finished, click Finish.

Configuration Wizard – Client

When the installation is complete, the cc|formatted documents - Configuration Wizard window opens. The Wizard assists with setting up the connection to the previously installed “cc|formatted documents – Server”.

: cc|formatted documents Client Configuration Wizard, Welcome window

  1. Click Next. The Product Key window opens.

: cc|formatted documents Client Configuration Wizard, Product Key window

  1. Enter the company name and the product key provided by COSMO CONSULT.

Note

The product key is company specific. For companies of affiliated enterprises more product keys are issued free of charge.

  1. Click Next. The Connection Settings window opens.

: cc|formatted documents Client Configuration Wizard, Connection Settings window

 

 

  1. Confirm or adjust the address, port and service name of your “cc|formatted documents –Server”:
    • The address of the “cc|formatted documents – Server” (FD Server) corresponds to the network address of the host, on which the “cc|formatted documents – Server” was installed.
    • The FD Port and FD Server Service Name only have to be adjusted if incorrect.
  1. Click Next. The Connection Test window opens.
    This window provides the ability to test the connection of the “cc|formatted documents – Client” to the “cc|formatted documents – Server”.

: cc|formatted documents Client Setup Wizard, Connection Test window

  1. Perform a connection test by clicking Test Connection.
    If the connection is established successfully, the “cc|formatted documents – Server” indicates the connection status to Microsoft Dynamics NAV.

Note

The “cc|formatted documents – Server” establishes the connection on demand – therefore it is not crucial if no connection to Microsoft Dynamics NAV exists.

  1. Click Next. The NAV Add-ins window opens.

: cc|formatted documents Client Configuration Wizard, NAV Add-ins window

To use cc|formatted documents in Microsoft Dynamics NAV, the Add-Ins path for Microsoft Dynamics NAV is required.

  1. Specify the path to the Add-In Folder of the Microsoft Dynamics NAV Client, which is used for the connection to the correct Microsoft Dynamics NAV Instance and then click Finish to complete the configuration wizard.
