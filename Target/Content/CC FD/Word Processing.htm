Word Processing

cc|formatted documents includes a built-in Microsoft Dynamics NAV word processor (Text Editor) that allows for any number of records of different tables to create and store any number of formatted text.

The word processing includes substantial functionality to mark text. Moreover, lists, QR codes and images can be inserted. The included clipart library consists of more than 900 clipart images that can be extended (refer to topic "Extend the Clipart Gallery" on page 1).

All text is spell checked; incorrect or unknown words are highlighted with a red, wavy line.
