Activating/Deactivating cc|formatted documents – Report Output

In addition to manually creating reports and document, one way of integrating cc|formatted documents into a Microsoft Dynamics NAV database is to import the modified standard objects that include the sample reports/documents.

To use cc|formatted documents to render a single report, the standard report selection must be set up in Microsoft Dynamics NAV.

To set up the standard report selection, follow these steps:

  1. Open the Report Selection Sales page in the Microsoft Dynamics NAV Client.

: Activating Report Output (1)

  1. Select the Use cc|formatted documents check box for report 205, Order Confirmation.

: Activating Report Output (2)
