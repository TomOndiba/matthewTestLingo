Introduction

cc|formatted documents expands Microsoft Dynamics® NAV by allowing users the ability to enhance and improve the appearance of their Microsoft Dynamics NAV reports through the use of an integrated word processor directly in Microsoft Dynamics NAV. Whether it is adding formatted pre-texts or post-texts (headers or footers) to order confirmations or integrating images or graphics in quotes, users have countless ways of creating attractive, understandable and customized reports and documents within the scope of their corporate design.

With cc|formatted documents, it is not necessary to re-create the report layouts. To access full functionality, it is only necessary to place an empty subreport in an existing report. In this sub-report, the contents are generated from the text processing. The entire content is stored in the database (e.g., in the transaction or master data).

Comparable solutions are based on alternative report generators that require a rebuild of the existing report definitions. In typical Microsoft Dynamics NAV reports, program code is also housed with “Business Logic,” which is not run in an alternative report generator. These constraints bring disadvantages and require a high additional integration effort.

Key Features

  • Text processing for Microsoft Dynamics NAV
    • Embedded XHTML
    • Additional option for plain text (Transition from/to plain text lines), so that text processing can be used as flowtext editor
  • Output of contents (from text processing) in Microsoft Dynamics NAV reports
    • Output of all formatting, images and so on
  • Integration of any Microsoft Dynamics NAV report as sub-report in other Microsoft Dynamics NAV reports
    • Formulation of the reporting instructions for the subreport in the main report
    • Sub-reports of other Microsoft Dynamics NAV Service Instances and Companies
  • OUT-OF-THE-BOX usage after completing the Configuration Wizard
  • Modern Microsoft Office Design
  • Modern XHTML Standard, suitable for delivering content to e-commerce solutions, catalog and print providers, and so on
    • Strict (configurable) cleaning of non-allowed HTML elements and attributes
  • Ensurance of company owned Corporate Design specifications
  • Unicode (also included for Microsoft Dynamics NAV 2009)
