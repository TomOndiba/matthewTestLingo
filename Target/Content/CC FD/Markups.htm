Markups

This section explains the basic formatting options in the ribbon.

Format Text as Bold

  1. Mark the text to be changed.
  2. Click the action in the START tab in the Formatting group.

Format Text as Italic

  1. Mark the text to be changed.
  2. Click the action in the START tab in the Formatting group.

Format Text as Underlined

  1. Mark the text to be changed.
  2. Click the action in the START tab in the Formatting group.
