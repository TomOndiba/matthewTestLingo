Report Output

The standard report output in Microsoft Dynamics NAV renders a standard Microsoft Dynamics NAV report. Microsoft Dynamics NAV reports rendered in cc|formatted documents can be output:

  • For printing (direct printing after selecting a printer)
  • As PDF file (requests save)
  • As Excel file (requests save)
  • As PDF file as email attachment
  • For page view (default option)

The type of operation of the report output depends on the integration and the call for the report in the Microsoft Dynamics NAV database. Theoretically, appropriate calls can be integrated to the pages for all of the individual options.
