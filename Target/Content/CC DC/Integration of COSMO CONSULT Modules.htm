Integration of COSMO CONSULT Modules

The text formatting capabilities of cc|formatted documents along with the additional document header, footer and line text capabilities of cc|text module have been integrated into cc|document configurator Document Layout templates to provide greater flexibility in producing documents from Microsoft Dynamics NAV.

  • With cc|formatted documents, users have the ability to enhance and improve the appearance of reports or documents through the use of an integrated word processor directly in Microsoft Dynamics NAV.
  • With cc|text module, users have the ability to add more information to records such as items, vendors, and customers which can then be inserted as headers, footers and line texts to be included in reports or documents.

The integration of cc|formatted documents into cc|text module makes it easy to add formatting to the header, footer and line text that appears in printed documents.

This section describes the cc|formatted documents and cc|text module functionality that is used with Document Layout templates in cc|document configuration.

For information about setting up and using cc|formatted documents, refer to the current “cc|formatted documents” documentation.

For information about setting up and using cc|text module, refer to the current “cc|text module” documentation.
