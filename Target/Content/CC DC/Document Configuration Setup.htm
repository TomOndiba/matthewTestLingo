Document Configurator Setup

The Document Configurator Setup page is opened from the cc|document configurator menu, under Administration, by clicking Document Configurator Setup. This page includes setup to define label codes that are used by the system to print the top part of a report. The lookup of the fields goes to the Document Texts (refer to topic “Document Text” on page 1); it also contains a setup of Font Family and Font Size that is used for layouts and fields where those are not specified.
