DC - Generate Totals Codeunit

This Codeunit calculates the amounts for totaling of documents printed from the Sales Header and Purchase Header tables. The way this codeunit is used is to set up the Section (Document Layout Line) to run it once per record, which then updates the Variables values that are used in the Document Layout.
