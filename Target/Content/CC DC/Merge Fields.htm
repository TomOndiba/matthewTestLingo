Merge Fields

This example shows how the Description and Description 2 fields are merged into one field to be displayed on the Sales Quote printout. Merging fields are done with Variables.

Note

Before starting this example, add Description 2 text to some items and assign to a new Sales Quote.

Review Variables in the Document Layout

Follow these steps to review variables in a document layout:

  1. Open the SQ-01, Sales Quote, Document Layout.
  2. Select the “Lines” section in the Sections FastTab and then click Line > Variables to open the Document Layout Variables page.
  3. Notice that two variables exist for descriptions:
    1. Description1 for the Description field
    2. Description2 for the Description 2 field

: Variables for descriptions for the sales quote lines

  1. Click OK to close the Document Layout Variables page.

Review Field with Merged Variables

The next step is to review the settings of the merged variables for the Description field in the Document Layout Fields.

Follow these steps to review the merged fields for printing:

  1. With the “Lines” section still selected, click Line > Fields on the Sections FastTab.
  2. Locate the line for the Description field.
  3. Notice the following:
    1. The merged description variables have been entered in the Text field as [Description1] [Description2].
    2. The Allow Line Break check box is selected.

: Merged variables of Description fields for the sales quote lines

  1. On the ribbon, click Run Report to view the report changes.
  2. Select the Show Request Page check box and then click OK.
  3. In the request page, click Preview.

: Merged variables of Description fields in the Sales Quote lines

Note

Since the Allow Line Break check box is enabled on the Description line in the Document Layout Field page, the system will print the descriptions longer than the width of the field on two lines (with a line break where there is a space, if possible).

  1. Close the Print Preview page.
  2. Click OK to close the Document Layout Fields page.
  3. Change the Status to Certified to make it available to the users.
  4. Click OK to close the Document Layout page.
