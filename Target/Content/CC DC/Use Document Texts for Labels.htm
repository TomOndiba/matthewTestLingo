Use Document Texts for Labels

Labels (Label = Caption) for fields can be entered as Text in a Document Layout Field or as Document Text. The advantage of using a Document Text is that it can be translated to several languages that can then be used in several different Document Layouts.

This example reviews Document Text used for labels in the Order Confirmation document layout.

Review and Change Document Text for Shipment Date

Follow these steps to review document text:

  1. On the Configuration menu, click Document Texts.
  2. Locate and open the SHIPMENTDATE document text.

: Document Text for Shipment Date

Document Texts are created using a Code and a Text. The Text should be either English or the Local language; the lines of the Document Text consists of the translation in the different languages. The system is programmed to use the Text of the line matching the language of the report and if no match, the Text from the header will be used.

  1. In the Lines FastTab, in the Text field for the line of the local language, add the language code to the end of the text (in this case, “Shipment Date ENG”) to see the change in the printout.
  2. Click OK to close the Document Text page.

Review Document Text in Document Layout

To use a Document Text in a report, select Document Text in the Type field of the Document Layout Fields and then enter the code in the Document Text Code field.

Follow these steps to review document text in the document layout:

  1. On the Configuration menu, click Document Layouts.
  2. Locate and open the OC-01, Order Confirmation, Document Layout.
  3. With the first line selected in the Sections FastTab, click Line > Fields.
  4. Locate the line for “Shipment Date Label”.
  5. Note the following:
    1. Type = Document Text
    2. Document Text Code = SHIPMENTDATE

: Document Text for Shipment Date Label in Document Layout

  1. On the ribbon, click Run Report to view the report changes.
  2. Select the Show Request Page check box and then click OK.
  3. In the request page, click Preview.

: Modified Document Text for Shipment Date updated in Order Confirmation report

  1. Close the Print Preview page.
  2. Click OK to close the Document Layout Fields page.
  3. Click OK to close the Document Layout page.
  4. Go back to the Document Text for SHIPMENTDATE and remove the text (e.g., ENG) added to the end of the local language text.
