Lookup a Value in a Related Table

In this example, the Campaign No. added in the previous example will be changed to print the Campaign description instead of the Campaign number.

Change the Document Layout Field

Follow these steps to change the Document Layout field:

  1. Open the SQ-01, Sales Quote, Document Layout.
  2. Change the Status to Under Development.
  3. Select the first line in the Sections FastTab and then click Line > Fields to open the Document Layout Fields page.
  4. On the line for Campaign No. (Line = 11), enter the following:
    1. Table No. = 5071
    2. Field No. = 2

: Changed Campaign No. line in the Document Layout Fields

Add Filters

Since the system now will look up into another table, filters must be added to define what record to use. This is done by going to the Document Layout Filters for the Campaign No. field and specify a filter on Field No. = 1 (which is the No. field for the Campaign) and selecting the filter to be a value equal to Filter Field No. = 5050 (which is the Campaign No. field in the Sales Header table).

Follow these steps to add filters to define which record to use:

  1. With the line for Campaign No. (Line = 11) selected, click Filters on the ribbon.
  2. Enter the following:
    1. Field No. = 1
    2. Filter Field No. = 5050

: Filter added to the Campaign No. line in the Document Layout Fields

  1. Click OK to close the Document Layout Filters page.
  2. On the ribbon, click Run Report to view the report changes.
  3. Select the Show Request Page check box and then click OK.
  4. In the request page, click Preview.

: Campaign No. field value changed from No. on Sales Header to Name from Campaign table

  1. Close the Print Preview page.
  2. Click OK to close the Document Layout Fields page.
  3. Change the Status to Certified to make it available to the users.
