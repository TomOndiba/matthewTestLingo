Home Tab

The Home tab of the Document Layout page includes six actions; each is described in the following table.

: Document Layout, actions on the Home tab

Action

Description

Export

Exports a Document Layout (or several by changing the filter) as an XML file. This function is useful for doing backups or if several companies use the same layouts.

Copy From

Copies all the Sections (lines) from other Document Layouts into the current Document Layout. This function is useful when creating a new version of a layout.

Run Report

Generates a printed output for testing the Document Layout configuration. This function uses the current Document Layout independent of its status (when running a report from another place in the system, the user can only select from certified layouts with the valid starting and ending dates).

Run Formatted Documents Report

Generates documents that include any special formatting added to the document using cc|formatted documents (refer to topic “Run Formatted Documents Report” on page 1).

Quick View Layout

Generates a bitmap representation of the Document Layout using the first page of the last record in the database.

Using this function when setting up a Document Layout is quicker than running the “Run Report” function.

Quick View Field Positions

Generates a bitmap representation of the document layout with each field showing the indentation, horizontal alignment and field width.
