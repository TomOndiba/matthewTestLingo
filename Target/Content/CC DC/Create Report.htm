Create Report

Creating a new report is done through the Object Designer in the Microsoft Dynamics NAV Development Environment. Open a template report in the Report Designer and save it as a different report number named “Item Printout”.

Create Report

Follow these steps to create the report:

  1. In the Microsoft Dynamics NAV Development Environment, open the Object Designer.
  2. Click Report and then locate report 12032022, DC – Generic Report.
  3. Click Design and then File > Save As.
  4. Enter the following:
    1. ID = report number in your object range
    2. Description = Item Printout
  5. Click OK.
  6. In the Report Designer, enter the following:
    1. Data Source = Item
    2. Name = Item

: New report for Item Printout

  1. Click View > Properties.
  2. In the Item - Properties page, enter the following:
    1. DataItemTableView = SORTING(No.) ORDER(Ascending)
    2. ReqFilterFields = No.

: Properties changed for new Item Printout report

  1. Close the Item - Properties page.
  2. Go to the blank line in the Report Designer and then click View > Properties to open the Report properties.
  3. Change the Caption property to Item Printout.

: Caption of report changed to Item Printout

  1. Close the Report - Properties page.
  2. Go back to the Item DataSet line and then click View > C/AL Code.

: C/AL editor, highlighting the code to be changed

  1. In the Item - OnAfterGetRecord section:
    1. change “Integer.GETPOSITION” to “Item.GETPOSITION”
    2. insert a space then “{Language Code}” after the single quotes

: C/AL editor, highlighting the changed code

Note

For this report, the client language is used so no other changes are needed. If a report is needed where the language will be based on a field in a record, then code similar to this would be used: (SalesHeader.”Language Code”).

  1. In the Item - OnPostDataItem section, change “Integer” to Item”.

: C/AL editor, highlighting the changed code

  1. Close the C/AL Editor page.
  2. Close the Report Designer and with Compiled = Yes, click Yes to save the report.
  3. Make note of the new report number and then close the Object Designer.
