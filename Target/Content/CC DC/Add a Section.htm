Add a Section

In this example, a Comments Section is added to the Sales Quote layout to print quote comments.

In the Document Layout, the new section will be inserted below the Total Section line and use Section Type = Line since it will be printed once between the header and footer. This new section will reference Table No. 44, Sales Comments Line, where the comments on a Sales Quote are stored.

Note

Before starting this example, add some comment text to the existing Sales Quote header.

Add Section to Document Layout

Follow these steps to add a section to a document layout:

  1. Open the SQ-01, Sales Quote, Document Layout.
  2. Change the Status to Under Development.
  3. Insert a line after the Total Section line.
  4. Enter the following:
    1. Section Type = Line
    2. Description = Comments
    3. Table No. = 44

: New section added to the Document Layout

Add Filters to Section

Filters must be set up for the new section to define the table relation, which is done through the Document Layout Filters.

In the Document Layout Filters, specify that the Document Type and Document Lines No. filters should always be 0 (the Document Type is an option field where Quote is the first option, therefore this field is 0. “Quote” can also be entered as a filter but then it will not be possible to run the report in languages other than English) and the No. field should be filtered according to the No. field in the Sales Header table (which is the sales quote number).

Follow these steps to add filters to the section:

  1. With the section line for “Comments” selected, click Line > Filters on the Sections FastTab.
  2. On the first line, enter the following:
    1. Field No. = 1
    2. Filter Type = Text
  3. On the second line, enter the following:
    1. Field No. = 2
    2. Filter Type = Value
    3. Filter Field No. = 3
  4. On the third line, enter the following:
    1. Field No. = 7
    2. Filter Type = Text

: Filters added to the new section

  1. Click OK to close the Document Layout Filters page.

Specify Field

The last step is to specify the field to be printed for the Comments Section. This is done through the Document Layout Fields as described earlier. Field No. 6 from table 44 (no filters are required since it is from the same table as specified in the Section) will be printed in Italics.

Follow these steps to specify the field to be printed in the Comments section:

  1. With the section line for “Comments” selected, click Line > Fields on the Sections FastTab.
  2. On the first line, enter the following:
    1. Line = 1
    2. Description = Comment
    3. Type = Value
    4. Table No. = 44
    5. Field No. = 6
    6. Font Style = Italic

: Field added to the new section

  1. On the ribbon, click Run Report to view the report changes.
  2. Select the Show Request Page check box and then click OK.
  3. In the request page, click Preview.

: Comment text from the Sales Quote header added to the report

  1. Close the Print Preview page.
  2. Click OK to close the Document Layout Fields page.
  3. Change the Status to Certified to make it available to the users.
  4. Click OK to close the Document Layout page.
