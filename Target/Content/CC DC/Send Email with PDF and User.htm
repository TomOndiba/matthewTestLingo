Sales documents use the “Email” and “Print and Email” actions provided with the standard Microsoft Dynamics NAV functionality for emailing sales documents.

Along with sending the documents as PDFs, the emails can be set up with specific cc|text module entries for general emails or specific customers/vendors/contacts/responsibility centers. It is also possible to upload attachments to the codes based on various setup criteria.
