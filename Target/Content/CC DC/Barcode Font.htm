Barcode Font

To use the barcode printing option, install the Free 3 of 9 Extended Regular (FRE3OF9X.ttf) font on each machine printing documents with barcodes. The barcode font can be downloaded from here: http://www.barcodesinc.com/free-barcode-font.

Other barcode fonts can be used by changing the font family of the barcode fields in the layouts.
