Number of Copies

cc|document configurator has been programmed to set the number of copies to be printed to the variable called “NoOfCopies”. This variable can be declared to have the system assign it a number during runtime and copies of the document will be printed accordingly. This is an alternative to manually specifying the No. of Copies in the request page when running the report.
