Document Layouts with cc|text module

cc|text module is integrated into the Document Layout lines using the Text ID from the Text Module Header Document as well as fields in the Document Layout Fields page.

: Document Layout with cc|text module integration

As illustrated in Fig. 1, each line that has a “TM -” reference also includes a Phantom Layout where the header, line and footer text for the cc|text module information is defined.

: Phantom Document Layout with Document Layout Fields and Filters for Text Module Line Document table

When cc|text module header, footer and/or line text is added to a document, e.g. the Sales Order, and the Order Confirmation is printed based on the Document Layout with cc|text module integration, the header, footer and/or line text is printed in the document.

: cc|text module header, line and footer text added to sales order, printed in Order Confirmation based on Document Layout with cc|text module integration

Additionally, cc|text module line formatting values such as italic, bold, underline and color apply to cc|document configurator reports. Setting the values for a cc|text module line will format the text according to the configuration in the cc|document configurator render report.
