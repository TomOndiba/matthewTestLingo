Document Text

Document Text contains texts with related translations. When setting up a Document Layout, a Document Text can be used to print the document in different languages. Document Texts also makes it easier to change texts used in multiple documents in one place.

: Document Text

The system will first look for a text from the lines based on the language code; if it does not exist in that specific language it will use the Text from the header.

Field

Description

Code

Unique identifier of the Document Text.

Language Code

Indicates the language of the text.

Text

Identifies the actual text.
