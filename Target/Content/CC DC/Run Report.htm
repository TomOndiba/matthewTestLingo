Run Report

cc|document configurator uses process only reports that insert records into a Document Print Queue which are then printed using the rendering report defined on the Document Layout.

: Report dialog opened from Run Report action on Sales Quote document layout

The fields on the report dialog page are described in the following table.

Field

Description

No. of Copies

Indicates the number of copies to be printed. The default value is 0.

This can also be specified at runtime; refer to topic “Number of Copies” on page 1.

Document Layout No.

Indicates the document layout to use for the printout. The available layouts are those that are Certified, configured for the same report and have a starting and ending date before and after the workdate.

Printer Name

Indicates the printer name to where the report should be printed. This field defaults according to the standard Microsoft Dynamics NAV printer selection table, but can be changed. If left blank it will print to the default printer.

Show Request Page

If selected, will show the request page of the rendering report.
