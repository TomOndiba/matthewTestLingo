Create Totals and Averages

In this example, the total quantity and the average price per piece will be displayed as part of the Total Section of the Sales Quote layout.

Calculations such as Totaling and Averages are done using Variables and the Formula functionality. Variables will be created for the Line Section that summarize the Quantity field for the lines, which is done using two Variables:

  • one that gets assigned the Quantity for the current line (“LineQuantity”).
  • one with Type Formula that adds itself with the quantity of the line (“TotalQuantity”).         The Formula function is similar to the standard formula functionality in Account Schedules.

Add Variables to the Lines Section

Follow these steps to add quantity variables to the Sales Quote lines:

  1. Open the SQ-01, Sales Quote, Document Layout.
  2. Change the Status to Under Development.
  3. Select the “Lines” section in the Sections FastTab and then click Line > Variables to open the Document Layout Variables page.
  4. In the line below “Description2”, enter the following:
    1. Variable = LineQuantity
    2. Type = Value
    3. Table No. = 37
    4. Field No. = 15
    5. Format String = <Precision,2:2><Standard Format,0>
  5. In the next line, enter the following:
    1. Variable = TotalQuantity
    2. Type = Formula
    3. Formula = [TotalQuantity]+[LineQuantity]

: Variables for quantities for the sales quote lines

  1. Click OK to close the Document Layout Variables page.

Add Variables to Calculate Average

In the next step, a Variable will be created to calculate the average in the Total Section. This “Average” variable will use the formula: [TotalExclVAT]/[TotalQuantity]. Criterion will also be specified to only calculate the Average if the “TotalQuantity” is not zero.

Follow these steps to add variables and criterion to calculate average:

  1. Select the “Total Section” section in the Sections FastTab and then click Line > Variables.
  2. In the line below the second “CurrencyCode”, enter the following:
    1. Variable = Average
    2. Type = Formula
    3. Formula = [TotalExclVAT]/[TotalQuantity]

: Variable for quantity average for the sales quote totals

  1. With the last line for Variable = Average selected, click Criteria on the ribbon.
  2. In the Description field, type Only if [TotalQuantity] not 0.
  3. In the Table No. field, enter 12032010.
  4. Ensure that the Validation Type field is set to Exist.

: Criterion to only calculate the Average if the TotalQuantity is not zero

  1. Click OK to close the Document Layout Criteria page.
  2. Click OK to close the Document Layout Variables page.

Specify Fields

The last step is to add the Document Layout Fields to the Total Section, which includes a separator using underscore characters, text, variables, rounding precisions and text formatting.

Note

Due to regional settings, some of the positions (indentation/field width) may need to be adjusted.

Follow these steps to specify the fields to print the variable:

  1. With the Total Section line selected, click Line > Fields on the Sections FastTab.
  2. In the line below “Total Text Value”, enter the following:
    1. Line = 10
    2. Indentation (pt) = 305
    3. Field Width (pt) = 250
    4. Description = (underscore) ___________________________________________
    5. Type = Text
    6. Text = (underscore) ____________________________________________________
