Create a New Version of a Layout

In this example, a new copy of the Sales Quote Document Layout will be created.

Creating a new version is useful for making changes to the layout at the same time as users print the “old” version. When it is time to publish the new version, certify the Document Layout and close the old version. Versions can also have Starting and Ending Dates, which allows for setting up a layout that will replace another at a certain date.

Create New Layout Version

The new Sales Quote document layout version will be named SQ-02. The same information as in SQ-01 will be entered in the header of the new Document Layout and then the Sections (and all the setup related to the Sections) will be copied into the new from the old version.

Follow these steps to create a new layout version:

  1. On the Document Layouts ribbon, click New.
  2. In the No. field, type SQ-02.
  3. Enter the following:
    1. Description = New Sales Quote Layout
    2. Top DataItem Table No. = 36
    3. Report No. = 12032015
    4. Render Report No. = 12032027
    5. Document Label Code = SALESQUOTE-HEADER

: New document layout for Sales Quote

  1. On the ribbon, click Copy From.
    The Document Layout List page opens, displaying only the document layouts that use the same Top DataItem Table No. specified in the new layout.

: Document Layouts available for copy to the new Sales Quote layout

  1. Select the line for SQ-01, Sales Quote and click OK.
    The following message appears, confirming that the copy process will replace the Sections in the new layout with existing layout.

: Copy confirmation message

  1. Click Yes.

: New Document Layout with sections copied from Document Layout SQ-01

  1. Change the Status to Certified to make it available to the users.
  2. Click OK to close the Document Layout page.
