Example – Add Formatted Text to Sales Order Confirmation

In this example, special formatted text from cc|formatted documents is added to the cc|text module header, footer and lines in a new sales order through manual entry, opening of an HTML file and import of an image.

For this example, a new Text Module Print Management was created for usage with Sales Order Confirmations.

: Text Module Print Management setup for use with Order Confirmation documents

Note

The picture attached to item 1920-S, ANTWERP Conference Table has been deleted for this example.

Create a Sales Order and Add Header Text

Follow these steps to create a Sales Order and add header text:

  1. In the navigation pane, click Home > Sales Orders.
  2. On the ribbon, click New and then press ENTER or TAB.
  3. In the Sell-to Customer No. field, enter 01454545.
  4. On the ribbon, click Navigate > Header Texts.

: Edit Text page for header text

  1. In the Edit Text page, click Word Processing on the ribbon.
  2. In the first line of the Text Editor window, type “Dear Ms. McDonald,”.
  3. Press ENTER twice and then type “We are pleased to receive your order. Our logistic department will get in contact with you soon to arrange the delivery terms.”
  4. Highlight the words “delivery terms” and then click the Bold icon on the ribbon.

: Header text with formatting added to sales order

  1. Click OK to close the Text Editor.
    The text line information appears in the Edit Text page as Embedded Text. If the “Word Processing (with Line Conversion)” function had been used, the formatted text would have been converted into cc|text module text lines that can be printed without the cc|formatted documents report renderer, but is thereby losing its formatting.
  1. In the Print Mgt. Code field, enter S-ORDERS.

: Header text and Print Management Code for the Sales Order Confirmation added to Edit Text page

  1. Click OK to close the Edit Text page.

Add Footer Text to Sales Order

Follow these steps to add footer text to the Sales Order:

  1. On the Navigate tab, click Footer Texts.
  2. In the Edit Text page, click Word Processing on the ribbon.
  3. On the Start tab, click the Open File icon.
  4. Locate and select the signature file. The text and signature from the file is added to the Text Editor. Notice that the file information appears in the status bar.

: Signature File added to the footer text of the Sales Order

  1. Click OK to close the Text Editor. The text line information appears in the Edit Text page as Embedded Text.
  2. In the Print Mgt. Code field, enter S-ORDERS.

: Footer text and Print Management Code for the Sales Order Confirmation added to Edit Text page

  1. Click OK to close the Edit Text page.

Add Items, Graphic and Formatted Text to Sales Order Lines

Follow these steps to add items, a graphic and formatted text to the Sales Order lines:

  1. On the Lines FastTab, enter a line for six units of item 1980-S to the BLUE location.
  2. Go to the next line and enter one unit of item 1920-S to the BLUE location.
  3. With the line selected for item 1920-S, click Line > Line Texts on the Lines FastTab.
  4. In the Edit Text page, click Word Processing.
  5. On the ribbon, click the Insert tab and then click Image.
  6. Locate and insert the graphic of the ANTWERP Conference Table.
  7. On the ribbon, click the Start tab.
  8. Go to the next line and type “Both practical and stylish, this all-in-one table is professional enough for meetings, durable enough for luncheons. Abrasion- and stain-resistant laminate wipes clean for easy care. Adjustable levelling glides help compensate for uneven floors.”
  9. Highlight the “Abrasion- and stain-resistant laminate wipes clean for easy care” text and then click the Fore Color icon on the ribbon.
  10. In the fifth column, select the purplish-blue color in the second row from the bottom and then click the Bold icon on the ribbon.
  11. Press ENTER twice and then type “Features”.
  12. Press ENTER.
  13. In the ribbon, click the Unordered List icon and then type “All-in-one table is professional enough for meetings, durable enough for luncheons”
  14. Press ENTER and then type “Abrasion- and stain-resistant laminate wipes clean for easy care”
  15. Press ENTER and then type “Adjustable levelling glides help compensate for uneven floors”
  16. Highlight the “Features” text above the bulleted list, click the Heading icon on the ribbon and then select Heading 3.

: Line text with formatting added to sales order

  1. Click OK to close the Text Editor. The text line information appears in the Edit Text page as Embedded Text.
  2. In the Print Mgt. Code field, enter S-ORDERS.

: Line text and Print Management Code for the Sales Order Confirmation added to Edit Text page

  1. Click OK to close the Edit Text page.

 

Run the Formatted Documents Order Confirmation Report

Follow these steps to run the Formatted Documents Order Confirmation report:

  1. On the ribbon of the Sales Order, click Actions > Print Confirmation.
  2. In the report request page, set the Document Layout No. to OC-01-TM.

: DC - Order Confirmation report request page

  1. Click OK to run a preview of the report.

: Sales Order Confirmation document with formatted header, line and footer text
