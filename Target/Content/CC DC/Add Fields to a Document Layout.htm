Add Fields to a Document Layout Section

This example uses the SQ-01 (Sales Quote) Document Layout to add the Campaign No. field and a field label to the header.

Note

Before starting this example, ensure that a Sales Quote exists with an assigned campaign number.

Open the Document Layout Fields

Follow these steps to open the Document Layout Fields page:

  1. In the navigation pane, click Departments > cc|document configurator > Configuration > Document Layouts.
  2. Locate and open the SQ-01, Sales Quote, Document Layout.
  3. Change the Status to Under Development.
  4. Select the line in the Sections FastTab where the field needs to be added. For this example, select the first line for Section Type = Header.
  5. Click Line > Fields to open the Document Layout Fields page.

: Selecting the Fields menu item for the Header of Document section

Insert Three Lines with Positions

Three lines will be inserted above the line for the “Customer No. ### Label” field. These lines will be used to add one field to hold the label, one field to hold the value and one field to create a blank line.

When adding a field, the position it will have is specified in the Line and Indentation (pt) fields. The Line represents the vertical position and the Indentation (pt) represents the horizontal position in the Section. In this example, the lines will be set to have them just below each other, and the indentation to have them lined up in the right side of the page.

Follow these steps to insert the three lines with positions:

  1. Locate the line for the “Customer No. ### Label” field and insert a new line above it.
  2. In the Line field, enter 10.
  3. In the Indentation (pt) field, type 250.
  4. Insert a line below the new line.
  5. In the Line field, enter 11.
  6. In the Indentation (pt) field, type 250.
  7. Insert a line below the new line.
  8. In the Line field, enter 12.
  9. In the Indentation (pt) field, type 250.

: New lines added to the Document Layout Fields

Renumber Field Positions

Fields are generated one-by-one in the order as they are listed. This means that if two or more lines are at the same positions, then the later fields might overwrite the first fields. Therefore, the positions of the lines below the new lines need to be changed by incrementing the value of the Line field.

Follow these steps to renumber the field positions:

  1. In the Line field for “Customer No. ### Label”, type 13.
  2. On the remaining lines with overlapping numbers (Phone No., Fax No. and E-mail), increment the value in the Line field as illustrated in Fig. 3.

: Renumbered lines in the Document Layout Fields

Define Field Information

The remaining information in the line defines what should be printed in the fields and how it should appear in the printout.

Note

It might be necessary to create a Document Text Code for CAMPAIGNNO set up with {local language} = Campaign No.

Follow these steps to define the remaining field information:

  1. On the first new line (Line = 10), enter the following:
    1. Description = Campaign No. Label
    2. Type = Document Text
    3. Document Text Code = CAMPAIGNNO
    4. Font Size = 0
    5. Font Weight = Bold
  2. On the second new line (Line = 11), enter the following:
    1. Description = Campaign No.
    2. Type = Value
    3. Table No. = 36
    4. Field No. = 5050
      There is no need to specify a filter since it will be from the same record on which the section is based.
  3. On the third new line (Line = 12), enter the following:
    1. Description = Empty Line
    2. Type = Text

: New lines in the Document Layout Fields

  1. On the ribbon, click Run Report to view the report changes.
  2. Select the Show Request Page check box and then click OK.
  3. In the request page, click Preview.

: New lines added to the Document Layout appear in the header of the Sales Quote report

  1. Close the Print Preview page.
  2. Click OK to close the Document Layout Fields page.
  3. Change the Status to Certified to make it available to the users.

Note

It is helpful to export a Document Layout before making changes. That way, the original can be restored if needed. Another option is to create a new version of the Document Layout (refer to topic “Create a New Version of a Layout” on page 1 for more information).
