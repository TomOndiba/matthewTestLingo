Introduction

Built on Microsoft Dynamics NAV, cc|document configurator simplifies the process of modifying RoleTailored Client printouts.

cc|document configurator takes the design work from the native report designer and Visual Studio and brings it into a setup environment in Microsoft Dynamics NAV. This allows customers and consultants to do modifications without any programming skills; but remains flexible enough for programmers to write special code through codeunits, if needed.

cc|document configurator comes with more than 20 predefined layout templates in more than 20 different languages that are ready to use as they are or as a base for making further modifications. This add-on was developed as a standalone solution that requires minimal modifications to the standard objects.

Key Features

  • Reduces implementation time and cost
  • No programming skills required
  • Design documents without Visual Studio or the native report designer
  • Allows end users to take ownership of designing the printouts
  • More than 20 ready to use “out of the box” documents
  • Multilingual support
  • Company specific layouts
  • Multiple layouts for a single document

cc|document configurator functionalities are accessed in the cc|document configurator menu.
