Document Layouts with cc|formatted documents

cc|formatted documents is integrated into the Document Layout lines using fields in the Document Layout Fields page.

: Document Layout Fields for Lines in document layout with cc|formatted documents integration

Document Layout Fields

The fields/options in the Document Layout Fields page that are used with cc|formatted documents are described in the following table.

Field

Option

Description

Type

Subreport (via cc|formatted documents)

Provides the ability to insert a subreport into the current position within the Document Layout.

Subreports can only be printed together with cc|formatted documents and are configured by entering the report number in the Subreport No. field.

BLOB Field Usage

 Formatted Documents

Defines the rendering of a BLOB field; used e.g. in the cc|text module Document Layout templates (header, footer and lines), to switch between normal text or the cc|formatted documents BLOB text. When this option is selected, the BLOB field will be rendered as formatted text.

This option only works in combination with flow layouts and cc|formatted documents, not with the classic layouts.

Subreport No.

  

Used with Type = Subreport (via cc|formatted documents) to identify the report object ID of the subreport to be printed on the current position.

This field only works in combination with flow layouts and cc|formatted documents, not with the classic layouts.

Refer to topic “Document Layout Fields” on page 1 for more information about the other fields in this page.

Run Formatted Documents Report

The Document Layout page contains the “Run Formatted Documents Report” function, which uses the “Run Report” function with the cc|formatted documents report renderer.

: Run Formatted Documents Report function on Document Layout

Documents that are generated using the “Run Formatted Documents Report” function include any special formatting added to the document using cc|formatted documents.
