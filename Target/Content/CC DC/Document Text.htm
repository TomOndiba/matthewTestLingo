Document Text

Different texts and languages are managed through Document Texts. A Document Text carries translations to different languages and allows for making changes to labels and texts used on all printouts in one place. cc|document configurator comes preloaded with Document Texts in more than 20 different languages (for texts used on the predefined Document Layouts).
