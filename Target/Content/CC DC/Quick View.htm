Quick View

The quick view options in the Document Layout page is designed to be used while configuring a layout. It is a quicker way of seeing the result of the configuration instead of using the report viewer.

There are two quick view options:

  • Quick View Layout – shows the layout with content from the first page of the last record
  • Quick View Field Positions – shows information about the positions of the fields

When a quick view option is used, the system displays a Document Layout Quick View card with a bitmap representation of the layout.

: Quick View Layout

Right-click the image to copy or save the bitmap.

Double-click the image to open it in the default picture viewer.

: Double-clicked image in the Document Layout Quick View for Layout

When the “Quick View Field Positions” function is run, the information in the fields are the Indentation followed by a L, C or R to indicate the horizontal alignment and then the field width.
