Flow Layout FastTab

The fields on this FastTab are described in the following table.

Field

Description

Show Design Marks

Determines if the design marks, such as fold marks and address border, appear.

Note

This field only works in combination with flow layouts, not with the classic layouts.

Carry Over Label Code Header

Indicates the carry over label caption to be used for the header label on the cc|document configurator reports. Refer to topic “Add Carry Overs for Document Sums” on page 1 for more information.

Carry Over Label Code Footer

Indicates the carry over label caption to be used for the sum label at the end of the cc|document configurator report. Refer to topic “Add Carry Overs for Document Sums” on page 1 for more information.
