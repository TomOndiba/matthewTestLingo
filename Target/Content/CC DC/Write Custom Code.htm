Write Custom Code

Developers can write C/AL code in Codeunits that are run when running the report. Creating Codeunits that will work together with cc|document configurator requires some special code in order to retrieve the record being processed. To simplify this process, the Sample DMP Database includes a Codeunit that can be copied and used as a starting point, named “DC - Template Codeunit” (50000). Alternatively, refer to the following screen shots for information about the special code.

The RecRef variable in this Codeunit is the Record currently being processed, and from there on regular C/AL code can be written.
