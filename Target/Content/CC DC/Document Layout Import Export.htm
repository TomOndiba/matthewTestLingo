Document Layout Import/Export

The base document layouts included with cc|document configurator can be imported by running the “Import Document Layout” function in the RoleTailored Client.

  1. In the navigation pane, click Departments > cc|document configurator > Export and Import > Import Document Layout.
  2. Browse to the DocumentLayouts_DCXXXX.xml file and then click Open.

To export document layouts, use the same path as in step 1 except choose Export Document Layout and then browse to the location to save the export file.
