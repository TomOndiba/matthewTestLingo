Set Up Document Layout

After the new report is created through the Object Designer, a new Document Text is created for Item Report, and a new Document Layout is created with No. = ITEM-01 and Description = Item Report. The document layout is set up to be printed from the Item record, links to the new Item Printout report and the DC - A4 Portrait render report.

Create Document Text

Follow these steps to create the document text for the new report:

  1. In the RoleTailored Client, click Document Texts in the Configuration menu.
  2. On the ribbon, click New.
  3. Enter the following:
    1. Code = ItemReport
    2. Text = Item Report
    3. Language Code = {local language}
    4. Text = {text for Item Report in local language}
  4. Click OK to close the Document Text page.

Create Document Layout

Follow these steps to create the document layout for the new report:

  1. In the Configuration menu, click Document Layouts.
  2. On the ribbon, click New.
  3. In the No. field, type ITEM-01.
  4. Enter the following:
    1. Description = Item Report
    2. Top DataItem Table No. = 27
    3. Report No. = as noted in step 18 of the previous section
    4. Render Report No. = 12032027
    5. Document Label Code = ITEMREPORT

: New Document Layout for new Item Printout report

Add Sections

The Sections of the document layout are set up with the following: Item table as the header (sections almost always start with the same table from which the report is run), Production BOM Header table, Production BOM Lines table, Routing Header table, Routing Line table and the Integer table to print a footer. Certain Sections are indented to create the table relation hierarchy.

Follow these steps to add Sections to the document layout:

  1. On the first line of the Sections FastTab, enter the following:
    1.  Section Type = Header
    2.  Description = Header
    3.  Table No. = 27
  2. On the next line, enter the following:
    1. Section Type = Line
    2. Description = Header for BOM
    3. Table No. = 99000771
  3. On the next line, enter the following:
    1. Section Type = Line
    2. Description = BOM Lines
    3. Table No. = 99000772
  4. On the next line, enter the following:
    1. Section Type = Line
    2. Description = Header for Routing
    3. Table No. = 99000763
  5. On the next line, enter the following:
    1. Section Type = Line
    2. Description = Routing Lines
    3. Table No. = 99000764
  6. On the next line, enter the following:
    1. Section Type = Footer
    2. Description = Footer
    3. Table No. = 2000000026
  7. Select the second line and click Line > Move Right.
  8. Select the third line and click Line > Move Right.
  9. Repeat for the third line so it is indented below the second line.
  10. Select the fourth line and click Line > Move Right.
  11. Select the fifth line and click Line > Move Right.
  12. Repeat for the fifth line so it is indented below the fourth line.

: Sections added to the Document Layout for the new Item Printout report

Set Filters on Sections

Filters need to be set for the Sections to define the relations between the tables. The first line connects the Item header to the Top DataItem Table (specified in the header of the Document Layout).

Follow these steps to add filters to the sections:

  1. Select the first section line “Header” and then click Line > Filters.
  2. On the first line, enter the following:
    1. Field No. = 1
    2. Filter Type = Value
    3. Filter Field No. = 1

: Filter added to the Header section of the new ITEM-01 document layout

  1. Close the Document Layout Filters page.
  2. Select the second section line “Header for BOM” and then click Line > Filters.
  3. On the first line, enter the following:
    1. Field No. = 1
    2. Filter Type = Value
    3. Filter Field No. = 99000751

: Filter added to the Header for BOM section of the new ITEM-01 document layout

  1. Close the Document Layout Filters page.
  2. Select the third section line “BOM Lines” and then click Line > Filters.
  3. On the first line, enter the following:
    1. Field No. = 1
    2. Filter Type = Value
    3. Filter Field No. = 1
  4. On the next line, enter the following:
    1. Field No. = 3
    2. Filter Type = Text
    3. Text = ‘’ <blank>

Note

A blank (‘’) filter is specified on the Version Code to omit BOM versions from printing.

: Filters added to the BOM Lines section of the new ITEM-01 document layout

  1. Close the Document Layout Filters page.
  2. Select the fourth section line “Header for Routing” and then click Line > Filters.
  3. On the first line, enter the following:
    1. Field No. = 1
    2. Filter Type = Value
    3. Filter Field No. = 99000750

: Filter added to the Header for Routing section of the new ITEM-01 document layout

  1. Close the Document Layout Filters page.
  2. Select the fifth section line “Routing Lines” and then click Line > Filters.
  3. On the first line, enter the following:
    1. Field No. = 1
    2. Filter Type = Value
    3. Filter Field No. = 1
  4. On the next line, enter the following:
    1. Field No. = 2
    2. Filter Type = Text
    3. Text = ‘’ <blank>

: Filters added to the Routing Lines section of the new ITEM-01 document layout

  1. Close the Document Layout Filters page.
  2. Select the last section line “Footer” and then click Line > Filters.
  3. On the first line, enter the following:
    1. Field No. = 1
    2. Filter Type = Text
    3. Text = 1

: Filter added to the Footer section of the new ITEM-01 document layout

  1. Close the Document Layout Filters page.

Defining the Sections and the related filters are similar to setting up the DataItems in a native report.
