Document Layout Codeunits

Document Layout Codeunits are used to run Codeunits at the time of printing. This is useful for updating data or doing special calculations when printing. The codeunits specified here will run once for each record being iterated.

: Document Layout Codeunits

The fields on this page are described in the following table.

Field

Description

Codeunit No.

Indicates the codeunit number to run for the section.

Codeunit Name

Displays the name of the codeunit.

Refer to topic “Write Custom Code” on page 1 for more information about how to develop a Codeunit to run with cc|document configurator.
