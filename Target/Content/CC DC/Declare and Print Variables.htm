Declare and Print Variables

In this example, the Balance from the Customer Card will be assigned to a variable and the Variable will be used for displaying the amount on the Sales Quote printout.

The new Variable called “CustomerBalance” will be added to the Header of Document section in the layout. The value from the Balance field (58) in the Customer table (18) will be assigned to the Variable. Since this is a decimal value, the “<Precision,2:2><Standard Format,0>” Format String will be used to format the value with two decimals.

Add Variable to the Document Layout Section

Follow these steps to add a variable to a document layout section:

  1. Open the SQ-01, Sales Quote, Document Layout.
  2. Change the Status to Under Development.
  3. With the first line selected in the Sections FastTab, click Line > Variables to open the Document Layout Variables page.
  4. In the line below “CustomerNo”, enter the following:
    1. Variable = CustomerBalance
    2. Type = Value
    3. Table No. = 18
    4. Field No. = 58
    5. Format String = <Precision,2:2><Standard Format,0>

: Variable added to Document Layout section

Add Filter to Variable

As with fields, filters must be specified if using tables other than the table used in the Section. Specifying the filters for the Customer table is done in the Document Layout Filter page; where a filter is defined for the No. field in the Customer table to be equal to the Bill-to Number of the Sales Header table.

Follow these steps to add filters to the variable:

  1. With the line for “CustomerBalance” selected, click Filters on the ribbon.
  2. On the line, enter the following:
    1. Field No. = 1
    2. Filter Type = Value
    3. Filter Field No. = 4

: Filter added to the Variable

  1. Click OK to close the Document Layout Filters page.
  2. Click OK to close the Document Layout Variables page.

Specify Fields

The last step is to define the Document Layout Field to print the Variable just below the Bill-to Name. The Type = Text because variables are always entered within square brackets in the Text field (e.g., “Outstanding Balance: [CustomerBalance]”). This is valid for both Document Layout Fields and Document Layout Filters.

Follow these steps to specify the fields to print the variable:

  1. With the first line selected, click Line > Fields on the Sections FastTab.
  2. Insert a new line below the Bill-to Name line.
  3. Enter the following:
    1. Line = 17
    2. Description = Amount from Customer Card
    3. Type = Text
    4. Text = Outstanding Balance: [CustomerBalance]

: Field with variable text added to the section

Note

Entering “Outstanding Balance: [CustomerBalance]” in a Text field will not make it multilingual enabled. To do this, Document Texts must be used. Variables can also be used in the Text field of Document Texts by using square brackets.

  1. On the ribbon, click Run Report to view the report changes.
  2. Select the Show Request Page check box and then click OK.
  3. In the request page, click Preview.

: Variable added to the report

  1. Close the Print Preview page.
  2. Click OK to close the Document Layout Fields page.
  3. Change the Status to Certified to make it available to the users.
  4. Click OK to close the Document Layout page.
