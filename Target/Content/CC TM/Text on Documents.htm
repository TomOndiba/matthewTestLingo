Text on Documents

Once the text has been assigned to the master data items, it can be used on the documents. In this section, the sales order is used to review text setup and the Order Confirmation report is used to preview the document text.

Document Text Review

When a document, such as a sales order, is assigned to a customer that has been assigned text, a page appears to display the header text, the footer text, or both.

: Customer-related header and footer Text Module notification on the sales order

Click Close to close the Text Info page.

The text relating to the customer can be accessed on the Sales Order by clicking Navigate > Header Texts or Footer Texts.

: Accessing customer-related text

To review the text, click Header Text or Footer Text on the ribbon to open the Edit Text page.

: Edit Text page for the customer header text

In the Edit Text page, existing text can be viewed or edited, or new text can be added (refer to topic “Edit Text Page” on page 1 for more information).

Notification pages also appear when items with text are added to a document.

: Item-related Text Module notification on the sales order

On the Lines FastTab, click Line > Line Texts to access the assigned item-related text.

: Accessing item-related text

Similar to the customer text, the item text can be viewed and modified in the Edit Text page.

: Edit Text page for the item text

Edit Text Page

As previously stated, the Edit Text page is used to view or modify existing text or to add new text. This page also includes the Print Code field, which is used to filter the lines by a Print Mgt. Code if multiple lines exist with different codes. The fields in the lines are the same as in the Lines FastTab on the Text Module Card (refer to topic “Lines FastTab” on page 1 for descriptions of these fields).

: Actions on the Edit Text page

The Actions tab on the Edit Text page includes functions for copying and reverting changes to text. Each is described in the following table.

Action

Description

Copy Text

Copies the text and all settings of the selected lines to the Clipboard.

Paste Text Above

Pastes the copied text from the Clipboard above the selected line.

Paste Text Below

Pastes the copied text from the Clipboard below the selected line.

Paste Above

Pastes standard text above the selected line (refer to topic “Standard Text” on page 1).

Paste Below

Pastes standard text below the selected line (refer to topic “Standard Text” on page 1).

Undo Changes

Reloads the text from the document and removes all changes made since opening the text.

For example, text has been assigned to a customer card. A user creates a sales order for this customer, opens the Edit Text page and accidentally deletes a line. Clicking Undo Changes will revert to the text that existed when the Edit Text page was opened.

 

Exercise – Create Sales Order and Review Test on Order Confirmation

On January 24, 2019, Andy Teal from customer 10000, The Cannon Group PLC, places an order for one unit of item 1936-S.

Create the Sales Order

Follow these steps to create the sales order:

  1. In the navigation pane, under Sales & Marketing, click Order Processing > Sales Orders.
  2. On the ribbon, click New and then press ENTER or TAB.
  3. In the Sell-to Customer No. field, enter 10000.
  4. Press ENTER or TAB.
  5. Review and then close the Text Info page that displays the customer header/footer text.
  6. On the Lines FastTab, enter Item in the Type field.
  7. In the No. field enter 1936-S and then press ENTER.
  8. Review and then close the Text Info page that displays the item text.
  9. In the Location Code field, enter BLUE.
  10. In the Quantity field, type 1.

Report Text Review

Once the report is ready, it can be printed or previewed with the assigned text.

Exercise – Review Text on the Order Confirmation

Andy Teal receives an order confirmation when he places an order. Preview the Order Confirmation and confirm the text is printed correctly.

Preview the Order Confirmation

Follow these steps to preview the Order Confirmation:

  1. On the Sales Order, click Home > Print Confirmation.
  2. Click Preview.

: Order Confirmation with customer and item text

  1. Note the following:
    • Above the item information, the greeting to Mr. Teal appears (bolded and underlined).
    • Under the item description, the item text appears (bolded and red; italicized, bolded, underlined and black).
    • Under the VAT information, the closing to Mr. Teal appears (italicized and blue).
  2. Close the Print Preview page.

Adjust Text Settings

Follow these steps to remove the item text from the document:

  1. On the Lines FastTab, click Line > Line Texts.
  2. Clear the Print Mgt. Code field on both lines.

: Removal of print management code for item text on the Order Confirmation

  1. Click OK to close the Edit Text page and then escape back to the Sales Order.
  2. On the Sales Order ribbon, click Print Confirmation.
  3. Click Preview.
  4. Note that the customer-related header/footer text is displayed, but the item text is not.

: Order Confirmation with customer text only

  1. Close the Print Preview page.
  2. Reactivate the item-related text on the Order Processing report and then preview to confirm that it is displayed.
