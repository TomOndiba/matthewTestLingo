Standard Sales Text / Standard Purchase Text

With the Sales Text functionality, users can enter standard texts to be printed and displayed respectively in sales documents/pages. The Purchase Texts functionality offers the ability to predefine standard texts to be printed and displayed respectively in purchase documents/pages.

Setup

Standard Sales/Purchase text is created in the Text Module Sales Text Card or Text Module Purchase Text Card page, respectively. These pages are accessed from the cc|text module menu by clicking Standard Texts > Sales Texts or Purchase Texts.

The setup of standard sales and purchase text is generally the same as for other text with the exception of the usages text, which is described in the following section.

Usages Text

Usages text limits for which customer/vendor that Standard Sales/Purchase Text should be copied to a document. Access the Text Module Usages page by clicking Navigate > Use for on the Sales Text Card or Purchase Text Card.

: Use for menu item on Text Module Sales Text Card

In the Use for Type field, users can set limits on several customers/vendors, a whole Country Code, or set filters on fields in the customer/vendor records. In the Use for Code field, the appropriate customers and vendors, respectively, are defined for non-filter entries. The Name field displays the selected customer, vendor or filter field caption.

: Text Module Usages page displaying Use for Type options

Field Filters

To set up field filters, the Use for Type field must be set to Customer (Field Filter) or Vendor (Field Filter), respectively. In the Field Filter No. field, select the field number on which the filter is to be set, and in the Field Filter Term field, enter the filter expression.

For example, to define a text that should be copied for all customers/vendors, insert one entry with Use for Type = Customer (Field Filter) and one entry for Use for Type = Vendor (Field Filter). On each entry, set the Field Filter No. = 1 and Field Filter Term = * (‘*’ is a wild card filter).

Exercise – Set Up Standard Purchase Text and Use on Purchase Order

Your company has decided to notify all vendors of the impact on vendor rating when damaged packages are received. This notification will be in the footer of all purchase orders. It will be created in English (ENG) and some other languages, but the English text must be used if translated text is not available. Instead of setting this text on each vendor, standard purchase text is used.

Define Standard Purchase Text

Follow these steps to define standard purchase text:

  1. On the cc|text module menu, click Standard Texts > Purchase Texts.
  2. On the ribbon, click New.
  3. In the Language Code field, enter ENG.
  4. Select the All Languages check box.
  5. In the Description field, type “Damaged packages”.
  6. In the Header/Footer field, enter Footer.
  7. In the first line of the Text field, type “In case of damaged packaging, the entire delivery will be returned”.
  8. Select the Bold check box and in the Color field, click the arrow and select Red.
  9. In the Print Mgt. Code field, enter P-ORDERS.
  10. In the next line, type “and a negative entry will be created in our vendor rating system.”
    Note that the line format from the above text line transferred to the next text line.

: Assigned default settings for Standard Purchase Text

  1. Click OK to close the Text Module Purchase Text Card page.

Once setup is complete, these standard texts will appear directly on documents.

Create Purchase Order and Review Text

You place an order to vendor 10000 for 50 units of item 1900-S. Print the order and confirm that the standard purchase text appears.

Follow these steps to create the purchase order and preview the Purchase Order document:

  1. In the navigation pane, click Purchasing > Order Processing > Purchases Orders.
  2. In the Buy-from Vendor No. field, enter 10000 and then press ENTER or TAB.

: Standard Purchase Text notification on the purchase order

  1. Click Close to close the Text Info page.
    The standard text can be accessed on the Purchase Order by clicking Navigate > Header Texts or Footer Texts.

: Edit Text page for the Standard Purchase Text opened from the purchase order

  1. Click OK to close the Edit Text page.
  2. On the Lines FastTab, enter Item in the Type field.
  3. In the No. field enter 1900-S.
  4. In the Location Code field, enter BLUE.
  5. In the Quantity field, type 50.
  6. On the ribbon, click Home > Print.
  7. Click Preview.
    Note that the standard purchase text appears in the footer (bolded red).

: Standard Purchase Text on the Purchase Order document (bolded red)

  1. Close the Print Preview page.
