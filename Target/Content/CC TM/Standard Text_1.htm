Standard Text

Standard Text is the general text that can be used in multiple areas.

Setup

General standard text is created in the Text Module Standard Text Card page – accessed from the cc|text module menu by clicking Standard Texts > Standard Text – and is set up in the same way as in the Text Module Card page (refer to topic “Text Module Card” on page 1).

: Text Module Standard Text Card

Once setup is complete, this standard text can be brought into master data text using the “Get Standard Text” function on the Text Module Card by clicking Navigate > Get Standard Text or into documents using the “Paste Above” or “Paste Below” functions on the Edit Text page.

Note

Language and date ranges are not considered when these functions are used; all standard text will be available for selection.

Exercise – Use Standard Text on a Customer

Your company has decided to use generic text functionality. You create the standard text and then run the “Get Standard Text” function on text for customer 20000 to access the new text.

Define Standard Text

Follow these steps to define standard text:

  1. On the cc|text module menu, click Standard Texts > Standard Texts.
  2. On the ribbon, click New.
  3. In the Language Code field, enter ENG.
  4. In the Description field, type “Thank you”.
  5. In the Header/Footer field, enter Footer.
  6. In the first line of the Text field, type “Thank you for your order. We appreciate your business.”
  7. Select the Italic and Bold check boxes.
  8. In the Color field, click the arrow and select Green.
  9. In the Print Mgt. Code field, enter S-ORDERS.

: Assigned default settings for Standard Text

  1. Click OK to close the Text Module Standard Text Card page.

Run the Get Standard Text Function on Customer Text and Create Sales Order

You create a new text record for customer 20000 using the “Get Standard Text” function to access the generic text and then create an order for five units of item 1900-S. Preview the order confirmation and confirm that the standard text appears.

Follow these steps to create the customer text, run the function, create the order and preview the Order Confirmation document:

  1. In the navigation pane, click Sales & Marketing > Order Processing > Customers.
  2. Locate customer 20000 and open the customer card.
  3. Click Navigate > Text Module.
  4. On the ribbon, click New.
  5. In the Language Code field, enter ENG.
  6. In the Header/Footer field, enter Footer.
  7. On the ribbon, click Navigate > Get Standard Text to open the Text Module List page with the available standard texts.

: Accessing and running the “Get Standard Text” function from the Text Module Card on a customer

  1. With the line for the “Thank you” text selected, click OK.

: Standard Text added to the Text Module Card

  1. Click OK to close the Text Module Card page.
  2. On the ribbon of the Customer Card, click Home > Sales Order and then press ENTER or TAB.

: Standard Text in the footer text of the sales order

  1. Close the Text Info page.
  2. On the Lines FastTab, enter Item in the Type field.
  3. In the No. field enter 1900-S.
  4. In the Location Code field, enter BLUE.
  5. In the Quantity field, type 5.
  6. On the ribbon, click Print Confirmation.
  7. Click Preview.
    Note that the standard text appears in the footer (italicized, bolded and green).

: Order Confirmation document with standard text in bolded green italics

  1. Close the Print Preview page.

Exercise – Use Standard Text on a Sales Order

Your company also uses the generic text functionality directly on sales orders. You create an order for customer 30000 for five units of item 1900-S and create footer text from the “Thank you” Standard Text and modify the settings. Print the order confirmation and confirm that the standard text appears.

Follow these steps to create the sales order, assign the text, and preview the Order Confirmation:

  1. In the navigation pane, click Sales & Marketing > Order Processing > Sales Orders.
  2. On the ribbon, click New and then press ENTER or TAB.
  3. In the Sell-to Customer No. field, enter 30000 and then press ENTER or TAB.
  4. On the Lines FastTab, enter Item in the Type field.
  5. In the No. field enter 1900-S.
  6. In the Location Code field, enter BLUE.
  7. In the Quantity field, type 5.
  8. On the ribbon, click Navigate > Footer Texts.
  9. On the ribbon, click Actions > Paste Above or Paste Below to open the Text Module List page with the available standard texts.

Note

If text exists in the Edit Text page, the standard text can be pasted above or below the selected line. If no text exists, either function will add the standard text to the page.

: Accessing the Text Module List from the Edit Text page on a sales order

  1. Select the line for the “Thank you” text and then click OK.
  2. Clear the Italic check box and in the Color field, click the arrow and select Blue.
  3. On the next line, type “We look forward to serving you again soon!”
  4. Select the Italic check box and clear the Bold check box.

: Standard Text added and settings changed in the Edit Text page

  1. Click OK to close the Edit Text page.
  2. On the ribbon, click Home > Print Confirmation.
  3. Click Preview.
    Note that the modified standard text appears in the footer (bolded/blue, italics/blue).

: Order Confirmation document with standard text

  1. Close the Print Preview page.
