Time Sheet Integration

The integration of the standard Microsoft Dynamics NAV Time Sheet functionality to cc|project allows users to enter their time in a simple way. The integrated approval process must be used. Additionally, all approved Time Sheet entries from the Time Sheet Manager are captured and inserted to a Job Time Entry Job Journal for posting.

Setup

This section provides an overview of how to set up a user to approve Time Sheet entries and a user to make entries in a Time Sheet.

Approval User Setup

Follow these steps to set up the approval user:

  1. On the navigation pane, click Departments > Administration > Application Setup > Users > User Setup.
  2. For the user who needs to approve Time Sheets, select the Time Sheet Admin. check box.

: Time Sheet approval user selected

  1. Click OK.

Set Up User for Time Sheet Entries

Follow these steps to set up the user to make entries in the Time Sheet:

  1. On the navigation pane, click cc|project> Administration > Resources.
  2. Locate and open the card of the resource that needs to be use Time Sheets.

: Highlighting Time Sheet fields on Resource Card

  1. Select the Use Time Sheet check box to enable Time Sheet usage for the resource.
  2. In the Time Sheet Owner User ID field, click the arrow and select the resource from the User Setup table.
  3. In the Time Sheet Approver User ID field, click the arrow and select the resource responsible for approving Time Sheet entries.

: Resource set up for Time Sheet usage and approval

  1. Click OK.

Create Time Sheet and Submit Entries

Once the users are set up, a Time Sheet can be created for the resource and time can be entered directly for a job line, and then submitted for approval.

Create Time Sheet

Follow these steps to create a Time Sheet:

  1. Login as the user to create a Time Sheet.
  2. On the navigation pane, click Home > Time Sheets.
  3. On the ribbon, click Create Time Sheets.
  4. In the Options FastTab, fill in the fields as needed.
  5. In the Resource FastTab, set the user filter.

: Create Time Sheets filtered on resource P002

  1. Click OK.
    A message appears stating the number of time sheets that have been created.
  2. Click OK to close the message.

Enter and Submit Time

Follow these steps to enter and submit time in the Time Sheet:

  1. On the ribbon of the Time Sheets page, click Edit Time Sheet.

: Open the new Time Sheet for editing

  1. In the Time Sheet page, enter Job in the Type field.
  2. In the Job No. field, enter the job to which time needs to be entered.
  3. In the WBS field, enter the job line to which the time needs to be entered.
    The Description field is populated with the description from the job line.

: Job information entered in the Time Sheet

  1. In the day fields (e.g., 28 Mon, 29 Tues, and so on), enter the hours worked for the selected job line.

: Hours worked for the job entered in the Time Sheet

The Time Sheet Status and Actual/Scheduled Summary FactBoxes are updated with the entered time .

  1. To make changes to the line, such as to change the Description, click Activity Details on the ribbon.
  2. In the Time Sheet Line Job Detail page, make changes as needed.

: Description from job line changed to include details

 

  1. Click OK.
    The changes are made to the Time Sheet line.

: Description field updated in Time Sheet line

  1. On the ribbon, click Submit.
  2. A message appears asking to submit all open lines or selected lines only.

: Preparing to submit Time Sheet entry

Note

If the "All open lines" option is selected, any line with Status = Open will be submitted.

  1. With the All open lines option selected, click OK.
    The Status is changed to Submitted, the Time Sheet Status FactBox is updated, and the lines remain in the Time Sheet.

: Submitted Time Sheet

  1. Click OK.

Approve Submitted Time Sheet

After submission, the Time Sheet is approved by the approval manager assigned to the resource.

Approve Time Sheet

Follow these steps to approve a Time Sheet:

  1. Login as the approval user.
  2. On the navigation pane, click Home > Manager Time Sheets.
    Notice that a submitted Time Sheet entry exists for the resource.

: Preparing to open the Submitted Time Sheet

  1. On the ribbon, click Edit Time Sheet.
    The Manager Time Sheet page opens with the same information as in the Time Sheet page, except now the approver’s name appears in the top section, and approval/rejection options are available.

: Manager Time Sheet view of submitted Time Sheet

  1. Additional details about the Time Sheet entry can be viewed by clicking Activity Details in the ribbon.

: Time Sheet Line Job Details

  1. Click Cancel to close the page without making changes.
  2. On the ribbon, click Approve to transfer the entries to the Time Job Journal.
    A message appears asking to approve all submitted lines or selected lines only.

: Preparing to approve Time Sheet entry

  1. With the All submitted lines option selected, click OK.
    The Status is changed to Approved, the Time Sheet Status FactBox is updated, and the lines remain in the Time Sheet.

: Approved Time Sheet

  1. Click OK.

Post Job Time Entry Journal

After approval, the Time Sheet entries are transferred to the Job Time Entry Journal for posting.

Post Time Sheet Entries in Job Time Entry Journal

Follow these steps to post entries in the Job Time Entry Journal:

  1. Login as the Time Sheet Administrator user.
  2. On the navigation pane, click Home > Journals > Time Entry Journals.
  1. On the ribbon, click Edit Journal; it does not matter which resource is selected.

: Preparing to open the Job Time Entry Journal with approved Time Sheet entries

  1. On the ribbon, click Suggest Lines from Time Sheets.
  2. In the Resource No. Filter field, enter the number of the resource with approved Time Sheet entries.

: Suggesting lines from Time Sheets

  1. Enter any other filters to narrow the suggestion, and then click OK.
    The Job Time Entry lines are populated with the approved entries per day.

: Approved Time Sheet entries suggested in Job Time Entry Journal

  1. On the ribbon, click Post.

: Preparing to post the Job Time Entry Journal

  1. Click Yes to post the lines and then OK to close the confirmation message, and then OK to close the Job Time Entry Journal.

The posted entries for Time Sheets are accessible from the Manager Time Sheet view by clicking Posting Entries.

: Time Sheet Posting Entries opened from Manager Time Sheet

Create Time Sheet with Previous Entries

To reduce data entry, it is possible to create a Time Sheet using a previously entered Time Sheet. This is useful when a resource works on the same job line for a long period of time.

Create Time Sheet using Previous Entries

Follow these steps to create a Time Sheet using previous entries:

  1. Login as the user to create a Time Sheet.
  2. On the navigation pane, click Home > Time Sheets.
  3. On the ribbon, click Create Time Sheets.
  4. In the Options FastTab of the Create Time Sheets page, enter the relevant information.
  5. In the Resource FastTab, the set the user filter.

: Create Time Sheets filtered on resource P002

  1. Click OK and then click OK to close the creation message.
  2. Open the new Time Sheet and then click Copy lines from previous time sheet.

: Preparing to copy lines from previous time sheet

  1. Click Yes to copy the lines.

: Previous Time Sheet line copied to new Time Sheet

New time entries can be entered for the copied Time Sheet line, new lines can be added and the Time Sheet submission/approval process continues as previously described.
