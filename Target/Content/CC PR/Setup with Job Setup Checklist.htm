Setup with Job Setup Checklist

The cc|project module is equipped with a setup wizard that can be used for the initial setup.

The Job Setup Checklist is opened from navigation pane by clicking Departments > cc|project > Administration > Setup Checklist.

: Job Setup Checklist page

To access the Job Setup Wizard for line-by-line setup, click Run Wizard by Step on the ribbon of the Job Setup Checklist page.
