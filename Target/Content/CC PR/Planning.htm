Planning

The Planning menu contains setup pages relating to planning tasks. Each is defined in this section.

Standard Planning Setup

The Standard Planning Setup page is used as a template for the job-specific Planning Setup page accessed on the Job Planning and Processing view.

: Standard Planning Setup

Planning Resources

Job planning requires planning of Persons and Person Groups, which are managed in the Planning Resources page.

: Planning Resources page

Planning resources can be single persons as well as resource groups.

For more transparency of responsibilities and job assignments, the planning resources can be individually assigned to each job line or hierarchical level.

Units of Measure

On the Units of Measure page, the Duration Format field defines standard units that can be used to reference existing codes.

: Units of Measure page
