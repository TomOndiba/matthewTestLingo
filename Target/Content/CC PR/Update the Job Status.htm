Update the Job Status

When a job moves from a planning or quote status to an order, the job Status must be changed to Order for any posting to occur.

cc|project offers two options on the Sales FastTab on the Jobs Setup page to automate this process:

  • Select Optional or Always in the Set Job Status Quote field to ask or automatically change the job Status to Quote when a quote is created.
  • Select Optional or Always in the Set Job Status Order field to ask or automatically change the job Status to Order when an order is created.

If these options are not selected, the Status must be manually changed on the job.

In this example, the job Status was manually set to Order because the option to automatically change the status during sales document creation was declined.
