Purchase Quote Manager

The Purchase Quote Manager (PQM) is an add-on to cc|project. This functionality is used to manage multiple quote requests with varying prices, from multiple vendors, for one or more job lines.

Once the vendor is chosen, the Purchase Quotes Manager provides the ability to move relevant quote information (vendor number, direct unit cost, quote date, etc.) to the job line(s).

Note

The Purchase Quote Manager can also be used for Requisitions Worksheet lines.

PQM Setup

The general setup for the Purchase Quote Manager is entered in the PQM Setup page, accessed from the Departments menu by clicking cc|project > Administration > PQM Setup.

: PQM Setup

These are the default options for the Purchase Quote Manager Cards that can be changed for individual purposes later. The fields on this page are described in the following table.

Field

Description

PQM Nos.

Indicates the number series to be used for PQM.

Assign Vendors autom.

If selected, all linked vendors of an item will be assigned to the PQM automatically.

Assign Vendors per Line

Vendors can be allocated to the PQM Header or to the PQM Lines. If selected, the vendors of the PQM Lines are used for creating quotes.

Show Exp. Receipt Date

If selected, the value in the Expected Receipt Date field of the quote line will be displayed in the Purchase Quote Matrix.

Show Vendor Lead Time Calc.

If selected, the value in the Vendor Lead Time field of the quote line will be displayed in the Purchase Quote Matrix.

 

Items

Essential data for the Purchase Quote Manager process can be defined in the Replenishment FastTab of the Item Card.

: Item Card, Replenishment FastTab

The Vendor No. field is used to assign a main Vendor to the Purchase Quote Manager card.

The vendors in the Item Vendor Catalog are also used to assign Vendors to Purchase Quote Manager cards.
