Calculations for Document BOMs

Using functionality from both cc|calculation and cc|configure to order, calculations can be created for Document BOMs in the Job Budget Calculation line. Additionally, calculated values from the Document BOM can be transferred back to the unit cost and price on the job line, as well as the item master.

As with running a calculation for the Job Budget Calculation lines, the lines of the Document BOM must contain relevant Calculation Groups Codes.

Create Calculation for Document BOM

Once a Document BOM has been created for a Job Budget Calculation line, the calculation can be created by clicking Navigate > Calculation on the ribbon of the Document BOM.

: Document BOM created from Job Budget Calculation line, accessing Calculation function

The process of creating a calculation for a Document BOM is generally the same as creating one for a Job, except that instead of a dialog to select a job calculation type in the new Calculation Card, the Calculation Schedule List page appears, where the calculation schedule for the Document BOM calculation must be selected.

: Calculation Schedule List for selecting calculation schedule for Document BOM

Click OK to select the calculation schedule; the Calculation Card is updated with the information from the Document BOM as well as the lines from the calculation schedule. Note that the Document FastTab refers to the Document BOM for the job.

: Calculation Card for Document BOM

To see calculated values, click Update Calculation on the ribbon, and refresh the Calculation Chart to see the values in a chart.

: Updated calculated values

Transfer Values

The “Transfer Value” function on the Calculation Card transfers the following based on the Value Transfer Option setting in the Calculation Schedule:

  • calculated unit cost/price to the item
  • calculated unit cost/price to the job line

This function can only be run if the Status is set to Certified; if it is not certified when the function is run, a message appears stating that it will certify the calculation and then open the Transfer Calculation Value page.

: Calculated values set to be transferred to the Job Budget Calculation line only

The Current Value field shows the current unit cost/price on the Item Card and Job Budget Calculation line. The Calculation Value shows the calculated value on the Calculation Card.

Lines with the Transfer check box selected will be transferred to the item master and/or Job Budget Calculation line. Click OK to complete the transfer process.

Note

The Sales Price transfer option has no functionality in cc|project.

: Calculated values transferred to the Job Budget Calculation line
