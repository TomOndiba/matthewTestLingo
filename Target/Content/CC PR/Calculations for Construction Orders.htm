Calculations for Construction Orders

Using functionality from both cc|calculation and cc|configure to order, calculations can be created for Construction Orders created from the Job Planning and Processing line. Additionally, calculated values from the Construction Order can be transferred back to the unit cost and price on the job line, as well as the item master.

As with running a calculation for the Job Budget Calculation lines, the lines of the Construction Order must contain relevant Calculation Groups Codes.

Create Calculation for Construction Order

Once a Construction Order has been created for a Job Planning and Processing line, click Yes in the Procurement Release field to open the related Construction Order.

: Opening the Construction Order created from the Job Planning and Processing line

In the Construction Order, click Navigate > Calculation on the ribbon of the Document BOM.

: Construction Order, accessing Calculation function

The process of creating a calculation for a Construction Order is generally the same as creating one for a Job, except that there is no dialog to select a job calculation type in the new Calculation Card.

: Calculation List for creating calculation for Construction Order

After creating a new calculation in the Calculation List page, either select a calculation schedule if a default has not been assigned to the item, or the default schedule will be used automatically. The Calculation Card is updated with the information from the Construction Order as well as the lines from the calculation schedule. Note that the Document FastTab refers to the Construction Order for the job.

: Calculation Card for Construction Order

To see calculated values, click Update Calculation on the ribbon, and refresh the Calculation Chart to see the values in a chart.

: Updated calculated values

Transfer Values

The “Transfer Value” function on the Calculation Card transfers the following based on the Value Transfer Option setting in the Calculation Schedule:

  • calculated unit cost/price to the item
  • calculated unit cost/price to the job line

This function can only be run if the Status is set to Certified; if it is not certified when the function is run, a message appears stating that it will certify the calculation and then open the Transfer Calculation Value page.

: Calculated values set to be transferred to the Job Budget Calculation line only

The Current Value field shows the current unit cost/price on the Item Card and Job Budget Calculation line. The Calculation Value shows the calculated value on the Calculation Card.

Lines with the Transfer check box selected will be transferred to the item master and/or Job Budget Calculation line. Click OK to complete the transfer process.

Note

The Sales Price transfer option has no functionality in cc|project.

: Calculated values transferred to the Job Budget Calculation line

Concurrent Calculations

The concurrent calculations for Construction Orders are also the same as for job calculations.
