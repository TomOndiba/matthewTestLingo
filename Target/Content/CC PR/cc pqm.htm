cc|purchase quotes manager

cc|purchase quotes manager (PQM) is an add-on to cc|project. This functionality is used to manage multiple quote requests with varying prices, from multiple vendors, for one or more job lines.

Once the vendor is chosen, PQM provides the ability to move relevant quote information (vendor number, direct unit cost, quote date, etc.) to the job line(s). However, since the PQM is independent from the procurement process, there are two scenarios for handling the information once the PQM process is complete:

  • The quotes can be deleted because only the prices were needed; the job line can move normally through the procurement process.
  • The placed quote(s) are used later for procurement but are not used in the procurement planning.

Note

PQM can also be used for Requisitions Worksheet lines as well as directly from the PQM page.
