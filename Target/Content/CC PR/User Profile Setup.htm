User/Profile Setup

This topic describes the users and profiles setup for use with cc|project. Each is accessible from the navigation pane by clicking Departments > Administration > Application Setup.

User Setup

The User Setup page – accessed from the Application Setup menu by clicking Users > Users Setup – gives user-specific rights to post Job Time Entries and Inventory Requests. Additionally, the Job Time Entry Administration field determines whether a user can see all Job Journal Batches. If this check box is not set, the user can only see their own time entry journal.

Assign a Profile to User Personalization

A profile can be assigned to each user in Microsoft Dynamics NAV through the User Personalization Card.

Follow these steps to create a new User Personalization Card and add the PROJECT MANAGER PR profile:

  1. In the RoleTailored Client menu, click User Personalization.
  2. Locate the user to assign a profile and then click Edit.
  3. In the Profile ID field, enter the Project Manager PR profile.

: User Personalization Card page

  1. Click OK to close the User Personalization Card.
