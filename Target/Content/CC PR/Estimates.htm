Estimates

Estimates are necessary, because:

  • in addition to the calculation values, more current estimates of the project expenses often exist.
  • the data can be used for the WIP calculation as expected project values.

It is only necessary to enter the estimated cost in the Job Planning and Processing view if there is a deviation from the calculated cost values.

: Estimates on Job Planning and Processing view

The structure calculation logic is different from the calculation because budgets are not necessary.

  • Bottom up – all higher levels will be calculated.
  • Modifications in higher levels will not affect changes in lower structures.
