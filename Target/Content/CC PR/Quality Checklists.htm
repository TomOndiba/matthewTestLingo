Quality Checklists

The Quality Check functionality uses pre-defined question and answer templates for the purpose of quality inspection. These questions and answers are assigned in the QC Templates, which are then used in QC testing. The templates can also be used for gathering additional information about a Job.

QC Templates

The QC Template is the basis of the QC test and is accessed from the Departments menu by clicking cc|project > Administration > QC Template List. Initially, the QC Template List page appears.

: QC Template List

Click New to create a template or double-click an existing template to open the QC Template page.

: QC Template card

For detailed information about QC Templates, refer to the Quality Check topic in the “cc|industry solution: Process Manufacturing” documentation.

Job Types

The QC Template can be assigned to a Job Type, which will indicate the QC Test Card for the Job that will be created with this Job Type.

: Job Types Setup with assigned QC Template

Note

Additional checklists can be added to the job manually using the “Add QC Test” function on the Actions tab of the Job Card.
