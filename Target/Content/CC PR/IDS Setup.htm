IDS

The IDS Webshop interface is the (semi-) automatic communication between the commercial software by using the Internet presences of the wholesaler. The IDS connector in cc|project provides the ability to import and export carts from Wholesale Webshops.

This topic describes the set up required to use the IDS Connector in cc|project.

Note

For assistance with the installation of IDS, refer to your COSMO CONSULT consultant.

IDS Setup

The general setup for IDS is entered in the IDS Setup page, accessed from the Departments menu by clicking cc|project > Administration > IDS Setup.

: IDS Setup

The fields on the IDS Setup page are described in the following table.

Field

Description

Webhook URL

Indicates the URL that is used to communicate between the Webshop (internet) and the local Microsoft Dynamics NAV server. For instance, that could be a URL using the service of ngrok.

Default Item Template Code

Defaults the item template that is used for the item mapping of items from Webshops to new Microsoft Dynamics NAV items.

IDS Item Nos.

Indicates the number series used for new items from item mapping.

 

IDS Code Mapping

The IDS interface requires unique units of measure for communication, which do not match the units of measure codes in Microsoft Dynamics NAV. Therefore, a connection must be made between the IDS codes and Microsoft Dynamics NAV codes in the IDS Code Mapping page.

The IDS units are created automatically when the IDS Code Mapping page is opened (Departments > cc|project > Administration > IDS Code Mapping); however, the assignment to the Microsoft Dynamics NAV codes must be done manually.

: IDS Code Mapping page with mappings to NAV units of measure and alloy types

The fields in the IDS Code Mapping page are described in the following table.

Field

Description

Type

Indicates if the IDS Code to be mapped is a Unit of Measure or a Raw Material.

IDS Code

Indicates the actual IDS Code needed for mapping to a code in cc|project.

Description

Short description of the IDS Code.

NAV Code

Identifies the Unit of Measure or Alloy Type (raw material) in cc|project to be mapped to the IDS Code.

 

Vendor Card

The IDS - Connector FastTab on the Vendor Card contains fields that must be populated in order to create a connection with the vendor’s Webshop. The access data for the IDS communications are provided to the company by the Webshop operator (vendor).

: Vendor Card, IDS - Connector FastTab

The fields on the IDS - Connector FastTab are described in the following table.

Field

Description

IDS Webshop URL

Indicates the website of the vendor’s IDS Webshop.

IDS Webshop Cust. No.

Indicates the customer number assigned in the vendor’s IDS Webshop.

IDS Webshop Cust. Name

Indicates the customer name assigned in the vendor’s IDS Webshop.

IDS Webshop Password

Indicates the password for accessing the vendor’s IDS Webshop.

Note

All fields may not be required for every vendor Webshop.

Item Cross-References
