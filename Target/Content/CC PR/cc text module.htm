cc|text module

With cc|text module, header and/or footer text can be added to a job, which will then transfer to the header or footer text of the following documents if linked by the job number:

  • Purchase Quote / Order / Invoice / Credit Memo
  • Sales Quote / Order / Invoice / Credit Memo
  • Job Shipment

Whether this text appears on the printed documents or not depends on the report settings defined in the Print Management Setup for cc|text module.

This section describes the functionality that is specific to using cc|text module in cc|project and does not include standard cc|text module field descriptions, etc. For detailed setup, configuration and usage information, refer to the current “cc|text module” documentation.

Setup

In standard cc|text module, many reports (documents) and tables are set up by running a default function (such as purchase and sales documents). To use cc|text module with non-standard documents, such as the Job Shipment, the related reports and tables must be added to the Text Module Objects Setup page.

Fig. 1 is a sample of the setup that could be used with Job Shipments.

: Text Module Objects Setup with sample Job Shipment setup

Assign Text to Job

Header and footer text is assigned to a job by clicking Navigate > Text Module on a Job Card.

: Accessing Text Module from Job Card

In the Text Module List page, click New to create text or Edit to open existing text in the Text Module Card page.

In the General FastTab, enter language and whether it is header or footer information. In the Lines FastTab, enter the actual text, along with any formatting to transfer to the printed document. If text is to be printed, enter the relevant code in the Print Mgt. Code field.

: Header text that will not be printed on Job Shipment document

Click OK to close the Text Module Card or OK button > OK & New to add more text.

: Footer text that will be printed on Job Shipment document

Click OK to close the Text Module Card and then close the Text Module List page.

Process and Print Documents with Text

When a document is generated during job processing/planning/release, the text is transferred to the corresponding Header Text and Footer Text. For example, if a Job Shipment is planned and released and a document is created, the header and footer text assigned to the job are accessible from the Navigate tab of the Job Shipment page.

: Header and Footer text on the Job Shipment

When the document is printed, any text that has a Print Mgt. Code that includes the report will be printed in the header, footer or both.

: Job Shipment report with only footer text, no header text

In either Edit Text page, existing text can be modified or new text can be added. For example, the Header Text on the Job Shipment was changed to be external and a Print Mgt. Code was assigned.

: Header text changed on Job Shipment

When the Job Shipment report is printed now, the Header Text will also appear.

: Job Shipment report with header and footer text

Additionally, any standard text that has been set up within the company can be assigned to the Edit Text pages by clicking Actions > Paste Above or Paste Below in the Get Standard Text group. Refer to the current “cc|text module” manual for more information about standard text.
