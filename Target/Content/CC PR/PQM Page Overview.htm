PQM Page

The process of creating the price quotes to the vendors begins in the PQM page. This topic describes the fields and functions on this page.

The PQM page opens when the “Create” or “Show PQM” functions are run from the Navigate tab on the Job Budget Calculation view.

: PQM page

This page is also accessible directly from the cc|project menu by clicking Administration > Purchase Quote Manager.

General FastTab

The fields on the General FastTab are described in the following table.

Field

Description

No.

Unique identifier for the PQM.

Description

Short description for the PQM:

  • Entered manually for PQMs created from the PQM page
  • Job description when created from Job Budget Calculation line
  • Batch name when created from Requisition Worksheet

Description 2

Additional description field.

Person Responsible

Identifies the person responsible for the PQM; accessed from the Resource table.

Assigned Vendor

Indicates the number of vendors that have been assigned to the PQM. This is done automatically if the Assign Vendors autom. check box is selected in the Options FastTab and vendors have been assigned to the item added in the Lines FastTab.

Status

Indicates the current state of the PQM. The options are:

  • Open
  • Released
  • Finished

 

Lines FastTab

The fields on the Lines FastTab are described in the following table.

Field

Description

Type

Identifies the type of PQM line; currently only the Item option can be manually entered in the Lines FastTab. However, job budget lines with general components or service lines that use G/L accounts can be included in the PQM lines for processing.

No.

Number of the appropriate item, item charge, fixed asset, or G/L account.

Description

Defaults from the item card.

Quantity

Indicates the number of items to be quoted.

Unit of Measure Code

Defaults from the Purch. Unit of Measure field on the Replenishment FastTab of the Item Card.

Source Vendor No.

Defaults from the Vendor No. field on the Replenishment FastTab of the Item Card.

Source Unit Cost

Defaults from the Direct Unit Cost on the:

  • Item Card or the Purchase Prices if set for the item/vendor when created from the PQM page
  • Job Budget Calculation line when created from the Job Budget Calculation
  • Requisition Worksheet

Assigned Vendors

Indicates the number of vendors that have been assigned to the PQM line.

Due Date

Indicates the Expected Receipt Date for the Purchase Quote line.

Status Created

Indicates whether purchase quotes have been created or not. Click the value in this field to open the PQM Vendors page.

Status Verified

Indicates whether the status of the created purchase quotes have been changed to Verified or not. Click the value in this field to open the PQM Vendors page .

Status Placed

Indicates that a purchase quote has been placed. Click the value in this field to open the PQM Vendors page to view the quote that has been placed.

Status Not Placed

Indicates that created purchase quote(s) have not been placed. Click the value in this field to open the PQM Vendors page to view the quote(s) that have not been placed.

Placing Vendor No.

Indicates the vendor number of the placed purchase quote.

Placing Unit Cost

Indicates the unit cost of the placed purchase quote.

Difference Unit Cost

Indicates the difference between the Source Unit Cost and the Placing Unit Cost.

Assigned

If selected, a purchase quote has been assigned to the PQM line.

Source Status

Indicates the current status of the source of the PQM line:

  • blank – created from the PQM page
  • From Document – created from the Job Budget Calculation or Requisition Worksheet
  • Updated – indicates that the Job Budget Calculation has been updated
  • Release – currently has no functionality

Currency Code

Identifies the alternate currency if different than local currency.

Currency Date

Identifies the current date of the currency exchange rate for the selected Currency Code.

Direct Unit Cost

Displays the Direct Unit Cost in foreign currency if the Currency Code is used, otherwise in local currency.

Direct Unit Cost (LCY)

Displays the Direct Unit Cost in local currency.

Line Discount %

Identifies a discount percentage for the PQM line.

Line Discount Amount

Identifies the amount of the discount for the PQM line.

Order Date

Indicates the system date the PQM line was created from the Job Budget Calculation.

Planned Lead Time Calculation

Calculates the due date of the PQM line based on the Order Date field.

Total Direct Cost (LCY)

Displays the calculated cost of Quantity * Source Unit Cost.

Variant Code

Indicates if an item variant is used for the PQM line.

 

Administration FastTab

The fields on the Administration FastTab indicate the date the PQM was created and last modified.

Options FastTab

The values in the fields on the Options FastTab default from the settings in the PQM Setup page, but the values can be changed per PQM. Refer to topic “PQM Setup” on page 1 for field definitions.

Actions Tab

The actions on the Actions tab are described in the following table.

Action

Description

Reset Status

Resets the status from Released to Open and from Finished to Released.

Finish

Sets the status to Finished.

Assigned Vendors

Drop-down arrow includes two actions:

  • Copy from Header to Lines – transfers information from the Assigned Vendors
  • Update Line Status – changes the status in the Assigned Vendors page to the previous status (e.g., Verified back to Created)

Create Purch. Quotes

Creates purchase quotes for all Vendors, and opens the PQM Placing Matrix for quote selection (placing).

 

Navigate Tab

The actions on the Navigate tab are described in the following table.

Action

Description

Purchase Quotes Matrix

Opens the PQM Matrix request page, where filters can be set then the PQM Placing Matrix page opens, where the quote information for each purchase quote can be viewed.

Assigned Vendors

Opens the PQM Vendors page, where Vendors from the PQM lines have been copied or Vendors are added manually, lead times can be added, and the quote status is available once quotes are created.
