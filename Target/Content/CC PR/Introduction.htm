Introduction

cc|project is an industry solution for project management applications. 

This module is built for Microsoft Dynamics® NAV and can be integrated with other Microsoft Dynamics NAV solutions where the need for event or project management is required. It covers all procedural aspects of job planning, job control and project management, offering full integration into company processes. 

The underlying design basis aims to find a shared platform for the integrative components of the solution that offers flexible design options. Thanks to the structural layout of the data, a close link to Microsoft® Project® is realized in the process.

Key Features

  • Project calculation
  • Cost monitoring
  • Flexible project structure
  • Resource planning
  • Time management
  • Sales (quote/order/invoice/credit note)
  • Purchasing (quote/order/invoice/credit memo)
  • Advanced and down payment
  • Microsoft Project interface (bi-directional)
  • Stock integration
  • Manufacturing integration

cc|project functionalities are accessed in the cc|project menu.
