Job WIP View

There is a special view just for the WIP calculation accessed from the Job Card by clicking Navigate > WIP.

: Job WIP view for Job J000001.01

The fields and functions on the Job WIP view are described in the following sections.

Field

Description

WBS / Description / Sales Code

Transferred from the Job Budget Calculation view.

Calculation Total Cost (LCY)

Transferred from the Total Cost Budget (LCY) field on the Job Budget Calculation view.

Calculation Total Price (LCY)

Transferred from the Sales Line Amount Budget (LCY) field on the Job Budget Calculation view.

Total Price Released (LCY)

Transferred from released sales orders; used to calculate the WIP Amount.

WIP Total Price Released (LCY)

Transferred from released sales orders; used to calculate the WIP Amount.

Note

This is used if WIP is calculated independently from the Sales Code.

Procurement Total Cost (LCY)

Displays the total cost of procurement (in local currency) from the processed procurements assigned to the sales position.

WIP Total Cost Proc. SP (LCY)

Displays the total cost of procurement (in local currency) from the processed procurements assigned to the sales position.

Note

This is used if WIP is calculated independently from the Sales Code.

Expected Total Cost (LCY)

Used to calculate the WIP Amount.

This is calculated using the calculation values, released sums and/or estimates. The user can decide which amounts should be included.

Expected Total Price (LCY)

Used to calculate the WIP Amount.

This is calculated using the calculation values, released sums and/or estimates. The user can decide which amounts should be included.

Total Cost Usage SP (LCY)

Sum of the posted usage costs in the Job Ledger Entries; used to calculate the WIP Amount.

WIP Total Cost Usage SP (LCY)

Sum of the posted usage costs in the Job Ledger Entries; used to calculate the WIP Amount.

Note

This is used if WIP is calculated independently from the Sales Code.

Total Price Posted (LCY)

Sum of the posted sales amounts in the Job Ledger Entries; used to calculate the WIP Amount.

WIP Total Price Posted (LCY)

Sum of the posted sales amounts in the Job Ledger Entries; used to calculate the WIP Amount.

Note

This is used if WIP is calculated independently from the Sales Code.

WIP Method

Determines how WIP is calculated. The options are:

  • Cost Value
  • Percentage of Completion
  • Sales Value
  • Completed Contract
  • Cost of Sales
  

Note

Cost Value, Cost of Sales and Completed Contract use the same calculation logic.

Last WIP Date

Indicates the last date on which WIP was calculated.

WIP Amount (LCY)

Displays the calculated WIP amount (in local currency) between the last and the current WIP date.

Prospective Cost (LCY)

Displays the calculated amount (in local currency) for future expenses between the last and the current WIP date.

Impending Losses (LCY)

Displays the calculated amount (in local currency) for pending losses between the last and the current WIP date.

WIP Not Posted

Indicates whether WIP has been posted or not.

Comment

Indicates if a job comment line exists for the WBS position.

Cost of Sales (LCY)

Displays the calculated amount (in local currency) for revenue-based WIP (refer to the WIP Revenue-Based (LCY) field in the Job Line WIP Entries page).

Estimated Total Cost (LCY)

Indicates the estimated total cost (in local currency); this field is editable.

Estimated Total Dir. Cost (LCY)

Indicates the estimated total direct cost (in local currency); this field is editable.

Estimated Total Price (LCY)

Indicates the estimated total price (in local currency); this field is editable.

Invoicing Total Price RF (LCY)

Transferred from posted sales invoices; used to calculate the WIP Amount.

Job Posting Group

Defaults from the Job Card.

Outline Code

Defaults from the Job Budget Calculation view.

Realized % Based on cost

Displays the completion level at the WBS position level; non-editable field that is calculated over substructures.

Responsible

Identifies a resource (person or group) that is responsible for the WBS position.

No. / Quantity / Type / Unit of Measure Code / Work Type Code

Transferred from the Job Budget Calculation view.

 

Navigate Tab

The Navigate tab includes three action groups: Line, Views and WIP. The actions within each are described in the following sections.

Line Action Group

The actions in the Line group are described in the following table.

Action

Description

Position Card

Opens the Job Position Card page, which provides a card view of all the information in the Job WIP view for the selected WBS position.

Dimensions

Opens the Job Line Dimensions page, which shows dimensions relating to the selected WIP line.

Comments

Opens the Job Comment Lines page, where existing comments can be viewed or new comments can be added for the selected WIP line.

 

Views Action Group

The Views group provides direct access to the other views relating to the job. Each view is described in more detail in other topics.

WIP Action Group

The actions in the WIP group are described in the following table.

Action

Description

Calculate and Post WIP

Opens the Calculate WIP batch job, which calculates the project value for all sales positions with a Sales Code (refer to topic “Calculate WIP” on page 1 for more information).

WIP Entries

Opens the Job Line WIP Entries page, which displays posted entries for the selected WIP line.

WIP G/L Entries

Opens the General Ledger Entries page, which displays posted G/L entries for the selected WIP line.

 

Calculate WIP

This batch calculates the job value for all sales WBS positions. These are all lines with a sales code.  There are different options for calculation. After calculation (or at the same time with calculation) the calculated values can be posted to G/L.

Several valuation options can be selected which will influence the maximum value of WIP.

The calculation is started by clicking Calculate and Post WIP on the ribbon the Job WIP view.

: Calculate WIP

The fields on the Calculate WIP batch job are described in the following table.

Field/Field Group

Description

WIP Posting Date

Indicates the date on which WIP is to be calculated/posted.

Expected Amounts

Cost and Sales Values are calculated using the calculation values, released sums and estimates. Select to determine what is included in the calculation.

WIP Max. Cost Range

Indicates the maximum amount of cost to be calculated as WIP. The options are Cost Amount and Sales Amount.

WIP Adm./Sales %

Percentage value to reduce the WIP Max. Cost Range.

Calculate

Select to start the calculation of the WIP values.

Post

Select to post the calculated WIP values to the general ledger.

Post Impending Losses

Select to post the calculated impending losses to the general ledger.

Post Prospective Costs

Select to post the calculated prospective costs to the general ledger.

Cost of Sales

Select to post the calculated cost of sales value to the general ledger.
