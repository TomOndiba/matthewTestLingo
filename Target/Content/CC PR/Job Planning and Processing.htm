Job Planning and Processing View

In the Job Planning and Processing view, a significant number of fields are populated with default values from Job Budget Calculation and the work packages (Type and No. of the selected components). Many of these fields were defined in topic “Job Budget Calculation View” on page 1 and others are defined later in this documentation. Therefore, this section does not provide a complete description of each field.

Actions Tab

The functions on the Actions tab are described in the following sections.

Expand/Collapse

The directory levels can be expanded and collapsed by clicking Actions > Expand/Collapse:

  • Current Position – only relevant on Position lines; expands or collapses depending on the current state, e.g., expanded lines in the Position will be collapsed when the function is run.
  • Expand All – all levels are expanded.
  • Collapse All – all levels are collapsed to the Position level.

Recalculate Rollup Amounts

This function recalculates the rollup amounts if usage has been posted and the Rollup Type has been changed.

IDS - Connector

This action contains functions that are used when processing job lines with the IDS Webshop interface. Refer to topic “IDS Connector” on page 1 for more information.

Planning Action Group

The Planning group provides various setup and process pages relating to planning. Each is described in this section.

Action

Description

Planning Setup

Opens the Job Planning Setup page, which is defaulted from Standard Planning Setup and can be changed for individual Job planning parameters.

Planning Process

Contains actions related to planning processing. The options are:

  • Planning – plans a WBS position for planning processing.
  • Release – releases a planned WBS position for planning processing.
  • Planning and Release – plans and releases the WBS position(s) concurrently.
  • Set to “Finished” – sets the Planning Status = Finished.

View

Opens the Planning Processes page, where the WBS positions can be viewed. The options are:

  • Per Line – Only planning processes for the selected line are displayed.
  • Per Block – All planning processes for the main position are displayed.

Resource Allocation

Opens the Planning Res. Allocation page, where resources can be assigned to WBS planning positions (refer to topic “Resource Allocation” on page 1).

FactBoxes

The Job Planning and Processing view includes two cc|project-specific FactBoxes:

  • WBS Processing – displays planning, procurement, job shipment and invoicing information about the job planning line.
  • Job Comment Lines – displays all comments assigned to the planning and processing line.
