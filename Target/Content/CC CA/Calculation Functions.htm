Calculation Functions

This section describes the various functions on the ribbon of the Calculation Card.

Update Calculation

The “Update Calculation” function creates the initial calculation and recalculates the cost estimate and the values of the calculation. This function cannot be run if the Status is set to Certified.

Transfer Value

The “Transfer Value” function transfers the following based on the line setup:

  • calculated cost, sales price, or discount values to the item
  • calculated unit cost, unit price and line discount to the sales document line

This function can only be run if the Status is set to Certified; if it is not certified when the function is run, a message appears stating that it will certify the calculation and then open the Transfer Calculation Value page.

: Calculated values set to be transferred to the item and sales line

Lines with the Transfer check box selected will be transferred to the item and/or sales document line.

Transfer Values to Sales Document Lines

Calculations with references to sales documents can be used to transfer the calculation values to the document line.

: Calculated values set to be transferred to the sales order line

Note

The option (Type = Document Unit Price) to transfer the Unit Price value to the document is not available if the currency codes are different between calculation and document.

Click OK to complete the transfer process and update the unit price and line discount % on the sales order line.

: Calculated values transferred to the sales order line

Transfer Sales Price and Sales Line Discount to Item

In addition to the values transferred to the sales document line, the calculated sales price and sales line discount percent can be transferred to the item’s Sales Price and Sales Line Discount tables, respectively, based on the lot size of the calculation.

: Calculated value set to be transferred to the Sales Price and Sales Line Discount tables

Click OK to complete the transfer process and update the sales price and line discount % on the item.

: Calculated values transferred to the sales prices and sales line discounts for the item

Transfer Unit Cost to Sales Document Line

For items with costing method FIFO, the Document Unit Cost line type appears in the Transfer Calculation Value page.

: Document Unit Cost set to be transferred to sales line

If the Transfer check box is selected, the calculated value will be transferred to the Unit Cost field in the Sales Line.

: Unit Cost updated in the sales line from Value Transfer function

Note that the Unit Cost is only transferred to the sales line, not the item.

: Unit Cost not updated on Item Card

Lot Size Calculation

The “Lot Size Calculation” function is used for price comparison of multiple lot sizes.

In the Lot Size Calculation Lines page, enter the various lot sizes to compare in the Lot Size field.

: Four lot sizes added for comparison

To run the price comparison, click Show on the ribbon and then click Yes in the message to copy the information from the calculation. The Lot Size Calculation Matrix page opens, displaying the calculation lines based on the different lot sizes.

: Sales calculation lines for the four lot sizes

On the General FastTab, select how to display the column names. The options are:

  • Lot Size – Lot Size specified in the Lot Size Calculation Lines page
  • Description – Description from the Lot Size Calculation Lines page
  • Calculation No. – Calculation No. automatically assigned to each lot size

Close the Lot Size Calculation Matrix page and note that the following values were transferred to the Lot Size Calculation Lines page:

  • net cost calculation line for each lot transfers to the Standard Cost field
  • net price calculation line for each lot transfers to the Unit Price field

: Calculated cost and price values from the matrix

Calculated lines can be deleted as needed; the related calculation is also deleted.

The lot size calculation can be transferred to the item by selecting the line and then clicking Transfer Value on the ribbon. The Transfer Calculation Value page appears, displaying the value to transfer; click OK to complete the transfer process.

: Transfer Calculation Values from selected lot size calculation

For example, if the lot size of 500 is selected and the “Transfer Value” function is run, the lot size pricing will be transferred to the Sales Price line for item 1000.

: Lot size pricing transferred from calculation to the item

Reverse Calculation (Target Costing)

cc|calculation provides market-oriented target cost management that is essential in the early phases of product development. Using the “Reverse Calculation” function in an existing Calculation, the value of a calculated line can be manually adjusted and then the difference can be reallocated to another line to offset the cost/price change.

Fig. 14 illustrates a sample target cost flow in cc|calculation. Starting from a competitive net price, the system calculates backwards and reduces each of the subtotals by the specified overhead, down to a target value.

: Target costing flow sample in cc|calculation

After creating a calculation with costs/prices that need to be adjusted, click Reverse Calculation.

: Running the Reverse Calculation function in Calculation Card

The fields in the Reverse Calculation page are the same as the lines of the Calculation Card, with two additions.

: Reverse Calculation page

The additional fields are described in the following table.

Field

Description

Target Value

Detemines the charge line to which the adjusted cost is to be allocated.

Only rows of Totaling Type = Charge can be selected as Target Value.

New Value (LCY)

Indicates the adjusted cost/price for the calculation line.

These values are calculated backwards, starting from total (highest row number where adjusted prize is added) to row marked as Target Value.

Example

The calculated price for a bicycle is 723.13, which is not an ideal price. The sales manager knows there is a better chance to sell these bicycles for 699.99 instead. Therefore, 699.99 is entered in the New Value (LCY) field in row 1200 , Net list price.

: New Value entered for list price

After the new value is entered, all charge lines that are available for cost/price reallocation are changed to a blue font. In this example, the offset (23.14) for the list price will be applied to the Profit charge, row 0970. The sales manager selects the Target Value check box in row 0970.

: Profit and related charges adjusted due to reallocation of cost/price

By selecting the Profit as the target for the reallocation, any charges that have a reference to this charge are also reduced proportionally. Therefore, the Profit was reduced by 19.67 and the related Reduction line (row 1100, which is related to row 1000, from which the calculation includes row 0970), is reduced by 3.47. Thus, the reduction of 23.14 is reallocated.

If the reallocations are not acceptable, the Reverse Calculation page can be closed and the “Reverse Calculation” function can be run to start the process again.

If the reallocations are acceptable, click Make Calculation to create a new calculation and then click OK to close the Reverse Calculation page. Close the original Calculation Card and then open the new calculation.

: New calculation created with new values

The new calculation includes the new values, and shows the percentage changes that occurred due to the target costing adjustments. In this example, the original percentage for the profit charge was 25 (refer to Fig. 15), and now it is 20.77355.

Calculation Detail Lines

The Calculation Detail Lines page displays an overview of all cost estimates for the calculation or the detail of the cost calculated on a specific line.

On the Calculation Card, click Navigate > Calculation Detail Lines to open the overview page.

Note

To see calculations in this page, assign an item and run the “Update Calculation” function.

: Calculation Detail Lines for an entire calculation

Each level can be expanded or collapsed by clicking the arrows in the Type field.

On a calculation line in the Calculation Card, click the AssistEdit button in the Value (LCY) field to open the detailed calculations.

: Calculation Detail Lines for a calculation line
