Calculation Views per User

An authorization level can be assigned to each Calculation Schedule line in the Calculation View Authorization field. There are five levels available.

Note

This set up must be performed in the Calculation Schedule Card page.

Calculation lines without a calculation view authorization level are not subject to restrictions.

: Calculation Schedule Card displaying the calculation view authorization levels

Note

The order of the calculation view authorization can be ascending or descending.

In the Calculation View Authorization field in the User Setup page, the level is indicated, up to which a user can see values within a calculation. There are five levels available.

: User Setup displaying the calculation view authorization levels

The user can see all values of a calculation if there is no level indicated in the User Setup page.
