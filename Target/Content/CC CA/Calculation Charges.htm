Calculation Charges

Calculation Charges are used to define percentile or absolute charges to a calculation, such as discounts, freight costs, or mark-up.

: Calculation Charges

The fields in this page are described in the following table.

Field

Description

Code

Unique identifier for the calculation charge.

Starting Date

Controls when the price increases and discounts for the calculation should apply.

Description

Description of the calculation charge.

Type

Determines if the calculation charge is a percentage of cost or an absolute value.

Rate

Value of the percentage or absolute value defined in the Type field.

Type of Reference Value

To calculate the correct discount and payment discount in calculations, set this field to Reduced Basic Value. The charge is calculated with the basic value if this field is empty or set to Basic Value.

Example – Type of Reference Value

As illustrated in Fig. 1, the CDISCOUNT and DISCOUNT Calculation Charges are set to Reduced Basic Value.

: Reduced Basic Value discount calculation on the Sales Calculation

When the Sales Calculation is run, the reduced basic value is calculated as follows:

Net list price = Target retail price / (100 - Reduction) * 100

Therefore, the Net list price was calculated as 275.34 = 234.04 / (100 - 15) * 100.
