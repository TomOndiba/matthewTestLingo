Calculation Chart

The Calculation Chart FactBox on the Calculation Card displays calculations based on Item Groups, Capacity Groups or both, and by amount or time. Additionally, a variety of chart styles can be used, such as Pie, Line, or Column.

: Pie-type Calculation Chart with no filtering

The Show menu includes the following options:

  • All Groups – no filtering
  • Item Groups – Lines with Totaling Type = Item Calc. Group
  • Capacity Groups – Lines with Totaling Type = Capacity Calc. Group

: Calculation Card displaying a Pie-type Calculation Chart filtering Item Calculation Groups by Amounts

The Options menu includes two menus:

  • Value to Calculate – Determines what type of calculation values are shown:
    • Amount / Amount (LCY) – calculated cost
    • Time – calculated time; only relevant for capacity calculation groups
  • Chart Type – Determines what type of chart to display the calculations, such as Pie, Column, and Funnel

: Calculation Card displaying a Column-type Calculation Chart filtering Capacity Calculation Groups by Time Values

Use the “Refresh” function to update the chart when changes are made to the calculations.

Along with the legend at the bottom which identifies the colors in the chart, hovering the mouse over sections of the chart accesses Tooltips that provide additional information.
