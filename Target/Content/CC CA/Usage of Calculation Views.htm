Usage of Calculation Views per User

When using a calculation schedule on creating a calculation, the values of the Calculation View Authorization field are transferred from the Calculation Schedule line to the Calculation line.

The calculation lines with calculation view authorization levels higher than the level assigned in the user setup are suppressed.
