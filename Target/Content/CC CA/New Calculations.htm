New Calculations

New cost estimates are created in the Calculation Card either manually or from an existing calculation schedule.

After creating a new Calculation Card, the Calculation Schedule List page appears when the user attempts to insert a value in the No. field. Select an existing schedule or click Cancel to create a calculation manually.

: New Calculation Card with Calculation Schedule List

If a template is selected, it will automatically fill in the lines based on the calculations defined in the template.
