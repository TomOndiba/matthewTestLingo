Calculation Chart on Concurrent Calculation

The Calculation Chart on the Concurrent Calculation has fewer Chart Types and a major element like a bar for each calculation, as opposed to a single piece of pie, for example, in the Calculation Card for each line in the Calculation Card. 

The size of each color part in the bar is based on the value from a calculation line and the values are stacked to a total calculation value, but always show the resulting calculated value for quantity = 1.

: Stacked Column-type Calculation Chart on Concurrent Calculation with no filtering

The Show menu includes the following options:

  • All Groups – no filtering
  • Item Groups – Lines with Totaling Type = Item Calc. Group
  • Capacity Groups – Lines with Totaling Type = Capacity Calc. Group

: Calculation Card displaying a Stacked Area-type Calculation Chart filtering Item Calculation Groups by Amounts

The Options menu includes two menus:

  • Value to Calculate – Determines what type of calculation values are shown:
    • Amount / Amount (LCY) – calculated cost
    • Time – calculated time; only relevant for capacity calculation groups
  • Chart Type – Determines what type of chart to display the calculations, including Stacked Column, Stacked Column %, Stacked Area and Stacked Area %

: Calculation Card displaying a Stacked Column-type Calculation Chart filtering Capacity Calculation Groups by Time Values

Use the “Refresh” function to update the chart when changes are made to the calculations.

Hover the mouse over sections of the chart to access Tooltips, which provide additional information.
