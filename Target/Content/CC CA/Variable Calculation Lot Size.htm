Variable Calculation Lot Size on Items

Items can be marked for variable calculation lot size by selecting the Variable Calculation Lot Size check box on the Invoicing FastTab of the Item Card.

: Variable Calculation Lot Size check box on the Item Card

If these items are used as components in a multi-level calculation, the lot size of the calculation header is used instead of the lot size of the component item.

This check box is automatically selected on the Item Card if the Reordering Policy = Order and Manufacturing Policy = Make-to-Order.
