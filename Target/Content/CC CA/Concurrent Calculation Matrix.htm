Concurrent Calculation Matrix

Use the Concurrent Calculation Matrix page to compare the calculations within a concurrent calculation.

On the ribbon of the Concurrent Calculation page, click Concurrent Calculation Matrix.

: Concurrent Calculation Matrix access

Note

The Concurrent Calculation Matrix can also be opened from the Calculation Card and Calculation List pages.

The Concurrent Calculation Matrix page displays all the calculations within the selected concurrent calculation.

: Concurrent Calculation Matrix page showing quotes, orders, and production order calculations

On the General FastTab, select how to display the column names. The options are:

  • Type – Line Type from the Concurrent Calculation line
  • Description – Reference Type and Reference No. from the Concurrent Calculation line
  • Calculation No. – Calculation No. from the Concurrent Calculation line

On the Concurrent Calculation Matrix FastTab, click Functions > Update Prod. Order Calculations to update the production order calculations on the matrix.
