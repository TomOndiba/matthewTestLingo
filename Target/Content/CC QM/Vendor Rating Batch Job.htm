Vendor Rating Batch Job

On the cc|vendor rating menu, click Reports and Analysis > Vendor Rating to access the “Vendor Rating” batch job.

: Vendor Rating batch job

Note

To run this report, the “Calculate Vendor Rating” batch job must have already been processed (refer to topic "Calculate Vendor Rating Batch Job" on page 1).

The report calculates a respective result with an according score for the hard and the soft rating criteria. These two results are combined to an overall result based on the vendor-specific weighting (Vendor Rating FastTab on the Vendor Card). If no weighting has been specified for the vendor, the default weighting will be applied (refer to topic "Weighting" on page 1). Thus, a vendor-specific rating / weighting can be established without maintaining all vendors.

The hard rating criteria are consolidated and displayed per item. In regards to the soft rating criteria, the system determines valid data records for each type of soft criteria (audit, visit report, self-assessment) and always uses the highest version. Consequently, there are just as many data records available as there are types of soft rating criteria defined in the system.

Note

This batch job will automatically process a re-classification of the vendor. The current classification is displayed in the New Categorization field on the Vendor Rating FastTab of the Vendor Card.

: Rating History page

Note

If the Vendor Rating report is run multiple times on the same day, only one entry will appear for that date with the most recent rating information.
