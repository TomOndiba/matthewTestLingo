Assign Sample Planning to Item

Sample Planning can be added to the Test Plan Setup and is needed for every Unit of Measure Code. On the Test Plan Setup page, click Sample Planning.

: Sample Plannings page opened from Test Plan Setup for item Q-40012

The fields in the Sample Plannings page are described in the following table.

Field

Description

Description

Short description of the sample planning.

Unit of Measure Code

Identifies the Base Unit of Measure Code from the item

Sample Occurrence Code

Indicates the relevant sample planning for the item
