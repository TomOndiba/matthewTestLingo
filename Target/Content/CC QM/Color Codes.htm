Color Codes

Color Codes are used to provide lot-/serial number status visualization in a variety of pages (refer to topic “Lot-/Serial No. Status” on page 1 for more information).

: Color Codes page

The fields in this page are described in the following table.

Field

Description

Code

Unique identifier of the color code.

Description

Short description of the color code.

Color Name

Defines the color of the status information to which the code is assigned. The options are black, blue, red [italicized], green [bolded] and yellow.

Bold

If selected, the color of the status information to which the code is assigned is bolded.

Note

This field has no style influence when Color Name = yellow (never bolded) or green (always bolded).
