8D Report

The 8D Report can be generated from the Sales Return Order or opened from the cc|quality management menu.

To generate the report from the Sales Return Order page, complete the return order and then click Functions > Create 8D Report.

: Sales Return Order, preparing to create the 8D Report

Note

If the “Get Posted Document Lines to Reverse” function was used to populate the lines, ensure that the item line is selected before running the “Create 8D Report” function.

Click Yes to create the 8D report and then accept the creation confirmation message.

To open the generated 8D Report, click Line > 8D Reports. In the 8D Reports page, click Edit. The 8D Report Card opens with the following information:

  • Status = Created
  • Customer No. and Quantity transferred from the Sales Return Order
  • Texts transferred from the default template assigned in the Quality Management Setup page (if used)

: 8D Report Card created from Sales Return Order

Click Show Item Tracking on the ribbon to view the selected Lot-/Serial No. information.

: 8D Item Tracking for 8D Report Card created from Sales Return Order

Before any steps can be used in the 8D report, at least one Team Member must be added in the 8D Team Member FactBox. In the FactBox, click Edit > Open.

: Preparing to add Team Members to the 8D Report Card

The 8D Team Members page opens. In the Team Member Code field, the following can be entered:

  • Microsoft Dynamics NAV User; populates Team Member Name with user name
  • Code or Number and a name for external responsibility

In the Function / Responsibility field, enter a description of the user’s function on the team.

Select the Lead check box for the team leader. The Team Member Name of this user will be shown in bold in the 8D Team Member FactBox. Only one user can be selected as team lead.

: External and Microsoft Dynamics NAV users assigned as Team Members to the 8D Report Card

Click Release on the ribbon to change the Status to Analysis, which allows team members to be assigned to the steps.

: Status changed to Analysis for team member assignment

Note

The “Finish” and “Reject” functions close the 8D Report, which means it is only visible from the cc|quality management, Archive menu. Refer to topic "Closed 8D Reports" on page 1 for more information.

: Description updated in step D2, users and dates assigned to steps in 8D Report Card
