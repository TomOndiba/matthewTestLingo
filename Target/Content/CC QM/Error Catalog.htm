Error Catalog

The Error Catalog provides the ability to assign error codes to a failed value recording of measurement tests. Each setup page is described in the following sections.

Error Types

Error Types explain the kind of error that occurred.

: Error Types page

Error Causes

Error Causes explain why the error occurred.

: Error Causes page

Error Actions

Error Actions indicate what action must be taken to address the error.

: Error Actions page

Error Categories

Error Categories classify the intensity of the error.

: Error Categories page

Error List

In the Error List page, the various error settings are combined to create the error catalog, which can be used during the Value Recording process (refer to topic “Exercise – Process a Test Order” on page 1 and topic “Test Orders/Value Recording” on page 1).

: Error List page

In the QM Value Recording page, the Error Code field only appears for measuring tests; in the Test Order/Value Recording Card page, an Error Code can only be entered on measuring test lines.

: Error list accessed from the Error Code field on the QM Value Recording page
