Test Incidents are opened from the Lines FastTab (Lines > Test Incidents) of the source document from which the test order was generated. In this example, the test orders were generated from the Posted Purchase Receipt.

: Test Incidents list opened from the Posted Purchase Receipt for item Q-40012

On the Test Incident Card, the General FastTab contains the information that is common for all of the Test Orders. The Lines FastTab shows the list of Test Orders related to this Test Incident. The actions on the ribbon and the Lines FastTab are similar to actions on a Test Order.

The External Document, the Sample Taking Instruction and the Sample Handling Instruction are copied from the Sample Occurrence to each Test Incident where each can be edited.

: Test Incident Card for LOT_A001 for item Q-40012

The Test Orders created for one level contain the Test Incident No. and a Test Incident Order No. and are shown together on the Test Incident Card.
