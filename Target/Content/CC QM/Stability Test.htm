Stability Test

Stability Tests determine how a lot will behave under various inventory conditions, such as temperature or humidity, within a certain period of time. Each setup page is described in the following sections.

Inventory Conditions

Inventory Conditions are different conditions like temperature, brightness, or humidity for carrying out stability studies. Each inventory condition is marked with a code and a significant description for the condition.

: Inventory Conditions page

Stability Test Schedules

Stability Test Schedules define a default set of date formulas that can be automatically assigned to stability test lines. Double-click a line to view an existing schedule or click New on the ribbon to insert a new schedule.
