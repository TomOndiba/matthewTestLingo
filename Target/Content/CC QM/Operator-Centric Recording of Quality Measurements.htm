Operator-Centric Recording of Quality Measurements

A worksheet page has been added to show the Test Order Properties in a table where filters can be set on Test Device / Test Location and the value recording can be done directly.

Note

Test Orders must be set to Status = Value Recording to appear in the Value Recording Worksheet.

The Value Recording Worksheet page is accessed from the cc|quality management menu.

: Value Recording Worksheet with no filtering

The Value Recording Worksheet opens with a filter on the properties that are not tested and (if exists) a Responsibility Center from the User Setup. This view can be filtered by Test Device, Item No., Test Location and Test Incident (refer to topic “Test Incidents” on page 1).

: Value Recording Worksheet filtering on Test Device TD0008

Other filtering includes the Show Tested check box, which shows all lines with Property Tested = TRUE.

: Value Recording Worksheet showing tested lines

The Show Order Information check box adds columns from the Test Order(s) to the view with information of the Source Document.

: Value Recording Worksheet showing order information

On the ribbon, related information from the Test Order can be opened, including the Test Order itself, the Source Document from which the Test Order was created from, the Test Order Property Card, the Test Devices list page and the comments pages.

Value Recording

The value recording is done the same as in the Test Order/Value Recording Card by completing the appropriate fields for Counting Testing (“Failed Qty. …”) or Measuring Testing (“Determined Value”). The Property Tested check box may need to be set manually for counting testing.

: Value Recording Worksheet with counting and measuring testing results

When all properties of a Test Order are tested, the standard “Test Order ###### is Testing Finished” message appears and the test order Status is set to Testing Finished.

: Value Recording Worksheet with Testing Finished message for tested Test Order

Once the status is changed to Testing Finished, the lines are removed from the Value Recording Worksheet, regardless of the setting in the Show Tested check box.

: Value Recording Worksheet with only test order lines that have Status = Value Recording

 

As with the Test Order/Value Recording Card, all value recording is updated in the Test Order Card.

: Test Order finished from processing in the Value Recording Worksheet
