Example – Post Output with Retain Samples

This section demonstrates how samples are retained during output.

Based on the settings in Fig. 1, at least 10 units of item Q-40005 must be produced for 1 PCS sample to be retained. If a sample is created, it will be retained one month from the posting date.

First, a production order will be processed for 5 units of item Q-40005 to demonstrate how a sample is not created if the threshold is not met.

: Output posted for 5 units of item Q-40005; did not meet threshold for sample retention

The retained samples can be viewed from the cc|quality management, Archive menu.

: No samples retained from the output posting

Now a production order will be processed for 10 units of item Q-40005.

: Output posted for 10 units of item Q-40005; did meet threshold for sample retention

The Retained Samples page shows that a sample has been retained for one unit of item Q-40005.

: Sample retained from the output posting

Additionally, the Retained Samples page shows the following:

  • Lot No. – from the item’s lot number series
  • Elimination Date – one month from output posting date (1/24/2019)
  • Location code – from the Retain Sample Instructions
  • Sales/Cost Amounts – from item card, depending on costing method
  • Status – from the Retain Sample Status Code in Quality Management Setup
