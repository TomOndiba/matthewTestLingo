Test Activities

Test Activities define how to operate testing. Test Activities are used in the Test Order Scheduling process to relate one or more Test Properties to a group that represents the test step. Refer to topic “Master Data Setup” on page 1 for more information.

: Test Activities page

The fields on the Test Activities page are described in the following table.

Field

Description

Code

Unique identifier of the test activity.

Description

Short description of the test activity.

Test Activity Group Code

Indicates the category of test activity for grouping purposes.

Resource No.

Indicates the standard Microsoft Dynamics NAV Resource to be used for the Activity.

Resources are linked to work centers in order to use the Work Center structures like capacities. Work Center Capacities are used instead of Resource Capacities for Test Order Scheduling.

Work Center No.

Indicates the Work Center to be used for the activity; defaults from the linked Resource No.

The use of the Work Center results in usage of related working time structures so that no Activities are planned outside of the working time.

Work Center Group Code

Defaults from the Work Center specified in the Work Center No. field.

Exec. Time Unit of Measure Code

Defaults from the Work Center specified in the Work Center No. field.
