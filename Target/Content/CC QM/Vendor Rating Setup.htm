Vendor Rating Setup

The central setup of cc|vendor rating can be defined in the Vendor Rating Setup page.

: Vendor Rating Setup page

The fields in this page are described in the following table.

Field

Description

Assert Delivery Terms on Release

If selected, the Assert Terms of Delivery check box will automatically be selected in all purchase order lines when the order is released (refer to topic "Purchase Line" on page 1).

Use Post. Date as Arrival Date

If selected, the posting date of the receipt is used as the arrival date. If not selected, the arrival date must be populated manually.

Quality Rating on Release

If selected, the Quality Rating Code Document page opens automatically when a user posts a Status change for a Purchase Receipt from the Lot/Serial No. Administration page to make the user enter the missing rating information.

Allow Rating in Posted Rcpt.

If selected, ratings can be added to posted receipts through the “Quality Rating” function on the Quality Evaluation Purchase page or the Posted Purchase Receipt line.
