Exercise – Set Up Ratings for Vendor and Add a Soft Criteria Rating

In this exercise, set up the soft to hard criteria ratio for vendor V00010, activate the ratings for this vendor for all items and then add a soft criteria rating.

Set Up Ratings for a Vendor

Follow these steps to set up ratings for a vendor:

  1. In the navigation pane, click Purchasing > Order Processing > Vendors.
  2. Locate vendor V00010 and then double-click the line to open the Vendor Card page.
  3. Expand the Vendor Rating FastTab.
  4. In the Vendor Rating Factor Soft field, type 2.
  5. In the Vendor Rating Factor Hard field, type 5.
  6. Select the Duty of Rating check box.

: Vendor Rating information updated on Vendor Card

Add a Soft Criteria Rating

Follow these steps to add a soft criteria rating to a vendor:

  1. On the ribbon, click Navigate > Soft Criterion.
  2. In the Rating Type field, select Visit Report.
  3. In the Category Code field, enter 20.
  4. In the Version field, type 1.
  5. In the Valid from field, enter 01/01/2019.
  6. In the Valid to field, enter 06/30/2019.
  7. In the Rating field, type 90.

: Vendor Rating Soft added to vendor V00010

  1. Click OK to close the Vendor Rating Soft page and then click OK to close the Vendor Card page.
