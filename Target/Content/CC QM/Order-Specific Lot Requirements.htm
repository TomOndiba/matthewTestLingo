Order-Specific Lot Requirements

In addition to suggesting lots based on Test Plans at the customer level, it is possible to base the suggestion on the sales order line level.

This function allows users to copy the existing sales order Test Plan to a sales order line and edit the accepted values or tolerances (upper/lower limit) of each property. This criterion will be used in the “Suggest Lots” function to search for lots in inventory that fulfill the accepted value or that are within the tolerance.

Example

In this example, customer 20000 orders 250 units of item Q-40005 that has 300 units on hand in three lots: LOT1000, LOT1001 and LOT1002. This customer has defined that net weight must be between 200–205 but only LOT1000 and LOT1002 meet this criterion; and therefore 50 units cannot be fulfilled. The customer decides that a slight variance is acceptable so the entire order can ship immediately.

: Lot Selection page showing lots that are available for the customer

The order processor creates an order-specific test plan for the sales line and runs the “Suggest Lots” function again.

On the Lines FastTab, click Line > Order-specific Test Plan.

: Accessing the order-specific test plan setup

The Test Plan Setup page opens with an Order-specific line that uses the customer-specific Test Plan in the Test Plan No. field.

: Order-specific test plan with customer-specific test plan number

Click Copy Test Plan on the ribbon.

A new test plan is created from the customer-specific test plan and is inserted into the Test Plan No. field.

: Order-specific test plan

On the ribbon, click Customize Test Plan to make the order-specific changes. In the Test Plan Card page, update information on the General FastTab as needed.

: Test Plan Card for order-specific test plan

Select the first line and then click Property Set > Card on the Lines FastTab.

: Test Plan Prop. Set Card for order-specific test plan

In the Lower Limit field for the net weight test property field, change the value from 200.00 to 198.00.

: Modified Test Plan Property Set for order-specific test plan

Click OK to close the Test Plan Prop. Set Card page.

On the General FastTab of the Test Plan Card page, enter a date in the Valid from field and then change the Status to Certified.

: Certified order-specific test plan

Click OK to close the Test Plan Card page and then close the Test Plan Setup page.

On the Sales Order, click Functions > Suggest Lots on the Lines FastTab. Now all three lots (LOT1000, LOT1001 and LOT1002) appear in the Lot Selection page since each meet the criteria for this sales order line.

: LOT1001 added to the lots that meet order-specific test plan criteria appear for selection

The Order Processor selects LOT1000 and LOT1002 to take full inventory from these lots and then runs the function again to select the remaining 50 from LOT1001 to fulfill the order.

: Lots reserved for the sales order based on the order-specific test plan criteria
