Print a Test Order

On the Test Order page, click Test Order on the ribbon to open the Test Order request page for printing the report.

: Test Order print request page

To only print the test order for the current test – meaning the first test in the Test Order that has not been tested – a filter for the test order must be set on the Test Order FastTab.

 

On the Options FastTab, users can select the language and whether to show actual values.

: Test Order report set to show actual values
