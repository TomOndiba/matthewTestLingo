Exercise – Process a Recall

This exercise explains the basic steps of creating and issuing a Recall.

Create a Recall

Follow these steps to create a Recall:

  1. In the navigation pane, click cc|quality management > Recall > Recalls.
  2. On the ribbon, click New and then press ENTER to automatically populate the No. field.
  3. On the General FastTab, enter the filters to be included in the Recall search.
  4. In the Posting Date field, enter the current date.
  5. Select the Add Return Receipts check box to include customer return receipts.
  6. On the ribbon, click Suggest Recall Lines.

Issue a Recall

Follow these steps to issue a Recall:

  1. On the ribbon, click Navigate > Comments.
  2. In the Comment field, type a comment for the Recall.
  3. Click OK to close the Comment Sheet page.
  4. On the ribbon, click Issue.
  5. Select the Print check box to print a letter for each customer in the Recall.
  6. Select the Replace Posting Date check box if a date other than the date in the Posting Date field on the General FastTab of the Recall page should be used for issuing the callback.
  7. Enter a date in the Posting Date field if the Replace Posting Date check box was selected.

: Issue Recall page

  1. Click OK.
  2. If the Print check box was selected, a message appears asking if you want to print letters to customers. Click Yes to print the letters or No to stop the letter printing process.

The Recall is issued and removed from the Recall page.

Review Issued Recall

Follow these steps to review the Issued Recall:

  1. In the navigation pane, click cc|quality management > Archive > Issued Recalls.
  2. Double-click the line for the Issued Recall.

: Issued Recall page, with replaced Posting Date

  1. On the ribbon, click Print to generate the Issued Recall List report and print the letters to customers.
  2. Comments, shipment information and inventory information entered on the Recall can be accessed from the Navigate tab.
  3. Close the Issued Recall page.
