Due Date Parameters

In the Vendor Rating Param Date page – accessed on the cc|vendor rating menu by clicking Administration > Due Date Parameters – scores are allocated for the number of days the actual delivery date differs from the promised delivery date. These scores are included in the calculation of the hard vendor rating factors for the vendor.

: Vendor Rating Param Date page

In the Item Category Code field, this setup in can be limited to a certain item category. If no item category has been specified, the setup applies to all items. The item category is assigned on the General FastTab of the Item Card.
