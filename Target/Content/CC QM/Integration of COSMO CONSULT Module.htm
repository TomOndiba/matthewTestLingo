Integration of COSMO CONSULT Module

The “Send PDF as Email Distribution” functionality of cc|document configurator has been integrated into cc|quality management. This makes it possible to use the Email Distribution Setup functionality – combined with a setting on the Item Categories page – to send emails from the Lot-/Serial No. Administration page when a status change is made for lot- and/or serial numbers.

This section describes only the functionality that relates to cc|quality management. For detailed information about setting up and using the Email Distribution functionality, refer to the current “cc|document configurator” documentation.
