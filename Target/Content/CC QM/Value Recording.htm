Value Recording

On a Test Incident, the Test Orders can be released for value recording in two ways:

  • All by clicking Release Test Orders on the ribbon.
  • For individual test order by selecting the line and then on the Lines FastTab, clicking Function > Release.

: Release Test Orders function run on Test Incident

Click Value Recording to open the Value Recording Worksheet filtered on the Test Incident. The view can be sorted to the Test Order Properties for combined recording of all samples (refer to topic "Operator-Centric Recording of Quality Measurements" on page 1 for more information about using the Value Recording Worksheet).

: Value Recording Worksheet opened from Test Incident for all released test orders

After value recording has been executed, the Test Results for all Test Orders of a Test Incident can be compared in a matrix view. On the Test Incident Card, click Compare Test Results.

: Test Incident Properties Compare page

Further processing of the test orders, such as finishing a test order, can be handled by clicking Function > Finish for a line, or the test order can be opened by clicking Line > Test Order > Card to open the Test Order.
