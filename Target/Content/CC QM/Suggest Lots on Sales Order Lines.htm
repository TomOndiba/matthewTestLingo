Suggest Lots on Sales Order Lines

Lots that match a customer’s specifications can be selected on a Sales Order line. On the Lines FastTab, click Functions > Suggest Lots.

: Accessing the Suggest Lots function on the Sales Order

The Lot Selection page opens for items that use Lot No. Information and if lots are available for the customer.

: Lot Selection page showing lots that are available for the customer (note that LOT0005 is not listed as available for this customer because the measurement values were not acceptable in the test order for LOT0005)

To view the test results that the original Test Order is based on, the Test Order Properties Compare page can be opened from the Lot Selection page by clicking Navigate > Test Result.

: Test Order Properties Compare page for selected lot

Select one or more lots and then click OK.

: Selected lots to be assigned to sales order item tracking

The Item Tracking for the required quantity is automatically created using the marked lot numbers as long as the quantity is available.

: Item Tracking Lines page, after Lot Selection

In addition, the assigned quantity is reserved for the Sales Order.

: Sales Order page, reserved quantity after Lot Selection

Note

Only lots with Status = Released are transferred to Item Tracking and are automatically reserved.
