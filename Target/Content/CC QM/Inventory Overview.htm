Inventory Overview

The Inventory Overview page combines information from the Item Ledger Entries and the Warehouse Entries.

: Inventory Overview page opened from the cc|quality management menu

This inventory overview is indispensable, especially for companies who use the standard Warehouse module of Microsoft Dynamics NAV and therefore must be able to see their inventories on the warehouse level.

The fields on the Inventory Overview page are described in the following table.

Field

Description

Item No. / Item Description

Number and description of the respective item.

Location Code

Location code of the item / lot / serial number.

Bin Code

Bin code of the item / lot / serial number.

Lot No.

Lot number of the item.

Quantity

Quantity of the item / lot / serial number (not the Available Quantity) in the posted or base unit of measure, depending on the setting in the Inventory Overview Display field on the Quality Management Setup page (refer to topic "Quality Management Setup" on page 1).

Qty. Picked

Picked lots are displayed. Click the field to open the Warehouse Entries page to view related entries.

Status

Status of the lot / serial number.

Entry Date

Entry date of the lot / serial number.

Expiration Date

Expiration date of the lot / serial number.

Variant Code

Variant code of the item.

External Lot No.

External lot number of the vendor.

Item Category Code

Default field for the item category code.

Product Group Code

Default field for the product group code.

Serial No.

Default field for the serial number.

Warranty Date

Warranty date of the lot / serial number.

Retest Date

Date of the next quality management testing of the lot / serial number.

Quarantine Date

Quarantine date of the lot / serial number.

Availability Date

Availability date of the lot / serial number.

Quantity (Base)

Quantity of the item / lot / serial number (not the Available Quantity) in the base unit of measure of the Item Card.

Unit of Measure Code

Based on the setting in the Inventory Overview Display field in the Quality Management Setup page of whether the Quantity is displayed in the current unit of measure or the Base Unit of Measure from the Item Card.

Note

The Inventory Overview page can also be accessed by clicking the Inventory (IO) field on the Inventory List page (refer to topic "Inventory List" on page 1). This way, the inventory overview is filtered on the respective item.
