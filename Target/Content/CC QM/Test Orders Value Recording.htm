Test Orders/Value Recording

To enter values directly on the Test Order lines instead of entering values in the QM Value Recording page, click cc|quality management > cc|quality management > Test Orders/Value Recording. The Test Orders/Value Recording list page displays all orders regardless of status.

The Lines FastTab on the Test Order/Value Recording Card page contains the same fields that appear on the QM Value Recording page. The Test Property menu on the Lines FastTab contains menu items that relate to the selected Test Order/Value Recording line.

: Test Order/Value Recording Card opened from Test Orders/Value Recording list page

To process the test order, click Release on the ribbon and then enter testing results:

  • For Testing Method = Counting (AQL):
    • Select the Property Tested check box if there are no units with errors.
    • Enter the number of units with errors in the corresponding Failed Qty. X fields and then select the Property Tested check box.
  • For Testing Method = Measuring, enter the result in the Determined Value field.

When the last value is entered, a message appears stating that testing has finished; click OK. The Inspection Result field is updated and the Status is updated to Testing Finished.

: Value recording performed on Test Order lines

On the ribbon, click Finish to complete the testing process.
