Test Setup Oeser/Strohecker

In the Test Setup Oeser/Strohecker page, the test plan according to Oeser/Strohecker – which calculates the number of trading units from where samples must be extracted – is defined.

: Test Setup Oeser/Strohecker page

This setup is used to calculate the No. of Units to Test field on the Test Order based on the No. of Units field on the Test Order that comes from the source document (Sales Line, Purchase Line, Item Ledger Entry).
