Item Card

Lot creation and test-related values are defined on the Item Card; most are described in more detail throughout this training material. This section highlights the significant additions.

: Item Card page highlighting fields on the Item Tracking and QM FastTabs

Item Tracking FastTab

The fields on the Item Tracking FastTab are described in the following table.

Field

Description

Retest Calculation

Formula of the Retest Date; based on work date unless Expiration Date is entered then calculated from the expiration date.

Quarantine Calculation

Formula for the Quarantine Date; based on work date.

Automatic Lot No. Creation

If selected, the lot number is created automatically with the creation of the production order of the item.

Autom. Create Lot No. at Status

Indicates the status of the production order at which the automatic lot number creation should occur. The options are:

  • Firm Planned – When creating a production order with the status Firm Planned.
  • Released – When creating a production order with the status Released.

QM FastTab

The fields on the QM FastTab are described in the following table.

Field

Description

Inventory (Expired)

Displays the quantity of inventory that has expired. Click the value to open the Inventory Overview page for further details (refer to topic “Inventory Overview” on page 1).

Inventory (XX)

Displays the inventory per status for all statuses that have the Show in Inv. List / Item Card check box selected in the Lot-/Serial No. Status page. For space reasons, the number of the viewable status-dependent inventory is limited to a maximum of ten.

Home Tab

The Home tab contains three cc|quality management-related actions. Each is described in the following table.

Action

Description

Test Plan Setup

Opens the Test Plan Setup page, where users can assign Test Plans for the item to different Source Types (such as Sales Order, Production Order and so on). Users are also able to enter vendor-specific or customer-specific test plans. Refer to topic “Test Plan Setup on Items” on page 1 for more information.

Test Status

Opens the Test Status page, which displays the Current Test Type and the corresponding Current Test No. for the item and its respective vendor. This information is used by the system for determining the Inspection Severity of the next test.

This page is populated from the Inspection Severity page. It is based on the test result of the latest processed test order and the resulting test type when finishing the test order.

Retain Sample Instruction

Opens the Retain Sample Instruction page, where users can set up how and under what conditions a sample must be retained for the item.

Refer to topic “Retain Sample Management” on page 1 for more information.
