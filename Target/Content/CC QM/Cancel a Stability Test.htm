Cancel a Stability Test

If a test result lies outside of the specification, the Stability Test can be cancelled manually using the “Cancel” function on the test order. With this function, it is possible to cancel only the test order or both test order and stability test.

If afterwards the “Create Stability Test” function is run, no new test orders are generated for the cancelled stability tests. The cancellation of a separate test order has no consequences for the status of the stability test.

Cancelling Only a Test Order

When only the Test Order is cancelled (Yes to cancel the Test Order, No to cancel the Stability Test), a message appears that the Stability Test is still active and after clicking OK, the Status and Stability Test Status fields are set to Cancelled on the Test Order.

: Test Order cancelled for a Stability Test without cancelling the Stability Test

The Test Order No. is removed from the General FastTab of the Stability Test Card and only the corresponding test line(s) are cancelled, thus allowing the user to proceed with the Stability Test on the next line.

: Stability Test with cancelled test order lines

Cancelling a Test Order and Stability Test

When both the Test Order and Stability Test are cancelled, (Yes to cancel the Test Order, Yes to cancel the Stability Test), the Status and Stability Test Status fields are set to Cancelled on the Test Order. On the Stability Test Card, the Status is set to Cancelled and the corresponding test line(s) are cancelled.
