Exercise – Process a Test Order

In this exercise, process the Test Order that was automatically created for item Q-40014.

Release a Test Order for Value Recording

Follow these steps to release a test order:

  1. In the navigation pane, click cc|quality management > Test Orders per Status > Test Orders - Created.
  2. Double-click the line for test order created in the “Generate a Test Order Automatically” exercise.
  3. On the ribbon, click Release.
    Notice that the Test Order page closes (or a different order is active if one exists); the released order was removed from this view because of the status change.

Record Values in the Test Order

Follow these steps to record values:

  1. In the navigation pane, click Test Orders per Status > Test Orders - Value Recording.
  2. Double-click the line for the released test order.
  3. On the ribbon, click Value Recording.
    The Details FastTab opens with Test Property Code TP00038.
  4. Click Det. Value within Exp. on the ribbon to determine that there are no scratches found.

: QM Value Recording page

  1. The next line with Test Property Code TP00039 opens. Click Det. Value within Exp. to determine that there are no dents found.
  2. The next line with Test Property Code TP00102 opens. Click Det. Value within Exp. to determine that there are no faults found in the packaging.
  3. The next line with Test Property Code TP00006 opens. In the Details FastTab, type 99.50 in the Determined Value field and then press ENTER. 

: QM Value Recording page with Determined Value entered

NOTE

The Error Code field now appears in the Details FastTab because TP00006 is a measuring test.

  1. Click Next Property on the ribbon.
  2. The next test line with Test Property Code TP00005 opens. In the Details FastTab, type 19.60 in the Determined Value field and then press ENTER.
  3. Click Next Property on the ribbon.
  4. The next test line with Test Property Code TP00008 opens. In the Details FastTab, type 15.00 in the Determined Value field and then press ENTER.
  5. Click Next Property on the ribbon.
  6. The next test line with Test Property Code TP00020 opens. Click Det. Value within Exp. to determine that the overall appearance is OK.
  7. The next test line with Test Property Code TP00021 opens. Click Det. Value within Exp. to determine that there is no unusual smell.
  8. The next test line with Test Property Code TP00041 opens. Click Det. Value within Exp. to determine that the items all have the same color (BLUE).

The QM Value Recording page closes automatically when all properties are tested and the Test Order disappears from the view because the Status field has changed to Testing Finished.

Finish the Test Order

Follow these steps to finish the test order:

  1. In the navigation pane, click Test Orders per Status > Test Orders - Testing Finished.
  2. Double-click the line for the tested order.
  3. On the General FastTab, note that the Inspection Result field is set to Items are OK.
  4. On the Lines FastTab, note that the Tested field is set to Yes on each line.

: Test Order with testing completed

  1. On the ribbon, click Finish.
  2. Click Yes to close the test order.

The Test Order disappears from the view because the Status field has changed to Finished.
