Test Property Translations

The translation feature in Test Properties makes it possible to add multilingual capabilities to the testing process. This is helpful for multilingual testing facilities or generating reports that are required in a customer’s language.

Translations can be added to the Test Properties and/or Test Property Sets based on the business process. The translations transfer from these settings to the Test Plans through to Value Recording. These translations can be modified in the various pages as needed, and viewed in the Test Plan, Test Order and Analysis Certificate reports.

Translations in Test Property Sets

Although translations can be added to Test Properties, they are ideally added to Test Property Sets, which are directly used in Test Plans. In addition to the translations of the Test Properties, a single translation can be added to the Test Property Set.

On the Navigate tab of the Test Property Set Card, click Translations to open the Test Translation page. In the Language Code field, enter the relevant language and then the translation in the Description field.

: German translation for “Visual Inspection” added to Test Property Set TPS0002

This translation will be used as the Description of the Test Property Set in value recording and reports.

On the Lines FastTab, click Line > Translations to open the Test Translation page for the individual Test Properties. The fields are populated as described for the Test Property Set.

: German translation for “Scratch” added to Test Property TO00038

Note

Translations added to existing Test Properties/Test Property Sets are not automatically updated through to the Test Plan. Therefore, the “Update Test Plan” function must be run from the Test Property Sets to update all related Test Plans at once (refer to topic “Mass Updates of Test Plans” on page 1). Translations added to new or existing Test Properties/Test Property Sets will be included in new Test Property Sets and Test Plans.

Translations in Test Plans

Once the translations are added to the Test Properties and the “Update Test Plan” function is run (if added to an existing Test Property Set), the translations of Test Properties and the Test Property Set are available in the Test Plan.

: German translation for Test Property Set TPS0002 available in Test Plan

The Test Properties translations are visible by opening the Test Plan Prop. Set Card (Lines FastTab > Propery Set > Card > Lines FastTab > Property > Translations).

: German translation of Test Property TP00038 available in Test Plan Property Set

Translations in Test Orders

Translated test properties and test property sets are visible in the Test Order if a QM Language Code is defined in the User Setup.

: German language code used in translations defined for user in User Setup

If a language code used in the translations is not defined per user or a language code does not exist for the Test Property Set, then the default language of the system is shown in description field of the Test Order line.

: German translation of Test Property Set shown in first line of Test Order; default language (English) shown in other lines which are not yet set up with translations

The settings in the User Setup are also enabled for viewing the Test Order Properties with translations. The Caption field in the General FastTab shows the translated Test Property Set description and the Description field in the Lines FastTab show the translated Test Properties.

: German translation of Test Property Set shown in General FastTab and Test Properties in the Lines FastTab

Translations in Value Recording Pages

The settings in the User Setup also apply to the various Value Recording pages. In the QM Value Recording page (opened from the Test Order), the translated Test Property is visible in the Description field.

: German translation of Test Property shown in Description field of QM Value Recording page

In the Test Order/Value Recording Card and the Value Recording Worksheet pages, the translated Test Properties are listed in the Description field of the Lines FastTab.

: German translations of Test Properties shown in Description field of Test Order/Value Recording Card and Value Recording Worksheet pages

Translations in Reports

Translations of Test Properties and Test Property Sets can be included in reports by entering the relevant Language Code in the report request page. The reports with translation capabilities are the Test Plan, Test Order and Analysis Certificate.

Similiar to the visibility of translations in the Test Order and Value Recording pages, translations will appear for test properties/test property sets that match the Language Code; otherwise the report text will print in the default system language.
