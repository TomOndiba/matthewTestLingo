Example – Post Shipment with Retain Samples

This section demonstrates how samples are retained during shipment.

Based on the settings in Fig. 1, there is no threshold of item Q-40008 that must be shipped for a sample of .10 KG to be retained. If a sample is created, it will be retained ten days from the posting date.

A sales order will be processed for 50 units of item Q-40008 to demonstrate how a sample is retained on shipment.

: Shipment posted for 50 units of item Q-40008

The Retained Samples page shows that a sample has been retained for 0.1 units of item Q-40008.

: Sample retained from the shipment posting

Additionally, the Retained Samples page shows the following:

  • Lot No. – from the item’s lot number series
  • Elimination Date – ten days from posting date (1/24/2019)
  • Location code – from the Retain Sample Instructions
  • Sales/Cost Amounts – from item card, depending on costing method
  • Status – from the Retain Sample Status Code in Quality Management Setup
