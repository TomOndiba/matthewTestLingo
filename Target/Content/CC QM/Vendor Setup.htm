Vendor Setup

In the Vendor Setup page, define the weighting of the Hard (date, quality, quantity) towards the Soft rating criteria (audit, self-assessment, visitor report) for the vendor rating per vendor. The values entered here are copied to the fields on the Vendor Rating FastTab of the Vendor Card. If no vendor-specific value is defined here, the system will use the general weighting by way of comparison (refer to topic "Weighting" on page 1).
