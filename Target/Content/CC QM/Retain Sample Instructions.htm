Retain Sample Instructions

The Retain Sample Instructions manage how item samples are to be retained. The Retain Sample Instructions need to be entered for the item or item category code, and can also be reached from the Item Card and Item List.

: Retain Sample Instruction page opened from the cc|quality management, Administration menu

The fields on the Retain Sample Instruction page are described in the following table.

Field

Description

Item No.

Identifies the item for which samples need to be retained.

Item Category Code

Identifies an item category code for which samples need to be retained.

Note

Instructions set on an Item override settings on item category codes.

Description

Indicates the description of the selected item or item category code.

Quantity / Unit of Measure Code

Together determines the item portion to be taken for retention, for example 0.5 kg.

Retain Location Code

Identifies the location where the retained samples need to be stored.

Per Lot

If selected, samples are retained at production order output.

Per Shipment

If selected, samples are retained at sales shipment.

Retention Time

Indicates the length of time in which the samples need to be retained.

Comment

User-defined description of the instructions.

Quantity Threshold

Indicates the minimum quantity to be posted on the production/sales order to create a Retained Sample.

If the quantity is less, no sample will be retained.

Refer to topic “Retain Sample Management” on page 1 for more information.
