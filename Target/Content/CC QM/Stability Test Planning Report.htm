Stability Test Planning Report

The Stability Test Planning report displays an overview of all test orders that are to be generated via a stability test within a specified time frame.

Access this report from the ribbon of the Stability Test page or the cc|quality management menu by clicking Stability Test > Reports and Analysis > Stability Test Planning.

: Stability Test Planning report request page

In the request page, set filters for a certain stability test as well as enter a date range, for which the analysis should be run.

Afterwards the system determines all test orders, according to planning, that are supposed to be created within the specified time frame. The results are sorted by date.
