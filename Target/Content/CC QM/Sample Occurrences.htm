Sample Occurrences

The Sample Occurrence is used with Sample Planning to control the number of Test Orders to be created. Once the number series are defined and assigned in Quality Management Setup, the Sample Occurrences must be created.

: Sample Occurrences page

Click New to create a new occurrence or Edit to open an existing occurrence.

The Sample Occurrence contains a relative and an absolute value of Test Orders to be generated. The relative number of Test Orders depends on the quantity of the source document position for which the Test Order(s) are created.

: Sample Occurrence Card

The fields on the General FastTab are described in the following table.

Field

Description

Code

Unique identifier of the sample occurrence.

Description

Short description of the sample occurrence.

Test Order per each (Units)

Determines the quantity for that one Test Order is created.

Example

If 100 is entered in this field and the source document line contains a quantity of 200, two Test Orders are created.

Absolute No. of Test Orders

Assigns a number of Test Orders that will be created regardless of the quantity on the source document line; works in combination with the value in the Test Order per each (Units) field.

Example

Entering 1 in this field, along with the 100 in the Test Order per each (Units) field, will generate one Test Order per 100 Units plus one extra Test Order. Therefore, with a total quantity of 200 in the source document line means that three Test Orders are created.

In general, the number of test orders created based on source document line quantity follows this pattern:

From Quantity To Quantity No. of Test Orders
0 99.99 1
100 199.99 2
200 299.99 3
300 399.99 4
...    

Status

Indicates the current state of the sample occurrence. The options are New, Under Development, Certified and Closed. Only certified Sample Occurrences can be assigned to a Test Plan Setup, and therefore, used for Test Order creation.

External Document Path

Assigns an External Document to the sample occurrence. To open the document, click Show External Document on the ribbon.

Sample Location

Indicates a location for the sample; can be filled in with any value or a Location Code can be selected from the Location table.

The remaining FastTabs are used for instructing users how to manage the sample:

  • In the Sample Taking FastTab, enter the instructions for taking the sample.
  • In the Sample Handling FastTab, enter the instructions for handling the sample.
