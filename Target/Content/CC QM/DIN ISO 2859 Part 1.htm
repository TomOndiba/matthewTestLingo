DIN ISO 2859 - Part 1

The procedure of DIN ISO 2859 - Part 1 only applies to those test properties that have been allocated to a Test Property Set with the testing method of Counting (AQL).

Each setup page is described in the following sections.

Allowed AQL Values

In the Allowed AQL Values page, the Acceptance Quality Level (AQL) values to be applied must be defined according to the requirements of the supplier and the receiver. The AQL logic is only to be used for component testing.

: Allowed AQL Values page

The values in this page are automatically transferred to the Sample Instruction for Test Type page.

Test Types

In the Test Types page, users can define the various test types that form the basis of the Sample Instruction for Test Type page.

: Test Types page

Inspection Levels

By means of various inspection levels, the key letters for the sample size are allocated to a certain lot size (refer to topic "Sample Size for Inspection Level" on page 1). According to DIN ISO 2859 - Part 1, section 10.1 “Inspection Level”, inspection level II must be used if nothing else has been specified.

: Inspection Levels page

Sample Size for Inspection Level

In the Sample Size/Inspection Level page, users can select the appropriate inspection level and then assign Key Letters for the sample size to the according lot size interval.

This page is accessed by clicking Card on the ribbon of the Inspection Levels page.

: Sample Size/Inspection Levels page

Sample Instruction for Test Type

The Sample Instruction for Test Type menu item opens the Sample Instruction Setup List page; click View on the ribbon to open the Sample Instruction Setup page for the selected Test Type.

: Sample Instruction Setup for the Normal test type

The fields on the Lines FastTab are described in the following table.

Field

Description

Key Letter

Manual setup of the key letter for the sample size from the Sample Size/Inspection Level page.

Sample Size

Based on the AQL Values and the Key Letter for the sample size, the sample size must be gathered from the Sample Instruction of DIN ISO 2859 - Part 1.

AQL Values

Based on the values specified in the Allowed AQL Values page.

The values in the matrix display the maximum allowable acceptance limits depending on the sample size and the defined AQL Values. Thus, the allowable acceptance number indicates how many error parts the sample is allowed to contain so that the tested quantity (= shipped quantity) is still accepted.

Inspection Severities

The switching rules between the various test types of items to be tested (simultaneously considering the vendor) are defined in the Inspection Severity page.

When the Inspection Severities menu item is accessed, the Inspection Severities list page appears which displays all existing inspection severities. To open the Inspection Severity page, select the relevant line and then click Edit or View.

: Inspection Severities page

The respective current test type / test number of an upcoming test order for an item is entered into the Test Status page (Item Card > Navigate > Test Status) based on the result of the test order and the determined value of this page.

The fields in this page are described in the following table.

Field

Description

Current Test No.

Next Test No. when OK

Indicates the next test number (in connection with the next test type) if the testing has been classified as OK.

Indicates the next test type (in connection with the next test number) if the testing has not been classified as OK.

Indicates the next test number (in connection with the next test type) if the testing has not been classified as OK.

Description

If an item has not been tested yet, the system automatically selects the test type with this check box selected (usually, this is the Normal 1 inspection type).
