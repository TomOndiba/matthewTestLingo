Automatic Email with Lot-/Serial No. Status Change

On the Item Categories page, click Actions > Lot Status Email Setup to define whether an email should be sent in case of a lot-/serial number status change for that Item Category and what setup must be used when the status changes from Status A to Status B.

The emails can be set up for changes of From Status > To Status, only To Status or only From Status.

: Lot status change settings for e-mail distribution on RAW MATERI item category

The email setup is managed in the Email Distribution Setup page, accessed from the Departments menu via Administration > Application Setup > General. Here the usage of Outlook or SMTP Mail can be defined as well as standard Recipients, Subject line format, or special usage filters like Customer/Vendor/Contact/Responsibility Center and Language Code.

Subject text can be entered in the Subject Text field. The text may contain placeholders to use dynamic parts in the Subject line as well as in the Email Body Text. Some possible placeholders are:

%1 – Item No.

%4 – From Status

%2 – Variant Code

%5 – To Status

%3 – Lot No.

%6 – Serial No.

Note

Numerous placeholders can be used in the Body Text. Refer to the current “cc|document configurator” documentation for the complete list.

: Email Distribution settings for lot status change with Navigate tab showing Text Module and Attachment access

Attachments can be assigned to the specific email distribution setup entries. Attached file(s) will be included in the emails sent using this distribution.

Additionally, cc|text module functionality can be used on each Email Distribution Setup line to define the text to appear in the body of the email.

: cc|text module text added to the LOTCHANGE Email Distribution code

With these settings, an email is generated or automatically sent when a lot status change is posted from the Lot-/Serial No. Administration page.

: Email generated from changing lot status from RELEASED to QUARANTINE for item with RAW MATERI category

The Send Email page includes the Edit in Outlook check box, which will open the email in Outlook to allow for further functionality. To send the email without opening Outlook, clear the Edit in Outlook check box and click OK.

: Email opened in Outlook from the Send Email page
