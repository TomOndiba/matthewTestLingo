Interface Architecture

The interface transfers data between Microsoft Dynamics NAV and the 3rd party application in a generalized format. Data that should be transferred is always reformatted to the generalized format where each single value is stored as a record with field name and field value.

The interface consists of a 4-layer architecture and is designed to provide secure transactions. An external device or application will send data via various transportation methods and in various data formats.
