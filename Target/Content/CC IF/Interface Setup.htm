Interface Setup

The Interface Setup page contains all settings relating to cc|interface. This page is accessed on the Departments menu by clicking cc|interface > Administration > Interface Setup.

: Interface Setup

General FastTab

The field on this FastTab is described in the following table.

Field

Description

Default Endpoint Address

Indicates a default endpoint address that will be pre-populated in the Endpoint / Filename field in the Data Formats, which then default to associated Project Functions.

 

Datafox FastTab

The fields on this FastTab are used in the Datafox interface and are described in the following table.

Field

Description

Import Folder Datafox

Indicates the folder where the Datafox Talk server save the files containing the data from the terminal.

Export Folder Datafox

Indicates the folder from where the Datafox Talk server will read the files and send the data to the Terminals.

Folder Processed Files

Indicates the folder where the files that have been successfully processed are stored.

 

TDC FastTab

The fields on this FastTab are described in the following table.

Field

Description

Standard Journal Template

Identifies the TDC Journal where time data from the Datafox interface will be processed.

Standard Journal Name

Identifies the batch of the TDC Journal where time data from the Datafox interface will be processed.

Post TDC Journal

If selected, the time data from the Datafox interface is sent to the TDC Journal and is immediately posted. If not selected, the Datafox terminal data is sent to the TDC Journal and remains open for manual posting.

Product Version Information

On the ribbon, click About to display the current product version information.
