Transaction Entries

A new transaction entry is created for each step within a transaction and therefore attached to a transaction. The transaction entries are, from a Microsoft Dynamics NAV point of view, inbound or outbound entries.

The Transaction Entries page can be accessed from either the Transactions or Project Transactions pages by clicking Entries on the ribbon.

: Transaction Entries

The fields on the Transaction Entries page are described in the following table.

Field

Description

ID

Unique identifier of the entry within a transaction.

Type

Specifies the type of the transaction entry from the Microsoft Dynamics NAV perspective. The options are Inbound and Outbound.

Status

Indicates the current process status of the transaction entry. The options are:

  • <blank>
  • Received
  • Processed
  • Sent
  • Error

Data Format Code

Indicates the data format used in the function and transaction.

Endpoint / Filename

Displays the endpoint or file name used in the transaction.

Created By User Name

Identifies the name of the user who created the record.

Created Date-Time

Indicates the date and time in which the record was created.

Created By User ID

Indicates the GUID of the user who created the record.

Attached to Entry ID

Contains a transaction entry ID if the transaction entry is directly attached to another transaction entry. For example, if the processing of an inbound entry results in an outbound entry, the outbound entry contains the inbound entry ID in this field.
