Introduction

Businesses often require a connection from the hardware devices they use – such as weighing machines in bulk processing, density meters, RFID scanners, and time clocks and external applications like master control station, CAD application – to Microsoft Dynamics NAV.

In many cases, a separate interface has been implemented for every device or application and all interfaces worked in a similar way and differ only in the format and way of transportation. To provide a generalized interface with a normalized data structure, cc|interface was developed.

Key Features

  • Interface Framework
  • Layered Architecture
  • Normalized Data Structure
  • Pre-implemented Text File Support
  • Pre-implemented Web Service Support

cc|interface functionalities are accessed in the cc|interface menu.
