Interface Records

A transaction can contain one or more records. These records are attached to the Transaction Entries and can be accessed by clicking Records on the ribbon of the Transaction Entries page.

: Interface Records

The fields on the Interface Records page are described in the following table.

Field

Description

Record ID

Unique identifier within the transaction entry.

Project Code

Indicates the project code of the related transaction.

Project Function Code

Indicates the project function code of the related transaction.

Status

Indicates the current process status of the transaction entry. The options are:

  • <blank>
  • Processed
  • Sent

 

Interface Record Card

A record consists of several values that can be reviewed in the Interface Record Card page, accessed by clicking View on the ribbon.

: Interface Record Card

General FastTab

The General FastTab shows information regarding the current record within the transaction. The fields on this FastTab are described in the following table.

Field

Description

Transaction ID

Refers to the ID field of the Transaction table.

Transaction Entry ID

Refers to the ID field of the Transaction Entry table.

Record ID

Same as on the Interface Records page.

Project Code

Same as on the Interface Records page.

Project Function Code

Same as on the Interface Records page.

 

Lines FastTab

The Lines FastTab displays the fields belonging to a record. The fields on this FastTab are described in the following table.

Field

Description

Field Sequence

Indicates the sequence number of the field within a record.

Depending on if the transaction is Inbound or Outbound, the sequence number defines the order in which the fields have been received or the order in which they are sent.

Field Name

Indicates the field name from the record.

Field Value

Indicates the field value from the record.
