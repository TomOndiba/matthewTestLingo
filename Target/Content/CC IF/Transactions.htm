Transactions

A transaction record with a unique ID is created for each process that is initiated by either Microsoft Dynamics NAV or an external component. Transactions can be viewed for all projects (cc|interface > General > Transactions) or filtered by project (cc|interface > General > Projects > Transactions).

: Transactions and Project Transactions

The fields on the Transactions and Project Transactions pages are described in the following table.

Field

Description

ID

Unique identifier of an interface process.

Project Code

Indicates the Project to which the transaction is related.
Only on the Transactions page.

Project Description

Displays the description of the Project.

Created By User ID

Indicates the GUID of the user who created the record.
Only on the Project Transactions page.

Created By User Name

Indicates the name of the user who created the record.
Only on the Project Transactions page.

Created Date-Time

Indicates the data and time in which the record was created.
Only on the Project Transactions page.
