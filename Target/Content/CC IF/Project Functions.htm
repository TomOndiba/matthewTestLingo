Project Functions

Within a project, specific functions can be called to process requests and data. These functions are defined in the Project Functions page. Possible functions can be sending or exporting master data or receiving or importing data.

: Project Functions

The fields on the Project Functions page are described in the following table.

Field Name

Description

Code

Unique code of project function.

Description

Short description of the project function.

Processing Codeunit

Indicates the interface codeunit assigned to this function, e.g. for processing the data like a Codeunit.

Data Format Code

Defines the data format and transportation layer.

Continue Processing After Error

If selected, the interface will continue processing after an error.

Endpoint / Filename

Indicates the endpoint for the project function, which can be a file name, a web service address or an FTP server address.

Defaults from the assigned Data Format Code if one exists.
