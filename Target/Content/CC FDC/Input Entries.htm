Input Entries

The Input Entries list single check ins and check outs. Therefore during posting of Ending Input Entries (“End – Run/Presence Time”, “End – Break Time”..) a related Beginning Input Entry (“Begin – Run/Presence Time”, “Begin – Break Time”..) is searched automatically. Such an Input Entry pair is connected via the Linked Input Entry field.
