FDC Terminal

To use the FDC Terminal, it is necessary to copy the required DLL in the Service Tier directory under “Add-ins”.

The “CosmoConsult.Dynamics.NAV.FDCW1xxxx.Terminal.dll” file must be copied to any subfolder in the Microsoft Dynamics NAV 2015/2016 server add-in directory (middle tier), for example, “% Program Files % \ Microsoft Dynamics NAV \ xx \ Service \ Add-ins \ Cosmo Consult AG \ FDC”.

The necessary add-ins are registered automatically in Microsoft Dynamics NAV when the FDC Setup page is opened.
