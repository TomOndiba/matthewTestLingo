FDC Compare Expected/Actual

The Comparison of Expected/Actual report offers an overview of differential times and quantities of all work orders. The calculation of this report is based on the values from FDC Work Order Lines. For a better view, only work order lines containing values are displayed.

The request page of the reports offers the following options:

  • With deviation larger than (in %) – Only detailed lines with a deviation of at least the entered value are used.
  • Show detailed Lines – If selected, the corresponding Detailed Lines of all Work Orders are visible. If not, only the total lines of the Work Orders are shown.
