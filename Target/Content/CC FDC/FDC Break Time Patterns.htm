FDC Break Time Patterns

With FDC Break Time Patterns, breaks can be assigned to shifts with each shift having exactly one break time pattern. Each break time pattern can have multiple breaks that are registered in the break time pattern lines. The sum of these breaks will affect the net times of the shifts to which they were assigned. In the settlement (refer to topic "Invoicing" on page 1), breaks in FDC Entries can be deducted automatically.

: FDC Break Time Pattern

The fields in the FDC Break Time Pattern page are described in the following table.

Field

Description

Code

Unique identifier of the break time pattern; maximum 20 alphanumeric characters.

Description

Brief description of the break time pattern.

Break Time

Indicates the sum of all breaks in this pattern.

Description

Brief description of the break time pattern line.

Break Time

Indicates the calculated break time.

Start Time

Indicates the time when the break begins.

End Time

Indicates the time when the break ends.

Start Time Next Day

Must be activated when the break begins on the following day.

End Time Next Day

Must be activated when the break ends on the following day.
