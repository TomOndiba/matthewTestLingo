Order Stock

The Order Stock determines the selectable FDC Work Order Lines for an FDC Resource and is defined in the Order Stock Type field of an FDC Resource. In the FDC Journal, the “Show Order Stock” function shows the Order Stock after entering the FDC Resource into a line. The Selected check box on the lines can be selected to indicate which lines should be transferred to the journal.

The options of the Order Stock Type field filters the FDC Work Order Operation information as follows:

  • Posting Permitted – displays all FDC Work Order Operations to which the FDC Resource is permitted to post times or quantities (refer to topic "Posting Permission" on page 1).
  • FDC Resource on Work Order Line – limits the view to all FDC Work Order Operations to where the FDC Resource is directly assigned (FDC Resource No. field).
  • Current Work Area and Posting Permitted – reduces the list to all FDC Work Order Operations where a posting permission exists and that are located in the FDC Resource’s current FDC Work Area.

In the FDC Terminal, the lookup of the order stock of the previously logged in FDC Resource is shown. If the Terminal List option is selected in the Terminal Lookup Type field, only one FDC Work Order Operation can be selected for capturing.
