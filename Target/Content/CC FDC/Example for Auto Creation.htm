Example for Auto Creation of Order Bundle

Starting point:

  • FDC Resource is not logged in.
  • For the FDC Resource, the Auto Order Bundle Creation option is selected in the Concurrent Beginning field.

Sequence:

  1. Start Operation A ð no Concurrent Beginning detected ð no Auto Order Bundle creation

In Addition: Input Entry (Begin) created with reference to Operation A

  1. Start Operation B ð Concurrent Beginning detected ð Order Bundle created, including Operation A and B

In Addition:

  1. Start Operation A and Input Entry 1 cancelled
  2. Start changed to Order Bundle with Start Time of Operation A.
  3. Input Entry (Begin) generated with reference to Order Bundle and Start Time of Input Entry 1
  1. Start Operation C ð Concurrent Beginning detected ð Operation C added to Order Bundle

In Addition:

  1. Start Operation B and Input Entry 2 cancelled
  2. Start changed to Order Bundle with Start Time of Operation B.
  3. Input Entry (Begin) generated with reference to Order Bundle and Start Time of Input Entry 2 = Start Time of Input Entry 2
  1. Stop Operation A, B or C ð Detection of Auto Order Bundle ð Stop of Order Bundle

In Addition:

  1. Input Entry (End) generated with reference to Order Bundle
  2. FDC Entry generated with reference to Order Bundle
  3. Automatic Distribution occurs
  4. ü Times get distributed to Operations A, B and C
