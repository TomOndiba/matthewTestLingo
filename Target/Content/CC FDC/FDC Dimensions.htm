FDC Dimensions

Dimensions are data that can be attached to entries in order to perform appropriate analysis. FDC Default Dimension values can be stored for both FDC Resources as well as FDC Work Order Lines. The handling of the FDC Dimensions are the same as those of standard Microsoft Dynamics NAV dimensions. It is also possible to restrict the FDC Dimension values (refer to topic "Permitted FDC Dimension Values" on page 1).

In journals (refer to topic "Capturing (FDC and STDC)" on page 1) the FDC Dimensions get passed to FDC Input Entries, FDC Entries and STDC Entries. The Dim. Values Mandatory on End field in FDC Setup enables checking of FDC Dimensions at Input Entries only for Entry Type = End. Therefore, on processing of Input Entries to FDC Entries, only FDC Dimensions of Entry Type = End are transferred.

Permitted FDC Dimension Values

The setup of Permitted FDC Dimension Values occurs on FDC Resources or FDC Work Orders by clicking Dimensions > Permitted FDC Dimension Values.

Permitted FDC Dimension Values

Permitted FDC Dimension Values control the types or combinations of dimension values that can be used for Work Order posting. Similarly, the default dimension values will be defined at the resource or the FDC Work Order Line. For the selection and posting of Input Entries and FDC Entries, only the intersection of permitted dimension values is possible. No definition of a dimension code allows all values, while an empty dimension value does not correspond to a permitted value.

The fields in the Permitted FDC Dimension Values page are described in the following table.

Field

Description

Dimension Code

Identifies the FDC Dimension code, for which a limitation of Permitted FDC Dimension Values should be made.

Dimension Value Code

Indicates the corresponding dimension value code, for which every Permitted FDC Dimension Value a single line is necessary. An empty value corresponds to “no permitted value” on posting the selection.

 

FDC Dimension Analysis

All FDC Dimensions are associated with the entry if specified at posting. All FDC Dimension Analysis can be created using Business Intelligence (BI) software (e.g. QlikView from QlikTech).
