Redirection

The Redirect to FDC Work Order No. and Redirect to FDC Work Order Line No. fields refers to other released FDC Work Order Operations. A redirected FDC Work Order Operations is indicated by the color grey.

: Redirecting to other Work Order Operations

A Redirect allows users to capture data on one Work Order Operation without posting to the operation by transferring the data to other operations.

For example, operations for Work Orders WO-0010 and WO-0012 are contained in Order Bundle WO-0015. Data is captured in Order Bundle WO-0015 but the postings need to be processed in Work Orders WO-0010 and WO-0012. By setting the Redirect to FDC Work Order No. and Redirect to FDC Work Order Line No. fields on WO-0015 to the corresponding operations on Work Orders WO-0010 and WO-0012, the Redirect function transfers all the captured from Order Bundle WO-0015 to Work Orders WO-0010 and WO-0012 for posting.

During posting, all information like permissions or dimensions from the original capture line are checked.
