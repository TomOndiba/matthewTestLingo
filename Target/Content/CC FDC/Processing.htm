Processing

On processing of an Input Entry pair, an FDC or STDC Journal Line is created and – depending on FDC Setup (Autom. Posting When Processing Input Journal field) – is posted instantly. An Input Entry pair is built from the Entry Type = Begin and End with the same Entry Subtype, FDC Work Order, FDC Work Order Line and FDC Resource. This value is stored in the Linked Input Entry field. Only linked Begin and End entries are processed. Dimensions, as well as additional fields like Output Quantity, Scrap Quantity or Finished get transferred to the FDC Journal from the End Entry.

Entries with Entry Type = Output or Consumption get processed directly as FDC Journal Line and without linked Entry, while Work Area Change cannot be processed any further.
