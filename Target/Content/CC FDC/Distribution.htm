Distribution

Distribution is a function specific to Order Bundles. Therefore a Bundle Line (also called “0-Line”) is distributed to all underlying Order Bundle Lines, according to the settings of the Order Bundle Time Distribution and Order Bundle Qty. Distribution fields on the Order Bundle FastTab in the FDC Work Order. Times and quantities can be distributed as determined by the following options:

  • Uniform – distribution posts the same part of times/quantities to all lines.
  • Expected Time/Expected Qty. – distributes according to the Expected Time/Qty. of each line.
  • Actual Qty./Actual Time – distributes the quantity first and then the time, and vice versa for “Actual Time” (the time gets distributed first and then the quantity). According to this, a setting of “Actual Time” and “Actual Qty.” is invalid.

In FDC Setup, the Automatic Order Bundle Distribution When Posting field can be set to instantly apply a new distribution after each posting on an FDC Work Order. Alternatively, a new distribution can be applied by clicking Distribute on the ribbon of the FDC Work Order page. With this function, a later change of the distribution key and a new distribution can be made.

When distributing, a new FDC (Distribution) Entry per Order Bundle Line is created for each FDC Entry of a Bundle Line. This Distribution Entry contains the calculated part of the “0-Line” by using the key as an FDC Detailed Entry. On further distributions, the Detailed Entries get cancelled and new entries will be created. The Distribution Entry remains.
