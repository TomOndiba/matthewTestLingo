Interface Codeunit for Creating FDC Work Orders

The Interface Codeunit 5308656 FDC – Interface offers, amongst other things, simple functions for creating FDC Work Orders. So it is possible to create an FDC Work Order by referencing to a record using the Data Type RecordRef with the function “CreateWOFromRecRef” (Create FDC Work Order from RecordRef) for the following records: Production Order, Production Order Line, Job, Service Header and Assembly Header.

The function “GetWOFromRecRef” (Get FDC Work Order from RecordRef) returns the number of the (first) found FDC Work Order for the given record. The usable records are the same as above.

Please refer to your COSMO CONSULT consultant for more information about integrating these functionalities.
