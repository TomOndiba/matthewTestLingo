STDC Employee Statistics

The STDC Employee Statics Report can be used for analyzing the simple time data capturing (STDC), and also as an overview of the Net Work Time distribution.

The request page offers the following options:

  • View by – Specifies the period in which the data should be viewed.
  • Start Date / End Date – The range of time in which the data should be considered.

Only STDC-related FDC data is used for calculating the distribution of work time.

The column “Absence / Overtime” displays the difference between Expected Time and Actual Time. A red value indicates an Absence, a green value indicates Overtime.

The calculation of the distribution of worktime is based on the Actual Time.

  • Worktime with Source – The sum of all orders with Type = With Source or Order Bundle.
  • Worktime without Source – The sum of all orders with Type = No Source.
  • Non-allocated Time – Remaining time that was not posted on any orders.
