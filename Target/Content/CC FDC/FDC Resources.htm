FDC Resources

FDC resources provide a connection between cc|factory data capture and the Microsoft Dynamics NAV resources, and depending on source type, represent employees, workstations, work centers, resources or resource groups. Therefore FDC resources can be both machines and people.

: FDC Resource Card

An FDC Resource is required for the capturing of time and quantity. It can be located at one or no FDC work area at the same time (refer to topic "FDC Work Areas" on page 1) and can own both FDC Dimensions and permissible FDC dimension values (refer to topic "FDC Dimensions" on page 1). An FDC Shift Structure can be assigned to an FDC Resource for a special day, which provides an automatic break deduction (refer to topic "Invoicing" on page 1) and a representation of capacities.

An FDC Resource can be set for valid concurrent beginnings for different FDC Work Orders (refer to topic "Concurrent Beginning" on page 1), its order stock and the users that have a posting permission.

The fields in the FDC Resource page are described in the following table.

Field

Description

No.

Unique code for the resource; maximum of 20 alphanumeric characters.

Source Type

Specifies the type of resource. The options are: Resource, Resource Group, Machine Center, Work Center and Employees.

Source No.

Indicates the FDC Resource to be created based on the Source Type of the corresponding resource.

Name

Name or description of the resource.

First Name

First name of the resource

Card No.

Identifies the FDC Resource at the terminal.

Work Center Group

Assigns a work center group to the resource. The work center group is for grouping and filtering purposes only.

Valid to Date

Indicates the date upon which the resource is no longer valid. Exceeding this date results in blocking the FDC Resource.

Blocked

Prevents the resource from posting any processes.

Concurrent Beginning

Indicates the type of concurrent beginnings for the FDC Resource (refer to topic "Concurrent Beginning" on page 1).

Regular Work Area

Assigns a regular work area to the FDC Resource. This work area will be transferred on creating an FDC Work Order Line.

Order Stock Type

Indicates which orders are displayed in the order stock (refer to topic "Order Stock" on page 1).

Corresponds to FDC Resource No.

Identifies another an FDC Resource that should be used instead of this FDC Resource on exporting captured data.

Assigned User ID

Connects an FDC Resource to a Microsoft Dynamics NAV user. If an FDC Resource is assigned to a user, only this user can process postings with this FDC Resource.

The FDC administrator can process postings for FDC Resources that have different User IDs than the administrator.

The actions on the ribbon of the FDC Resource card are described in the following table.

Action

Description

FDC Work Areas

Opens the FDC Work Areas page where FDC Work Areas can be assigned to the FDC Resource for specific periods.

FDC Shift Structures

Opens the FDC Shift Structure page where FDC Shift Structures can be assigned to the FDC Resource for specific periods.

FDC Dimensions

Opens the FDC Default Dimensions page where multiple FDC Dimensions can be assigned to the FDC Resource.

Permitted FDC Dimension Values

Opens the Permitted FDC Dimension Values page where multiple valid FDC Dimension Values can be assigned to the FDC Resource.
