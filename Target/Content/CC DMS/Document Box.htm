Document Box

The Document Box is a FactBox, implemented on Microsoft Dynamics NAV pages in the RoleTailored Client on the right side of a page. It supports Drag and Drop from files on the logo or specific content types, and a context menu for the list items (content types and documents). The square brackets show the number of documents found for the content type configuration. The height of the client add-in can be defined in the DMS Setup page (refer to topic “DMS Setup” on page 1).

: Context Menu for Document and Content Type

The items for the context menu can be limited and configured on the content type configuration. The actions for a document are:

  • Open File – Opens the selected file in SharePoint in the web browser. Double-clicking the document executes this action as well.
  • Filter in SharePoint – Opens the document library with the default view in the web browser and filters only this document.
  • Copy URL – Copies the URL of the document into the user’s clipboard.
  • Show MetaData – Opens the Show DMS Document Entry MetaData page, which displays the MetaData according to the current content type’s configuration for the selected file.
  • Edit MetaData – Opens the Edit DMS Document Entry MetaData page, which displays the MetaData for the selected file according to the current content type’s configuration, and where values can be uploaded upon confirmation.
  • Download File – Downloads the file from SharePoint directly to a local path.
  • Delete File – Removes the file from the SharePoint document library.

For the content type, the following actions are possible:

  • Open SharePoint – Opens the default web browser and navigates to the default view of the document library.
  • Upload File – Opens the file dialog from Windows and uploads the document similarly to the DocumentBox Drag and Drop, including possible MetaData definition, into SharePoint.

: States of the DocumentBox

The DocumentBox can be in different states, to be distinguished by the logo on the top section, if visible. The different states are:

  • Colored Logo – The client add-in is activated and running.
  • Loading Circle – The client add-in is deactivated and either the document list is retrieved and/or other actions are in progress. The RoleTailored Client is useable, but closing the page might lead to errors, especially when uploading a file.
  • Grey Logo – The client add-in is deactivated. This might be caused by any shown error, such as an exceeded license.

As a FactBox, users can add or remove the DocumentBox from the implemented pages through the Customize page feature.
