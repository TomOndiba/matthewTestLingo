Server

Servers are configured in the DMS Server Card page, accessed from the navigation pane by clicking Departments > cc|dms > Setup > DMS Server List.

: DMS Server Card

A server identifies the address (URL) to a SharePoint server, which stores the document libraries and should be accessible by all users. The URL must indicate only the server part, not any website collection or library parts. Otherwise the connection to the library cannot be established.

The fields on this page are described in the following table.

Field

Description

Code

Unique identifier for the server; maximum of 20 characters.

Name

Short description of the server.

SharePoint URL

Indicates the SharePoint server address (URL) (e.g. “http://intranet.mycompany.com”, “http://office365.sharepoint.com”).
