Document Library

Document libraries are configured in the DMS Document Library page, accessed from the navigation pane by clicking Departments > cc|dms > Setup > DMS Document Library List.

: DMS Document Library Card

A DMS Document Library in Microsoft Dynamics NAV defines the connection to the corresponding document library in SharePoint. A document library owns several content types (refer to topic “Content Type” on page 1) which furthermore bind different tables (refer to topic “Table Setup” on page 1) with independent MetaData (refer to topic “MetaData” on page 1) and table filters (refer to topic “SharePoint Table Filter” on page 1 and topic “Microsoft Dynamics NAV Table Filter” on page 1).

: Hierarchy of Configuration

After configuring the document library, it is necessary to open the field list by clicking Fields on the ribbon and then click Update Fields to update the list from SharePoint. The list of fields will be retrieved in the current Microsoft Dynamics NAV’s signed in user context. Therefore, the Microsoft Dynamics NAV RoleTailored Client must access SharePoint and the user must have at least reading rights for the document library.

General FastTab

The fields on the General FastTab of the DMS Document Library page are described in the following table.

Field

Description

No.

Unique identifier for the document library.

Server

Identifies the server where the document library is stored.

Server URL

Displays the URL of the server.

Description

Short internal description of the document library.

Relative URL

Defines the part of the document library's address (URL), in-between the server url and the internal (static) name. This par is usually defined by website collections and websites. An address (URL) is structured in the following way:

“<Server>/<Relative URL>/<Internal Doc. Library Name>/Forms/<View>.aspx” e.g. “http://www.cosmoconsult.com/finance/documents/Forms/AllItems.aspx”

Other examples to retrieve the relative URL are:

  • http://intranet/marketing/corporate/Forms/AllItems.aspx
            Relative URL: “marketing”
  • http://www.mycompany.com/dms/operations/customer/Forms/AllItems.aspx
            Relative URL: “dms/operations”
  • http://localhost:80000/websites/documents/items/Forms/FlatView.aspx
            Relative URL: “websites/documents”

SharePoint List GUID

Populated with information of the selected Library via lookup through communication with SharePoint.

Absolute URL

Indicates the complete path of the server, relative URL and internal static name of the document library.

Root Folder

Populated with information of the selected Library via lookup through communication with SharePoint.

Static Name

Indicates the name in which the document library was created in SharePoint. If a document library is renamed, this value will always be the same. Refer to the Relative URL field for more information.

Display Name

Shows the (current) display name of the document library in SharePoint. When renaming (new caption) the library in SharePoint, this configuration must be changed too.

Service Timeout

Sets the timeout in milliseconds until the service aborts an action (such as retrieve document list for DocumentBox). “0” will use the settings from the DMS Setup.

Total Released

Selected automatically if all table setups are released.

Document Library Actions

The actions on the ribbon of the DMS Document Library page are described in the following table.

Tab

Action

Description

Actions

Open in Web Browser

Opens the SharePoint address for the document library.

 

Test Connection

Tests the connection to the server URL for the document library current Microsoft Dynamics NAV’s signed in user context.

 

Release All

Simultaneously releases all table setups for a document library.

 

Reopen All

Simultaneously reopens all table setups for a document library.

Navigate

Fields

Opens the DMS Document Library Fields page, which displays all fields used in the document library.

Use the “Update Fields” function after configuring the document library to update the list from SharePoint, in the current Microsoft Dynamics NAV’s signed in user context.

 

User Credentials

Opens the DMS User Credentials page, which displays any credentials assigned to the document library.

 

Document Entries

Opens the DMS Document Entries page, which displays any actions tracked or saved in Microsoft Dynamics NAV for the document library. Refer to topic “Document Entries” on page 1 for more information.
