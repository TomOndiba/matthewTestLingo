Introduction

cc|dms is used to share documents in Microsoft Dynamics® NAV and provide easy access through Microsoft SharePoint®. Searching efforts are reduced due to the display of only relevant files on a record. Reports can be automatically uploaded to save digital copies. Users without Microsoft Dynamics NAV or mobile staff can access those files in SharePoint.

Key Features

  • Display and filter only relevant documents for a record in Microsoft Dynamics NAV
  • Upload and delete new files by Drag and Drop or context menu
  • Manage MetaData to tag the files with important information
  • Automatic upload of report output (e.g., order confirmations)
  • Configure all filters and MetaData directly in Microsoft Dynamics NAV
  • Use SharePoint to provide additional features, such as versioning, in-document search and team management

cc|dms functionalities are accessed in the cc|dms menu.
