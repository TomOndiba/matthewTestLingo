Content Type

Content Types are subgroups of document libraries and are configured on the DMS Content Types FastTab in the DMS Document Library page. These can be used as a filter to upload a document for the same record with different MetaData settings, or to show different filtered document lists from SharePoint for the same document library.

: Content Types of the DMS Document Library page

The fields on the DMS Content Types FastTab are described in the following table.

Field

Description

Code

Unique identifier for the content type; maximum 20 characters.

Name

Indicates the name of the content type, used in the DocumentBox as caption if no other language-specific caption (translation) is configured (refer to “Captions” in the topic “Content Type Actions” on page 1).

Upload Type

Defines how to upload documents (refer to topic “DMS Setup” on page 1).

If DMS Setup is selected, the value in the Upload Type field on the DMS Setup page is used.

SharePoint File Duplicate

Specifies how to handle duplicate file names when uploading a new document into the document library (refer to topic “DMS Setup” on page 1).

If DMS Setup is selected, the value in the SharePoint File Duplicate field on the DMS Setup page is used.

Define MetaData

If selected, the Define MetaData page opens before a new file will be uploaded. Otherwise the default MetaData values will be used to upload the document without user confirmation (e.g. for automated report upload).

Visible in Document Box

Defines whether the content type will be visible in the DocumentBox, if an appropriate table setup for the current record is configured.

Open Document

Activates the “Open File” context menu for documents and enables users to double-click in the DocumentBox to open documents in the web browser.

Upload Document

Allows Drag and Drop of documents on a content type in the DocumentBox and activates the “Upload File” context menu.

Download Document

Enables the download of documents from the document box via the “Download File” context menu.

Delete Document

Enables the deletion of documents in the document box via the “Delete File” context menu.

Show MetaData

Enables the display of MetaData in the DocumentBox via the “Show MetaData” context menu.

Edit MetaData

Enables the modification of MetaData in the DocumentBox via the “Edit MetaData” context menu.

Expand Documents

Expands the content type in the DocumentBox after loading the document list to see all files immediately.

File CheckIn Type

Determines how the upload and modification CheckIns are handled. The SharePoint document library must have appropriate version settings to avoid errors. The options are:

  • Minor CheckIn – New minor version (e.g. 1.0, 2.0, 3.0, ...)
  • Major CheckIn – New major version (e.g. 1.1, 1.2, 1.3, ...)
  • Overwrite CheckIn – Overwrites without generating a new version (for major version configured). The file will be checked out if the SharePoint document library has a minor version configured.

CheckIn / CheckOut Document

Enables the “CheckIn” and “CheckOut” options (ContextMenu) for files in the DocumentBox.

This function calls the actions for the document in the SharePoint document library, without any further advantages.

File Caption-Formula

Identifies a formula used to change the displayed name of the file in the DocumentBox.

Possible variables are all retrieved static column names (e.g. “[FileLeafRef]”, “[ID]”, “[DocumentNo]”, “[Customer_No]”, ...).

File ToolTip-Formula

Identifies a formula to set the tooltip (mouse over description) of a file in the DocumentBox. Refer to the description for the File Caption-Formula field for possible variables.

Document Library No.

Displays the value from the No. field on the General FastTab of the DMS Document Library page.

To display a content type in the DocumentBox, the following criteria must be met:

  • The Visible in DocumentBox check box must be selected on the content type
  • The content type must have a released table setup for the corresponding table number of the current record, where the DocumentBox is displayed
  • The (current) record must match the filter criteria of the Microsoft Dynamics NAV table filter
  • The user needs at least reading rights for the document library to retrieve the document list for the DocumentBox

Content Type Actions

There are four actions on the DMS Content Types FastTab. Each is described in the following table.

Action

Description

Document Entries

Opens the DMS Document Entries page, which displays any actions tracked or saved in Microsoft Dynamics NAV for the selected content type.

Captions

Opens the DMS Translations page, where multiple languages can be defined for the selected content type. Values entered here override the Content Type Names and are displayed in the DocumentBox.

Table Setup > Release All

Releases all related table setup line(s) for the selected content type.

Table Setup > Reopen All

Reopens all related table setup line(s) for the selected content type.
