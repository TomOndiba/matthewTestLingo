Office 365

Office 365 capabilities can be used with cc|dms. The relevant installation and configuration settings are described in this section.

Windows Identity Framework

The Windows Identity Framework within the Windows Features must be activated for the server and each client for older cc|dms versions. This setting usually applies automatically for Windows 10.

User Credentials

A user login must be defined for Office 365. It is therefore recommended to manage a service user by selecting the Office 365 User check box in the DMS User Credentials page.

: Office 365 User designated in DMS User Credentials

Note

The permissions (shown documents) via cc|dms are no longer directly user-specific when a service user is defined.
