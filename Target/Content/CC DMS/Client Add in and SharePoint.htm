Client Add-in and SharePoint Communication DLLs

Two DLL files must be copied into the “Add-ins” directories of the Microsoft Dynamics NAV Service to use the DocumentBox and the automated report upload of documents into SharePoint. Therefore, an arbitrary substructure of folders can be used.

: Add-in Directory for Microsoft Dynamics NAV Service

For example, copy the files “CosmoConsult.DMS.SharePoint.Communication.<Version>.dll” and “CosmoConsult.Dynamics.NAV.DMSW1-<Version.>.dll” into the following service directory:

C:\Program Files\Microsoft Dynamics NAV\100\Service\Add-ins\Cosmo Consult\DMS\

The necessary client add-ins will be registered automatically into Microsoft Dynamics NAV when the DMS Setup page is opened for the first time and the client add-ins will be deployed automatically to the RoleTailored Clients.
