Report Upload

Instead of printing and scanning, reports can automatically be printed into a PDF and uploaded. Therefore, the default “Report.RUN” processing code in Microsoft Dynamics NAV will be changed to “REPORT.SAVEASPDF”.

In the Report Selections for Sales, Purchase and Service, the document library and content type can be configured to upload the PDF into SharePoint. These report selection pages can be accessed on the Navigate tab of the DMS Setup page and in the standard Microsoft Dynamics NAV menus.

: Report Selection - Sales

Many standard objects will include the “Upload PDF” and/or “Post and Upload PDF” functions to perform the automated report upload. When using these functions, the Request Dialog must be skipped because the “SAVEAS” function does not allow it.

Depending on the content type configuration, a MetaData define dialog will be opened (“Define MetaData”) and depending on the report selection configuration, a new Email or the report itself will be opened. If no content type is selected, the default behavior (REPORT.RUN) will be processed.

The fields specific to cc|dms in the Report Selection pages are described in the following table.

Field

Description

Document Library No.

Identifies the number of the document library where the document has to be uploaded.

Content Type Code

Identifies the content type to define the MetaData when uploading the document.

Report Upload Type

Defines the type of uploading for the report. The options are:

  • DMS Setup – The value in the Report Upload Type field on the DMS Setup page is used.
  • Only Upload – The report will be uploaded into SharePoint without any further action.
  • Upload and Open – After uploading the report, the local copy (from a temporary path) will be opened, e.g. for manual printing.
  • Upload and E-Mail – A new Email will be created by the default application with the PDF automatically attached.

 

Batch Sales Order Posting Upload

Multiple sales orders posted with the “Post Batch” function on the Sales Orders page can be uploaded to SharePoint at one time. In the Batch Post Sales Orders request page, select the Upload as Pdf check box to upload all included sales orders to SharePoint.

: Batch Post Sales Orders request page with Upload as Pdf selected to upload two sales orders for customer 50000

When the batch posting process is finished, the Posted Sales Invoices will be uploaded to SharePoint.
