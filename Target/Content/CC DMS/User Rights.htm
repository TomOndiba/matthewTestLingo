User Rights

The actions (refer to topic “Document Actions” on page 1) are processed in context of the RoleTailored Client’s user or by the NAS account user. Accordingly, both need to be able to connect with SharePoint to retrieve and send documents and data.

The reports (save as pdf) and the upload of files by the upload queue both use the temporary directory of the user to store files. Therefore, each Microsoft Dynamics NAV user needs rights to write into his or her temporary directory. If there are issues with rights, in urgent cases, replace the references for the “ServerTempFileName” with any local path in:

  • Codeunit “DMS - DocumentBox Management”, function “UploadFile()”
  • Codeunit “DMS - Upload PDF”, function “InitializeByReportSelection()”
