Configuration Export and Import

The Configuration Export and Import functionality – accessed from the cc|dms menu by clicking Setup > DMS Config.-Export/-Import – allows quick and easy export and import of document library configuration, including content types and table setups via an XML file.

: Configuration Export and Import

Configuration Export

When the DMS Config. Export/-Import page is opened, a list of all possible exports is created. Select the Select check box on each configuration line that need to be included in the export. Select the Export Report Selection check box to include all entries in the Report Selections table in the export file.

Click Export on the ribbon to create an XML file in the defined output directory with all selected content.

Configuration Import

Click Import on the ribbon to import all configurations of an XML file into Microsoft Dynamics NAV.
