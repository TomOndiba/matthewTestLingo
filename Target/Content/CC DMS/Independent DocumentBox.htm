Independent DocumentBox

An alternative integration is the parent-independent DocumentBox in an outsourced page, which can be called by any action. This makes it possible to increase the performance on the “parent” page and load the documents only for pertinent cases. This integration type also allows for an integration in document lines (SubPages).

: Independent DocumentBox on Sales Orders list

To open an independent page, create an action with the local variables DocumentBoxAsync (Page: DMS - Doc. Box FactBox Async) and RecRef (RecordRef). Similar to the standard implementation, the RecRef must be handed over to the new page by “DocumentBoxAsync.SetRecordRef(RecRef)”.

: Implementation of an independent DocumentBox on Sales Order List page

Additionally, run the “DocumentBoxAsync.SetIndependentPage(TRUE,CurrPage.CAPTION)” function to load the documents from SharePoint. Use either “DocumentBoxAsync.RUN” or “DocumentBoxAsync.RUNMODAL” to open the page.
