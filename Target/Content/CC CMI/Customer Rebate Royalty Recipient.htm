Customer / Rebate/Royalty Recipient Catalog

New rebate/royalty recipients must be defined on the customer. This can be done using the rebate/royalty recipient catalog. It represents the link between customers and an assigned rebate/royalty recipient.

From the customer card, the Customer Comm. Recipient Catalog page can be displayed by clicking Navigate > Commission Recipients.

Refer to topic “Customer / Commission Recipient Catalog” on page 1 for more information about this page.
