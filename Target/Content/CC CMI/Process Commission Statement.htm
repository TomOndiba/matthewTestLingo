Process Commission Statement

Commission statements can be processed after creation before they are registered.

Note

If the settings on the Invoicing FastTab are changed, the existing Commission Statement Lines are recalculated.

The statement lines can be changed or deleted, or new lines can be added. For modification and verification there are various possibilities. However, the statement date range cannot be modified. Therefore, to run the commission statement for a different date range, a new statement must be created.

: Special fields on Commission Statement lines

In addition to several other fields on the lines, there are two special fields. Both are described in the following table.

Field

Description

Fixed Line

Each commission statement line can be maintained as a fixed line. This means that in a new calculation, this line is excluded from the recalculation. Even if the basic statement settings are changed, selecting this check box prevents a change or deletion.

Ranking

Each statement line contains a ranking after a calculation. This value reflects the ranking of the commission rate within the commission search order. Lines that were combined for the scale commission receive the same ranking value.

Refer to online help for more information about the other fields on this page.

Line Menu Items

On the Lines FastTab, click Line to access the menu items that assist with reviewing the line information.

: Commission Statement, Line menu items

Each menu item is described in the following table.

Menu Item

Description

Dimensions

Displays the original dimension values. For new statement lines, dimension values can be entered.

Navigate

Displays the documents relating to the statement lines.

Calculation Information

Displays a detailed breakdown of the calculated commission amounts on the line.

Get Commission Rate

Displays commission rates. New statement lines can be assigned to a commission rate or the current commission rate of an existing line can be verified and modified.

Scale commissions are displayed if they are available.

 

Commission Statement Recalculation

When changes are required on a commission statement, a recalculation can be done by clicking Actions > Recalculate Commission Values on the ribbon. This causes a recalculation of the entire commission statement.

Commission Statement Statistics

Commission Statement Statistics (Home tab > Statistics) are based on the currency used in the Foreign Trade FastTab. The statement lines might be in different currencies but all lines are totaled in the header currency. If the Currency Code field is empty, the Commission Amount and Commissionable Amount (LCY) are the same.
