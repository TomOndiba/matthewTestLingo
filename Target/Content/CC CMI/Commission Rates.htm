Commission Rates

In the header of the Commission Rates page, the following filters are available to limit the list view:

  • Sales Type Filter
  • Comm. Recipient Code Filter
  • Sales Code Filter
  • Type Filter
  • Comm. Recipient Type Filter
  • Code Filter

The selected filter is used as a template for new lines. To clear the filters, select None in the Sales Type Filter field. There is also an option to filter the commission rate lines by start date and currency.

: Commission Rates page

The fields in the lines – in addition to the filtered fields – are described in the following table.

Field

Description

Minimum Quantity /    Minimum Amount

Determines the scale value for a commission rate line. The name of this field depends on the Scale Type selected in the Commission Setup page (refer to topic “Commission Setup” on page 1).

Note

Normally, the minimum amount is only used for customer bonus calculation with much higher values, or together with starting and ending dates to motivate salespeople to sell a particular item or items of a particular item commission group.

Commission Type

Determines how commissions are entered. The options are:

  • Percent – for example, four percent of the entire order
  • Amount per Unit of Measure – for example, 100 LCY for each bicycle sold

In the sales documents, quantity (basis) is used as Unit of Measure.

Commission

Based on the selection in the Commission Type field:

  • Percent – the commission rate can be entered as a percentage.
  • Amount per Unit of Measure – a nominal value can be entered.

Starting Date / Ending Date

Time period for which the commission rate line is valid.

Currency Code

The commission rate lines can have different currencies. This field has multiple functions:

  • A minimum amount (scale type amount scale) is entered in this currency.
  • The nominal commission for Commission Type = Amount per Unit of Measure is entered in this currency.
  • The currency is considered during the commission calculation for orders in foreign currency.

 

Example – Customer Commission Group

In this example, commission rates are created for the RETAIL customer commission group and the STANDARD commission recipient group, with variations on the item commission groups.

: Commission Rates for RETAIL customer commission group and STANDARD commission recipient group

The commission rates for each item commission group – respective of the Commission Basis and Calculation Base (refer to topic “General FastTab” on page 1) assigned to a recipient – are described in the following table.

Item Commission Group

Commissionable Rates within Accounting Period

FURNITURE
  • 7% if recipient sells furniture
  • 8% if recipient has a total sales revenue of 15000 <currency>
  • 9.5% if recipient is able to reach a sales revenue of 15000 <currency> in December

HARDWARE
  • 10 <currency> for each item sold
  • 11.5 <currency> for each item sold when recipient reaches a total of 2500 <currency>

STANDARD
  • 5% if recipient sells standard items
  • 5.5% if recipient has a total sales revenue of 1000 <currency>
  • 6.5% if recipient has a total sales revenue of 5000 <currency>
