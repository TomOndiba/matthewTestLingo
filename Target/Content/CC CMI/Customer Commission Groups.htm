Customer Commission Groups

Customer Commission Groups are used to group customers by commission type. Once set up, the groups can be set up with rates and assigned to customers.

The Customer Commission Groups page is opened from the cc|commission menu by clicking Customer Commission Groups.

: Customer Commission Groups page, Navigate tab

The Navigate tab contains two actions for commissions. Each is described in the following table.

Action

Description

Customers

Displays an overview of all customers assigned to the selected customer commission group.

Commission Rates

Displays the commission rates for the selected customer commission group. Rates can be modified in this page.

This function is also available on the Home tab.
