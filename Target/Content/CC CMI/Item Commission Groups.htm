Item Commission Groups

Item Commission Groups are used to group items by commission type. Once set up, the groups can be set up with rates and assigned to items.

The Item Commission Groups page is opened from the cc|commission menu by clicking Item Commission Groups.

: Item Commission Groups page, Navigate tab

The Navigate tab contains two actions relating to commissions. Each is described in the following table.

Action

Description

Items

Displays the list of items that are assigned to the selected item commission group.

Commission Rates

Displays the commission rates for the selected item commission group. Rates can be modified in this page.

This function is also available on the Home tab.
