Issue a Commission Statement

Before a commission statement is issued, a test report can be run by clicking Commission Statement - Test on the Home tab (or by clicking Test Report on the Actions tab).

If a commission statement is complete and correct, the batch processing for the registration can be triggered by clicking Issue on the ribbon.

On the Issue Commission Statement request page:

  • The settings on the Options FastTab determine whether the posting date of the commission statement needs to be replaced and if a printed document should be generated.
  • The Commission Statement Header FastTab is automatically filtered on the current commission statement number.

Issued Commission Statements

An issued commission statement can be viewed from the cc|commission menu by clicking Issued Commission Statements.

: Issued Commission Statement

All functions that are available from the commission statement can be used here. However, values cannot be changed. They are used for later processing in Finance and/or in Human Resources.

On the ribbon, click Print to print an issued commission statement.
