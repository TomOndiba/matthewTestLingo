Exchange of Rented Items

Occasionally, rented items may become damaged or need to be serviced during the rental period. If an item is still required during the rental period and a failure is unacceptable, an exchange must occur where the rented item is returned and a new item is delivered to the customer.

To exchange a rented item, run the “Register Change” function (Rent Order > Lines FastTab > Line > Register Change). This function is only available for items that use Serial Nos. and are already shipped.

: Rental Change Card page

The fields in this page are described in the following table.

Field

Description

Change Date

Date on which the item is to be exchanged.

Change Time

Time when the exchange will occur; this field is optional.

Item No.

Automatically populated from the No. field of the Rent Order line to be exchanged.

Serial No.

Serial No. to be exchanged:

  • If only one Serial No. is used in the Rent Order line, the field is automatically populated with that Serial No.
  • If more than one Serial No. is used in the Rent Order line, the field is blank and the Serial No. to be exchanged must be selected.

New Item No.

Item to be shipped as the replacement item. This can be the same or different item than the item to be exchanged.

New Serial No.

Serial No. of the replacement item to be shipped.

Description

Short explanation to specify why the item is to be exchanged.
