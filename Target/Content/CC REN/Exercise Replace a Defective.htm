Exercise – Replace a Defective Excavator

On February, 15 2019, customer R-10000 calls the service department to report that their rented excavator broke down and lost a lot of its hydraulic fluid from the power assisted steering. While the excavator is repaired, a replacement excavator of the same type is provided to the customer.

Create and Ship the Rent Order

Follow these steps to create the rent order:

  1. Insert a new rent order for customer R-10000.
  2. Expand the Rental FastTab and enter the following:
    1. Rental Rate Code = Month
    2. Rental Start Date = 02/01/19
    3. Rental End Date = 02/28/19
  3. On the first line of the Lines FastTab, enter the following:
    1. Type = Item
    2. No. = R-10000-01
    3. Quantity = 1
    4. Rate Unit Price = add if values are missing on Item or Rate Component lines
  4. On the Lines FastTab, click Line > Item Tracking Lines.
  5. With the Shipment option selected, click OK.
  6. On the ribbon, click Select Entries.
  7. Note which serial number is selected and then click OK.
  8. Click OK to close the Item Tracking Lines page.
  9. On the ribbon, click Post.
  10. With the Ship option selected, click OK.

Exchange the Rented Item

Follow these steps to exchange the rented item:

  1. Select the line with the item to exchange.
  2. On the Lines FastTab, click Line > Register Change.

: Rental Change Confirmation page

  1. Click Yes to post the change.
  2. In the Change Date field, enter 02/15/19.
  3. In the Change Time field, enter 12:00 PM.
  4. In the New Serial No. field, click the arrow, select a serial number and then click OK.

: Serial No. List page

  1. In the Description field, type “Fault in steering system”.

: Rental Change Card page

  1. Click OK to close the Rental Change Card page. Notice the following:
    • On the original item line (and any relevant rate components):
      • The Return Qty. to Receive field is set to 1
      • The Rental Return Date field is set to the Change Date minus one day
    • The exchanged item – and any related rate components – is added to the rent order with the proper serial number

: Rent Order with exchange information

  1. On the ribbon, click Post.
  2. Select the Receive option and then click OK.
  3. On the ribbon, click Reopen.
  4. Fill in the Rate Unit Price field on Item or Rate Component lines if values are missing on the new lines.
  5. On the ribbon, click Post.
  6. With the Ship option selected, click OK.

: Rent Order with updated return and exchange item shipment information

  1. Click the value in the Qty. Rent Changes field.
    The Rental Changes page opens displaying the change information as well as identifying the line number of the new rental item.

: Rental change information for replaced item

  1. Escape back to the rent order.
