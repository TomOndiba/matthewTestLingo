Exercise – Add Extra Usage from Counters to Invoices

Item R-10000-02, ReTex TC240 LC Crawler Excavator, is now equipped with a counter for Operation Hours. Based on a monthly invoice period, 160 hours are included in the rental price of 1500. If this number of hours has been exceeded upon return of the item, the extra usage will be added to the invoice at a price of 9.38 per hour (Rate Price / Free Operation Hours).

Customer R-10000 needs to rent item R-10000-02 for the entire month of January 2019. Upon shipment, the hourly counter in the Stock Movement shipment is updated to 10 to match the counter on the item. When the item is returned on January 31st, the counter is checked and 195 is entered in the Stock Movement return. Since the number of hours was exceeded by 25 (return counter - shipment counter - free operation hours (195-10-160)), an entry for extra hours is added to the invoice.

Note

The same process can be performed for miles counters using the Free Operation Miles in Rental Prices and Operation Miles in Stock Movements.

Set Up the Price Structure

Follow these steps to set up the price structure:

  1. Ensure that the work date is 01/01/19.
  2. In the navigation pane, click cc|rental > Contract Management > Items.
  3. Locate item R-10000-02 and then click Navigate > Prices (in the Rental group).
  4. In the Rental Prices page, enter the following:
    1. Sales Type = All Customers
    2. Unit of Measure Code = PCS
    3. Rental Rate Code = MONTH
    4. Unit Price = 1500
    5. Free Operation Hours = 160
    6. Include Counters = Yes

: Rental Prices for item R-10000-02

Create the Rent Order

Follow these steps to create the rent order:

  1. Insert a new rent order for customer R-10000.
  2. On the first line of the Lines FastTab, enter the following:
    1. Type = Item
    2. No. = R-10000-02
    3. Quantity = 1
    4. Rental Rate Code = Month
    5. Invoice Period = Month
    6. Rental Start Date = 01/01/19
    7. Rental End Date = 01/31/19
  3. Notice that the Include Counters check box is selected on the line.

: Rent Order with item that includes hourly counters in invoice

  1. On the Lines FastTab, click Line > Item Tracking Lines.
  2. With the Shipment option selected, click OK.
  3. On the ribbon, click Select Entries.
  4. Note which serial number is selected and then click OK.
  5. On the line for the Insurance rate component, enter 150 in the Rate Unit Cost field.
  6. On the line for the Cleaning rate component, enter 50 in the Unit Price Excl. VAT field.

Update Counter for Shipment

Follow these steps to manually update the counter prior to posting the shipment:

  1. On the ribbon, click Post.
  2. With the Ship option selected, click OK.
    A message appears stating that lines are using counter based invoicing, should the current counter be used as the default? Clicking Yes uses the current setting in the Machine Card; clicking No allows for a manual update of the counter in the Stock Movement.
  3. Click No to update the counter manually and close the message.

: Counter based invoicing message on shipment

  1. Another message appears asking to update the counters before posting.
  2. In the navigation pane, click cc|rental > Stock Movement > Shipments.
  3. In the Rental Shipments page, locate the correct rental order and then click Update Counter.

: Preparing to update counter for item R-10000-02 shipment

  1. In the Counter Input page, enter 10 in the Operation Hours field.

: Hourly counter updated manually prior to shipment

  1. Click OK to close the Counter Input page and then the Rental Shipments page.
  2. In the Rent Order, click Post.
  3. With the Ship option selected, click OK.

Process the Return

Follow these steps to process the return:

  1. In the Posting Date field, enter 01/31/19.
  2. In the Lines FastTab, enter 01/31/19 in the Rental Return Date field.
  3. In the Return Qty. to Receive field, enter 1.
  4. On the Lines FastTab, click Line > Item Tracking Lines.
  5. Select the Receive option and then click OK.
  6. Ensure that the noted serial number is selected and then click OK to close the Item Tracking Lines page.
  7. On the ribbon, click Post.
  8. Select the Receive option and then click OK.
    A message appears stating that lines are using counter based invoicing and update the counters before posting.

: Message to update counters before return posting

  1. Click OK.

  2. In the navigation pane, click cc|rental > Stock Movement > Return Orders.

  3. In the Rental Returns page, locate the correct rental order and then click Update Counter.

: Preparing to update counter for item R-10000-02 return

  1. In the Counter Input page, enter 195 in the Operation Hours field.

: Hourly counter updated manually prior to shipment

  1. Click OK to close the Counter Input page and then the Rental Returns page.
  2. In the Rent Order, click Post.
  1. Select the Receive option and then click OK.

Process the Invoice

Follow these steps to process the invoice:

  1. On the ribbon, click Post.
  2. Click the Invoice option and then click OK.
  3. In the Posting Date field, enter 01/31/19.
  4. In the Invoice to Date field, enter 01/31/19.
  5. Click OK to create the invoice and then accept all confirmation messages.
  6. On the ribbon, click Navigate > Invoices.
  7. Open the invoice and notice that a line has been added for the extra usage of 25 hours in the amount of 234.38, which is calculated at the Hourly Rental Rate of 9.38 (25 * 9.38).

: Invoice with extra hourly usage from counters

  1. Post the invoice and then close the Rent Order.

Review Updated Counter Information on Machine Card

Follow these steps to view the update counter information on the Machine Card:

  1. In the navigation pane, under Machines, click Machines > Machines.
  2. Double-click the line for item R-10000-02 with the serial number used in this exercise.
  3. Expand the Technical Data FastTab and notice that the Operation Hours field is updated to 195, which was the hourly counter entered in the Stock Movement on return of this item.

: Machine Card with updated hourly counter information

  1. Click OK to close the Machine Card.
