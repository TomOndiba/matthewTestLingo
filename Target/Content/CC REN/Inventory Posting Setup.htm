Inventory Posting Setup

When customer locations are created automatically, an Inventory Posting Setup entry is also created for the combination of customer location and rental location. To use this functionality, the rental location and inventory posting group combination must be set up.

Additionally, when 3rd Party rentals are used, the external location (specified in the External Rental Loc. Code field on the Rental Mgt. Setup page) and inventory posting group combination must be set up.
