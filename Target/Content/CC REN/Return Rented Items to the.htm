Return Rented Items to the Vendor

When a customer returns an item that has been rented from a vendor, it can be returned to the vendor. The Qty. to Return field on the Purchase Order can be used to post this process.

: Purchase Order page, 3rd Party Return

Additional options are available on the Post Purchase Order dialog for posting the return of a 3rd Party Purchase Order or the posting of losses when an item is not returned.

: Post Purchase Order options

Post a Lost Message to eliminate the step of returning the item to the vendor by creating a sort of negative shipment document. To post a loss, populate the Qty. to Return field on the purchase line and then post the Lost Message. The Qty. Lost field will be updated with the received quantity.

Return 3rd Party Item with Serial Number

Use the “Select Rent Return Serial Nos.” function on the purchase order to select the serial number of returned items that have already been posted in the Rent Order.

The steps for returning a 3rd Party item with a serial number are:

  1. Post the return for the serial number in the Rent Order (return of the Serial No. from the customer).
  2. Select the Serial No. for return in the Purchase Order (Lines FastTab > Line > Select Rent Return Serial Nos.).
  3. Post the Return in the Purchase Order (return of the Serial No. to the Vendor)
    This process posts the negative adjustment for the Serial No. that is no longer in stock nor available for rental.
