Internal Service Orders

When dealing with machine type items, it may be necessary to post a Service Order for non-customer related purposes. The invoicing should be done using the general purchase Unit Cost for items and an internal amount for services as Sales Price.

Exercise – Create an Internal Service Order

A new excavator must be assembled after it is received and before it can be sold or rented. The services that are expended for that have to be posted using an Internal Service Order.

Before the Service Order can be created, the Service Order Type for internal services must be set up.

Set Up an Internal Service Order Type

Follow these steps to set up an internal Service Order Type:

  1. In the navigation pane, click Service > Administration > Service Order Types.
  2. Use the “Choose Columns” function to add the Internal Cost field.
  3. On the ribbon, click New.
  4. In the Code field, type INTERNAL.
  5. In the Description field, type Internal Services.
  6. Select the Internal Cost check box.

: New Service Order Type for internal services

  1. Click OK to close the Service Order Types page.

Create an Internal Service Order

Follow these steps to create the internal service order:

  1. In the navigation pane, click Service > Order Processing > Service Orders.
  2. On the ribbon, click New and then press ENTER to automatically populate the No. field.
  3. In the Description field, type Internal Service Order.
  4. In the Customer No. field, enter R-99999.
  5. In the Service Order Type field, select INTERNAL.
  6. In the Lines FastTab, click the arrow in the Service Item No. field.
  7. Select the line for Item No. = R-10000-01 and Serial No. = RTW150-00003 and then click OK.

: Internal Service Order

  1. On the Lines FastTab, click Line > Service Item Worksheet.
  2. On the Lines FastTab, select Item in the Type field.
  3. In the No. field, enter 70104.
  4. In the Location field, enter BLUE.
  5. In the Quantity field, enter 1.
  6. Notice that the Unit Price is 1.40 (the Unit Cost from the Item Card).
  7. Click OK to close the open pages.
