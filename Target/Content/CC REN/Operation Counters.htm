Operation Counters

Items with machine-like attributes may have a counter to log which work is done in what time. cc|rental offers two types of counters to meet this requirement:

  • A counter for operation hours
  • A counter for operation miles

On a Machine Type Card, the type of counter can be assigned by clicking Navigate > Technical Data.

: Machine Type Card page, highlighting the Technical Data icon in the Item menu group

On the Technical Data Card page, general attributes for all Machines of that type can be set up.

: Technical Data Card page

The fields on the General FastTab are described in the following table.

Field

Description

Type of Counter

Enables the use of operation counters. The options are Hours, Miles, or Hours+Miles.

Performance (kW)

General value for the performance of the machines of the type in kilowatts (kW). The value in the Performance (HP) field is automatically updated if a value is entered here.

Performance (HP)

General value for the performance of the machines of the type in horsepower (HP). The value in the Performance (KW) field is automatically updated if a value is entered here.

Internet Link

Web site of the vendor or manufacturer.

The fields on the Color FastTab are described in the following table.

Field

Description

External Colour

External color of the Machine Type.

Internal Colour

Internal color of the Machine Type.

The values that are entered in the Technical Data Card page are transferred to the Machine Card on creation (purchase or positive adjustment).

Exercise – Set up an Operation Counter

The MIG Welder is now equipped with a counter for Operation Hours. To log the hour counter reading, the counter must be activated.  A purchase order for two new excavators with this type is created and then the counter is updated on the Service Order.

Note

The “Create a Machine Type” exercise must be completed prior to performing this exercise.

Activate the Counter for Operation Hours

Follow these steps to activate the counter for Operation Hours:

  1. In the navigation pane, click Machines > Types > Machine Types.
  2. Double-click the line for number R-50000-01.
  3. On the ribbon, click Navigate > Technical Data.
  4. In the Type of Counter field, enter Hours.
  5. In the External Colour field, enter BLUE.

: Technical Data Card page for item R-50000-01

  1. Click OK to close the open pages.

Buy Machines with Technical Data

Follow these steps to buy machines that generally are used with technical data (counters):

  1. In the navigation pane, click Purchasing > Order Processing > Purchase Orders.
  2. On the ribbon, click New.
  3. In the Buy-from Vendor No. field, enter R-50000.
  4. In the Lines FastTab, enter Item in the Type field.
  5. In the No. field, enter R-50000-01.
  6. In the Location Code field, enter BLUE.
  7. In the Quantity field, enter 2.
  8. On the Lines FastTab, click Line > Item Tracking Lines.
  9. On the ribbon, click Actions > Create Customized SN.
  10. In the Customized SN field, type R-50000-01-00006.
  11. Click OK.
  12. Click OK to close the Item Tracking Lines page.
  13. On the ribbon, click Post.
  14. Select the Receive option and then click OK.
  15. Click OK to close the Purchase Order.

Review the Machine Type Item Information

Follow these steps to review the updates to the Machine:

  1. In the navigation pane, under Machines, click Machines > Machines.
  2. Go to the end of the list.
  3. Double-click the line for the MIG Welder with serial number R-50000-01-00006.
  4. Expand the Technical Data FastTab and notice that the fields that were populated on the Technical Data Card page for the machine are populated in the Machine Card.

: Machine Card with information from the Technical Data Card page

  1. Click OK to close the Machine Card.

Update the Counter Value

Follow these steps to update the counter of a Machine:

  1. In the navigation pane, click Service > Order Processing > Service Orders.
  2. On the ribbon, click New and then press ENTER to automatically populate the No. field.
  3. In the Customer No. field, enter R-99999.
  4. On the Lines FastTab, click the arrow in the Service Item No. field.
  5. Select the line for Item No. = R-50000-01 and Serial No. = R-50000-01-00006 and then click OK.
  6. On the Lines FastTab, click Line > Service Item Worksheet.
  7. On the ribbon, click Actions > Update Counter.
  8. In the Operation Hours field, enter 23.

: Counter Input page

  1. Click OK.
  2. Click OK to close the open pages.

Note

The “Update Counter” function is also accessible on the Stock Movement pages (refer to topic “Stock Movements” on page 1).

Review the Updated Counter Information

Updated counter information is viewable on the related fields on the Machine Card.

Follow these steps to update the counter of a Machine:

  1. In the navigation pane, click Machines > Machines > Machines.
  2. Open the Machine Card for item R-50000-01, Serial Number R-50000-01-00006.
  3. Expand the Technical Data FastTab and notice that the Operation Hours is set to 23.

: Machine Card with updated counter information from service order

Counter Registration Worksheet

The Counter Registration Worksheet – accessed on the Machines menu by clicking Tasks or Machines – gathers all values for existing machines for a mass update. Operation Hours and Miles can be entered for multiple machines and posted at once with the “Register Changes” function.

: Counter Registration Worksheet

Each entry must be entered with a Registration Date and Time. When the hours or miles are entered, the Modified check box is selected.

: Counter History for item R-10000-02 and new counter entry in worksheet

Only when the date and time combination is greater than the latest value in the Counter History will the value be updated on the Machine Card.

: Operations updated on Machine Card

If changes are not registered, a warning message appears when the Counter Registration Worksheet is closed.

: Message upon closing Counter Registration Worksheet with modified, unregistered entries
