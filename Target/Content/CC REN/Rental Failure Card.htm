Rental Failure Card

The Rental Failure Card is accessed from the Rent Order by selecting the proper line, and then clicking Line > Enter Failure on the Lines FastTab. The following message will appear:

: Create Failure confirmation message

Note

Failures cannot be assigned to a line until it has shipped.

After clicking Yes to the message, the Rental Failure Card page appears, with the Rental Start Date and Rental End Date fields filled in from the Rent Order line.

: Rental Failure Card page

The fields on this page are described in the following table.

Field

Description

Rental Start Date / Rental End Date

Dates from the rental line.

Start Date Failure / End Date Failure

Date period of the failure to be processed.

Failure Type

Determines how the failure will be processed, as specified in the Failure Types page (refer to topic “Failure Types” on page 1).

Description

Description of the Failure Type.

Quantity

Number of units that need to be included in the Failure adjustment.

Create Credit Memo

If selected, a credit memo should be created for the failure.

Defaults from the Failure Type, but can be changed as needed here without affecting the Failure Type.

Discount %

Percentage of the rate to be credited.

Defaults from the Failure Type, but can be changed as needed.

Note

If the Create Credit Memo check box is not selected and a value is entered in this field, the check box is then automatically selected.

Status

Displays the status of the Failure, which cannot be processed until it is Released.

Invoice created until

Date the invoice/credit memo was created for the line; will be the same as the End Date Failure if the failure is completely processed.

Reduce Free Counter

Indicates if the calculated free counter is reduced completely or by the discount percentage when the failure is posted (refer to topic...

After filling in the Rental Failure Card page, it must be released by clicking Release on the ribbon.

: Preparing to release the failure

Note

If the failure is not released before closing this page, it can be done by re-opening the page from the Rental Failures page (refer to Fig. 5).

Once released, click OK (otherwise the information will not be saved and the failure entry will not be created). The Qty. Failures field indicates the number of failures assigned to it.

: Qty. Failures field on the Rent Order

Clicking the value in this field opens the Rental Failures page, which contains all failures assigned to the line.

: Rental Failures page

From this page, access the Rental Failure Card page by clicking Navigate > Card on the ribbon.

Edit Invoicing Information

When an invoice has more than one period from the same document line, the system resets the Next Invoice Period field on the rent order line to the latest invoicing period. With the “Edit Invoicing Information” function, users can set the Invoice created until field on the failure to a date before this point.

This function is accessed on the Rental Failure Card by clicking Navigate > Edit Invoicing Information on the ribbon.
