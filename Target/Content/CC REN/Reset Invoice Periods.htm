Reset Invoice Periods

To re-invoice a rental period, individual lines of a Rent Order can be reset by using the “Reset Invoice Period” function. The Set New Invoice Period request page is filled with the previous invoicing period by default.

: Set New Invoice Period request page

These values can be changed to invoice a different period.

Note

The due date is set to Invoicing Start or Invoicing End as the system does in other cases.

By clicking OK, the new Invoicing Start Date and Invoicing End Date will be written to the document line (as indicated in the Next Invoice Period field) and the underlying records.
