Credit Memos

For crediting of an invoice, use the Credit Memo from the cc|rental menu. The “Copy Document” function can then be used to create the credit memo for the posted invoice with new Rental Ledger Entries to be closed by posting the credit memo.

If an invoice period must be re-posted after processing a credit memo, use the “Reset Invoice Period” function from the Rent Order to adjust the invoicing period on the line so another invoice can be created (refer to topic “Reset Invoice Periods” on page 1).
