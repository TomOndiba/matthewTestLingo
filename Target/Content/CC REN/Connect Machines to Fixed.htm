Connect Machines to Fixed Assets

Machines can be related to Fixed Assets. The purchase of the Fixed Asset related to the Machine is done by selecting the Fixed Asset No. in the document line and for sales, added automatically to the sales line when selecting the serial number assigned to the Fixed Asset Machine.

Note

When using this function, the document line can only have a quantity of one.

In the navigation pane, click Machines > Fixed Assets > Fixed Asset Machines to open existing fixed asset machines or create a new fixed asset machine.

: Fixed Asset Machine Card page without assigned serial number

Refer to online help for descriptions of the fields on this page, and refer to the Microsoft Dynamics® NAV Fixed Assets training materials for more information about working with Fixed Assets.

Fixed Asset Purchase for Machines

To purchase a Fixed Asset Machine, set up the document line as follows:

  • Type = Item
  • No. = Item number assigned to the Fixed Asset Machine
  • Quantity = 1
  • Item Fixed Asset Purchase = Yes
  • Item Fixed Asset No. = Fixed Asset Machine number

Open the Item Tracking Lines page and assign the Serial No. to the purchase line.

: Serial number assigned to Fixed Asset Machine from purchase order

The Serial No. field is updated on the Fixed Asset Machine Card to match the serial number assigned on the purchase line.

: Serial number transferred to Fixed Asset Machine assigned to purchase order

Fixed Asset Sale for Machines

As opposed to purchases, the Item Fixed Asset No. field on the sales line is automatically populated if the:

  • Item Fixed Asset Sale check box is selected on the sales line
  • Serial No. assigned to the Fixed Asset Machine is assigned to the sales line

For example, serial number NTGX26-00024 is assigned to the Fixed Asset Machine Card FA000110 for item R-20000-02.

: Fixed Asset Machine Card page with assigned serial number

On the sales line, item R-20000-02 is assigned for a quantity of one and then the Item Fixed Asset Sale check box is selected. The Item Fixed Asset No. field remains empty on the sales line until the serial number is assigned.

: Assigning the serial number to the sales line

After serial number NTGX26-00024 is assigned to the line, the Item Fixed Asset No. field is automatically populated with the corresponding Fixed Asset Machine number (FA000110).

: Fixed Asset Machine assigned to sales order
