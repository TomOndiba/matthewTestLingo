Exercise – Use Rate Component Percentages in Rent Order

Due to recent damages to item R-20000-02, CRONUS International Ltd. has decided to change the current insurance fee of 100 to 10% of the rental price.

On January 31, 2019, customer R-20000 requests one unit of item R-20000-02 for the period of 02/01/19 through 02/28/19. While processing the order, you find out the that price has increased to 1505 per month. The customer approves the price change as well as the 10% insurance fee. Due to a temporary arrangement, the one-time invoicing for the cleaning fee is waived for this order.

Note

Ensure that a Rental Price has been set up for item R-20000-02 / customer R-20000 with a monthly rate of 1480.

Set Up the Price Structure

Follow these steps to set up the price structure:

  1. Ensure that the work date is 01/31/19.
  2. In the navigation pane, click cc|rental > Contract Management > Items.
  3. Locate item R-20000-02 and then click Navigate > Rate Component Prices (in the Rental group).
  4. In the Rent Component Prices page, enter/update the following:
    1. Rate Component Type = Rate Part
    2. Rate Component No. = INSURANCE
    3. Sales Code = R-20000
    4. Unit of Measure Code = PCS
    5. Rental Rate Code = MONTH
    6. Unit Price = 0.00
    7. Percentage of Item Rate = 10

: Insurance rate component set to 10% of rental rate

  1. Click OK to close the Rent Component Prices page.

Create and Ship the Rent Order

Follow these steps to create and ship the rent order:

  1. In the navigation pane, click cc|rental > Contract Management > Rent Orders.
  2. Insert a new rent order for customer R-20000.
  3. Expand the Rental FastTab and enter the following:
    1. Rental Rate Code = Month
    2. Rental Start Date = 02/01/19
    3. Rental End Date = 02/28/19
    4. Prepaid = Yes
  4. In the Lines FastTab, select Item in the Type field.
  5. In the No. field, enter R-20000-02 and then press ENTER. Notice that the Rate Unit Price for the INSURANCE line is 10% of the Rate Unit Price on the rent item line.

: Rent Order with rate component lines

  1. On the rent item line, enter 1 in the Quantity field.
  1. On the line for the TAX rate component, enter 80 in the Rate Unit Price field.
  2. Delete the last line for the One-Time Invoicing CLEANRETURN to remove the cleaning fee.
  3. On the rent item line, change the Rate Unit Price from 1480 to 1505. Notice that the Rate Unit Price in the INSURANCE rate component line has changed to 150.50.

: Rent Order with changed rate unit prices and deleted Cleaning fee

  1. Select the rent item line and then click Line > Item Tracking Lines on the Lines FastTab.
  2. With the Shipment option selected, click OK.
  3. On the ribbon, click Select Entries.
  4. Note which serial number is selected and then click OK.
  5. Click OK to close the Item Tracking Lines page.
  6. On the ribbon, click Post.
  7. With the Ship option selected, click OK.

Invoice the Rent Order

Follow these steps to invoice the rent order:

  1. On the ribbon, click Post.
  2. Select the Invoice option and then click OK.
  3. In the Posting Date field, enter 02/01/19.
  4. In the Invoice to Date field, enter 02/28/19.
  5. Click OK and accept all confirmation messages.
  6. On the ribbon, click Navigate > Invoices.
  7. Double-click the line to open the Service Invoice Rental page. Notice that the rental item and the two Rate Part components appear with the same values entered on the Rent Order line.

: Service Invoice for rental and rate part component positions

  1. On the ribbon, click Post and then click Yes to post the invoice.
  2. Escape back to the Rent Order.
  3. On the Navigate tab, click Entries > Rental Ledger Entries. Notice that the rental and rate part component lines appear with the adjusted amounts.

: Rental Ledger Entries for a posted invoice with rental and rate part component positions

  1. Escape back to the Rent Order.

Return the Rent Order

Follow these steps to return the rent order:

  1. In the Posting Date field, enter 02/28/19.
  2. On the rental item line, enter 02/28/19 in the Rental Return Date field.
  3. In the Return Qty. to Receive field, enter 1.
  4. Select the rental item line and then click Line > Item Tracking Lines on the Lines FastTab.
  5. Select the Receive option and then click OK.
  6. Confirm that the serial number is correct and then click OK to close the Item Tracking Lines page.
  7. On the ribbon, click Home > Post.
  8. Select the Receive option and then click OK.
  9. Close the Rent Order page.
