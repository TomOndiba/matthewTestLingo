Extensions

The Rental Extension functionality allows users to extend rental periods. This extension has an effect on the availability of items: the items are not available until the end of the specific rental.

When the “Rental Extension” function is run for a rent order line, the Rental Extension Card is used to enter the new Rental End Date and a description.

: Rental Extension Card

By clicking OK, this function updates the Rental End Date field on the rent order and the availability that is dependent on this change.

Note

If the OK button is not clicked, the extension process is cancelled.
