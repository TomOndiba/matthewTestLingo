Disposition Calendar Set Up

On the cc|graphical extension menu, under Administration, click Disposition Calendar Setup.

: Disposition Calendar Setup page

On the Misc FastTab, fill in the Default Rent Rate Code field with the default code that will be used when new rent reservations or rent orders are created from the Disposition Calendar View.

For information about the setup and handling of cc|graphical extension, please refer to the current “cc|graphical extension” training documentation.
