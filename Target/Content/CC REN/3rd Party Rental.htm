3rd Party Rental

If a customer wants to rent an item that is not in stock, it can be useful to rent the item from a vendor instead of buying it or not being able to rent the item to the customer. This type of renting from a vendor is referred to as 3rd Party Rental.

cc|rental offers two ways to initiate the 3rd party rental process:

  1. Create a 3rd Party Purchase Order for the Vendor
    A Purchase Order is created from a Rent Order (Lines FastTab > Functions > Create Purchase Order). Then a vendor is selected and the system subsequently creates the purchase order for the 3rd Party Rental. These purchase order lines have G/L Accounts to post the rental revenue and are linked to the Rent Order lines from which they are created.
  1. Create a Requisition Worksheet
    A Requisition Worksheet is filled with the lines from a Rent Order (Lines FastTab > Functions > Create Requisition). In the Requisition Worksheet, the demand can be covered by the creation of a 3rd Party Purchase Order, supply from an alternative location, or from a production order.

: Rent Order page, highlighting Create 3rd Party Requisition / Purchase Order functions

Using either of the functions opens a page to choose the vendor:

  • If creating a 3rd Party Purchase Order, the Vendor No. field must be populated.
  • If creating a Requisition Worksheet, the Vendor No. field can be left blank. The Vendor No. field in the Requisition Worksheet will not be pre-populated.

: Create Rental Purchase Order page

If a requisition worksheet is created, a message appears stating that the requisition was created but the requisition worksheet is not opened automatically. On the ribbon of the Rent Order, click Navigate > Requisition Lines to open the document.

If a purchase order is created, a message appears stating that the order was created and asks if you want to edit the order. Clicking Yes opens the purchase order; clicking No closes the message. The order can later be accessed by clicking Navigate > Purchase Lines on the ribbon of the Rent Order.

 

: Rent Order, View created supply documents

A list with the created lines appears from where the purchase order or requisition worksheet can be opened.

: Requisition Lines page

In the Requisition Worksheet, there are six Replenishment System options that can be selected:

  • Purchase – A Purchase Order is created
  • Prod. Order – A Production Order is initiated
  • Transfer – The item is transferred from another location to the rental location before shipment
  • Assembly – An Assembly Order is created
  • Rent – A 3rd Party Purchase Order is created (3rd Party Rental)
  • Alternative Location – The item is shipped from another rental location

: Requisition Worksheet with 3rd party rental information

If the Rent option is selected, the Vendor No. field must be populated to create the Purchase Order when the “Carry Out Action Message” function is run. Upon creation of the Purchase Order, the Location Code in the Rent Order is changed to the 3rd Party Location Code that is entered in the External Rental Loc. Code field in the Rental Mgt. Setup page.

: Rent Order page, 3rd Party Location

If the item to be rented requires serial number-specific tracking, a new serial number must be entered for the Rent Order line. Refer to topic “Return 3rd Party Item with Serial Number” on page 1 for more information about returning serialized items to the 3rd party vendor.

The shipment of the Purchase Order posts the positive adjustment for the items to make them available on the 3rd Party Location.
