Exercise – Create a Machine Type

A MIG Welder is a new item that will be available for sale. It must set up as a Machine Type and purchased before it can be sold to customers. The unit cost is 4500, the unit price is 5900, and it must be set up for serial number tracking.

A new vendor, Weller Electric Manufacturing Co., must be set up for the purchase of this item.

Create a Machine Type Item

Follow these steps to create a Machine Type:

  1. In the navigation pane, click Departments > Machines > Types > Machine Types.
  2. On the ribbon, click New.
  3. In the No. field, type R-50000-01.
  4. Notice that the Base Unit of Measure field is filled automatically.
  5. In the Description field, type MIG Welder.

: Machine Type Card page

  1. Expand the Invoicing FastTab and notice that the Costing Method field is set to Specific.
  2. In the Unit Cost field, enter 4,500.
  3. In the Unit Price field, enter 5,900.
  4. In the Gen. Prod. Posting Group field, enter RETAIL.
  5. In the Inventory Posting Group field, enter RESALE.
  6. Expand the Item Tracking FastTab and notice that the Item Tracking Code field is set to SNALL.
  7. Click OK to close Machine Type Card page.

Set Up a Vendor and Process a Purchase Order

Follow these steps to set up a vendor and process a Purchase Order:

  1. In the navigation pane, click Purchasing > Order Processing > Vendors.
  2. On the ribbon, click New.
  3. Click Cancel to close the templates page.
  4. In the No. field, type R-50000.
  5. In the Name field, type Weller Electric Manufacturing Co.
  6. Expand the Address & Contact FastTab.
  7. In the Post Code field, enter GL78 5TT.
  8. Expand the Invoicing FastTab.
  9. In the Gen. Bus. Posting Group field, enter EU.
  10. In the Vendor Posting Group field, enter EU.
  11. On the ribbon, click Purchase Order to create a new purchase order for this vendor.
  12. In the Vendor Invoice No. field, enter R-50000-001.
  13. On the Lines FastTab, select Item in the Type field.
  14. In the No. field, enter R-50000-01.
  15. In the Quantity field, enter 5.
  16. In the Direct Unit Cost field, enter 3200.

: Purchase Order page

  1. On the Lines FastTab, click Line > Item Tracking Lines.
  2. On the ribbon, click Actions > Create Customized SN.
  3. In the Customized SN field, type R-50000-01-00001.
  4. Click OK.
  5. Click OK to close the Item Tracking Lines page.
  6. On the ribbon, click Post.
  7. With the Receive and Invoice option selected, click OK.
  8. Close the open pages.

Review the Machine Type Item Information

Follow these steps to review the updates to the Machine Type item:

  1. In the navigation pane, click Machines > Types > Machine Types.
  2. Double-click the line for number R-50000-01 and notice that there are now five units of this machine type item in inventory.

: Machine Type Card page showing on-hand inventory

  1. Click OK to close the Machine Type Card page.
  2. In the navigation pane, under Machines, click Machines > Machines.
  3. Go to the end of the list and notice that five new Machine records for the MIG Welder exist and contain the created Serial Nos.

: Machines created for item R-50000-01

  1. Double-click the line for the first MIG Welder.
  2. Expand the Customer FastTab and notice that the Machine Card is related to the Internal Customer after purchase.

: Machine Card showing relation to the Internal Customer

Note

When a Machine Type Item is sold, the customer on the Rent Order or Sales Order is transferred to the Customer No. field on the Service Item/Machine Card.

  1. Close the open pages.
