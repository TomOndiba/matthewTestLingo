Failures

For every Rental position on a Rent Order, a Failure can be assigned. A Failure may be necessary when the item has downtime within a rental period that prevents the customer from using the rented item, such as planned maintenance.

Note

Since rate components – such as taxes and insurance – have to be paid regardless of a failure, an item’s Rate Components are not automatically credited for a failure period.

Posting a Failure allows the user to credit that portion of time from the invoice period. This can be handled in one of the following ways:

  • Creating a Credit Memo for a posted invoice period
  • Including credit lines on the invoice if within the invoice period

Therefore a failure period and the Failure Type must be specified in the Rental Failure Card page.
