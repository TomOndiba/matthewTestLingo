Exercise – Create a 3rd Party Purchase Order from a Rent Order

Customer R-40000 needs to rent a 20 Feet Store Container (item R-40000-01) from March 1, 2019 through April 30, 2019 which is not currently available. The container must be rented from vendor R-40000 in order to rent it to the customer.

Create the Rent Order

Follow these steps to create the Rent Order:

  1. Set the Work Date to 02/15/19.
  2. Insert a new rent order for customer R-40000.
  3. On the first line of the Lines FastTab, enter the following:
    1. Type = Item
    2. No. = R-40000-01
    3. Quantity = 1
    4. Rental Rate Code = Month
    5. Month Based = Yes
    6. Rental Start Date = 03/01/19
    7. Rental End Date = 04/30/19
    8. Rate Unit Price = add if a value is missing

Create the 3rd Party Purchase Order

Follow these steps to create the 3rd Party Purchase Order:

  1. On the Lines FastTab, click Functions > Create Purchase Order.
  2. In the Vendor No. field, enter R-40000.

: Create Rental Purchase Order, Vendor input page

  1. Click OK.
  2. Click Yes to edit the purchase order. Notice the following:
    • The line contains the G/L Account entered in the 3rd Party Rent Account field in the General Posting Setup page.
    • Location Code is set to EXTERNAL, the location entered in the External Rental Loc. Code field in the Rental Mgt. Setup page.
    • Rental Start Date is set to 3/1/2019 and the Rental End Date is set to 4/30/2019.

: 3rd Party Purchase Order created from the Rent Order

  1. Close the Purchase Order page.
  2. Notice that the Location Code field in the selected line of the Rent Order is set to EXTERNAL.
  3. On the Lines FastTab, click Line > Item Tracking Lines.
  4. With the Shipment option selected, click OK.
  5. In the Serial No. field, type EXT001.
  6. In the Quantity (Base) field, type 1.

: Item Tracking Lines, with no availability

  1. Click OK and then click Yes to dismiss the Availability Warnings and close the Item Tracking Lines page.
  2. On the ribbon, click Navigate > Purchase Lines.
  3. On the ribbon, click Show Document.
  4. In the Rent Cost per Rent Base Unit Excl. VAT field on the line, enter the daily rental cost.
  5. Update the value in the No. of Rent Base Units field as needed.
  6. In the Rental Start Date and Rental End Date fields, enter the rental date range from the vendor invoice.

: 3rd Party Purchase Order with updated rental information

Post the 3rd Party Purchase Order and Rent Order

Follow these steps to receive/invoice the 3rd Party Purchase Order and ship/invoice the Rent Order:

  1. In the Vendor Invoice No. field, type the number from the vendor invoice.
  2. On the ribbon, click Post.
  3. Select the Receive and Invoice option and then click OK.
    Note that the Qty. to Return = 1 on the purchase order line.

: Received and Invoiced 3rd Party Purchase Order

  1. Close the Purchase Order and Purchase Lines pages.
  2. On the ribbon of the Rent Order, click Home > Post.
  3. With the Ship option selected, click OK.
  4. On the ribbon, click Post.
  5. Select the Invoice option and then click OK.
  6. In the Posting Date field, enter 03/01/19.
  7. In the Invoice to Date field, enter 03/31/19.
  8. Click OK and accept all confirmation messages.
  9. Open and post the Service Invoice Rental.
  10. On the ribbon of the Rent Order, click Home > Post.
  11. Select the Invoice option and then click OK.
  12. In the Posting Date field, enter 04/01/19.
  13. In the Invoice to Date field, enter 04/30/19.
  14. Click OK and accept all confirmation messages.
  15. Open and post the Service Invoice Rental.

Post the Rent Order Return and the 3rd Party Purchase Order Return

Follow these steps to post the return to the Rent Order and the return to the 3rd Party Purchase Order:

  1. In the Posting Date field, enter 04/30/19.
  2. On the rental item line, enter 04/30/19 in the Rental Return Date field.
  3. In the Return Qty. to Receive field, enter 1.
  4. Select the rental item line and then click Line > Item Tracking Lines on the Lines FastTab.
  5. Select the Receive option and then click OK.
  6. Confirm that the serial number is correct (EXT001) and then click OK to close the Item Tracking Lines page.
  7. On the ribbon, click Home > Post.
  8. Select the Receive option and then click OK.
  9. On the ribbon, click Navigate > Purchase Lines.
  10. On the ribbon, click Show Document.
  11. On the Lines FastTab, click Line > Select Rent Return Serial Nos.
  12. Confirm that the serial number is correct (EXT001) and then select the To Return check box on the line.

: 3rd Party Purchase Order with updated rental information

  1. Click OK to close the Service Contract Line List page.
  2. On the ribbon, click Post.
  3. Select the Return option and then click OK.
    Note that the Qty. Returned field is set to 1, to indicate the rental item return to the 3rd party vendor.

: Received, Invoiced and Returned 3rd Party Purchase Order

  1. In the Vendor Invoice No. field, add a variation, such as “-2” to the number from the vendor invoice (since the number was already used in a previous step).
  2. On the ribbon, click Post.
  3. Select the Invoice option and then click OK to finish the 3rd Party Purchase Order process.
  4. Close the open pages.
