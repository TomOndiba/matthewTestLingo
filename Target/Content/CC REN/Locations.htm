Locations

cc|rental needs two location codes for shipment and returns because the system uses item transfers, not sales or negative adjustments. To do so it needs a source location and a destination location. On shipment the system posts a transfer from the Rental Location, to the In-transit Location and then to the Rented Location. On return it will do a transfer from the Rented Location, to the In-transit Location and then to the Rental Location.

Rental locations must be marked as either:

  • Use As Rental Location – main rental location
  • Use As - Rented – customer locations (rented location)

: Location Card page

Locations can only be used as Rental Location or Rented Location – never as both. These location types should be used exclusively for Rental purposes and processes.
