Create Rental Contract Invoices

As an alternative to posting invoices for each rent order individually, use the “Create Rental Contract Invoice” batch job to create invoices for multiple rent orders.

When the batch job is run, the program automatically fills in the relevant fields on the invoices with information from the rent order.

: Create Rental Contract Invoice batch job opened from the Periodic Activities menu

When running this function from the Periodic Activities menu, the Posting Date field is populated with the work date and the Invoice to Date field is empty.

Note

If a filter is set on a Rent Order No. in the Rental Instance FastTab, delete the value in the filter before running the function.

Exercise – Run the Create Rental Invoices Batch Job

On January 1, 2019, you create and ship two orders:

  • Two units of item R-30000-01 to customer R-30000 for the period of January 1, 2019 – February 28, 2019.
  • Four units of item R-30000-02 to customer R-30000 for the period of January 1, 2019 – January 31, 2019.

Instead of creating separate invoices for the month of January, you decide to run the “Create Rental Contract Invoice” batch job through January 31, 2019.

Create the Rent Orders

Follow these steps to create the rent orders:

  1. Set the Work Date to 01/01/19.
  2. Insert a new rent order for customer R-30000.
  3. In the lines, enter the following:
    1. Type = Item
    2. No. = R-30000-01
    3. Quantity = 2
    4. Rental Rate Code = Month
    5. Month Based = Yes
    6. Rental Start Date = 01/01/19
    7. Rental End Date = 02/28/19
  4. On the ribbon, click Post.
  5. With the Ship option selected, click OK.
  6. Insert another order for customer R-30000.
  7. In the lines, enter the following:
    1. Type = Item
    2. No. = R-30000-02
    3. Quantity = 4
    4. Rental Rate Code = Month
    5. Rental Start Date = 01/01/19
    6. Rental End Date = 01/31/19
    7. Rate Unit Price = 280 (if missing)
  1. On the ribbon, click Post.
  2. With the Ship option selected, click OK.
  3. Close the Rent Order.

Run the Create Rental Contract Invoices Batch Job

Follow these steps to run the “Create Rental Contract Invoice” batch job:

  1. In the navigation pane, under cc|rental, click Periodic Activities > Create Contract Invoices.
  2. In the Posting Date field, enter 01/01/19.
  3. In the Invoice to Date field, enter 01/31/19.
  4. In the Bill to Customer No. filter, enter R-30000.
  5. In the Rent Order No. filter, delete the value if one exists.

: Create Rental Contract Invoice batch job with the Rent Order No. filter cleared

  1. Click OK and accept the confirmation messages. The following message appears stating how many invoices were created:

: Invoice creation confirmation message

  1. Click OK to close the message.
  2. In the navigation pane, under cc|rental, click Contract Management > Invoices.
  3. Open the first invoice and notice that the invoice period is 01/01/19 – 01/31/19, not through the rental end date of 02/28/19 because the invoice to date was set to 01/31/19.

: Invoice created from the Create Rental Contract Invoice batch job

  1. On the ribbon, click Post and then click Yes to post the invoice.
  2. Open the second invoice and notice that the invoice period is 01/01/19 – 01/31/19.

: Invoice created from the Create Rental Contract Invoice batch job

  1. On the ribbon, click Post and then click Yes to post the invoice.
  2. In the navigation pane, under cc|rental, click Contract Management > Rent Orders.
  3. Open each rent order created in this exercise and then click Navigate > Entries > Rental Ledger Entries on the ribbon. Notice that the posted entries have been closed.

: Rental Ledger Entries for posted invoices created from the Create Rental Contract Invoice batch job
