Machine Setup

Machine setup is performed in pages found in the Machines, Administration and the Service, Administration menus.

Machine Mgt. Setup

The Machine Mgt. Setup page is opened from the navigation pane by clicking Departments > Machines > Administration > Machine Mgt. Setup. The following sections explain the fields on this page.

General FastTab

The General FastTab provide the basic settings for the Machines module.

: Machines Mgt. Setup page, General FastTab

The fields on this FastTab are described in the following table.

Field

Description

Def. Item Tracking Code

Used as the Default Item Tracking Code when creating a Machine Type.

Def. Unit of Measure

Used as the default Unit of Measure Code when creating a Machine Type, Supplementary Part Type or Machine Component Type.

 

Numbering FastTab

On this FastTab, number series for the Machine master data are defined.

: Machine Mgt. Setup page, Numbering FastTab

The fields on this FastTab are described in the following table.

Field

Description

Machine Nos.

Number Series to be used for the enumeration of Machines.

Machine Type Nos.

Number Series to be used for the enumeration of Machine Types.

Suppl. Part Type Nos.

Number Series to be used for the enumeration of Supplementary Parts.

Component Type Nos.

Number Series to be used for the enumeration of Machine Components.

 

Product Version Information

On the ribbon, click About to display the current product version information.

: Product version information accessed by clicking About on the ribbon

Machine Colors

As a part of technical data machines can have a color. The Machine Colors page is opened from the navigation pane by clicking Departments > Machines > Administration > Machine Colors.

: Machine Colors page

The fields in the Machine Colors page are described in the following table.

Field

Description

Code

Unique identifier for the machine color.

Description

A short description for the machine color.

Service Setup

To enable the system to work with Internal Service Orders, fields in the Service Mgt. Setup page must be populated. The Service Mgt. Setup page is opened from the navigation pane by clicking Departments > Service > Administration > Service Setup.

: Service Mgt. Setup page, Internal FastTab

The fields on the Internal FastTab are described in the following table.

Field

Description

Work Type Code Internal

Identifies the Work Type Code to be used for internal services.

Def. Internal Customer No.

Identifies the Customer No. that is used when a Machine is created automatically by the system.
