Customers

Renting is always processed with a customer. To ship a rental item to a customer, a customer location must be assigned in the Customer Location Code field on the sales line. This Location Code describes the place where the item is located during rental (with the customer).

In the Customer record, a location can be specified for the default rental location and for the customer location.

: Customer Card, Rental FastTab

For each customer location, an entry in the Inventory Posting Setup is required. In addition, the Transfer Routes from Rental Locations to Customer Locations and back must be inserted. These processes can be done manually or automatically using the “Create Customer Location” function.

Create Customer Location Function

The “Create Customer Location” function creates a customer location using the information from the customer and assigns it to the customer record. In addition, this function automatically creates the Inventory Posting Setup and the Transfer Routes that are needed from the new location code to all existing Rental Location Codes that exist at that time.

The “Create Customer Location” function is accessed from the customer card by clicking Actions > Create Customer Location on the ribbon.

: Accessing the “Create Customer Location” function from the Customer Card

A confirmation message appears, stating that the customer location was created, and does the user want to open it. Click Yes to open the Location Card or click No to close the confirmation message.

On the Location Card, the code is the same as the customer number, the Use As - Rented check box is selected, and the address and communication information is transferred from the customer card.

: Location Card for the customer location

On the Customer Card, the customer location is assigned in the Customer Location Code field.

: Customer Location assigned to the customer

The Inventory Posting Setup is completed for the customer location using the Inventory Posting Setup information specified in the Def. Invt. Post Setup Location field on the Rental Mgt. Setup page.

: Inventory Posting Setup for the customer location

The Transfer Routes are set up for the rental and customer locations, using the In-Transit Location specified in the Def. In-Transit Location Code field on the Rental Mgt. Setup page.

: Transfer Routes for rental and customer locations

Self-Insured Customers

Customers who have their own insurance can be specified as Self-Insured on the Rental FastTab of the Customer Card.

: Customer set up as Self-Insured

When this check box is selected, Rent Rate Components with Usage = Insurance will not be included in the rental documents for the customer. Refer to topic “Self-Insured Customers” on page 1 for more information.
