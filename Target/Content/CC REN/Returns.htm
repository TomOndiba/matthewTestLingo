Returns

The following information must be entered in the Rent Order lines to post a return of the shipped goods:

  • The return date must be entered in the Rental Return Date field. This is the date that is used to return the full or partial quantity that was shipped back to the Rental Location.
  • The quantity to be returned must be inserted to the Return Qty. to Receive field.

If the period of time between the last invoiced date and the Return Date is not invoiced yet, it will be regarded in the next invoice(s) that is created. If the period is invoiced before the return, a credit memo is created if necessary.

Note

For rental returns that use serial tracking, the proper serial number must be selected for return if the quantity is >1 and processed automatically when quantity = 1.

The return will be considered within the batch for invoicing. If the Return Date is before the current Invoice Period, a separate Credit Memo is created with the lines. In any other cases, the return will be accounted in the Invoice (invoicing until Return Date).

Exercise – Return the Rented Items

On February 28, 2019, you receive the rental item back from customer R-30000.

Post the Item Return

Follow these steps to post the item return:

  1. Open the rent order created in the “Record a Rental Position on a Rent Order” exercise.
  2. In the Posting Date field, enter 02/28/19.
  3. In the Lines FastTab, enter 02/28/19 in the Rental Return Date field.
  4. In the Return Qty. to Receive field, enter 1.
  5. On the ribbon, click Post.
  6. Select the Receive option and then click OK.
  7. Use the “Choose Columns” function to show the Next Invoice Period field in the lines.
  8. Expand the Rental FastTab and then click Show more fields.
  1. Notice the following:
    • Qty. to Invoice and Return Qty. to Receive fields are blank
    • Return Qty. Received and Quantity Invoiced fields are 1
    • Next Invoice Period field is blank because all invoicing is complete
    • Availability is 33 in the Rent Line Details FactBox
    • Closed check box is selected on the Rental FastTab
