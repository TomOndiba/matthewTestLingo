Disposition Calendar Usage

The Disposition Calendar can be used to obtain a graphical view of items that are handled with serial numbers and are created from the Machine Type Card, as well as items that do not use serial numbers.

For items that use serial numbers, the Rent Item field must be set to Yes on the Rental FastTab of the Machine Type Card. For items that do not use serial numbers, the Rent Item field must be set to Yes on the Rental FastTab of the Item Card.

: Machine Type Card and Item Card with Rent Item field set to Yes

The Disposition Calendar is separated into these main areas:

  • Calendar (top left pane), Timeline and Search (top right pane)
  • Rent Item or Location Views (bottom left pane)
  • Details View (bottom right pane)
  • Calendar, Timeline and Search

The Calendar shows the same date range that is visible in the Timeline. The displayed date area can be changed by clicking and dragging the mouse on the calendar control.

The Timeline controls the date area for the rendered entries. Scroll horizontally by clicking and holding the mouse on the Timeline, and then dragging it to the proper date range. The Previous and Next buttons can also be used to scroll to the next or the previous period.

To scale the Timeline, hold down the right mouse button and drag the mouse cursor horizontally or click the Day, Week or Month buttons.

: Calendar and Timeline highlighted on the Disposition Calendar

In the Search field, enter a search description (using the standard Microsoft Dynamics NAV filtering syntax for the search text) and then click the Find icon. When the matching criteria are met, the view automatically scrolls to the record. Click the Find icon again to move to the next record that meets the search criteria.

Rent Item and Locations Views

The Disposition Calendar View is grouped by Rent Item (order by Item No.). For each item, the list of related serial numbers can be shown by expanding the folder.

: Disposition Calendar with Rent Item R-10000-02 folder expanded

The Locations view shows all the serial numbers that are used and may be available on the selected location.

: Disposition Calendar with Rental Locations displayed

Details View

The Details View pane contains a column of buttons to the left and the related information in the calendar matrix to the right:

  • The first line in the column of buttons contains the item number. The calendar matrix shows the availability in graphical form as a bar chart. The red line at the top of the row is the point of full use of the capacity.
  • The second line – “without Serialno.” – contains documents (Rent Reservations or Rent Orders) that have the item number attached but do not yet contain serial numbers.
  • The following lines contain a single serial number. The calendar matrix contains Rent Reservations and Rent Orders that exist for that specific item and serial number.

Creating Rental Documents from Disposition Calendar

To create a document from the Disposition Calendar (Rent Reservation or Rent Order), select the item that is intended to be rented to the customer. On the first day of the period – and in the row for the specific serial number – press the right mouse button and then select Create Reservation or Create Rent Order.

: Creating a rental document from the Disposition Calendar

The Customer List page appears; select the customer requesting the rental and then click OK.

The system now creates a Rent Reservation or a Rent Order (as selected) using the customer, item number and serial number for the selected date. It returns to the Disposition Calendar View that now has a new box on the selected date for your new document. You can now extend the rental period by dragging the right margin of the box.

After dragging the box to the right date, the document can be released by right-clicking the mouse on the box and then selecting Release. A released document can also be reopened or a rent order can be created for a Rent Reservation by right-clicking the box.

: Releasing a rental document from the Disposition Calendar

Note

Periods of released documents cannot be changed by dragging.

If no serial number is available, the document can be created on the “without Serial No.” line.

Lines that are not released can be pulled to a different period and to another serial number within the same item.

Right-clicking on an area in the Details View offers the possibility to open the related card; that is, the Item Card (first line), the Service Item Card (for each serial number), and the Rent Reservation or the Rent Order Card (document entries). Right-click a document box and then select Open Rent Reservation or Open Rent Order to open the related document.

: Released Rent Reservation opened from the Disposition Calendar

Updating Documents with Disposition Calendar

If the rental document is created from the Rent Reservation or the Rent Order Card, the Disposition Calendar can be used to select the specific serial number and add the rental period for the item.

To do so, first open the appropriate card from the Rental > Contract Management menu and create the document with the following line information:

  • Rental Position = Rental
  • Type = Item
  • No. = Item number
  • Quantity = 1

Afterwards, open the Disposition Calendar by clicking the AssistEdit button in the No. field.

: Opening the Disposition Calendar from a rental document

The Disposition Calendar View opens to create an entry for the document line. Right-click a free serial number and then select Create here.

: Adding a document line to the Disposition Calendar opened from a rental document

A new box appears for the document line; drag the ends of the box to adjust the rental period. Click OK to close the Disposition Calendar View to return to the source document.

: New document added to the Disposition Calendar

The selected Rental Start Date, Rental End Date and Serial No. are added to the document.

: Serial number and rental date information added to new document from the Disposition Calendar

Using Disposition Calendar for Items without Serial Numbers

The same functions used to create and update Rental documents with the Disposition Calendar is available for items that do not use serial numbers. The differences are that a serial number is not selected in the Disposition Calendar line or the Rental document line, and all documents for the item appear in a single row in the Disposition Calendar line.

: Disposition Calendar lines for items with no serial numbers

Create Rental Document from Disposition Calendar

To create a document from the Disposition Calendar (Rent Reservation or Rent Order), select the item that is intended to be rented to the customer. On the first day of the period, right-click and select Create Reservation or Create Rent Order.

: Creating a rental document from the Disposition Calendar

The Customer List page appears; select the customer requesting the rental and then click OK.

The system creates a Rent Reservation or a Rent Order (as selected) using the customer and item number for the selected date. In the Disposition Calendar View, a new box is added on the selected date for the new document. The rental period can be extended by dragging the right margin of the box, and can be released. The released document can also be reopened or a rent order can be created for a Rent Reservation by right-clicking the box.

: Options for the rental document created from the Disposition Calendar

Lines that are not released can be pulled to a different period within the same item.

Right-clicking on an area in the Details View offers the possibility to open the related card; that is, the Item Card (first/second line), and the Rent Reservation or the Rent Order Card (document entries). Right-click a document box and then select Open Rent Reservation or Open Rent Order to open the related document.

: Rent Reservation opened from the Disposition Calendar

Updating Documents with Disposition Calendar

If the rental document is created from the Rent Reservation or the Rent Order Card, the Disposition Calendar can be used to update the rental period for the item.

To do so, first open the appropriate card from the Rental > Contract Management menu and create the document with the following line information:

  • Rental Position = Rental
  • Type = Item
  • No. = Item number
  • Quantity = 1
  • Rental Start Date = starting date of the rental
  • Rental End Date = ending date of the rental; if not known, choose the same as Rental Start Date for easier movement in the Disposition Calendar

Afterwards, open the Disposition Calendar by clicking the AssistEdit button in the No. field.

: Opening the Disposition Calendar from a rental document

The Disposition Calendar View opens to update the entry for the document line. A new box appears for the document line.

: New document added to the Disposition Calendar

Drag the ends of the box to adjust the rental period. Click OK to close the Disposition Calendar View to return to the source document.

The updated dates are updated in the document.

: Rental date information updated in new document from the Disposition Calendar
