Self-Insured Customers

Some customers may have their own insurance for machines or other rental equipment. For those customers, the rate component for insurance can be excluded on Rental documents if the:

  • Self-Insured check box is selected on the Customer Card
  • Rent Rate Component/Item Rent Rate Component for Insurance has Usage = Insurance

The Usage setting is transferred from the Rent Rate Component to the Item Rent Rate Component, but can be changed per item. Alternatively, the Usage setting can be handled directly on the Item Rent Rate Component instead of the Rent Rate Component.

Exercise – Create Rent Order for Self-Insured Customer

Customer R-10000 has notified CRONUS International Ltd. that they have obtained insurance for their machine operators to use the ReTex TC240 LC Crawler Excavator, CRONUS item R-10000-02.

After receiving verification of their insurance, you set their customer as Self-Insured and add the Usage of Insurance to the Item Rent Rate Component for item R-10000-02.

When you create a Rent Order, you confirm that the rent rate component for Insurance is not included in the Rental line.

Set Up the Customer and Item Rent Rate Component

Follow these steps to set up the customer and Item Rent Rate Component:

  1. Ensure that the work date is 01/31/19.
  2. In the navigation pane, click cc|rental > Contract Management > Customers.
  3. Locate customer R-10000 and open the card.
  4. Expand the Rental FastTab.
  5. Select the Self-Insured check box.

: Customer R-10000 set up as Self-Insured

  1. Click OK to close the Customer Card.
  2. In the navigation pane, click cc|rental > Contract Management > Items.
  3. Locate item R-10000-02 and then click Navigate > Rent Rate Components Prices (in the Rental group). Notice that there are two components: Insurance and a Return Cleaning fee.
  4. Add the Usage field using the “Choose Columns” function.
  5. On the line for Rate Part = Insurance, select Insurance in the Usage field.

: Usage set to Insurance for the insurance rate part for item R-10000-02

Create the Rent Order

Follow these steps to create the rent order:

  1. In the navigation pane, click cc|rental > Contract Management > Rent Orders.
  2. Insert a new rent order for customer R-10000.
  3. Expand the Rental FastTab and enter the following:
    1. Rental Rate Code = Month
    2. Rental Start Date = 02/01/19
    3. Rental End Date = 02/28/19
  4. In the Lines FastTab, select Item in the Type field.
  5. In the No. field, enter R-10000-02 and then press ENTER. Notice that only the Rate Component for the Return Cleaning fee has been added; the Rate Component for Insurance has been excluded for this customer/item combination.

: No rate component for insurance added to self-insured customer R-10000 for item R-10000-02

  1. Click OK to close the Rent Order.
