Exercise – Ship Item with Preconditions

For legal purposes, item R-10000-01 can only be used by machine operators who are at least 18 years old. Prior to shipping this item to a customer, the name and date of birth of the person who will be operating the machine must be entered in the Shipment Preconditions page. This item also includes some information that needs to be relayed to the customer.

When customer R-10000 calls to rent the excavator, they are asked to provide the name and date of birth of the machine operator. John Tippett – born February 05, 2001 – was one of the people scheduled to operate the machine. Since he has not yet turned 18, the system shows the date in red. While the date can be overridden and the shipment processed, the customer provides the name of the other person, Mike Nash, born May 16, 1997.

Set Up the Preconditions

Follow these steps to set up the preconditions:

  1. In the navigation pane, click cc|rental > Administration > Rental Preconditions.
  2. On the ribbon, click New.
  3. In the Code field, enter AGE.
  4. In the Description field, type Required age to operate equipment.
  5. In the Action Type field, enter Confirmation.
  6. In the Confirmation Type field, enter Name+Date of Birth.
  7. In the Minimum Age field, enter 18Y.

: Customer R-10000 set up as Self-Insured

  1. Go to the next line.
  2. In the Code field, enter Info - Equip.
  3. In the Description field, type Operating Instructions on floor of equipment.
  4. Leave the Action Type field set to Message.

: New preconditions for rental equipment

  1. Click OK to close the Rental Preconditions pages.
  2. In the navigation pane, click cc|rental > Contract Management > Items.
  3. Locate item R-10000-01 and then click Navigate > Rental Preconditions (in the Rental group).
  4. In the Precondition Code field, enter AGE.
  5. Go to the next line.
  6. In the Precondition Code field, enter INFO - EQUIP.

: Usage set to Insurance for the insurance rate part for item R-10000-02

  1. Click OK to close the Item Rental Preconditions page.

Create and Ship the Rent Order

Follow these steps to create and ship the rent order:

  1. In the navigation pane, click cc|rental > Contract Management > Rent Orders.
  2. Insert a new rent order for customer R-10000.
  3. Expand the Rental FastTab and enter the following:
    1. Rental Rate Code = Month
    2. Rental Start Date = 02/01/19
    3. Rental End Date = 02/28/19
  4. On the first line of the Lines FastTab, enter the following:
    1. Type = Item
    2. No. = R-10000-01
    3. Quantity = 1
    4. Rate Unit Price = add if values are missing on Item or Rate Component lines
  5. On the Lines FastTab, click Line > Item Tracking Lines.
  6. With the Shipment option selected, click OK.
  7. On the ribbon, click Select Entries and then click OK.
  8. Click OK to close the Item Tracking Lines page.
  9. On the ribbon, click Post.
  10. With the Ship option selected, click OK.
    The Shipment Preconditions page opens with the two preconditions added for item R-10000-01.

: Shipment preconditions for item R-10000-01

  1. In the first line, enter John Tippett in the Confirmation Name field.
  2. In the Confirmation Date of Birth field, enter 02/05/2001. The value is in red italics because John is not yet 18 years old.

: Machine operator younger than 18 years old entered in precondition

  1. Clicking OK with a date of birth that doesn’t meet the required age (or if the required confirmation information is not entered) will generate a message that “Not all confirmations have been filled in. Continue anyway?” Clicking Yes completes the shipment process; clicking No stops the shipment process.

: Message for incomplete confirmations

  1. In this case, click No to enter the other machine operator.
  2. In the Confirmation Name field, enter Mike Nash.
  3. In the Confirmation Date of Birth field, enter 05/16/1997. The value appears as normal because Mike is 21 years old when the rental period will begin.

: Machine operator exceeding 18 years old entered in precondition

  1. Read the message to the customer and then click OK to ship the item.

Review the Shipment Precondition Logs

Follow these steps to review the shipment precondition log for the rent order and the item:

  1. On the ribbon of the rent order, click Navigate > Precondition Log.
    The Item Shipment Precondition Log page opens with the two precondition entries.

: Item Shipment Precondition Log opened from rent order

  1. Close the Item Shipment Precondition Log page.
  2. In the Rent Line Details FactBox, click item number R-10000-01.
  3. In the Item Card, click Navigate > Item Shipment Precondition Log (in the Rental group).
    The Item Shipment Precondition Log page opens with the two precondition entries.

: Item Shipment Precondition Log opened from Item Card

  1. Close the Item Shipment Precondition Log page.
  2. On the Item Card, expand the Invoicing FastTab, and in the Item Category Code field, click the arrow and select Advanced.
  3. In the Item Categories page, click Actions > Item Shipment Precondition Log.
    The Item Shipment Precondition Log page opens with the two precondition entries.

: Item Shipment Precondition Log opened from Item Categories

  1. Close the open pages.

Note

Remove the preconditions from item R-10000-01 to prevent the precondition requirements from appearing in the remaining exercises in the manual.
