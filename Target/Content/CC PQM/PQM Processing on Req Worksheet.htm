PQM Processing on Requisition Worksheet

The Purchase Quote Manager can also be used for Requisitions Worksheet lines. Similar to the running the functions on the PQM page, purchase quotes can be created for each assigned vendor and then the quote that is most cost-effective and/or provides the best delivery time can be chosen for order placement.

This topic provides a demonstration of how to use the Purchase Quote Manager with the Requisition Worksheet.

Note

This demonstration uses the setup described in the topic "Setup" on page 1.

Demonstration – Place Quote from PQM in Requisition Worksheet

Scenario: Item D-7014, Hydraulic Ventile Type 70/B, has been set up with a Reorder Quantity of 100. When Alicia, the purchasing agent at CRONUS International, Ltd., runs the Calculate Plan function in the Req. Worksheet, a line is created for 100 units of item D-7014.

Three vendors supply this item to the company, each with different prices and lead times. Instead of creating separate quotes and checking the purchase prices, she creates a PQM for this item, creates the quotes and checks for the best price/delivery date.

Create PQM for Item D-7014

Follow these steps to create a PQM for item D-7014:

  1. On the navigation pane, click Purchase > Planning > Requisition Worksheets.
  2. Click Calculate Plan for the correct date range and filter on item D-7014.
  3. Select the line for item D-7014 and then click Navigate > Create in the PQM menu group.

: Creating PQM for item D-7014 on Req. Worksheet line

  1. Notice that the No. and Description fields are populated in the General FastTab and the PQM Lines FastTab contains the following information from the Req. Worksheet line:
    • Item number and description
    • Quantity
    • Vendor No.
    • Unit Cost
    • Order Date
    • Total Direct Cost

: PQM created from Req. Worksheet line

  1. Click Navigate > Assigned Vendors.
  2. Enter the following Lead Time Calculations:
    • Vendor 60000, Lead Time Calculation = 21D
    • Vendor 61000, Lead Time Calculation = 28D
    • Vendor 62000, Lead Time Calculation = 2W
  3. Click OK to close the Assigned Vendors page.
  4. Click Actions > Assigned Vendors > Copy from Header to Lines.
    Notice that the value of Assigned Vendors is 3 on the PQM Lines FastTab.

: PQM page with header and line information

Create Purchase Quotes for PQM Lines

Follow these steps to create the purchase quotes for the PQM lines:

  1. On the Actions tab, click Create Purch. Quotes.
  2. On the ribbon, click Show Matrix.
    Notice that the price is populated for each quote as 49.00, which is based on the purchase price of 10+ units of item D-7014 for vendor 60000 (source vendor number from requisition worksheet line).

: PQM Placing Matrix, quotes with pricing and delivery dates from vendor 60000

  1. Click Close to close the PQM Placing Matrix page and then click OK to return to the PQM page.
  1. In the PQM Lines FastTab, notice that the:
    • Status Created field = Yes
    • Status Verified, Status Placed and Status Not Placed fields = No

: Status fields updated on PQM Lines

  1. Click Home > Purchase Quotes Matrix.
  2. On the ribbon, click Show Matrix.
  3. Click the value for vendor 60000 to open the purchase quote.
  4. Revalidate the item number in the No. field and then click OK to return to the PQM Placing Matrix.
  5. Repeat steps 7-8 for vendors 61000 and 62000.
  6. Refresh the PQM Placing Matrix page.

    Notice that the prices are updated for each quote.

: PQM Placing Matrix, quotes with updated pricing and delivery dates

Change Quote Status

Depending on business processes, the status of each quote can be changed to Verified once the quotes are validated by the vendors. To do so, follow these steps:

  1. Click Close to close the PQM Placing Matrix page and then click OK to return to the PQM page.
  2. In the PQM Lines FastTab, click the value in the Assigned Vendors field to open the PQM Vendors page and then click Edit List.

: PQM Vendors with quote Status = Created

  1. For each quote, change the Status to Verified.
  2.  Click OK to close the PQM Vendors page.
  3. If necessary, refresh the PQM page and notice that the:
    • Status Verified field = Yes
    • Status Created, Status Placed and Status Not Placed fields = No

: Status fields updated on PQM Lines

Place Best Quote

Follow these steps to place the best quote:

  1. Based on the price and delivery date, the quote for vendor 62000 should be chosen.
  2. Open the PQM Placing Matrix page (Home > Purchase Quotes Matrix > Show Matrix).
  3. On the ribbon, click Placing.
  4. Click OK to select all lines for placement.
  5. In the PQM Placing page, allow the system to place by best price (which will be vendor 62000). Alternatively, enter 62000 in the Vendor No. field.
  6. In the Options FastTab, select the Update source data check box.

: Settings on the PQM Placing page

  1. Click OK.
  2. In the PQM Placing Matrix page, notice that the:
    • Assigned check box is selected
    • Values are cleared for vendors 60000 and 61000

: PQM Placing Matrix, quote assigned for vendor 62000

  1. Click Close to close the PQM Placing Matrix page and then click OK to return to the PQM page.
  2. In the PQM Lines FastTab, notice that the:
    • Status Created and Status Verified fields = No
    • Status Placed and Status Not Placed fields = Yes
    • Placing Vendor No. = 62000
    • Placing Unit Cost = 48.50 (direct unit cost from placed purchase quote)
    • Difference Unit Cost = -0.05 (Source Unit Cost - Placing Unit Cost)
    • Assigned check box is selected
    • Source Status = Updated (due to Update Source Data check box selected)

: Status and placing fields updated on PQM Lines

  1. Click the value in the Status Placed field.
    Notice that the quote for vendor 62000 appears.
  2. Go back to the PQM page.
  3. Click the value in the Status Not Placed field.
    Notice that the quotes for vendors 60000 and 61000 appear.

: Placed quote for vendor 62000 and Not Placed quotes for vendors 60000 and 61000

  1. Go back to the PQM page.
  2. If no further changes need to be made, click Finish on the ribbon to complete the processing for this PQM.

Review Requisition Worksheet Line

Follow these steps to review the changes to the Req. Worksheet line:

  1. Click OK to close the PQM page.
  2. Refresh the Req. Worksheet page, if necessary. Notice that the:
    • Vendor No. = 62000
    • Direct Unit Cost = 48.50

: Updated Req. Worksheet line for item D-7014

To access the PQM again at any time for this Req. Worksheet line, click Show PQM on the Navigate FastTab.

: PQM page reopened from Req. Worksheet line
