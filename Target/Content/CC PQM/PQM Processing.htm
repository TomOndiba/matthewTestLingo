PQM Processing

Once setup is complete on the PQM page, purchase quotes can be created for each assigned vendor and then the quote that is most cost-effective and/or provides the best delivery time can be chosen for order placement.

This topic provides a demonstration of how to use the Purchase Quote Manager.

Note

This demonstration uses the setup described in the topic "Setup" on page 1.

Demonstration – Place Quote from PQM

Scenario: Alicia, the purchasing agent at CRONUS International, Ltd. needs to order 100 units of item D-7014, Hydraulic Ventile Type 70/B. Three vendors supply this item to the company, each with different prices and lead times. Instead of creating separate quotes and checking the purchase prices, she creates a PQM for this item, creates the quotes and checks for the best price/delivery date.

Create PQM for Item D-7014

Follow these steps to create a PQM for item D-7014:

  1. On the navigation pane, click Purchase > cc|purchase quote manager > Purchase Quote Manager.
  2. Click New and then press ENTER to automatically populate the No. field.
  3. In the Description field, type “Quotes for item D-7014”.
  4. In the PQM Lines FastTab, enter Item in the Type field.
  5. In the No. field, enter D-7014.
  6. Refresh the page, notice that the value of Assigned Vendors is 3 on the General FastTab.
  7. Click Navigate > Assigned Vendors.
  8. Enter the following Lead Time Calculations:
    • Vendor 60000, Lead Time Calculation = 21D
    • Vendor 61000, Lead Time Calculation = 28D
    • Vendor 62000, Lead Time Calculation = 2W
  9. Click OK to close the Assigned Vendors page.
  10. Click Actions > Assigned Vendors > Copy from Header to Lines.
    Notice that the value of Assigned Vendors is 3 on the PQM Lines FastTab.

: PQM page with header and line information

  1. Scroll down the line and notice that the Direct Unit Cost is 52.00, which is the Last Direct Cost of the item.

Create Purchase Quotes for PQM Lines

Follow these steps to create the purchase quotes for the PQM lines:

  1. On the Actions tab, click Create Purch. Quotes.
  2. On the ribbon, click Show Matrix.
    Notice that the price is 0.00 for each quote; this is because no quantity was entered in the PQM Lines FastTab.

: PQM Placing Matrix, quotes with no pricing

  1. Click Close to close the PQM Placing Matrix page and then click OK to return to the PQM page.
  2. In the PQM Lines FastTab, notice that the:
    • Status Created field = Yes
    • Status Verified, Status Placed and Status Not Placed fields = No

: Status fields updated on PQM Lines

  1. Click Home > Purchase Quotes Matrix.
  2. On the ribbon, click Show Matrix.
  3. Click the value for vendor 60000 to open the purchase quote.
  4. In the Quantity field, enter 100 and then click OK to return to the PQM Placing Matrix.
  5. Repeat steps 4-5 for vendors 61000 and 62000.
  6. Refresh the PQM Placing Matrix page.
    Notice that the prices are updated for each quote.

: PQM Placing Matrix, quotes with pricing

Change Quote Status

Depending on business processes, the status of each quote can be changed to Verified once the quotes are validated by the vendors. To do so, follow these steps:

  1. Click Close to close the PQM Placing Matrix page and then click OK to return to the PQM page.
  2. In the PQM Lines FastTab, click the value in the Status Created field to open the PQM Vendors page.

: PQM Vendors with quote Status = Created

  1. For each quote, change the Status to Verified.
  2.  Click OK to close the PQM Vendors page.
  3. If necessary, refresh the PQM page and notice that the:
    • Status Verified field = Yes
    • Status Created, Status Placed and Status Not Placed fields = No

: Status fields updated on PQM Lines

Place Best Quote

Follow these steps to place the best quote:

  1. Based on the price and delivery date, the quote for vendor 62000 should be chosen.
  2. Open the PQM Placing Matrix page (Home > Purchase Quotes Matrix > Show Matrix).
  3. On the ribbon, click Placing.
  4. Click OK to select all lines for placement.
  5. In the PQM Placing page, allow the system to place by best price (which will be vendor 62000). Alternatively, enter 62000 in the Vendor No. field.

: Settings on the PQM Placing page

  1. Click OK.
  2. In the PQM Placing Matrix page, notice that the:
    • Assigned check box is selected
    • Values are cleared for vendors 60000 and 61000

: PQM Placing Matrix, quote assigned for vendor 62000

  1. Click Close to close the PQM Placing Matrix page and then click OK to return to the PQM page.
  2. In the PQM Lines FastTab, notice that the:
    • Status Created and Status Verified fields = No
    • Status Placed and Status Not Placed fields = Yes
    • Placing Vendor No. = 62000
    • Placing Unit Cost = 48.50 (direct unit cost from placed purchase quote)
    • Assigned check box is selected

: Status and placing fields updated on PQM Lines

  1. Click the value in the Status Placed field.
    Notice that the quote for vendor 62000 appears.
  2. Go back to the PQM page.
  3. Click the value in the Status Not Placed field.
    Notice that the quotes for vendors 60000 and 61000 appear.

: Placed quote for vendor 62000 and Not Placed quotes for vendors 60000 and 61000

  1. Go back to the PQM page.
  2. If no further changes need to be made, click Finish on the ribbon to complete the processing for this PQM.
