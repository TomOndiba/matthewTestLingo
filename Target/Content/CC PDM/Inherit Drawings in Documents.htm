Inherit Drawings in Documents

When creating a document, the drawing set of the previous document or the valid drawings of the item are always inherited. Changing the drawing set in a document will pass it on to the next document. It is only important that the drawing set has a status that allows it to be passed on to the new document.

The item is assigned the mechanical drawing PDM-1000 in revision A. This is the set of drawings with the internal number 100. This is the drawing set with the internal number 100. This set number is passed on to the sales quote when the quotation is created and from there to the sales order. In the sales order, the assembly drawing PDM-1000, revision B is assigned. The new combination of both drawings corresponds to drawing set number 120, which is now inherited in all subsequent documents.
