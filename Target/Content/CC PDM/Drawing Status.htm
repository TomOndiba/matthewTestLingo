Drawing Status

On the Drawing Status page, the statuses that a drawing can assume are set up. The drawing status is only checked if the Status Check field in the PDM Setup page has the option Show Error or Confirm Individually.

Note

If CAD / PDM or PLM systems are connected using the module cc|product data management – cad connector, the drawing status must correspond to that of the connected system.

Important

An item may only have ONE drawing per drawing type, which has a status code marked “Valid”. When replacing the revision status, always change the item first.

: Drawing Status Setup

The fields in the Drawing Status page are described in the following table.

Field

Description

Code

Code of a drawing status. The code must be unique and can have up to 20 characters.

Description

Descriptions of the drawing status, at maximum 50 characters.

Valid

Defines the status of a valid drawing. Only drawings with the status “Valid” can be assigned to an item.

Use in Item

Controls whether drawings can be assigned to an item or not. The prerequisite is that a drawing of the same type (e.g. Mechanical) has not already been assigned to the item.

Use in BOM

Controls whether drawings can be assigned to a production BOM or not. The prerequisite is that a drawing of the same type (e.g. Mechanical) has not already been assigned to the production BOM.

Use in Not Released Prod. Order

Controls whether drawings can or cannot be assigned to a production order that has not been released (status simulated, planned or firm planned). A prerequisite for an assignment is that a drawing of the same type (e.g. Mechanical) has not already been assigned to the production order.

In addition, a production order cannot be created if the underlying BOM contains a component whose drawing status prevents it from being used in a production order that has not been released.

Use in Released Prod. Order

Controls whether drawings can be assigned to a released production order or not. A prerequisite for an assignment is that a drawing of the same type (e.g. Mechanical) has not already been assigned to the production order.

In addition, a production order cannot be created if the underlying BOM contains a component whose drawing status prevents it from being used in a production order that has not been released.

Use in Sales Line

Defines whether a drawing can or cannot be used in a sales document (sales quote or sales order). Also, it is not possible to manually assign a drawing to a sales document if the status does not allow this.

Use in Purch. Line

Defines whether a drawing can or cannot be used in a purchasing document (purchase quotes or purchase order). Also, it is not possible to manually assign a drawing to a purchasing document if the status does not allow this.

Use in Service

Defines whether a drawing can or cannot be used in a service document or for service items / service BOMs. Nor can a drawing be assigned manually to a service BOM or service item / service BOM if the status does not allow it.

Allow Posting

Controls whether documents can be posted if they contain a drawing set that is not permitted for this purpose. This setting applies to the following documents:

  • Purchase Orders
  • Warehouse Receipts
  • Sales Orders
  • Warehouse Shipments
  • Production Orders
  • Service Orders
