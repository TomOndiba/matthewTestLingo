Engineering Change Management FastTab

The Engineering Change Management FastTab is used to set up change management. The fields on this FastTab are described in the following table.

Field

Description

Eng. Change Notice Nos.

Indicates the number series that will be used when creating a new engineering change notice (ECN) for the technical department.

Eng. Change Order Nos.

Indicates the number series that will be used for a new engineering change order (ECO) created by the technical department.
