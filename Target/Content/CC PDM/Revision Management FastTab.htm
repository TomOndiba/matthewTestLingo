Revision Management FastTab

The fields on the Revision Management FastTab are described in the following table.

Field

Description

Link New Revision to BOM

Automatically assigns new revisions to active production BOMs. To activate this field, the Status Code for Revision field must allow a usage in production BOMs.

Create New BOM Version

Determines whether a new production BOM version is to be created for each new revision, instead of updating Link New Revision to BOM the original production BOM. This field can be activated only if is also set.

Prod. BOM Version Nos.

Indicates the number series that will be used when creating a new production BOM version. This field must be populated if the Create new BOM Version field is activated.

BOM Status after Update

Specifies the production BOM (version) status after a revision has been updated.
