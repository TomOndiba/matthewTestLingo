BOM Changes

The changes made to the BOM are summarized in the Bom changes tab.

: Change Report in CADLink

The following table describes the fields for BOM changes in the CADLink window.

Field

Description

Part Number

Item / Drawing number of the material.

Description

Item description for item / drawing number.

CadLink Qt

Quantity in CADLink.

Cad Qty

Quantity copied from CAD.

Erp Qty

Quantity in ERP from the BOM line.

Notes

Information about what exactly will happen when importing into the ERP, e.g. that an item will be removed or added.
