Bills of Materials

In the CADLink Bills of Materials tab, the BOMs from CAD and Microsoft Dynamics NAV are compared.

: Bills of Materials View in CADLink with Color Display

The following table describes the fields for BOMs in the CADLink window.

Field

Description

Save

If selected, the item is copied to the ERP system, updated or removed from the BOM.

Part Number

Item number of the assembly or material.

Revision

Revision code with which a drawing is to be created.

Description

Description of the item.

Unit Qty

Quantity that is required as the base quantity in the BOM.

Base Uom

Base unit of measure based on quantity.

Category Code

Category assigned to the item.

ERP-Item

Item in ERP that was added manually but is not to be imported into ERP.

Note

In the Microsoft Dynamics NAV production parts lists, BOM components are marked as ERP items.

Drawing Link

Path that refers to the storage location of the drawing.

File

Storage path of the CAD file.

There is also the ERP Item check box in the Microsoft Dynamics NAV BOMs. If this check box is selected, the component is not considered when comparing with CADLink. In this way, parts lists can be added manually in Microsoft Dynamics NAV without the components being deleted again by CADLink.
