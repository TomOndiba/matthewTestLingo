Drawing FastTab

The fields on the Drawing FastTab are described in the following table.

Field

Description

Copy from Item to BOM

Specifies at first allocation of a production BOM to an item, whether the drawings will be copied from item to production BOM.

The options are:

  • Never
  • Question
  • Always

Status Check

Determines whether the drawing should be checked. The use in certain tables/documents or the posting can be restricted. A restricted drawing can result in either an error message or a confirmation message.

The options are:

  • Show Error
  • Confirm Individually
  • No Check

First Revision Code

Specifies the revision code to be used for first drawing versions. If no value is specified, counting will start at 000001.

If the value A is stored, the following revision level is assigned the value B.

Revision Mandatory

Specifies whether the revision code is populated automatically at drawing insertion.

Status Code for Revision

Specifies the status code with which new revisions will be created.
