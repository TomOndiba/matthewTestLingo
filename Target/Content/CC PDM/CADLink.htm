CADLink

Every company working with CAD systems is faced with the problem of transferring data developed in CAD to ERP. With cc|product data management – cad connector the customer can create, update or remove design data from the CAD in Microsoft Dynamics NAV.

Note

This topic provides an initial overview of the most important functions of CADLink. For more information, please contact the supplier QBuild.

Fig. 1 illustrates how the connection between CAD and cc|product data management works.

: How CADLink works

CADLink is to be understood as middleware between CAD system and Microsoft Dynamics NAV. Web services are used to read information from both systems and compare it in CADLink.

CADLink is opened from a CAD, PDM or PLM system and is installed there in the respective client.

Note

The following screen shots are taken from a specific CAD system. When using other CAD systems, the appearance may differ. However, the functionality of CADLink remains the same.

: CADLink with the CAD Viewer eDrawings as an Example

The following table describes the actions in the menu in the CADLink window.

Action

Description

Open File

Opens an Explorer for searching a CAD file.

Close File

Closes the open file and returns CADLink to its initial state.

Run CADLink

Calculates the materials of the drawing and opens the CADLink BOM management.

If a CAD system is connected, it is possible to use this function directly from the CAD system.

 

: eDrawings with Open Drawing

After the file has been opened in the CAD system, use the “Run CADLink” function to open the CADLink system.

: Opening CADLink

When opening CADLink, the Web services are initialized, and all materials are derived from the CAD model, determined, and summarized in a BOM.
