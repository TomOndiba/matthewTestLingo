Example – Formula Setup Time and Formula Run Time

Similar to the calculation in the Formula Quantity field, the Formula Setup Time and Formula Run Time fields can be dynamically calculated.

: Formulas for Setup and Run Time

In this example, the setup time is calculated using a dynamic function (refer to topic "Dynamic Functions" on page 1) while the run time uses a manually captured formula. In the Document BOM, the value in the Exp. Quantity field calculates the Setup and Run Times accordingly.

: Calculated Setup Time and Run Time in the Document BOM

The calculated Setup and Run Time values are transferred to the routing plan of production orders.

Note

These functions are only relevant if the Microsoft Dynamics NAV Manufacturing modules are being used.
