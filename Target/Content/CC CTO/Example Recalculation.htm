Example – Recalculation

The “Recalculation” function of the Dynamic Master BOM corresponds with the Dynamic Sub-Checklists (refer to topic “Example – Sub-Checklists” on page 1).

The Master BOM contains the Recalculate check box. If this check box is selected, the Master BOM line will be copied to the Document BOM as many times as was entered/calculated in the Exp. Quantity field. Each created Document BOM line will be set with Exp. Quantity = 1. The number of created Document BOM lines will be the quantity that was entered in the Input field on the Checklist line (NOSUBCL).

: Master BOM for a lower-level BOM item setup for recalculation on Document BOM lines

In a lower-level BOM item, a counting variable “i” is available. In the description fields and the formula fields, the syntax “#i#” can be used as a substitute symbol.

: Substitute symbols in a lower-level Master BOM

When the Master BOM is opened in the Document BOM line:

  • The substitute symbol is replaced with the contents of the counting variables.
  • The variables produced correspond with those from the dynamic sub-checklist.
  • The values assigned in the checklist are available in the Document BOM.

In this example, item D-9103 appears three times in the Document BOM by requesting the Master BOM.
