Construction Inquiry

The construction inquiry is almost identical to the construction order except that the components of the construction inquiry do not initiate demand. In addition, some menu options are omitted, such as “Create Production Order.”

: Construction Inquiry for the Electronic User Interface assembly

A construction inquiry can be generated only from a document BOM by clicking Navigate > Construction Inquiry on the ribbon (from the Document BOM Header page, click Line > Construction Inquiry on the Lines FastTab). The construction inquiry can be created either for the entire document BOM of a construction inquiry or only for the assembly unit, based on the selected document BOM line. If a construction inquiry for an assembly unit is generated, the Constr. Inquiry Exists field is set to Yes on the Document BOM line.

: Document BOM, for the Construction Inquiry created for the assembly unit

To open the construction inquiry, select the line and then click Navigate > Construction Inquiry.

After a construction inquiry is generated, it can be opened directly from the main menu and processed the same way as a “normal” construction order. When all changes are made in the Construction Inquiry, return the changed BOM to the document BOM by clicking Transfer Back to Doc. BOM on the ribbon. This stores the user ID and the time of the return in the Transferred back on and Transferred back by fields on the General FastTab of the Construction Inquiry.
