Description Catalog

The Description Catalog is a function used to standardized item descriptions. In Configure to Order Setup, it is defined whether with a description catalog should be used.

: Configure to Order Setup, Item Description Catalog activated

If the Item Description Catalog check box contains a check mark, then the arrow in the Description field on the Item Card appears and opens the Item Description Catalog page. Manually entered descriptions are verified against the Item Description Catalog table.

: Checking of item description in the catalog

If an invalid item description is entered, an error message appears. Click the arrow in the Description field to select a valid (standardized) item description.
