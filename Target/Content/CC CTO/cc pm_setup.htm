Setup

This section describes the setup necessary to use cc|process manufacturing functionality in cc|configure to order.

Configure to Order Setup

On the Manufacturing FastTab of the Configure to Order Setup page, the Certify New Prod. Structure check box determines what happens when a new Production Structure is created from a document or BOM:

  • If selected, the Production Structure will be certified and the new item will get a newly generated Routing and Production BOM. The Production Structure, Routing and Production BOM will be linked to the new item.
  • If not selected, a question appears asking if the new Production Structure should be opened. This allows for modifications to the recipes prior to the manual certification of the Production Structure, plus the ability to omit the creation of new Routing/Production BOM.

: Configure to Order Setup, Certify New Prod. Structure field

BOM Lines

In the BOM lines of cc|configure to order (Master BOM, Document BOM and Construction BOM), the following fields are used with the Production Structure in cc|process manufacturing:

  • Type Production Structure – Specifies what should happen with each line if it gets transferred to a Production Structure. The options are: Share of Recipe, Tool, Main Recipe, and Insert. These options are the same as the options in the Type field in the Production Structure lines, thus providing the links to the cc|process manufacturing functionality.

Note

When using Share of Recipe or Main Recipe, the Quantity per field must be used instead of the Quantity field to properly define the Share percentage and therefore, create the Item/Production Structure.

  • Type – The Production BOM option allows for the inclusion of a Production BOM in the BOM lines.
