cc|process manufacturing

In cc|process manufacturing, the Production Structure is used to manage recipes, routings, BOMs, test plans and comments in one page. A Production Structure can be set up for one item but used for other items as well.

When used with cc|configure to order, the cc|process manufacturing functionality provides users the ability to create items and production structures from a CTO configuration or based on a Document BOM or Construction BOM. Once created, Production Structures are managed using the functionalities of cc|process manufacturing.

cc|process manufacturing functionalities that are specific to cc|configure to order are described in this section. For detailed setup, configuration and usage information about cc|process manufacturing, refer to the current “cc|industry solution: process manufacturing” documentation.
