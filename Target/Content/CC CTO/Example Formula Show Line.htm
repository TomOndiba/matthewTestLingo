Example – Formula Show Line

In this example, one question and two answers are defined in a checklist. The answers expect Boolean type of values (Yes/True or No/False). The variables “VAR1” and “VAR2” are assigned to the answers. The Show Line check boxes are selected as per the default and the Automatic Recalculation check boxes are selected on the answer lines.

: Formula Show Line

When this checklist is opened on a document, and the user clicks the Selected check box on one of the answer lines, the system automatically clears the Show Line check box for the other line, thereby removing it from view on the checklist. In this example, VAR1 is selected and the VAR2 line is removed. 

: Result to Formula Show Line specification

Note

The “Formula Show Line” function is applicable for all checklist lines, including headings and questions.
