General FastTab

Master BOMs are created in the Master BOM page, accessed from the cc|configure to order menu by clicking Lists > Master BOMs.

: Master BOM

The information in the General FastTab defines the entire Master BOM. Each field is described in the following table.

Field

Description

Code

Unique identifier of the Master BOM that can contain up to 20 characters.

Description and Description 2

Description of the Master BOM; each allows a maximum of 50 characters.

Status

Indicates the status to the Master BOM. The options are:

  • New – The status is set automatically to New with the creation of a Master BOM. In this status, the Master BOM can be worked on.
  • Certified – With this status, a Master BOM can be used in BOMs in sales documents and in construction orders. With this status, a Master BOM cannot be edited. If you use a Master BOM that is not certified, an error message appears.
  • Under Development – This status indicates that a Master BOM is being worked on. After the changes are made, the status must be set to Certified.
  • Closed – This status indicates that the Master BOM has become outdated and can no longer be worked on; this corresponds to a blockage of the BOM.

Routing Type

Used the same as the Routing Type field is used in standard Microsoft Dynamics NAV routings. The options are Serial and Parallel.

Refer to topic “Example – Parallel Routings in CTO BOMs” on page 1 for more information.

Quantity

Identifies the quantity to which the component data refers.

Unit of Measure Code

Quantity unit of measure that leads the Master BOM.

Production View Link

Identifies a production view link (refer to topic “Example – Production View” on page 1).

Version Nos.

Reference to the number series for Master BOM versions.

Last Status Modified Date

Date of the last status change of the Master BOM.

Last Status Modified User ID

User ID of the last person who changed the status of the Master BOM.

Last Date Modified

Date of the last change of the Master BOM.

Last User ID Modified

User ID of the last person who last changed the Master BOM.

Act. Version No.

Number of the current Master BOM version.
