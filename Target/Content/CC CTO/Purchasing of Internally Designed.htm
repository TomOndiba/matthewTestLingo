Purchasing of Internally Designed Items

The BOM of components that are simultaneously purchased and designed internally can be entered as text and transferred to the Purchase Order, which provides the vendor with the BOM structure.

The Copy BOM to Text field is available on the lines of the Master BOM, Document BOM, Construction Order, Service BOM and Prod. Order Component (and archived documents). This field has the following options:

  • Production BOM – The current production BOM of the item will be copied to the purchase line text. This option is not dependent on the document type that created the demand (Document BOM, Construction Order Line or Production Order Component).
  • Doc./Constr. BOM – The BOM from the Document BOM / Construction Order will be copied to the purchase line text. The Document BOM / Construction Order Lines will be copied based on the following criteria:
    • If a Document BOM / Construction Order line created the demand and the component has a Sub BOM.
    • If a Production Order has been created from a Document BOM / Construction Order line and the Prod. Order Component created the demand.

Example – Line Text from Document BOM

A sales order is created for one unit of item D-9000. In the Document BOM, two components are added below item D-9006, where the Copy to BOM to Text field is set to Doc./Constr. BOM.

: Document BOM set to copy BOM to purchase line text

After the Document BOM is released, the “Calculate Plan” function is run in the Requisition Worksheet for item D-9006.

: Requisition Worksheet for item D-9006 from Document BOM

Note

The text copied to the Requisition Worksheet line from the document BOM/construction order/production BOM line can be opened by clicking Navigate > Line Texts.

The “Carry Out Action Message” function is run and then the Purchase Order is opened to view the line text. The two components added for item D-9006 in the Document BOM are listed in the line text (Lines FastTab > Line > Line Texts).

: Purchase Line Text copied from Document BOM

Note

The same process occurs when a Construction Order is created except the purchase line text will be copied from the Construction Order line.

Example – Line Text from Production BOM

In this example, a production BOM with the two components is assigned to item D-9006 and then a sales order is created for 1 unit of item D-9000.

: Production BOM assigned to item D-9006

On the Document BOM for the sales order, set the Copy BOM to Text field to Production BOM on the line for item D-9006.

: Document BOM set to copy BOM to purchase line text from Production BOM

After the Document BOM is released, the “Calculate Plan” function is run in the Requisition Worksheet for item D-9006. The “Carry Out Action Message” function is run and then the Purchase Order is opened to view the line text. The two components from the Production BOM are listed in the line text (Lines FastTab > Line > Line Texts).

: Purchase Line Text copied from Production BOM
