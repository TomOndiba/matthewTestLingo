Example – Update Unit Costs/Prices in CTO BOMs

In this example, changes to unit costs/prices are made in the Document BOM and the “Update Unit Cost & Price” function is run to view the changes.

: Preparing to use the “Update Unit Cost & Price” function on Document BOM

The following changes are made to the Document BOM lines in Fig. 1:

  • Work Center 100: Unit Cost (CC) = 1.50
  • Item D-1056: Unit Cost (CC) = 520.00
  • Item D-7014: Unit Cost (CC) = 50.25, Unit Price 123.00

: Changes made to the Unit Costs/Prices in the Document BOM

Notice that the Changed check box is selected as changes were made to the page.

If the Document BOM is closed and reopened, the changes to the lines will remain but the Changed check box will be cleared.

: Reopened Document BOM clears Changed check box but keeps changes to lines

If the “Update Unit Cost & Price” function is run, the changed unit costs/prices will be updated (reset) to the current costs from the respective cards after the confirmation message is accepted. The Changed check box will also be selected again.

Note

If the Special Unit Cost field is set for a Work Center, the unit cost is not replaced on these Work Center lines when the "Update Unit Cost & Price" function is run.

: Result of running the “Update Unit Cost & Price” function on Document BOM

As mentioned, the same function is available in the Construction Order/BOM.

: “Update Unit Cost & Price” function on Construction Order/BOM
