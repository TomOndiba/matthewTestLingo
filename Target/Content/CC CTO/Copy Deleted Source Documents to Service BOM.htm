Copy Deleted Source Documents to Service BOM

When using Document Type = Construction Order or Document BOM, but the selected documents only exist as archived versions (i.e., the documents have been archived but deleted), the system automatically copies the last archived version of the Construction Order/Document BOM to the Service BOM.

The deleted documents will not appear in the list when opened from the Document No. field in the Copy BOM request page, but the used Construction Order or Document BOM number will be automatically entered in the Document No. field. Refer to the documents used in the Detail FastTab on the Service BOM.

Example – Deleted Documents Copied from Archived Sales Order

In this example, a Sales Order was created with a Document BOM and a Construction Order. After the related production order was processed, the Construction Order was archived and then deleted.

Next, the Sales Order was archived and shipped (thus creating the service item), and then the Document BOM was deleted from the Sales Order.

: Document BOM deleted from sales order but exists in Archived sales order

In the Service Item Card, all BOMs are listed in the Detail FastTab.

: Service Item Card, Detail FastTab with all used BOMs

In the “Copy BOM” function on the Service BOM, select the Document BOM option in the Document Type field. The Document No. field is populated with the Document BOM displayed in the Document BOM No. field on the Service Item Card. However, the lookup in the Document No. field has no available lines for this Document BOM.

: Preparing to copy the archived (deleted) Document BOM to Service BOM

Close the lookup and then click OK. The last archived version of this Document BOM is copied to the Service BOM.

: Archived (deleted) Document BOM copied to Service BOM

Delete the Service BOM lines and run the same process for the Construction Order. The Document No. field is populated with the Construction Order displayed in the Construction Order No. field on the Service Item Card. However, the lookup in the Document No. field has no available lines for this Construction Order.

: Preparing to copy the archived (deleted) Construction Order to Service BOM

Close the lookup and then click OK. The last archived version of this Construction Order is copied to the Service BOM.

: Archived (deleted) Construction Order copied to Service BOM
