Example – Formula Valid

In this example, the Formula Valid field is filled in with the variable “VAR1” in the Master BOM page.

: Formula Valid setup on Master BOM

In the checklist template, the Input Type fields are set to Boolean and the VAR1 and VAR2 variables are set in the Variable for Dynamic BOM fields in the corresponding lines.

In the document checklist, the user clicks the Selected check box on the relevant line. In this example, the VAR1 variable line for the Male Touring Bicycle frame, variant 55.

: Formula Valid selection on document checklist

Now when the Master BOM is opened, only the selected line is transferred to the Document BOM, not both lines as were set up in the Master BOM.

: Formula Valid results in Document BOM
