Items and Production Structures from Configurations

Items and Production Structures can be created from Sales Quote/Order lines and Construction Order headers using the “Create Item & Prod. Structure” function.

With this function, a new item gets created as a copy of the existing item, which replaces that item in the Sales document line/Construction Order header. Concurrently, a new Production Structure gets created, and depending on the setup in the Configure to Order Setup page, can be automatically certified with a new Routing/Production BOM.

Standard Business Process

Typically, a Checklist Template and Master BOM have been set up for an item configuration (typically a dummy item). When this item is entered in a Sales Quote, the standard cc|configure to order processes are performed: the Checklist is completed, the Document BOM is created and calculations for pricing are made. Once the customer accepts the pricing, an order is created.

If this item configuration needs to be produced using cc|process manufacturing functionalities, the “Create Item & Prod. Structure” function is run, a new Item is created along with a new Production Structure, and the dummy item in the sales line is replaced with the new item. At this point, all further processing is managed through cc|process manufacturing, such as the definition of recipes within the Production Structure, the final creation of the Routing and Production BOM, and the production of the item.

Example – Create Item and Production Structure from Sales Order

In this example, a plastics processing company orders a configurable plastic plate. This item has a Checklist Template and Master BOM that include configurations for dimensions and colors. The items in the Master BOM are set up as Share of Recipe so the Quantity per field is used in the Document BOM.

: Checklist and Master BOM for configurable plastic item

The entire configuration will be produced with the cc|process manufacturing functionalities. The design department will handle the detailed specifications after the order is placed; therefore, the Certify Production Structure check box is not selected in the Configure to Order Setup page.

: Sales Order for configurable plastic item

In the Checklist, the measurements are entered and the color is selected.

: Checklist with measurements and selected color

In the Document BOM, the quantity per is calculated for the measurement and selected color.

: Document BOM with measurements and selected color

Close the Document BOM.

In the Sales Order, click Line > Create Item & Prod. Structure on the Lines FastTab.

: Preparing to create Item and Production Structure from sales line

A message appears asking to keep the values for the Share of Recipe. Click Yes to transfer the shares to the Production Structure.

: Message to keep Share of Recipe values for transfer to Production Structure

After the process runs, a confirmation message appears with the new Item and Production Structure numbers. Click Yes to open the Production Structure.

: Confirmation message from Item and Production Structure creation

The Production Structure has been created for the new item with the calculated shares of the components.

: Production Structure for new configured item

The Design department will now check the Production Structure or add any further specifications before certifying the Production Structure and creating the Routing and Production BOM.

Click OK to close the Production Structure.

In the Sales Order line, the dummy item has been replaced with the new configured item.

: Sales Order line with new item

Item and Production Structure from Construction Order

The same process is used for creating an Item and Production Structure from a Construction Order header, except a reservation to the Construction Header cannot exist (i.e., order must be created manually), the “Create Item & Prod. Structure” function is accessed from the Actions tab and the Item No. is changed in the General FastTab.

: Create Item & Prod. Structure function in the Construction Order

Item and Production Structure from Manual Configuration

It is also possible to create an Item and Production Structure for a manually created Document BOM or Construction BOM using the “Create Item & Prod. Structure” function. Refer to topic “Items and Production Structures for Multilevel Products” on page 1 for information about using the function in those pages.
