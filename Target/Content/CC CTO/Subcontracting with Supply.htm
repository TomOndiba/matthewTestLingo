Subcontracting with Supply of Material

In contrast to standard Microsoft Dynamics NAV, cc|configure to order offers the option of mapping subcontracting with supply of material. In this case, the items provided are predefined in the Master BOM. However, these settings can be changed during order handling.

: Master BOM with Supply (Enclosure) of Material

To map the supply of material accurately, at least two Location Codes should be defined:

  • One Location Code from which material is provided.
  • One Location Code to which the material to be provided is transferred. This is generally the storage location of the subcontractor.

If several subcontractors are being used, define a separate Location for each subcontractor.

The transfer can occur from either the:

  • Item Reclassification Journal
  • Transfer Order

The manner in which the transfers are mapped is defined in the Configure to Order Setup page (refer to topic “Manufacturing FastTab” on page 1).

The subcontracting step is performed by storing a vendor number in the Subcontractor No. field on the Work Center Card as in standard Microsoft Dynamics NAV. The subcontractor’s storage location code to where the material provided is to be transferred is also entered there.

The connection between the subcontracting step and the material provided is established within the master BOM via a Routing Link Code (refer to Fig. 1).

: Subcontractor Work Center Group

The system recognizes the subcontracting steps from the Subcontracting Worksheet and generates the provision of material either from inventory Reclassification Journals or Transfer Orders.

: Subcontracting Worksheet

On the Navigate tab, click Planned Supplies to see if and which components will be enclosed to the subcontractor.

: Component to be enclosed to the subcontractor

Purchase orders are generated from the subcontracting worksheet as in standard Microsoft Dynamics NAV. On the purchase order, click Navigate > Enclosure to open either the Item Reclassification Journal or the Transfer Order (or a list of transfer orders) and the item transfer can be posted.

When the receipt is posted for the subcontracting purchase order, ledger entries are automatically posted for the corresponding series of operations on the production orders, and if the Flushing Method on the Prod. Order Components is set to Backward, the transferred material is automatically posted as consumption as well. This behavior corresponds to standard Microsoft Dynamics NAV.
