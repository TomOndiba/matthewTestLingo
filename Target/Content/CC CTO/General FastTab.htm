General FastTab

The fields on the General FastTab are described in the following table.

Field

Description

Special Characteristic Search with Variables

If selected, the special characteristic searching feature includes variables (refer to topic “Search Characteristics” on page 1).

Item Description Catalog

If selected, the Item Description Catalog functionality is activated (refer to topic “Description Catalog” on page 1).

Select Checklist to Copy

If selected and more than one Checklist Template is linked to an item, the system will ask the user if he/she wants to copy all linked Checklists or only one of them when he/she inserts the Item No. e.g. into a Sales Order.

Include in Supply Planning

Determines if Document BOMs or Construction Orders are included in supply planning:

  • If selected, released Document BOMs and/or Construction Order Lines will be included in supply planning.
  • If not selected, requirements are only planned if production orders were produced.

KANBAN

Selections made in this section are used with the Kanban system (refer to topic “General Setup” on page 1).
