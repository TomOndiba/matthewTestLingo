Items and Production Structures for Multilevel Products

Items and Production Structures can be created for multilevel products in a Document/Construction BOM using the “Create Item & Production Structure” function as well.

The difference from the sales document/Construction Order header creation is that the new Item and Production Structure are created from a line in the Document/Construction BOM. The new Item and Production Structure are based on this line, not the entire BOM.

Example – Create Item and Production Structure from Document BOM

This example uses the same item as in the example in topic “Items and Production Structures from Configurations” on page 1. However, the customer has asked for a change to the pearl green pigment so additional chemicals are added in the Document BOM. This new configuration is created as a new item and needs to be properly defined in the Production Structure.

: Checklist with measurements and selected color

In the Document BOM, the quantity per is calculated for the measurement and selected color.

: Document BOM with measurements and selected color

The new chemicals are added below item P-30106, thus creating the Sub BOM. Additionally, the item description is changed to be generic.

: Document BOM with new Sub BOM added below item P-30106

With the line selected, click Actions > Create Item & Prod. Structure.

: Preparing to create item and production structure for special color in Document BOM

A message appears asking to keep the values for the Share of Recipe. Click Yes to transfer the shares to the Production Structure.

: Message to keep Share of Recipe values for transfer to Production Structure

Another message may appear about density changes. Click OK.

After the process runs, a confirmation message appears with the new Item and Production Structure numbers. Click Yes to open the Production Structure.

: Confirmation message from Item and Production Structure creation

The Production Structure has been created for the new item with the calculated shares of the components, and the new description.

: Production Structure for new configured item

The Design department will now check the Production Structure or add any further specifications before certifying the Production Structure and creating the Routing and Production BOM.

Click OK to close the Production Structure.

In the Document BOM line, the item has been replaced with the new configured item.

: Document BOM line with new item

When the “Create Item & Prod. Structure” function is run for the Sales Order line, the new item in the Document BOM will be used in the new Production Structure.

Item and Production Structure from Construction BOM

The same process is used for creating an Item and Production Structure from a Construction BOM line, as long as a reservation to the Construction Header does not exist.
