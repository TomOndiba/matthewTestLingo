Lines FastTab, Line Menu

The following menu items appear on the Line menu, accessed from the Lines FastTab.

  • Checklist: Opens the assigned document checklist.
  • Attributes: Here attributes and attribute variants are assigned to the item in the sales line. This assignment applies only for the sales line. If attributes have already been assigned to the item in the item master, these are transferred to the sales line but can be modified here.
  • Document BOM: Opens the document BOM. If no document BOM exists when this menu item is selected, the “Request BOM” function is run automatically to create one.
  • Split Order Line: Opens the Sales Line Splitting page where sales lines – which already have checklists and/or Document BOMs – can be split into separate lines for individual configuration (refer to topic “Sales Line Splitting” on page 1).
  • Create Prod. Order: Generates a production order based on the document BOM if it was generated from the configuration or created manually. If the production order is generated here, there are two options:
    • Create Prod. Order without Variants – This function creates a production order without creating variant codes, production BOMs, or routings.
    • Create Prod. Order with Variants – The system generates a variant for all production items with the Part Type field set to Production and generates a manufacturing schedule and a manufacturing BOM. The automatically generated production BOMs and routings are linked to the item variant.

Whether the system creates production orders with or without variant codes by default is defined on the Configure to Order Setup page (refer to topic “Manufacturing FastTab” on page 1).

When the production order is generated for the Create Prod. Order with Variants option, the following steps are performed by the system:

  1. Generation of one variant per production item with the Part Type field set to Production.
  1. Creation of a manufacturing BOM using the default number series (refer to topic “Manufacturing FastTab” on page 1).
  1. Creation of a manufacturing routing using the default number series (refer to topic “Manufacturing FastTab” on page 1).
  1. Assignment of the manufacturing BOM and the routing on the Item Variant table.
  1. Creation of the production order based on the BOM entered and the routing.
  • PDB Interface: Enables the document BOM to be exported to the PDM system “ProfiDB”, or a document BOM created there can be imported.
  • Characteristic Search Tree: Opens the Characteristic Search Tree page where an item can be searched using characteristics (refer to topic “Search Characteristics” on page 1).
  • Document Schedule: Either opens a generated Document Schedule or the user is asked to create one (refer to topic “Operation Scheduling” on page 1).
  • Construction Order: Either opens the construction order generated or asks the user to create one. The Document BOM will be copied into the Construction Order.
  • Create Construction Orders: Creates construction orders for each selected line on a sales order.
  • Order Network: Refer to topic “Order Network” on page 1.
  • Line Texts: Opens the Edit Text page, where line-related text can be stored. Refer to topic “cc|text module” on page 1 for more information.
  • Calculation Doc. BOM: Opens the Calculation List page to create a new calculation or to view an existing calculation for the Document BOM (refer to topic “Document BOM Calculations” on page 1).
