Actions Tab

The functions on the Actions tab are described in the following table.

Function

Description

Change Status

Changes the status of the change order from New to Read. If Status = Read, the next status change opens a dialog to select Accept Changes or Reject Changes.

This function can be used to change from Accepted or Rejected back to Read to restart process.

Order Text

Opens the Change Order Text page (filtered by Text Type = Order Text), where additional information about the change order can be entered.

Existing text is indicated in the Order Text field on the General FastTab.

Answer Text

Opens the Change Order Text page (filtered by Text Type = Answer Text), where a response about the change order request can be entered.

Existing text is indicated in the Answer Text field on the General FastTab.
