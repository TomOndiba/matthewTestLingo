Navigate Tab

The following menu items appear on the Navigate tab of the Document BOM ribbon.

  • Move Left: Moves the marked position up one level in the BOM structure.
  • Move Right: Moves the marked position down one level in the BOM structure.
  • Open Next Level: In a multi-level BOM, opens a new page that displays entries of the next level only.
  • Attributes: Opens the attributes that were assigned to the document BOM line. Attributes can also be assigned manually here.
  • Characteristic Search Tree: Refer to topic “Search Characteristics” on page 1.
  • Calculation: Opens the Calculation List page to create a new calculation or to view an existing calculation (refer to topic “cc|calculation” on page 1 for more information).
  • Item Tracking Lines: Here item tracking can be assigned to components that are subject to management in batches or by serial numbers. This is standard Microsoft Dynamics NAV item tracking functionality.
  • Construction Inquiry: Opens the assigned construction inquiry (refer to topic “Construction Inquiry” on page 1).
  • Create Separate Prod. Order: Creates a separate production order for the document BOM line.
  • Comments: Comments can be informatively entered. The meaning and handling correspond to standard Microsoft Dynamics NAV.
  • Dimensions: Displays the dimensions assigned to the Document BOM line.
  • Tools: Tools information can be informatively entered. The meaning and handling correspond to standard Microsoft Dynamics NAV.
  • Personnel: Personnel information can be informatively entered. The meaning and handling correspond to standard Microsoft Dynamics NAV.
  • Quality Measures: Quality measures can be informatively entered. The meaning and handling correspond to standard Microsoft Dynamics NAV.
  • Line Texts: Opens the Edit Text page where line-related text can be stored.
  • Assembly Unit Export PDB: Enables a module to be exported to the PDM system “ProfiDB”. Further information on the ProfiDB interface is provided in the separate interface description document.
