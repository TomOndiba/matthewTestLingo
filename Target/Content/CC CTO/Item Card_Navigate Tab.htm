Navigate Tab

The following sections explain the menu items that have been added to the Master Data group on the Navigate tab.

Text Module

Opens the Text Module List page, where additional item text can be assigned to an item. Refer to topic “cc|text module” on page 1 for more information.

Master BOM

Click Master BOM > Master BOM to assign existing Master BOMs to an item.

: Item - Master BOM

The fields on the Item - Master BOM page are described in the following table.

Field

Description

Item

Identifies the item associated with the Master BOM.

Type

Indicates the type assigned to the Master BOM. This selection limits the possible Master BOM allocations.

Master BOM

Identifies the Master BOM associated with this item.

Description

Defaults from the Master BOM but can be modified.

Refer to topic “Master BOM Template” on page 1 for more information.

Master BOM Where Used

Click Master BOM > Master BOM Where Used to determine where the item is used as a component item or in a production BOM within the product structures.

: Master BOM Where-Used

The single or multilevel features can be used for maintenance or troubleshooting. The multilevel feature displays all effected BOMs.

Attributes

Opens the Attribute Values page, where attributes and the relevant values are assigned to an item.

: Item Attributes

In this example, attributes are assigned to item D-7017. Through development each attribute is provided a value. This item attribute is available in all documents relating to orders and can also be changed through the dynamic configuration.

Checklist

Opens the Item Checklist Relations page, where Checklist instructions can be assigned to an item.

: Item Checklist Relations

The fields on the Item Checklist Relations page are described in the following table.

Field

Description

Checklist Type

Indicates the type assigned to the Checklist. This selection limits the possible checklist allocations.

Starting Date

Date the item/checklist relationship becomes valid.

Checklist Code

Identifies the checklist associated with this item.

Purchase

Indicates if the checklist is valid for purchase documents.

Sales

Indicates if the checklist is valid for sales documents.

Construction

Indicates if the checklist is valid for construction orders.

Refer to topic “Checklists” on page 1 for more information.

Search Characteristics

Opens the Item Search Characteristics page, where the attributes for a search of the special characteristic list can be assigned to the item. Refer to topic “Search Characteristics” on page 1 for more information.

Profi DB

For more information about integration with the PDM system, refer to your COSMO CONSULT consultant.

KANBAN

Refer to topic “KANBAN System” on page 1 for more information.

Item Drawing

Opens the Item Drawing page, where designs and other similar documents are linked to the item.

: Design Administration
