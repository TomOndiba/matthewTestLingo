Actions Tab

The following menu items appear on the Actions tab of the Sales Quote ribbon.

  • Header Texts: Preceding text defined in the cc|text module for the quote can be opened and modified on the quote line. Refer to topic “cc|text module” on page 1 for more information.
  • Footer Texts: Subsequent text defined in the cc|text module for the quote can be opened and modified on the quote line. Refer to topic “cc|text module” on page 1 for more information.
