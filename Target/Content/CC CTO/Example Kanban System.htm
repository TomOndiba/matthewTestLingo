Example – Kanban System

In the Configure to Order Setup page, define the Kanban settings as illustrated in Fig. 1.

Open the Item Card for item 1160, clear the Reordering Policy field on the Planning FastTab and then open the KANBAN Setup page by clicking Navigate > KANBAN. Fill in the fields as shown in Fig. 1. For the Kanban Unit of Measure, define a BOX unit of measure with a quantity per unit of measure of 10.

Register three Kanban Cards by clicking Register Card on the ribbon three times (or press F7). The value in the Collected Quantity field increases with each registration.

: Three registered cards

Register the fourth and final Kanban Card; the following message appears.

: Requisition Line creation message

Click Yes and then click OK to close the message that four Requisition Lines were made.

The Collected Quantity field is set to 0 and the Supply Exists field is set to 4.

: Requisition Lines created from the Kanban system

Click Navigate > Supply to open the Requisition Lines page and then click Navigate > Show Worksheet to open the Requisition Worksheet. In the Name field, select the KANBAN journal batch.

The Requisition Worksheet contains the four Kanban lines, each with a quantity of 10 PCS (from the Base Quantity per Kanban Unit field) and the Created by KANBAN check box selected to indicate these lines were created by Kanban.

: Requisition Lines created from the Kanban system

If lines are deleted, the Collected Quantity field will be updated on the KANBAN Setup page. The user can now create and then post the Purchase Order.

From the KANBAN Setup page, the user can print new Kanban Cards and insert them in the delivered boxes. The Kanban cycle starts over from the beginning.
