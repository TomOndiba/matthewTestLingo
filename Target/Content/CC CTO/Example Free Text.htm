Example – Free Text

The Free Text field determines whether the contents of the Input fields appear in any description field in the Document BOM. In this way, for example, manually entered descriptions from the checklist can be transferred to the Document BOM.

To use free texts, the Free Text check box must be selected, the Input Type field must be set to Text, and the Variable for Dynamic BOM field must be filled in.

: Free Text in the Checklist Template Setup

The variable defined here can include the syntax for the substitute symbols (refer to Fig. 2). This setup can also be done in the Item Card in one of the description fields.

: Free Text in the Master BOM Setup

In the document checklist, enter the relevant text in the Input field.

: Free Text defined in the document checklist

When requesting the Master BOM, the contents of the Input field on the checklist (in this example, the text “The sky is blue”) is transferred to the variable “TEXT” and registered in the Description field on the Document BOM in place of the substitute symbol “#TEXT#”.
