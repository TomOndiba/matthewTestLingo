Example – Production View

The structure of a BOM in a construction order can be represented in two different views. The default is the construction view, which displays the BOM structure from the view of the design engineer. The other view displays the BOM structure from the view of the production.

For example, the BOM of an elevator during construction could be divided into mechanical and electronic components. From the production view, electronic components might be needed for the production of the mechanical components, although a majority of electronics is built in later.

: Construction Order in construction view

To switch between the construction and production view, two fields in the BOM tables are used: Production View Link and Belongs to Prod. View Link. In these fields, the Production View links are entered. The production view link provides a reference to an assembly unit, which can be assigned to additional components in the production view. Components or different assembly units, which are to be shifted then in such an assembly unit, must have the same assembly code assigned in the Belongs to Prod. View Link field.

It is possible to switch between the views by clicking Actions > Switch Const.-/Prod. View. The Production View check box on the General FastTab indicates whether the production view is in use.

: Construction Order in production view

The Production View Link and Belongs to Prod. View Link fields can also be set up in the master BOM and in the document list.
