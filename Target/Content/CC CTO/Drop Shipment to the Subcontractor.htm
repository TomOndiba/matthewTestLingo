Drop Shipment to the Subcontractor

An enclosure drop shipment to a subcontractor can be assigned in the Master BOM, Document BOM, or Construction Order Line and processed in the Subcontracting Prod. Order page as a Purchase Order. The vendor location address is copied to the Ship-to fields and the Vendor Location Code is copied to the Location Code field on the line.

Example – Process a Drop Shipment to a Subcontractor

In this example, a sales order and construction order have been created for 1 unit of item D-1000. The enclosed material will be ordered from a vendor other than the subcontractor and then the needed component will be sent directly to the subcontractor. Therefore, the Construction Order Lines have been set up with a work center and three enclosure components; one is set as a drop shipment.

: Construction BOM with Enclosure Drop Shipment line

To set the Enclosure Drop Shipment check box, the Enclosure check box must be selected. If the Reordering Policy for the item is not blank, a warning message appears.

Note

The drop shipment item must have a Vendor No. assigned on the Item Card.

After the Construction Order is released and the Production Order is created, use the Subcontracting Prod. Order page (refer to topic “Subcontracting Production Order” on page 1) to process the drop shipment to the subcontractor, along with the other subcontractor processes.

: Subcontracting Prod. Order with Enclosure Drop Shipment

After running the “Create Subcontracting Worksheet” and “Create Purchase Order” functions, the Subcontracting Prod. Order line is updated with a Purchase Line, Transfer Order and Encl. Drop Shipment order.

: Results of running the Create Subcontracting Worksheet and Create Purchase Order functions

First, post the Transfer Order.

Then open the Encl. Drop Shipment purchase order (click the field on the line or click Open Encl. Drop Shipment from the Line menu on the Subcontracting Routing Lines FastTab).

: Enclosure Drop Shipment Purchase Order

The Location Code field is set to the Vendor Location Code (assigned to the Work Center Card) and the Ship-to address fields are populated with the vendor location.

Post the Enclosure Drop Shipment purchase order and reopen the Subcontracting Purch. Order page.

: Results of posting the Transfer Order and Enclosure Drop Shipment purchase order

The Encl. Drop Shipment field displays the purchase receipt number, indicating that the purchase order was posted. Also note that the Posted Transfer Order field contains the number of the posted transfer order (done prior to posting the enclosure drop shipment order).

When the subcontractor delivers the completed item, the main subcontracting purchase order can be posted (from the Purchase Line field).

Note

All related subcontracting purchase orders and transfer orders can be opened from the main Purchase Order page by clicking Navigate > Enclosure.
