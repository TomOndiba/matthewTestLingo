Checklist Document, Calculation FastTab

The fields on the Calculation FastTab are described in the following table.

Field

Description

Calculation Price Base

Defaults from the sales price of the sales line.

Calculation Weight Base

Defaults from the Net Weight field on the Item Card.

Discount %

General discount rate that is deducted from the Calculation Price Base.

If the Discount Possible check box is selected on a checklist line, then the registered value will be recalculated from the Add. Charge/Price Reduction field and deducted from the Calculation Price.

Calculation Price

System calculated sales price according to the following formula:

Calculation Price = “Calculation Price Base” - Discount %
-/+ (Add. Charge/Price Reduction - Line Discount)

Calculation Weight

System calculated weight according to the following formula:

Calculation Weight = “Calculation Weight Base”
-/+ Add. Weight/Weight Reduction

In LCY

Determines whether the calculation price is to be converted into local currency.

Base Changed

Indicates whether the sales price of the sales line was changed since the last price calculation. If it has, then a new calculation is necessary.
