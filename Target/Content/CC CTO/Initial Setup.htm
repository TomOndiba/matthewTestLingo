Initial Setup

The Configure to Order Setup page has two translation fields on the Numbering FastTab.

: Translation fields on Configure to Order Setup page

Each field is described in the following table.

Field

Description

New Translation

Number series for new translations.

Alternative Language

Language that will be used if a selected language does not offer translated text; works with the Use Alternative Language field on the Translation Header page.

If an alternative language is not specified (and/or the Use Alternative Language check box is not selected on the Translation Header page) and a translation cannot be found for the selected location, a “No translations” message will appear in the Description (T) field on checklist lines.

If the alternative language is specified and used on translations, the system will look for a translation in this language instead of displaying the default message.

Additionally, user-specific language can be entered on the User Personalization page as the default translation for the user.
