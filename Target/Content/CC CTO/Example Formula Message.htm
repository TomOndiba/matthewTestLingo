Example – Formula Message

A message can be generated from a checklist calculation if the Formula Message field contains a formula and the Message Exists field is set to Yes.

: Formula Message Setup

In this example, a message is generated if the contents of the variable “SQM” (square meter) are larger and/or smaller than 6. The “SQM” variable is populated through the formula function. The message text can be entered by clicking the value in the Message Exists field, or from the Lines FastTab by clicking Checklist > Message.

In this example, the “SQM” variable is used as a substitute symbol with the syntax “#SQM#”. Now when the “Recalculate” function is run, the message appears and the contents of the variables replace the substitute syntax.
