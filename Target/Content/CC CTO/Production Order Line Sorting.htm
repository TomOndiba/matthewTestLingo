Production Order Line Sorting

In standard Microsoft Dynamics NAV, Production Order lines are always sorted by level. Most of the time this is not the same sorting that is shown in Document BOMs/Construction Orders.

As illustrated in Fig. 1, Assembly 3 and Assembly 4 are part of Main assembly 1, and Assembly 5 is part of Main assembly 2 in the Construction BOM.

: Construction Order highlighting the relation of the assembly lines (level 2) to main assembly lines (level 1)

When a Production Order is opened, the lines are sorted by level. As illustrated in Fig. 2, Assembly 3 – 5 all appear directly behind the Main assembly 2.

: Production Order with lines sorted by level (standard Microsoft Dynamics NAV)

To sort Production Order Lines to match the structure of the Document BOM/Construction Order, use the “Change Sorting” function (Lines FastTab > Line) in the Production Order lines.

As illustrated in Fig. 3, Assembly 3 and Assembly 4 are now shown directly behind Main assembly 1, and Assembly 5 is shown directly behind Main assembly 2 in the Production Order lines.

: Production Order with lines sorted with Change Sorting function
