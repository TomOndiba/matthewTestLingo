Example – Create Purchase Documents for Components

In this example, different purchase documents are created for items D-7014 (Requisition), D-7016 (Quote), D-7017 (Quote) and D-7019 (Order) using the “Purch. G/L Acc./Item” function.

The check for open Document BOM lines for creation of Purchase Quotes is reviewed, as well as the release of an entire Document BOM if open Purchase Quotes exist. Additionally, a production order and construction order are created to explain how reservations are maintained if the “Purch. G/L Acc./Item” function is run on other items in the Document BOM after a production order or construction order has been made.

: Document BOM lines to be used with the “Purch. G/L Acc./Item” function

Create Requisition Worksheet for Component

Select the line for item D-7014 and then click Actions > Purch. G/L Acc/Item on the ribbon.

: Purchase document creation dialog

With the Requisition Line option selected, click OK and then select the appropriate requisition worksheet from the Req. Wksh. Names page.

A confirmation message is generated and on the line for item D-7014:

  • Purchase Exists = Yes
  • Purchase Status = Purchase Created

: Document BOM line updated after Requisition line created

The “Purch. G/L Acc./Item” function is run again on the line to open the Requisition Lines page. Notice that a reservation has been made for this item.

: Requisition Line created from Document BOM

Close the Requisition Lines page and go back to the Document BOM.

Create Purchase Quote for Component

Select the line for item D-7016 and then click Purch. G/L Acc./Item. In the request page, select Purchase Quote and then click OK.

A confirmation message is generated and on the line for item D-7016:

  • Purchase Exists = Yes
  • Purchase Status = Purchase Created

Also notice that the Status = Open and therefore the purchase quote could be created for the Document BOM line.

: Document BOM line updated after Purchase Quote created for open line

Confirm Check for Open Document BOM Lines

On the line for item D-7017, change the Status to Released. Click Purch. G/L Acc./Item and notice that the Purchase Quote option is not available for selection in the purchase document creation dialog.

: Purchase Quote option not available for released Document BOM lines

For an additional check of the release function with purchase quotes, re-open the line for item D-7017 and then click Purch. G/L Acc./Item. In the purchase document creation dialog, select Purchase Quote.

Go back to the Document BOM. If the Status is changed to Released, an error message appears stating that a purchase quote exists for the line and it must be made into an order or deleted in order to release it.

: Error generated when attempting to release Document BOM line with a related Purchase Quote

Discard the release function; re-open the Purchase Quote and make it into a Purchase Order. Notice that a reservation has been made for this item to the Document BOM line.

: Purchase Order created from Purchase Quote with reservation created to Document BOM line

It is now possible to release the Document BOM line with the related Purchase Order.

: Document BOM line released after creating Purchase Order from related Purchase Quote

Select the line for item D-7019 and then click Purch. G/L Acc./Item. In the request page, select Purchase Order and then click OK.

A confirmation message is generated and on the line for item D-7019:

  • Purchase Exists = Yes
  • Purchase Status = Purchase Created

: Document BOM line updated after Purchase Order created

Now a requisition line, purchase quote, purchase quote to a purchase order, and a purchase order have been created. If the “Release” function is run for the entire Document BOM, an error will appear because of the open Purchase Quote.

: Error generated when attempting to release entire Document BOM with an open Purchase Quote

Reservations on Production/Construction Order

Change the purchase quote for item D-7016 into an order and then create a production order from the Sales Order.

Re-open the Document BOM, select the line for item D-7100 and then click Purch. G/L Acc./Item. A message appears stating that a production order already exists for this Document BOM line and confirms if the process should continue.

: Message generated when “Purch. G/L Acc./Item” function is run on Document BOM with production order

If the process continues, then everything works the same, only the reservation will be made directly between requisition/purchase order line and the production order component.

If a Construction Order was created instead of a production order, and an item should be purchased, a message appears stating that the construction order already exists.

: Message generated when “Purch. G/L Acc./Item” function is run on Document BOM with construction order

Cancel the process of creating the purchase document. Any further purchase processing for the item should be handled in the construction order.
