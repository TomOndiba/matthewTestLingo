Search Characteristics

The Search Characteristics is used for the quick selection of items using characteristics, which are set up the same as Attributes but are stored on the Item Card for each item as item search characteristics.

: Item Search Characteristics

The hierarchical structure of the various search characteristics is defined in the Characteristic Search Tree setup.

: Characteristic Search Tree Setup

Once set up, the special characteristics can be used for searching in various places within cc|configure to order. This is possible, for example, on the Sales Order by clicking Line > Characteristic Search Tree on the Lines FastTab or on the Construction BOM Line by clicking Navigate > Characteristic Search Tree.

: Characteristics Search Tree without Variables

The search page displays the highest level of the search characteristics. If sub-levels exist for a characteristic, the arrow icon ( ) will be available. Click the icon to open the next level of the search characteristics. The items assigned to the search characteristic are displayed in the Lines FastTab. To select an item, select the line in the Lines FastTab and then click OK. The item is added to the line and will overwrite an existing item if the line is not empty.

Note

The special characteristic search works only with the item master data. A search using an order-dependent characteristic is not possible.
