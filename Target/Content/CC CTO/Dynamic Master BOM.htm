Dynamic Master BOM

The Dynamic Master BOM is the second element of the configuration feature of cc|configure to order. The Master BOMs are opened “dynamically” if they are used for the automatic generation of BOMs relating to orders (Document BOM). In addition, within the BOM lines, formulas are entered. Usually checklists and Dynamic Master BOMs are used together for a configuration. The information entered in the checklist is stored as variables, which are then used for formula evaluation within the Dynamic Master BOM (refer to topic “Dynamic Checklist” on page 1).

: Master BOM

In addition to the fields previously described, the fields in the following table are specific for dynamic calculations within the Master BOM.

Field

Description

Formula Valid

In this field, enter a formula for use with calculation. Use the arrow to open the Formula Editor, which can assist with formula creation. The result of the formula evaluation (TRUE or FALSE) determines whether the line – when opening the Master BOM – is transferred to the Document BOM or not.

Formula Quantity

An alternative to the Quantity field, enter a formula for the calculation of the expected quantity. Use the arrow to open the Formula Editor. For work centers and machine centers, this field is used for the calculation of the time requirement according to the following formula:

Time requirement = Tr + Te /lot size * quantity

Note

If “quantity” is registered as a value in the field, it is overwritten by the result of the formula evaluation by opening the BOM.

Formula Quantity Per

An alternative to the Quantity per field; enter a formula for the calculation of the component quantity. Use the arrow to open the Formula Editor.

Recalculate

If selected on a BOM line, then this line is often transferred to the Document BOM, as the Exp. Quantity is calculated or registered. The variables used in a support BOM are supplemented before calculation of the formula around the contents of the counting variable “i”. In addition, the syntax #i# must be used (refer to topic “Example – Sub-Checklists” on page 1).

Formula Setup Time

For a work center or machine center, the necessary setup time can be calculated through a formula. Use the arrow to open the Formula Editor.

Note

If a value is registered in the Setup Time field, it is overwritten by the result of the formula evaluation by opening the BOM.

Formula Run Time

For machine centers or work centers, the necessary run time can be calculated through a formula. Use the arrow to open the Formula Editor.

Note

If a value is registered in the Run Time field, it is overwritten by the result of the formula evaluation by opening the BOM.
