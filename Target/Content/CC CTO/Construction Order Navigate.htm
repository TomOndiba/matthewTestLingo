Construction Order, Navigate Tab

The following menu items appear within the Construction and Print groups on the Navigate tab of the Construction Order ribbon.

  • Construction Line: Opens the editable construction BOM lines.
  • Checklist: Displays the associated document checklist.
  • Prod. Orders: Displays a list of all production orders that have already been generated for individual lines of the construction order.
  • Structure Representation: Displays the construction order in a tree-structure where the user can expand or collapse the different levels.
  • Header Texts: Opens the Edit Text page to view existing text (copied from the sales line to the construction order header when a construction order is created from the sales order) or to add text. Refer to topic “Line Text Transfer from Sales Line” on page 1 for more information.
  • Calculation: Opens the Calculation List page to create a new calculation or to view an existing calculation (refer to topic “cc|calculation” on page 1 for more information).
  • Attributes: Opens the attributes that were assigned to the construction order header. Attributes can also be assigned manually here.
  • Print BOM: Generates a printout of the construction order (header and BOM lines).
