Example – Create Purchase Order for G/L Account from Construction Order

A Sales Order and Construction Order are created for 1 unit of item D-1000. On the last line of the Construction BOM, a line is inserted for G/L Account = ACCOUNT, Exp. Quantity = 1 and Unit Cost (CC) = 100.

: Construction BOM with G/L Account line

Click Actions > Purch. G/L Acc/Item on the ribbon.

: Purchase G/L Account function request page

In the request page, select Purchase Order and then click OK. A confirmation message is generated and on the G/L Account line:

  • Purchase Exists = selected
  • Purchase Status = Purchase Created

: Document BOM with G/L Account purchase order created indicated

If the “Purch. G/L Acc/Item” function is run again, the following message is generated.

: Message to open purchase order

Click Yes to open the Purchase Lines page and then open the Purchase Order. The Purchase Order is created for the G/L Account for the Unit Cost entered on the Construction BOM line.

Note

If the value in the Direct Unit Cost Excl. VAT field is changed prior to posting the Purchase Order, the new amount will transfer to the Construction BOM line.

In this instance, change the value in the Direct Unit Cost Excl. VAT field to 200.

: Purchase Order created from the Purchase G/L Account function

Post the Purchase Order, go back to the Construction BOM and notice on the G/L Account line:

  • Unit Cost (CC) = 200.00 (changed value in purchase order)
  • Purchase Exists = not selected
  • Purchase Status = Purchase Posted

: Construction BOM line updated with Purchase Order posting
