Actions Tab

The Actions tab includes items that are relevant to cc|configure to order. Some of these menu items are available on the Home tab.

Set Standard Setup

This function automatically assigns standard setup in the system. Running this function will overwrite any other setup that has been completed in the Configure to Order Setup page.

Create Profile

This function creates the “CTO DESIGN ENGINEER” profile that is used in the RoleTailored Client.

CTO Overview

This function displays a cc|configure to order software structure graphic (refer to Fig. 1).

Note

The CTO Overview graphic can only be previewed; the print options do not work.
