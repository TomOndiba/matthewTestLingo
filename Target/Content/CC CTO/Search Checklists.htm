Search Checklists

The “Search Checklists” function provides the ability to search for checklists that were previously used. Search Checklists are templates and are located on the cc|configure to order menu by clicking Archive > Search Checklists. 

: Search Checklists

The fields on the Lines FastTab are described in the following table.

Field

Description

Search Field

In this field, searches can be entered for a Variable or a Description (based on the Description field in the Checklist line).

For example, the third line in Fig. 1 will set a filter on all checklist lines where the string @*kw* exists in the Description field. Uppercase or Lowercase will be ignored because the “at” symbol (@) is used. The first two lines will filter using variables: the APRON and GE Boolean should be set to Yes.

Variable/Description

This field is based on the selection in the Search Field:

  • If Variable is selected, use the arrow to select a value from the Variable/Attribute Types page.
  • If Description is selected, enter a filter for the Description field.

Search Type

Searching can be performed for alphanumerical and/or numerical values:

  • If searching for attributes or similar, use Numerical.
  • If searching for words, for example colors (green, blue), use Alphanumerical.

If numerical is used, the Input (numerical) field on the Checklist line will be used to filter the correct line. The Input (numerical) field contains the numeric value of the Input field. To initiate this field, the “Update Numerical Field” function must be run by clicking Actions > Update Numerical Field on the ribbon.

If the Input Type field in the Checklist line is set to Boolean, FALSE will be 0 and TRUE will be 1.

Filter Input

In this field, enter the filter that will be used on the Input field (or the Input (numerical) field if the Search Type field is set to Numerical).

To start a search, click New Search on the ribbon, or press F9. The Search Checklist Filters page opens with a copy of the lines from the Search Checklists page. The fields have the same meaning as in the Search Checklists page.

: Search Checklist Filters

Search Checklist Filters can be changed without any changes happening in the Search Checklists template. All changes that are made in the Search Checklist Filters page are not saved in the database.

To start the search from the Search Checklist Filters page, click Search on the ribbon, or press F9. If the search process finds checklists that match the search criteria, the Checklist Navigate page appears.

: Checklist Navigate

The Checklist Navigate page displays all documents that are linked to the found checklists. On the ribbon, click Show to look at these documents.

If multiple checklists are found (thus the string of all found checklist numbers is longer than about 1000 characters), a message appears and then these checklists are listed in the Checklists Found page. In this page, highlight the checklists to review and then click Actions > Navigate, or press CTRL+F7, to open the documents (such as Sales Lines, Construction Orders) that are linked to the selected checklists.
