Example – Formula Mandatory Input

In this example, one question with two selection options, and two additional answers are defined in a checklist for configuring a hydraulic or electric lift. The selection options have been set up with input values 1 and 2, respectively. The variable “LIFTTYPE” has been assigned to the question and then again in the Formula Mandatory Input field along with the input value “= 1” for the second answer line. The Mandatory Input check box is already selected for the first answer line.

Additionally, the Automatic Recalculation check boxes are selected on the question and answer lines.

: Formula Mandatory Input

When this checklist is opened on a document, and the user enters 1 in the Input field on the question line, the system automatically selects the Mandatory Input check box in the Hydraulic Oil line and hides the Electric Lift line.

: Result of Formula Mandatory Input with input value = 1

If the user enters 2 in the Input field on the question line, the system automatically hides the Hydraulic Lift line and the Mandatory Input check box is not selected in the Hydraulic Oil line.

: Result of Formula Mandatory Input with input value = 2
