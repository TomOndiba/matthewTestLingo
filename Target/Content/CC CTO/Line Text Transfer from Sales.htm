Line Text Transfer from Sales Line

Line text that is entered for a Sales Line is copied to the Construction Order header, where the text can be modified. When a Production Order is created from the Sales Line or Construction Order, the line text is copied to the first Production Order line.

For example, text has been added to a sales line for item D-9000 by clicking Lines FastTab > Line > Line Texts.

: Text added to Sales Line

When a construction order is created from the sales order, the text is copied from the sales line to the construction order header.

: Text copied to Construction Order header from Sales Line

If a production order is created from the construction order, the text is copied from the construction order header to the first production order line (opened from Lines FastTab > Line > Line Texts).

: Text copied to Production Order line from Construction Order header from Sales Line
