General FastTab

The fields on the General FastTab are described in the following table.

Field

Description

No.

Unique identifier for the change order.

Source Type

Indicates the source document of the change order. The options are:

  • Item
  • Planned Production Order
  • Production BOM
  • Firm Planned Production Order
  • Document BOM
  • Released Production Order

Source No.

Identifies the number of the source document of the change order.

Change Reason

Short description to explain the reason for the change order.

Status

Indicates the status of the change order. The options are:

  • New
  • Accepted
  • Read
  • Rejected

Order Text

Indicates if Order Text exists for the change order.

Answer Text

Indicates if Answer Text exists for the change order.

Created by

Identifies the user who created the change order.

Created on

Indicates the date on which the change order was created.

Modified by

Identifies the user who modified the status of the change order.

Modified on

Indicates the date on which the change order was modified.

Picture

Displays an image selected using standard Microsoft Dynamics NAV picture insertion functionality.
