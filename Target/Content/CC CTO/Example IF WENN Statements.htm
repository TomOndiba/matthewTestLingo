Example – IF/WENN Statements and String Variable Comparison

In this example, the IF statement is used in a dynamic formula to make a comparison with an empty value (using double quotes ""), a special string (value = blue) and a non-empty string (using not equal to brackets <>). This process works the same for the WENN statement.

: IF statements for string variable comparisons in Checklist Template

In the first line of the checklist template, a color specification is expected. The input becomes stored as the variable “CL_COLOR”. The results are listed in the next three lines: if the color variable matches the formula, then a true value (100) is returned; otherwise a false value (200) is returned.

If the variable has no value, then the result of the formula will be true (100) on the first comparison line and false (200) on the second and third lines.

: Empty value calculated in Checklist Template

If the variable “CL_COLOR” has a value of blue, then the result of the formula will be true (100) on the second and third comparison lines – because blue is the correct option for the second line and because the value is not blank for the third line – and false (200) on the first line.

: Valid value calculated in Checklist Template

If the variable “CL_COLOR” has a value that is not blank but is not blue (e.g., green), then the result of the formula will be true (100) on the third comparison line and false (200) on the first and second comparison lines.

: Non-blank, not blue, value calculated in Checklist Template
