Manufacturing FastTab

Integration with the Manufacturing modules in Microsoft Dynamics NAV is defined on the Manufacturing FastTab.

: Configure to Order Setup, Manufacturing FastTab

The fields on this FastTab are described in the following table.

Field

Description

Create Prod. Order Dialog

If selected, a dialog box will appear when a Production Order is created from a Document BOM or Construction Order that asks if the user wants to create variants for each BOM.

Create Prod. Order Default

This field, along with the Create Prod. Order Dialog check box, defines whether variants are created for each BOM when a Production Order is created from a Document BOM or Construction Order:

  • If the Create Prod. Order Dialog check box is selected, the option selected in the Create Prod. Order Default field will be suggested in the dialog box.
  • If the Create Prod. Order Dialog check box is not selected, then the method selected in the Create Prod. Order Default field will be used automatically.

New Variant Manufac.

Defines the number series for Variant numbers. When production orders are created from the Sales Order and/or Construction Order, Item Variants can be generated. The Prod. BOM No. and Routing No. will be assigned to the new Item Variant.

New Routing Manufac.

This number series is used for the new routing that is created when a production order is created from a Document BOM, or when production orders are created from a construction order, then new routings will be created using this number series.

New Prod. BOM Manufac.

This number series is used for the new production BOM that is created when a production order is created from a Document BOM, or when production orders are created from a construction order, then new Manufacturing BOMs will be created using this number series.

Enclosure by

During the creation of purchase orders for subcontractors, the enclosed part is moved to the stock location. The options for how the parts are moved are:

  • Reclass. Journal
  • Transfer Order

Enclosure Transfer Journal Template

Identifies the Item Journal Template that will be used if the Reclassification Journal is selected in the Enclosure by field.

Enclosure Transfer Journal Name

Identifies the Item Journal Batch that will be used if the Reclassification Journal is selected in the Enclosure by field.

Per Purchase New Journal Name

Determines if a new journal name needs to be used for each external purchase order. The order number will appear in the name of the Reclassification Journal.

Last Journal Name

Name of the last assigned Reclassification Journal.

In-Transit Code

Assignment of the in-transit code that is used with Transfer Orders created for external purchase orders.

Create Enclosure Whse. Shpt.

Automatically creates a Warehouse Shipment during the subcontracting process if a warehouse management location is used. The options are:

  • Create Whse. Shpt. Open
  • Create Whse. Shpt. Released

Refer to topic “Subcontracting with Warehouse Management” on page 1 for more information.

Due Date from

Determines if the due date of the construction order defaults from either the originating sales order header or line.

Dummy Routing Code

Assignment of a Routing number for dummy routings. This setting is only needed if the Document BOMs or construction orders create production orders using the “Create Prod. Order without Variants” function.

Certify New Prod. Structure

Used with the integration to cc|process manufacturing (refer to section “cc|process manufacturing” on page 1.
