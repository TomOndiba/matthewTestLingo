Find/Create Translation Function

The “Find/Create Translation” function is available on the Checklist Question Card, Checklist Answer Card and Checklist Template pages.

Translations on Checklist Questions and Answers

On the Checklist Question Card and Checklist Answer Card pages, the “Find/Create Translation” function populates the Translation Code field on the General FastTab with a unique number that is assigned to a Translation Header page, where the translated text is entered. Additionally, the Description (T) field is updated with the description in the language noted in the Language field.

: Translation information created on the Checklist Question Card

To enter the translation information, click the arrow in the Translation Code field and then click Advanced. In the Translation List page, click Edit.

The first line in the Lines FastTab contains the original text in the default language. On an empty line, enter the additional language in the Language Code field and then type the translated text into the Text field.

: German translation for Checklist Question entered in the Translation Header page

Select the Use Alternative Language check box to search for translations of this text in the specified alternative language (refer to topic “Initial Setup” on page 1).

Use the same process to create translations for the Checklist Answers.

: German translation for Checklist Answer entered in the Translation Header page

Translations on Checklist Templates

On the Checklist Template, enter the questions/answers in the lines. Existing translation information will populate the Translation Code and Description (T) fields on the lines.

: Checklist questions and answers added to Checklist Template, some with existing translations

On the ribbon, click Find/Create Translations to find existing translations or create new translations for lines with an empty Translation Code field. Click Yes to find/create translations.

: Translations created for checklist answers without translations

To add the translation to a new translation, click the Translation Code field on the line and then click Advanced > Edit and enter the translated text.

: Translated text entered with the Use Alternative Language check box selected

To see the translated text in the Description (T) field on the checklist line, change the Language field on the General FastTab to a language set up in the Translation Header page.

: Translated text appears in Description (T) field while default language for the user appears in Description field

If a language entered in the General FastTab has not been set up with translations, the Description (T) field will display one of the following, depending on the Use Alternative Language setting in the translation.
