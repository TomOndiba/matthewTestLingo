Example – Formula Function

In the following example, the Formula Function is explained using two types of square meter calculations:

  • Calculation of a function value directly in the checklist
  • Calculation of the function value with a dynamic function

: Formula Function

Two answers are defined, both with an Input Type of Decimal. For the first answer, a length specification is expected and in the second, an indication of width. The input becomes stored as the variables “VAR1” and “VAR2”, respectively.

The results are listed in the last two lines. On the first result line, the calculation occurs directly in the checklist as specified in the Formula Function field: The contents of the variable “VAR1” are multiplied by the contents of the variable “VAR2”.
