Lines FastTab

The following menu items are accessible from the Lines FastTab.

Line Menu

The following menu items appear on the Line menu, accessed from the Lines FastTab.

  • Move Left: Moves the marked position up one level in the Service BOM structure.
  • Move Right: Moves the marked position down one level in the Service BOM structure.
  • Attributes: Opens the assigned attributes for the current line.
  • Open File: Opens the file specified in the Drawing field for the current line.
  • Line Texts: Opens the Edit Text page where text can be stored for the current line. Refer to topic “cc|text module” on page 1 for more information.

Function Menu

The following menu item appears on the Function menu, accessed from the Lines FastTab.

  • Request BOM from Line: Opens the Master BOM for the component in a service BOM line, if it has its own master BOM.
