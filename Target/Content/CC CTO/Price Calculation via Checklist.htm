Price Calculation via Checklist

When calculating price with the checklist, it is possible to create a quotation costing without providing a document BOM and calculation scheme. The checklist takes over, thereby offering the additional function of a price list or sales manual, where certain options are provided with additional charges and/or if a standard option is deselected, a price reduction will calculate.

Calculating the price with the checklist can be used for the document calculation with the calculation scheme. The function of the document calculation is to verify the sales price based on a BOM.
