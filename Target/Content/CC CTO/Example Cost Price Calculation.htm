Example – Cost Price Calculation

In the Configure to Order Setup page, an attribute for cost price can be defined in the Variable for Unit Cost field (refer to topic “Dynamic BOM FastTab” on page 1). The assignment of the variables is similar to the variable for components in the Master BOM lines. In the Formula Value field on the Master BOM Attributes page, enter a formula for price calculation that relates back to the variable on the checklist.

: Variable for Cost Price on Master BOM

Thereby all variables of the checklists and the Master BOM can be accessed. The result of the calculation is transferred to the Document BOM in the Unit Cost (CC) field.

In this example, the value 200 is entered in the Input field on the document checklist line.

: Cost Price assigned on document checklist line

When the Document BOM is created, the system multiplies the value in the Input field by 2, as defined in the Master BOM Attribute, and 400 is transferred to the Unit Cost (CC) field.

: Calculated Cost Price in Document BOM
