Example – Line Text on the Checklist

Line text can be entered in checklists. This text can be opened and edited by clicking Line > Line Texts on the Lines FastTab (refer to topic “cc|text module” on page 1 for more information).

: Entering Line Text in the Checklist

Once text has been entered, the text identification number is displayed in the Text Set ID field, thus indicating that text exists for the line.

: Line Text entered in the Checklist

In the Configure to Order Setup page, a Variable for Extended Text can be entered (refer to topic “Dynamic BOM FastTab” on page 1).

: Variable for Extended Text set to TEXTBOX

In the Master BOM Attributes page, in the Attribute field, enter the same attribute that was entered in the Configure to Order Setup page. In the Formula Value field, enter the same variable that was entered in the Variable for Dynamic BOM field on the checklist line (that contains the extended text), in this example, the variable “APRON”. The value of the APRON variable must not equal zero or TRUE if the text should be copied to the BOM.

: Assignment of the Extended Text variable in the Master BOM

During the calculation of the BOM, the text that was entered in the checklist will be copied as line text to the BOM line.

: Line Text transferred to document BOM line
