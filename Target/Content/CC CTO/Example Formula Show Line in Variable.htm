Example – Formula Show Line in Variable Option

Similar to the Formula Show Line functionality in the checklist template, a Formula Show Line field exists in the Variable Options page to prevent some options from appearing for selection based on specific answers to questions in a checklist line.

In this example, a new Variable named OPTIONVAR was created with the following three options:

  • The first option should be shown always so the Formula Show Line field is empty
  • The second option should be shown when variable VAR1 is greater than 10
  • The third option should be shown when variables VAR1 and VAR2 are greater than 10

: Variable Options with Formula Show Line

A Checklist Template has been created with two question and two answer lines. The answer lines are defined with the variables VAR1 and VAR2, respectively, and the second question line has been defined with the OPTIONVAR variable, which provides the lookup to the variable options.

: Checklist Template with Formula Show Line in Variable Option setup

The formulas can be tested in the Checklist Template. If a user enters values in the Input field for the answer lines, e.g., 11 and 9, respectively, and then clicks the arrow in the Input field on the fourth line, the Variable Options page opens with all available options.

: Formula Show Line test in checklist template line

If the user clicks Test “Formula Show Line” on the ribbon, the values from the checklist template are uploaded to the formulainterpreter, the formulas in the Formula Show Line fields are calculated and if the result of a formula is false, the line(s) are hidden. In this example, Input = 30 was hidden because the value entered on the checklist template line was greater than 10 for Value 1 but not for Value 2.

: Result of Formula Show Line test in checklist template line

Click Test “Formula Show Line” again to clear the filter and show all options and then close the page by clicking Cancel. Additionally, clear any input values before certifying the checklist template for use.

NOTE

The “Test “Formula Show Line”” function can only be run from the Variable Options page when opened from a checklist line.

When this checklist is opened on a document, and the user enters 10 in the Input field for Value 1 and 9 for Value 2, and then opens the Variable Options page for the last line, only input line 10 appears because none of the correct variable values were met.

: Result 1 of Formula Show Line test in document checklist line

If an invalid option is manually entered in the Input field, an error message appears.

: Result of entering an invalid option in the document checklist line

If values are changed to meet the correct criteria, then the options will appear accordingly.

: Result 2 of Formula Show Line test in document checklist line

Any of the valid options can be entered in the Input field.

: Valid option entered in the document checklist line
