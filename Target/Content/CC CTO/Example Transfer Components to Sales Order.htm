Example – Transfer Components to Sales Order

In this example, the three documentation entries (D-7100, D-7105 and D-7110) in the Master BOM are set for transfer to sales document.

: Transfer to Sales Document field set on three lines in a Master BOM

When the Sales Order/Checklist/Document BOM are created, the check marks in the Transfer to Sales Document check boxes are transferred to the Document BOM.

: Transfer to Sales Document field setting transferred to Document BOM

Now, the Transfer to Sales Document check box is selected on the line for item D-1051 (an assembly group), and two new lines for items D-7100 and D-7105 are inserted at the end of the assembly group. Each documentation item is reserved against inventory.

: Transfer to Sales Document field set on assembly item, new items added and reservations made

When the “Transfer Comp. To Sales Document” function is run, the following message appears.

: Transfer Comp. to Sales Document function confirmation message

Accepting the confirmation message deletes the entire assembly group for item D-1051 and the three documentation components.

: Source lines deleted from Document BOM after Transfer Comp. to Sales Document function is run

In the Sales Order lines, assembly item D-1051 and the three documentation components have been transferred from the Document BOM.

: Assembly item and documentation components transferred from Document BOM to Sales Order line

The following field information is transferred from the Document BOM to the Sales Line:

Type = Item

(Exp.) Quantity

No.

Text Set ID

Variant Code

Unit Price

Description 1 – 5

“Line Discount %”

Unit of Measure

  

Additionally, the reservations made in the Document BOM have been transferred to the sales line.

Note

If this exercise was created with a Sales Quote, the reservations would not transfer because reservations cannot be made on sales quotes.

For item D-1051, a new Document BOM was created for the assembly group including the reservations.

: Document BOM created for assembly item

The inserted Sales Quote/Order Lines are assigned to the “main” Sales Line to which the source Document BOM is assigned. Therefore, if the “main” Sales Line is deleted, the other Sales Lines will be deleted as well.
