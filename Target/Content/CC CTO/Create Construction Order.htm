Create Construction Order from Service Item Worksheet

When an item is replaced or inserted into the Service BOM using the Service Item Worksheet, it is possible to create a Construction Order for the item if construction is necessary. After inserting or replacing the component (refer to topic “Substitution of Service Item Components” on page 1), click Line > Construction Order on the Lines FastTab of the Service Item Worksheet page. Click Yes to create a Construction Order.

If the component from the Service BOM was replaced, and the component has a BOM, the BOM will be copied within the new Construction Order. After all necessary updates are made to the Construction Order (refer to topic “Construction Order” on page 1), a Production Order can be created.

Changes that were made in the Construction BOM can be written back to the Service BOM by clicking Actions > Transfer Back to Service BOM on the Construction Order page. All changes will be copied into the Service BOM so a record of the current configuration of the Service Item can be made.

Note

Only replaced components – not new components – can be transferred back to the Service BOM.

For example, if a new component is added in the Service Item Worksheet and a Construction Order is created, the new component cannot be transferred back to the Service BOM because it currently does not exist in the Service BOM. But if the component was replaced in the Service Item Worksheet and a Construction Order is created, the replaced component can be transferred back to the Service BOM because the existing Service BOM lines are deleted and replaced with the Construction BOM.
