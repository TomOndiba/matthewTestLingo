Checklist Document, Lines FastTab

The fields described in the following table are relevant for price calculations on the checklist lines.

Field

Description

Add. Charge/ Price Reduction

If the checklist is used for the sales price calculation, an additional charge/price reduction can be entered on the checklist answer lines.

When the checklist is opened on the document, this price is included in the price calculation if the Selected check box is selected on the answer line or if the answer line was set up as Selection Option for the question.

Discount Possible

Determines whether the Add. Charge/Price Reduction on the checklist answers is discountable or not.

If this check box is selected, then the value in the Discount % field on the Calculation FastTab is deducted proportionally from the Add. Charge/Price Reduction and then added to the sales price.

Line Discount

Identifies a discount rate that is applied to the Add. Charge/Price Reduction. This value is calculated in addition to the checklist discount, if the Discount Possible check box is selected on the line.

Add. Weight/Weight Reduction

If the checklist is used for the weight calculation, an additional weight/weight reduction can be entered on the checklist answer lines.

When the checklist is opened on the document, this weight is included in the weight calculation if the Selected check box is selected on the answer line or the line was set up as Selection Option for the question.
