Order Network

Order Network uses a tree-structure to display all information that is connected with a process, for example, all the activity relating to a sales order.

: Order Network opened from cc|configure to order menu

In contrast to the “Navigate” function, all documents are displayed chronologically and hierarchically starting with the sales order. With this functionality, posted documents such as invoices and delivery documents are also displayed, as well as warning indicators

The Order Network page is opened from the Sales Order and the cc|configure to order menu under Order Processing.
