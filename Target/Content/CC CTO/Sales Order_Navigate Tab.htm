Navigate Tab

The following menu items appear on the Navigate tab of the Sales Order ribbon.

  • Header Texts: Preceding text defined in the cc|text module for the order can be opened and modified on the order line. Refer to topic “cc|text module” on page 1 for more information.
  • Footer Texts: Subsequent text defined in the cc|text module for the order can be opened and modified on the order line. Refer to topic “cc|text module” on page 1 for more information.
  • Order Network: Refer to topic “Order Network” on page 1.
