Example – Transfer Information into Checklists

Often it is necessary to have information e.g. from the Sales Header (Language Code) or Sales Line (Quantity) in a checklist line. Instead of entering this information manually into the checklist, the origin of the input can be defined on the Checklist Template line in the following fields.

Field

Description

Get Value from Table 

Indicates the table of origin. The options are:

  • Sales Header
  • Purchase Line
  • Sales Line
  • Service Item
  • Construction Header
  • Service Header
  • Purchase Header
  

Get Value from Table ID

Automatically populated with the number of the table selected in the Get Value from Table field.

Get Value from Field ID

Indicates the field from which data from the selected table is to be transferred.

Get Value from Field

Automatically populated with the name of the field selected in the Get Value from Field ID field.

Note

The Formula Function field cannot be populated if any of the Get Value from … fields are populated. In addition, if the Formula Function field is not empty, a table cannot be selected in the Get Value from Table field.

After recalculation of the Document Checklist, the values of the origin field will be transferred into the Input field on the checklist line. The Always Overwrite and Value of Function is Default fields should thereby also be used in the Checklist Template line(s). Additionally, the Free Text field should be used for any expected text values that should not be interpreted by the system.

In this example, a checklist template is set up to transfer the following information to the checklist:

Sales document type = order

Item number

Sales order number

Quantity

Shipment date

This checklist template is assigned to item 1001 and then a sales order is created for 2 units of the item to customer 20000, Selangorian Ltd. with shipment date of January 24, 2019.

: Document Checklist before recalculation

When the checklist is opened from the sales order and the “Recalculate” function is run, the specified fields will be populated with information from the linked sales header and line.
