Navigate Tab

The following sections explain the menu items on the Navigate tab of the Master BOM.

Version

This function opens the Overview Master BOM Version page, where new and existing versions of the Master BOM are created and displayed. Click New to create a version or double-click an existing version to open the Master BOM Version page.

: Master BOM Version page

The Master BOM Version page is structured like the Master BOM template, but it includes a validity date feature. The Starting Date is the date from which the version is to be valid. All versions with a starting date older than the here-entered date become invalid when this Master BOM version becomes valid. This allows users to set up new Master BOM versions, which become valid automatically at the defined start date.

Copy from Prod. BOM

This function provides the ability to create multiple master BOMs based on existing production BOMs, if the originating production BOM is a multi-level BOM.

: Copy Prod. BOM to Master BOM batch job

The fields on the batch job are described in the following table.

Field

Description

Prod. BOM No.

Number of the production BOM from which to be copied.

Prod. BOM Version

Version of the production BOM from which to be copied.

Routing No.

Number of the Routing from which to be copied.

Routing Version

Version of the Routing from which to be copied.

BOM Type

Type of the new master BOM, which must be filled in on the Master BOM to determine if only the first BOM level should be copied (BOM Type = empty) or if all levels should be copied (BOM Type <> empty.). It will be generated for the assembly units in the substructure Master BOMs with the stated BOM Type.

Replace Existing Master BOM-Item Relations

If selected, the copy function replaces all Master BOM lines in all Master BOMs that are related to the items. This setting is needed to copy multi-level BOMs.

If this check box is not selected, only new Master BOMs can be created the item. However, if a Master BOM is already related to the item (with the correct BOM Type), then the copy process with stop at this level.

Copy with Routings

If selected, and a multi-level BOM will be copied, then the Routings in each level will be copied also.

Status After Change

Status that the Master BOMs will have after the exchange and/or change of items. The options are:

  • New
  • Close
  • Certified
  • Maintain
  • Under Development
  

 

Syntax Check

This function validates the syntax in formulas within the Master BOM lines.

Use

This function opens the Item - Master BOM page, which shows the items to which the Master BOM has been assigned.

: Item - Master BOM page opened from Master BOM

Items can be assigned to the Master BOM in this page but are typically assigned from the Item Card (Navigate > Master BOM > Master BOM). Refer to section “Navigate Tab” on page 1 for more information.

Use in Lines

This function opens the Master BOM Line List, which shows all Master BOM Lines where the current Master BOM is used.

: Master BOM Line List, showing where Master BOM D-1056 is used in other Master BOM lines

Structure Representation

Displays the Master BOM in a tree-structure where the different levels can be expanded or collapsed. The Master BOMs must all have the same BOM type (refer to the BOM Type for Structure Representation field in the topic “Dynamic BOM FastTab” on page 1).

: Structure Representation for Master BOM
