Dynamic BOM Setup

The maximum levels for BOM structures are defined in the Dyn. BOM Setup page, accessed from the cc|configure to order menu by clicking Administration > Dyn. BOM Setup.

: Dyn. BOM Setup

For each document, to which a Document BOM can be assigned, the maximum number of levels for a BOM structure relating to orders is defined. cc|configure to order supports a structure depth of 50 levels. These settings apply both when dynamically opening BOMs and when copying from documents.

The Copy Information Lines check box determines whether the next BOM level is to be copied, even if the assembly group is not characterized relating to orders (refer to Part Type = Production in topic “Config. to Order FastTab” on page 1).
