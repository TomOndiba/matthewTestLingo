Construction Order Lines FastTab

The menu items on the Line and Function tabs are described in this section.

Lines FastTab, Line Tab

The following menu items appear on the Line tab, accessed from the Lines FastTab.

  • Attributes: Opens the attributes that were assigned to the construction line. Attributes can also be assigned manually here.
  • Open File: Opens the file specified in the Drawing field for the current line.
  • BOM: Opens the construction lines in an editable format. In contrast to clicking Construction Lines on the ribbon, this function presents only the current item and its substructure in the Construction BOM page.
  • Reserve: Enables reservations to be made manually (e.g. from the warehouse stock).
  • Reserve Ledger Entry: Displays the reserved quantity when reservation items have been generated for this position (e.g. via the date of promised delivery).
  • Item Tracking Lines: Here item tracking can be assigned to components that are subject to management in batches or by serial numbers; standard Microsoft Dynamics NAV functionality.
  • Order Promising: This function enables, for example, critical parts or long-running items to be planned in advance. This function is based on the standard function for “date of promised delivery”. The system calculates the earliest possible procurement date. If you accept the proposal, planning proposals are generated immediately which are connected with the document BOM line or the construction line via the reservation management of Microsoft Dynamics NAV.
  • Create Separate Prod. Order: Creates a production order for construction order lines that have the Separate Prod. Order check box selected. If the production order is generated here, there are two options:
    • Create Prod. Order without Variants – This function creates a production order for the overall construction order without creating variant codes, production BOMs, or routings.
    • Create Prod. Order with Variants – This function creates a production order based on the construction order. The system generates a variant for all production items with the Part Type field set to Production and generates a manufacturing routing and a manufacturing BOM. The automatically generated production BOMs and routings are linked to the item variant.

Whether the system creates production orders with or without variant codes by default is defined on the Configure to Order Setup page (refer to topic “Manufacturing FastTab” on page 1).

When the production order is generated for the Create Prod. Order with Variants option, the following steps are performed by the system:

  1. Generation of one variant per production item with the Part Type field set to Production.
  1. Creation of a manufacturing BOM using the default number series (refer to topic “Manufacturing FastTab” on page 1).
  1. Creation of a manufacturing routing using the default number series (refer to topic “Manufacturing FastTab” on page 1).
  1. Assignment of the manufacturing BOM and the routing on the Item Variants table.
  1. Creation of the production order based on the BOM entered and the routing.
  • Cut: This function enables individual items, multiple items and complete structures from the same or a different construction order to be cut and copied to the clipboard.
  • Copy: This function enables individual items, multiple items and complete structures from the same or a different construction order to be copied to the clipboard.
  • Paste: If a line was copied to the clipboard from the same or a different construction order it can be inserted here. Individual items, multiple items and complete structures can be inserted.
  • Comments: Comments can be informatively entered; standard Microsoft Dynamics NAV.
  • Dimensions: Displays the dimensions assigned to the construction order line.
  • Tools: Tools information can be informatively entered; standard Microsoft Dynamics NAV.
  • Personnel: Personnel information can be informatively entered; standard Microsoft Dynamics NAV.
  • Quality Measures: Quality measures can be informatively entered; standard Microsoft Dynamics NAV.
  • Line Texts: Opens the Edit Text page where line-related text can be stored. Refer to topic “cc|text module” on page 1 for more information.

Lines FastTab, Function Menu

The following menu item appears on the Function menu, accessed from the Lines FastTab.

  • Request BOM from Line: Opens the Master BOM for the component in a construction line, if it has its own master BOM.
