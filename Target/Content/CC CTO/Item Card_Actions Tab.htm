Actions Tab

The following sections explain the menu items that have been added to the Functions group on the Actions tab.

Create Change Order

The “Create Change Order” function creates an order to notify the design department of change suggestions for the item (refer to topic “Change Order ” on page 1).

Open Change Order

The “Open Change Order” function opens the Change Orders page from where existing change orders for the item can be viewed (refer to topic “Change Order ” on page 1).

Copy Item

The “Copy Item” function creates an item from an existing item. In the Item Copy page, either a target item number is entered or a number from a number series is produced automatically. Additional data to be copied can be selected.

: Item Copy

Search & Replace Item

Use the “Search & Replace Item” function to find and replace items in the following documents:

  • Master BOM Template
  • Document BOMs
  • Construction Orders

: Item Search and Replace

The fields/field groups on this dialog box are described in the following table.

Field / Field Group

Description

Search Item No.

Indicates the item to be found.

Replace with

Identifies the item number that the searched for item is to be replaced with. If the same item number in the Search Item No. field is entered in this field, then the item with master data relating to orders is overwritten based on the fields selected in the Overwrite Fields area.

This functionality is useful because all selected item data is replaced in the documents. For example, if a user needs to change the Description 5 field on an item, he or she does not have to change all documents manually.

Search & Replace in

In this field group, the documents to be included in the item search and/or replace process are selected.

Master BOM

Defines how the system is to behave with changes to the Master BOMs. The options are:

  • Create New Version – A new version of the Master BOM is to be added.
  • Change Actual Version – The current BOM version is to be changed.
  • Change All Versions – The exchange in all BOM versions is made.

Status After Change

Defines which status the Master BOMs will have after the exchange and/or change of items. The options are:

  • New
  • Closed
  • Certified
  • Maintain
  • Under Development
  

Only Status = Open

Determines whether approved Document BOMs and construction orders can be changed.

Note

If Document BOMs and construction orders are to be included in supply planning (as defined in Configure to Order Setup), purchase orders or production orders may have been already created for these lines. The “Item Search & Replace” function does not examine whether document lines, which should be replaced, have been created.

Overwrite Fields

Fields selected in this group will be overwritten for the items on the selected document(s) during the search and replace function.
