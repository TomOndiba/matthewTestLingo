Introduction

cc|mobile solution is a cross-industry and client-side platform-independent software providing a mobile data recording/processing solution built on Microsoft Dynamics® NAV. It uses a direct online connection and gives a complete integration of the mobile employee into the business processes of the company and the business logic of the ERP system.

Key Features

  • Two user interfaces (UI):
    • Browser (HTML – platform-independent)
    • Terminal Application (WPF – single application for Windows)
  • Many standard process dialogs (transactions) in the delivery of the product:
    • Supports all important processes of functional areas (Inventory/Warehouse, Manufacturing, Jobs/cc|project)
    • Supports all processes of cc|factory data capture
  • Customizable UI and logic (all UI use the same definition/scheme)
  • No middleware
  • Data in real-time
  • Elimination of redundant data management (including paper)
  • Efficient and simple system-controlled data input
  • Avoid data input errors (through barcodes, plausibility checks, etc.)
  • Process safety
  • Optimized for performance
