Application

This section provides a brief introduction to the User Interface (UI) of the Terminal Application.

: Terminal App

Navigation Bar

The buttons on the navigation bar are described in the following table.

: Navigation Bar

Button

Description

Show/Hide Virtual Keyboard

Shows/hides the virtual keyboard.

Forward/Back

Navigation between pages like in a browser.

Refresh

Refreshes the last requested page (F5).

Zoom in/out

Decreases/Increases the size of the content.

Auto-Zoom

Fits the view to the size of the content/resets the zoom to default.

Settings

Opens the Settings window.

Main Menu

Navigates to the Main Menu, as defined in Mobile Setup.

 

Additional Navigational Features

Additional navigational features for using the Terminal Application are:

  • Numbered tiles can be called up by typing the corresponding keys (NumPad too).
  • Forms can be sent by pressing Enter.
  • Scrollable lists can be navigated by PageUp/PageDown, and selected by double-clicking or pressing Enter.
  • Dropdown Menus can be expanded with F4.
  • Editable tables can be edited with F2.
  • Zoom is possible with CTRL+Mousewheel.
