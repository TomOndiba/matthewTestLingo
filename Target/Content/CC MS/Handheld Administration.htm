Handheld Administration

The handheld administration contains handhelds, handheld groups and handheld rights.

This section describes the settings and fields of the handhelds, explains how to create and administer handhelds in Microsoft Dynamics NAV, and shows how to assign handhelds to groups and grant/remove rights to handhelds.

Handhelds

The Handheld page represents a mobile device or a handheld user. The handheld setup contains the creation/administration of handhelds and additional functionalities that are useful for administrating the handhelds.

General FastTab

The General FastTab contains basic settings of the handheld.

: Handheld, General FastTab

The fields on this FastTab are described in the following table.

Field

Description

No.

Specifies the number of the handheld. This value is only important for Microsoft Dynamics NAV and does not have to be like the value entered in the PIN (Login) field.

Name

Specifies the (first) name of the handheld user or the name of the handheld.

PIN (Login)

Specifies the username, card number or PIN, with which the user can login.

Password

Indicates the password of the handheld user or device.

Language Code

Indicates the language of the content that will be displayed to the user.

Group Code

Specifies the handheld group of the handheld, which causes the handheld to inherit the group rights and group settings.

Start Page

Specifies the start page of the handheld. If no value is set, the value of the handheld group or Mobile Setup is used.

Layout Setup Code

Specifies a layout for the handheld, which allows different layouts for scanners or terminals to be used.

Blocked

Determines if the handheld is blocked.

Details FastTab

This FastTab contains user-specific settings of the handheld/user.

: Handheld, Details FastTab

The fields on this FastTab are described in the following table.

Field

Description

Phone No.

Indicates the phone number of the handheld/user.

E-Mail

Indicates the e-mail address of the handheld/user.

User ID

Specifies the Microsoft Dynamics NAV User ID if the handheld user also has a login to the Microsoft Dynamics NAV client.

Username

Displays the Microsoft Dynamics NAV Username depending on the selected User ID.

Home Directory

Displays the path where created files are stored.

This field is used for informational purposes only.

Default Printer

Specifies the default printer of the handheld. If no value is set, a printer selection will be shown before printing.

Last Used Printer

Displays the last printer that was used for printing.

Handheld Company

Specifies the Company Name of the handheld, in case the handheld is related to another Company but set up in this Company.

Application FastTab

In this FastTab the relations from the handheld to Microsoft Dynamics NAV Objects are set.

: Handheld, Application FastTab

The fields on this FastTab are described in the following table.

Field

Description

Salesperson Code

Specifies the Microsoft Dynamics NAV Salesperson Code to which the handheld should be assigned (if existing).

Res. Jnl. Batch Name

Assigns the handheld to an own Resource Journal Batch. This value is required if dialogues should be used for processing resource postings.

Resource No.

Assigns the handheld to a resource (if existing).

Item Journal Template Name

Assigns the handheld to an own Item Journal Template. This value is required if dialogues should be used for processes in inventory or production.

Item Journal Batch Name

Assigns the handheld to an own Item Journal Batch. This value is required if dialogues should be used for processes in inventory or production.

Location Code

Assigns the handheld to a location. This value is required if dialogues should be used for processes in inventory or production.

Whse. Jnl. Template Name

Assigns the handheld to an own Whse. Journal Template. This value is required if dialogues should be used for processes in inventory (only with Directed Put-away and Pick).

Whse. Jnl. Batch Name

Assigns the handheld to an own Whse. Journal Batch. This value is required if dialogues should be used for processes in inventory (only with Directed Put-away and Pick).

Job Journal Template Name

Assigns the handheld to an own Job Journal Template. This value is required if dialogues should be used for job processes.

Job Journal Batch Name

Assigns the handheld to an own Job Journal Batch. This value is required if dialogues should be used for job processes.

FDC Resource No.

For the usage of cc|factory data capture in cc|mobile solutions, an FDC Resource can be assigned to the handheld. In this case, in each dialogue this FDC Resource will be used for that handheld.

TDC Input Journal Template Name

Assigns the handheld to an own TDC Input Journal Template. This value is required if dialogues should be used for cc|factory data capture.

TDC Input Journal Batch Name

Assigns the handheld to an own TDC Input Journal Batch. This value is required if dialogues should be used for cc|factory data capture.

Home Tab

The Process group on the Home tab of the ribbon contains three actions. Each is described in the following table.

Action

Description

Rights

Opens the Handheld Right page and displays all rights that are assigned to the handheld. This includes group rights and individual rights.

Import/Export

Imports/exports data from/to other companies.

Actions Tab

The Process group on the Actions tab of the ribbon contains four actions. Each is described in the following table.

Action

Description

Create

Creates a subfolder in the root directory of the HTTP-Server with the name of the identifier (No.) of the handheld. This folder is required because all created pages are stored there.

Remove

Deletes all files in the created handheld folder and the folder itself.

Clear

Deletes all files in the folder.

Clear Directories of all Handhelds

Deletes the content of all handheld folders.

 

Handheld Groups

A handheld group combines multiple handhelds into a group. For example, handhelds of the type “inventory employee”, “production employee” and so on can be combined into groups.

: Handheld Group

General FastTab

The fields on the General FastTab are described in the following table.

Field

Description

Code

Defines the Handheld Group Code of the Handheld Group.

Description

Brief description of the group.

Number of Handhelds

Displays the number of assigned handhelds.

Start Page

Defines the start page that should be used for all handhelds in this group. This value is used when no start page is assigned to the handheld. If no value is set, the start page of the handheld or Mobile Setup is used.

Home Tab

The Process group on the Home tab of the ribbon contains four actions. Each is described in the following table.

Action

Description

Handhelds

Opens the Handheld List page, which displays all handhelds that are assigned to the group.

Rights

Opens the Handheld Group Right page, which displays all rights that are assigned to the group.

Import/Export

Imports/exports data from/to other companies.

Handheld Rights Administration

The Handheld Rights Administration handles the assigning of rights to handheld groups and handhelds. This rights are required for requesting the documents. Handheld rights and handheld group rights can be created and edited here.

Creation and editing of rights is performed in the Handheld Right page.

: Handheld Rights

General FastTab

The fields on the General FastTab are described in the following table.

Field

Description

No.

Displays the Handheld No. for which the rights are set.

PIN

Displays the username/card no./PIN of the corresponding handheld.

Start Page

Displays the start page of the handheld.

Name

Displays the name of the handheld.

Username

Displays the username (Microsoft Dynamics NAV) of the handheld.

Lines FastTab

The lines of this page display the rights for the documents that are assigned to the handheld.

Field

Description

Page Name

Identifies the name of the document to which the handheld should get access. If this entry is a right that is inherited by a handheld group, it cannot be edited or removed.

Module

Displays the module of the document; used for information purposes only and cannot be edited.

Page Available

Indicates if the corresponding page is available in the Document Editor.

Inherited

Indicates whether the right is inherited from a group or not.

Home Tab

The Process group on the Home tab of the ribbon contains two actions. Each is described in the following table.

Action

Description

Group Rights

Opens the Handheld Group Right page, which displays a list of all rights of the group of the handheld.

Add/Remove Module

Adds or removes the rights for all documents of a selected module.
