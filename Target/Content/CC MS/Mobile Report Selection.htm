Mobile Report Selection

The Mobile Report Selection page can be used for printing any report from a handheld device. The page can be accessed from the cc|mobile solution menu by clicking Setup > Mobile Report Selection. The corresponding process dialogue is called “Print Job”.

The Mobile Report Selection List page displays all configured reports for creating a print job. Click New to create a report selection or Edit to open an existing report selection.

General FastTab

The fields of this FastTab are described in the following table.

Field

Description

Report ID

Identifies the report to be printed.

Report Name

Displays the name of the selected report.

Report Description

Short description of the report; will be displayed at the “print job” process dialogue.

Table No.

Identifes the table number of the first DataItem of the report here (e.g. for “Item Register - Quantity” use table 23, Item).

Table Name

Displays the name of the selected table.

Mobile Rep. Sel. Filter Fields Subform FastTab

In this FastTab multiple filter fields can be configured. These fields correspond to the table selected in the General FastTab and will be displayed before running the report.

Mobile Rep. Sel. Parameter Subform FastTab

In this FastTab, up to five parameters that will be displayed on the report can be specified. The fields on this FastTab are described in the following table.

Field

Description

Report Param. Index

Identifies the index of the report parameter (required for referencing the parameter).

Report Param. Name

Indicates the name of the parameter to be displayed when asking for it in the dialogue.

Report Param. Value

Specifies a predefined value for the dialogue.

Applying the parameter requires an adjustment to the report. A textfield for displaying the parameter has to be created and the value of the parameter has to be called in the C/AL code. Therefore, the following variables are required:

  • Value – of type Text[50] (for storing the value)
  • PrintJobMgt – of type Codeunit 5307950 „MS - Print Job Q. Mgt./Disp.“

The value can be called in C/AL by using the following function:

Value := PrintJobMgt.GetActivePrintJobParam(<Parameterindex>);
