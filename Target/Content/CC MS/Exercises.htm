Solution – Exercise 1

This section provides a step-by-step solution for Exercise 1, Forms.

Task 1: Simple Form

Create a document with the following structure:

: Document TRAIN_1

Hint

Values shown in green are case-sensitive.

To create a document with the above structure, follow these steps:

  1. On the navigation pane, click Departments > COSMO CONSULT > cc|mobile solution > Application Management > Documents.
  2. Click New to create a document.
  3. In the Name field, type TRAIN_1.
  4. In the Module field, enter SYS_TRAIN.
  5. Add a card element to the document. The ID of the card is TRAIN1_CARD and the name attribute is assigned the value Task 1.
  6. Create a form with an input field and a result in the card.
  7. The ID of the input field is VALUE. Add the name attribute with the value Value to the element.
  8. Change the type of result to external using the lookup. Then change the value of the target attribute to _CHECKPARAMS.
  9. Add the Synchron element to the result.
  10. Validate the document just created with F11.

Task 2: Transmit Values in Forms

Create a document with the following structure: 

: Document TRAIN_2

To create a document with the above structure, follow these steps:

  1. Create a new document, name it TRAIN_2 and enter SYS_TRAIN in the Module field.
  2. Add a card element to the document. The ID of the card is TRAIN_2 and the name attribute is assigned the value data sheet.
  3. Create a label showing the text “You have input the following text”.
  4. Add a text field with the ID TEXT to the card TRAIN_2.
  5. Validate the document just created with F11. Now create a codeunit TRAIN_2 which processes the document.
  6. Enter the “MS – Mobile Journal Params” table in the codeunit as a record of the transmission.
  7. Create the codeunit variable GEN of MS - Generator type and a text variable with the name Value in the OnRun() function.
  8. Create the following code in the OnRun function:

GEN.Init(Rec);

GEN.CreateDeck(' TRAIN_2');

GEN.CreateCard('TRAIN_2');

getTextField('VALUE',Value);

GEN.SetOutputField('TEXT', Value,false);

GEN.CloseCard();

GEN.CloseDeck();

  1. Change the number of the codeunit on document TRAIN_2 of the document editor. The new card is created after this step. Now make the necessary changes to the TRAIN_2 document.
  2. Change the value of the destination attribute to TRAIN_2 in the form for the result element.
  3. Validate the document and test the newly created TRAIN_2 document.
