Setup

The Terminal Application is setup by clicking the Settings button on the navigation bar (refer to topic "Navigation Bar" on page 1).

: Settings Button

Settings – General

The fields on the General settings window are described in the following table.

: Settings / General

Field

Description

Username

Indicates a default username for the login page.

File extension

Determines how content is generated. The options are:

  • XML – The application uses XML documents for generating the content.
  • HTML – The application uses HTML documents for generating the content. Some features like auto-zoom are not available in this mode.

Language

Indicates the language of the application. This selection does not affect the language of the content.

Allow auto completion

Determines if a preselection of existing values is allowed. This enables a lookup in dialogues.

Allow animations

Determines if animations are allowed.

Note

Disable animations to increase performance.

Show virtual keyboard

Determines if a virtual keyboard appears when focusing an input field.

No. of most visited links

Indicates the number of links to be shown in the Info Panel; a value of 0 hides the links. History can be reset.

Settings – Connection

The fields on the Connection settings window are described in the following table.

: Settings / Connection

Field

Description

Host

Indicates the URL or IP Address of the HTTP-Server.

Port

Indicates the Port of the HTTP-Server. A green check mark shows if the server is reachable with the current settings.

Path

Indicates the path to the login page.

Start URL

Displays the path combination of host, port and path.

 

Settings – Layout

The fields on the Layout settings window are described in the following table.

: Settings / Layout

Field

Description

Nightmode

Inverts the color scheme of the application.

Design

Sets the colors used in the application.

Fontsize

Used as font size of the content in XML mode.

Show further links in

Determines if additional navigation links can be shown in the Info Panel or Navigation Bar.

Show Close Button

Determines if the Close button can be shown in content pages or not.

Navigation

Indicates the position of the Navigation Bar in HTML mode.
