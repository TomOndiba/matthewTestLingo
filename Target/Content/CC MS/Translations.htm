Translations

Translations are required to display the documents of cc|mobile solution in different languages.

This section describes how to use, create and administrate translations in cc|mobile solution using the Labeltext Insertion and Labeltext Card page.

Labeltext Insertion

The Labeltext Insertion page is used for gathering texts in the default language (no value in the Language Code field). No translations are entered here. The texts to be translated can be entered manually or automatically by the system.

If the text is to entered automatically, the Log Translated check box must be selected in the Mobile Setup. The system automatically creates an entry for each requested, non-translated text.

: Labeltext Insertion

The fields on the Labeltext Insertion page are described in the following table.

Field

Description

Code

Indicates the code of the page for which the translation should be performed, which allows different translations for the same text on different documents.

Text ID

Used as reference in the document definition; should have the format “_TEXT” without any space but an underscore “_” instead.

First Text Line

Displays the first translation, if existing.

Comment

Specifies additional comments for the translated text.

Translated

Indicates whether the text is already translated or not.

Languages

Displays the number of languages to which the text has been translated.

Home Tab

The Process group on the Home tab of the ribbon contains three actions specific to the labeltext. Each is described in the following table.

Action

Description

Translation

Opens the Labeltext Lines page where translations can be done.

Import / Export

Imports/exports data from/to other companies.

 

Labeltext Card

Translations of labeltexts are done in the Labeltext page.

: Labeltext Card

The General FastTab contains information about the text, some of which can already be set in the Labeltext Insertion page. The Lines FastTab contains the translations of the labeltext according to the selected language code.

The fields in the Lines FastTab are described in the following table.

Field

Description

Language Code

Defines the Language Code of the translation.

Text

Indicates the translated text that should be displayed.
