Mobile Application Setup

The Mobile Application Setup page contains additional fields to configure cc|mobile solution.

: Mobile Application Setup, General FastTab

General FastTab

The fields on this FastTab are described in the following table.

Field

Description

Records per Page

Specifies how many records should be displayed on a single page.

Print Job Queue User ID

Determines which client/NAS should be used for the Print Job Queue.

Print Job Queue Host Name

Specifies the hostname that should be used for the Print Job Queue.

Printer Selection Filter

Filters the list of printers that are available for printing.

 

Module FastTabs

The fields in the module FastTabs (Sales, Purchase, Inventory, CRM and Jobs) are used to specify the records per page for several dialogues that show different types of master data.

: Mobile Application Setup, Module FastTabs

Universal Dlg. Settings FastTab

The settings on this FastTab apply all process dialogues that use the Universal Dialogue.

: Mobile Application Setup, Universal Dlg. Settings FastTab

The fields on this FastTab are described in the following table.

Field

Description

Ask for Variant Code

Specifies if the Variant Code should be asked for on collection of item information.

Note

If the Variant Code is used, a Variant Code must not be empty (“”). Use a value like “empty” instead.

Show more Item Info

If selected, the value of the No. 2 field will be shown in addition to the Item No.

Changes only in Qty. (Base)

If selected, captured item quantity will be calculated and stored in the item’s Base Unit of Measure. For example, if a quantity is captured in unit of measure “KG” and the base unit of measure is “PCS”, the captured quantity will be recalculated in “KG”.
