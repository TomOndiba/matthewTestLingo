Mobile Setup

Access the Mobile Setup page from the navigation pane by clicking Departments > COSMO CONSULT > cc|mobile solution > Setup > Mobile Setup. In this page, basic settings for communication between Microsoft Dynamics NAV and HTTP server, the management of documents (Editor) and the layout of the application can be made.

When the Mobile Setup page is first opened, default values are set automatically for relevant fields.

General FastTab

The fields on the General FastTab are described in the following table.

: Mobile Setup, General FastTab

Field

Description

URL

Identifies the URL of the HTTP-Server according to the following pattern: “http://<computer name>|<url>:port/”

This field is mandatory and is auto-populated with the current host name of the Microsoft Dynamics NAV Service.

Note

Only local access is possible if address “127.0.0.1” or “localhost” is used. Ideally, the URL of the computer in the network should be used.

Version cc|mobile solution

Indicates the current version of cc|mobile solution.

Root-Path

Identifies the root directory of the HTTP-Server. In this folder, subdirectories are created and generated files will be stored there.

This folder must be identical to the folder specified in the HTTP-Server settings (refer to topic "General Tab" on page 1).

Image Directory

Identifies the directory where images are stored. This folder must be a subfolder of the root path.

 

Portal FastTab

The Portal FastTab contains fields that are used as default for all handhelds.

: Mobile Setup, Portal FastTab

The fields on this FastTab are described in the following table.

Field

Description

Start Page

Indicates the document to be used as the start page for handhelds. This value is used if no value has been entered in the Handheld or Handheld Group.

About Page

Indicates the document to be displayed when the About page link is clicked.

Server FastTab

The Server FastTab contains settings for the Mobile Server in Microsoft Dynamics NAV.

: Mobile Setup, Server FastTab

The fields on this FastTab are described in the following table.

Field

Description

Server Journal

Indicates what types of requests should be logged. The options are:

  • All – All requests will be logged.
  • Yes – Same as All (for later usage).
  • No – No requests will be logged.

Log Translated

If selected, the texts from documents are stored in Microsoft Dynamics NAV to provide a later translation.

Time Logging

If selected, the duration in milliseconds that was needed for creating a certain document will be specified. The recording of information occurs in the file “<Temp Path>\MobTime.log”.

Logging of Generator Buffer

If selected, the entries of the table will be logged in TmpTab. The recording of information occurs in the file “<Temp Path>\MobTmpTab.log”.

Session Idle Time (Min)

Specifies the idle time (in minutes) that may be passed until a session is declared as invalid and a new login of the handheld is required.

Search Timeout (s)

Specifies the duration (in seconds) in which the generator is allowed to create a document. The checking of this duration occurs by using given functions.

 

Editor FastTab

The Editor FastTab contains settings for the Document Editor.

: Mobile Setup, Editor FastTab

The fields on this FastTab are described in the following table.

Field

Description

Schema Path

Identifies the path to the provided XSD file. The default name of the XSD file is “SchemaVx.x.xsd”.

Temp Path

Identifies the folder that the system should use for storing temporary files. This folder is mainly used for Document Object Model (DOM) Validation (see next field).

DOM Validation

Disables/Enables the external validation. The documents can also be validated without using the MSXML parser.

Delete XML-File for DOM Validation

If selected, generated XML files will be deleted after DOM Validation.

If not selected, the generated XML file will be stored in the temp folder with the name “<Deckname>.xml”.

Schema Import Date

Indicates the date on which the XSD schema was last imported successfully.

Schema Exists

Indicates whether a correct schema exists or not.

Syntax Highlighting

Enables/Disables syntax highlighting in documents.

 

Layout FastTab

The Layout FastTab contains settings related to the view but are not dependent on documents.

: Mobile Setup, Layout FastTab

Field

Description

E-Mail/Phone Link Image

Identifies an image file to be displayed for E-Mail/Phone Links.

Company Logo

Identifies the image file to be displayed on top if the Company Logo field = show.

Navigation Position

Indicates the placement of the navigation links. The options are Left, Right, Down and Up.

Navigation Alignment

Determines if the navigation links are ordered vertically or horizontally.

Navigation Style

Determines if the navigation links are displayed as Links or Buttons.

Navigation on Login

Determines whether the navigation should be displayed on the login page or not.

Default Language Code

Indicates the language that will be used as the default value when handhelds are created.

Style Sheet File

Indicates the *.css file to be used for styles of HTML page. Create and select another file to customize the view.

Records per Page

Determines how many entries of lists should be displayed on a single page.

HTML Window Width/Height

Indicates the width and height value to be used for rendering the layout of HTML pages if not set on the handheld.

HTML One Window

If selected (recommended), all navigations to new pages will be rendered in the same window.

HTML Image Type

Indicates the file type of images found in the Image Directory path to be used for rendering images. Use png to ensure that all required images already exist. The options are png, jpg and gif.

Keep HTML Image Type

Indicates another image file type that can be used in addition to the type indicated in the HTML Image Type field.

Every image specified in documents having an file type <> to the value in the Keep HTML Image Type field will be replaced with the type specified in the HTML Image Type field. Therefore, to allow the display of gifs, set this field to gif.

Top Bar

Shows/hides the top bar, containing the company logo, title and so on.

Card Title

Shows/hides a bar on top, displaying the current card title.
