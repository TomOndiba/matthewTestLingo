Configuration of Process Dialogues

The process dialogues of cc|mobile solution offer multiple settings that can be partially applied in the Microsoft Dynamics NAV RoleTailored Client and also in the underlying Codeunit. The different settings are explained in the following sections.

Mobile Setup, Universal Dlg. Settings FastTab

The settings in this FastTab apply to all process dialogues that use the Universal Dialogue. The fields on this FastTab are described in section “Universal Dlg. Settings FastTab” on page 1.

Document, Universal Dlg. Settings FastTab

The settings in this FastTab apply to a specific process dialogue. The fields on this FastTab are described in topic “Universal Dlg. Settings FastTab” on page 1.

Underlying Codeunit Settings

In addition to the settings in the Microsoft Dynamics NAV RoleTailored Client, there are several other settings in the codeunits according to the documents.

Ask for Additional Fields

By using the Universal Dialogue, the user can be asked for additional fields that are not implemented in the default process dialogues.

Process dialogues have two parameters that can be used for getting two additional values. The following global variables can be used for collecting additional values:

  • SpecialFields[2] of type Integer
  • SpecialFieldCaptions[2] of type Text

SpecialFields represent a Field No. of the Table 83 “Item Journal Line”. To buffer those values, the table was extended by fields of different data types.

NOTE

It is possible that some of the fields are already used by the Codeunits. Therefore it is important to check that the desired field is “available” for SpecialFields usage. It is also possible to use default fields of the table, but in this case a check of “already-in-use” should be done.

In the process dialogues, an initial implementation for SpecialFields has to be adjusted.

The InitMD function contains the following lines:

SpecialFields[1] := 0;

SpecialFieldCaptions[1] := '';

SpecialFields[2] := 0;

SpecialFieldCaptions[2] := '';

The Field No. of the value to be asked for must be assigned to the variable “SpecialFields”. The variable “SpecialFieldsCaptions” specifies the desired caption to be displayed.

Since Data Type = Text, the GEN.TransDef() function can be used for Labeltexts to ensure Multilingual Support.

After running the Universal Dialogue, the values can be processed in the FinishMD(); function.
