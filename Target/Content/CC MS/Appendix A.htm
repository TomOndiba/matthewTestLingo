Appendix A

This appendix contains the API of the Document Generator and the Mobile Jnl. Param table. The functions of both are required to create or edit the underlying codeunits of documents. This appendix also contains the definitions of the Document Editor Schema.

API Document Generator

The functions of codeunit 5308001, MS - Basis are described in the following table.

Function

Description

CloseCard()

Closes the current card, generates it and writes the code of the output format to the file.

CloseDeck()

Closes the page and writes the end tags such as </body></html> or </wml> to the file. The file is then also closed.

ConvertText(piText : Text[200]) : Text[250])

Converts special characters and umlauts to hexadecimal or HTML notation.

CreateCard(CardID : Code[30])

Tells the generator which card is to be generated.

This function demarcates all following function call-ups to this card, i.e., in all functions in which an ID is specified, the search for this ID is confined to the area of the card. Before this function can be called up, the CreateDeck( text Deckname ) function must be called up.

Parameter
  • CardID = contains the ID of the card to be generated.

CreateDeck(Deckname : Code[20])

Tells the generator which page is to be generated.

This function calls up the initialization of the special generator codeunit according to the output format and specifies the page to which the subsequent function call-ups relate. Before calling up this function, the Init( var Record MS - Mobile Journal Params) function must have been called up.

Parameter
  • Deckname = name of page to be created.

DateToNavDate (DateTxt: Text [30]; VAR NAVDate: Date): Boolean

Converts client date to the Microsoft Dynamics NAV date format.

Parameter
  • DateTxt = date to be converted as text
  • NAVDate = variable containing the converted date

Return value

  • true if the conversion was successful, otherwise false.

EndTable()

Closes the current table.

Format(Number : Decimal;Decimals : Integer;Seperator1000 : Boolean;DecimalChar : Code[1];FieldLength : Integer;AddLeftSpaces : Boolean) Result : Text[30]

Formats the figure transmitted to a format to be specified and returns the figure as a text.

Parameter
  • Figure = figure to be formatted
  • Decimal places = number of decimal places
  • 1000 separation = with '.' as 1000 separation
  • Decimal point = decimal point e.g.: .
  • Field length = (optional 0) field length, right-aligned
  • leftEmpty = left-aligned with space
Return value
  • The figure as formatted as text.

Init(VAR MobileJnlParam : Record "MS - Mobile Jnl. Parameter")

Initializes the generator and transmits the variable Mobile Journal Params containing the parameters of the HTTP request. This is a variable of Record type (table).

This function must be the first function to be called up.

Parameter
  • MobileJnlParam = table containing all parameters

NextRec()

NextRec is called up once all data for a line within a dynamic table has been set.

This function closes the line and initializes the generator for a new line. Before calling up this function, the current table must be set using the StartTable( text TableID ) function.

NextSeq()

NextSeq is called up once all data for a cycle within a sequence has been set. This function closes the current cycle and initializes the generator for a new cycle.

SetExternLink(LinkID : Code[30];LinkName : Text[250];Destination : Text[250];TranslateLinkName : Boolean)

This function sets the necessary data for a dynamic link to another page.

Parameter
  • LinkID = ID of link to be generated
  • Linkname = name to be displayed for the link
  • Destination = name of page to which the link is to refer
  • TranslateLinkname = specifies whether the link name is to be translated.

SetGlobalLink(LinkID : Code[30];LinkName : Text[250];Destination : Text[250];TranslateLinkName : Boolean)

This function sets the necessary data for a dynamic link to any uniform resource identifier (URI) required.

Parameter
  • LinkID = ID of link to be generated
  • Linkname = name to be displayed for the link
  • Destination = name of page to which the link is to refer
  • TranslateLinkname = specifies whether the link name is to be translated

SetInternLink(LinkID : Code[30];LinkName : Text[250];Destination : Text[250];TranslateLinkName : Boolean)

This function sets the necessary data for a dynamic link to another card within a page.

Parameter
  • LinkID = ID of link to be generated
  • Linkname = name to be displayed for the link
  • Destination = name of page to which the link is to refer
  • TranslateLinkname = specifies whether the link name is to be translated

SetMailLink(LinkID : Code[30];LinkName : Text[250];Destination : Text[250];TranslateLinkName : Boolean)

This function sets the necessary data for a dynamic link to an e-mail address.

Parameter
  • LinkID = ID of link to be generated
  • Linkname = name to be displayed for the link
  • Destination = e-mail address to which reference is to be made
  • TranslateLinkname = specifies whether the link name is to be translated

SetOutputField(ObjectID : Code[30];Value : Text;TranslateValue : Boolean)

This function assigns a certain value to an element on the page. Object with the ID ObjectID and the value Value.

Parameter
  • ObjectID = ID of object to which the value is to be assigned
  • Value = the value to be assigned to the object
  • TranslateValue = specifies whether the value is to be translated

SetOutputValues(ObjectID : Code[30];ValueArr : ARRAY [4] OF Text[250])

This procedure sets dynamic entries in the lists of options or menu lists. The list is activated via ObjectID. The array Value contains one entry in each case.

Parameter
  • ObjectID = ID of object to which the entry is to be assigned
  • Value[1] = name of entry to be displayed
  • Value[2] = value of entry
  • Value[3] = not used
  • Value[4] = not used

SetParam(ObjectID : Code[30];Name : Code[20];Value : Text[250])

The procedure assigns a name and value to a dynamic parameter. The value and name are not translated.

Parameter
  • ObjectID = ID of parameter to which the value is assigned
  • Name = name assigned to the parameter
  • Value = value assigned to the parameter

SetParamValue(ObjectID : Code[30];Value : Text[250])

This procedure assigns a value to a parameter. The value is not translated.

Parameter
  • ObjectID = ID of parameter to which the value is assigned
  • Value = value assigned to the parameter

SetSMSLink(LinkID : Code[30];LinkName : Text[250];Destination : Text[250];TranslateLinkName : Boolean)

This procedure sets the necessary data for a dynamic link for recording an SMS.

Parameter
  • LinkID = ID of the link to which the data is assigned
  • Linkname = name which is displayed for the link
  • Destination = telephone number to which the SMS is to be sent
  • TranslateLinkname = specifies whether the link name is translated

SetTelLink(LinkID : Code[30];LinkName : Text[250];Destination : Text[250];TranslateLinkName : Boolean)

This procedure sets the necessary data for a dynamic link for choosing a telephone number.

Parameter
  • LinkID = ID of the link to which the data are assigned
  • Linkname = name which is displayed for the link
  • Destination = telephone number of person to be called
  • TranslateLinkname = specifies whether the link name is translated

StartTable(TableID : Code[30])

This function is called up to create a dynamic table and specify that all following IDs relate to objects within the table.

Parameter
  • Table ID = table ID of table to be created

Trans(Text : Text[250];Code : Code[20];LanguageCode : Code[10];InsertTrans : Boolean) : Text[250]

This function translates the specified text into the language designated by the language code. If specified, a non-found text is automatically entered into the translation table.

Parameter
  • Text = contains the table ID of the table to be created
  • Code = page name for which translation is requested
  • LanguageCode = contains the language code designating the target language or translation
  • InsertTrans = specifies whether the text is entered in the translation table

Mobile Jnl. Parameter API

The functions of table 5308007, MS - Mobile Jnl. Parameter are described in the following table.

Function

Description

ClearParams()

This procedure deletes all parameters transmitted apart from the page type, the page name and PIN (mobile_id).

GetBooleanField(ParamName : Code[30];VAR BooleanValue : Boolean) : Boolean

Calculates the value for the specified parameter and converts it to a Boolean where possible. The value is converted as follows:

true = ja, yes, 1

M

All characters.

N

Numerical characters.

x

Characters in lowercase.

X

Characters in uppercase.
